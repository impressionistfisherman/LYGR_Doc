# Applied Energistics 2 개요

AE2는 수백 개의 상자를 **하나의 디지털 네트워크**로 대체하는 저장 및 자동화 모드입니다.
아이템을 검색하고, 자동으로 제작하고, 외부 기계와 연동해 자동화 공장을 만들 수 있습니다.

> **정보: 인게임 가이드북이 있습니다**
>
> 이 모드팩에는 `guideme` 모드가 함께 설치되어 있습니다.
> 인게임에서 **guideme 책**을 만들면 AE2의 모든 내용을 인터랙티브 가이드로 볼 수 있습니다.
> 이 가이드와 함께 활용하면 훨씬 빠르게 이해할 수 있습니다.

---

## AE2가 하는 일

```
상자 100개
  → ME 네트워크 1개
     → 어디서든 검색 가능
     → 원클릭 자동 제작
     → 아이템 자동 수입/수출
     → 다른 기계와 완전 연동
```

## 핵심 개념 세 가지

AE2를 이해하려면 딱 세 가지 개념만 먼저 알면 됩니다.

### 1. ME 네트워크

모든 저장장치와 기기가 연결된 **디지털 네트워크**입니다.

```
ME Controller (핵심)
    ↕ 케이블
ME Drive (저장장치) ← Storage Cell (저장 셀)
    ↕ 케이블
ME Terminal (접근 터미널)
    ↕ 케이블
Import/Export Bus (자동 입출력)
```

### 2. 채널 (Channel)

기기를 네트워크에 연결하는 **데이터 연결선**입니다.

> **참고: 채널 수치 기준**
>
> 아래 채널 수치는 채널 기능이 켜진 AE2 기본 설정 기준입니다. 서버 설정에 따라 채널 기능이 꺼져 있거나 전달량이 조정될 수 있습니다.

```
일반 케이블: 최대 8채널 전달
Dense 케이블: 최대 32채널 전달
ME Controller 면 1개: 32채널 공급

채널이 부족하면 기기가 작동하지 않습니다.
```

채널은 AE2에서 가장 헷갈리는 개념입니다. [채널 상세 가이드](/docs/ae2/channels)에서 따로 다룹니다.

### 3. 저장 셀 (Storage Cell)

아이템이 실제로 저장되는 **디지털 저장 매체**입니다.

> **참고: 저장 셀 용량 기준**
>
> 저장 셀 용량은 AE2 기본 아이템 셀 기준입니다. 애드온, 서버 설정, 셀 종류에 따라 실제 저장량이나 타입 제한이 달라질 수 있습니다.

```
아이템 셀 티어:
  1k Cell  → 최대 1,024 아이템 (타입 63종)
  4k Cell  → 최대 4,096 아이템 (타입 63종)
  16k Cell → 최대 16,384 아이템 (타입 63종)
  64k Cell → 최대 65,536 아이템 (타입 63종)
  256k Cell → 최대 262,144 아이템 (타입 63종)
```

> **참고: 타입(Type) 이란?**
>
> 아이템의 **종류 수**입니다.
> 예를 들어 철 주괴 500개 + 금 주괴 300개 = 2타입 사용.
> 셀 하나에 저장할 수 있는 타입 수에 한계가 있으므로
> 아이템 종류가 너무 많으면 셀이 꽉 차 보여도 저장이 안 될 수 있습니다.

## 핵심 소재 — 서투스 쿼츠 (Certus Quartz)

AE2만의 고유 소재입니다.

### 획득 방법 — 운석(Meteorite)

```
1. Meteorite Compass 제작 (JEI 검색)
2. 나침반이 가리키는 방향으로 이동
3. 운석 충돌 크레이터 발견 → 중앙 파고들기
4. Mysterious Cube 발견 → 주변 서투스 쿼츠 클러스터 채굴
```

> **위험: Flawless Budding Certus는 절대 캐지 마세요**
>
> 운석 중앙에 있는 **Flawless Budding Certus** 블록은
> 실크터치로 캐도 등급이 내려가며, 한 번 내려간 등급은 복구 불가능합니다.
> 이 블록에서 서투스 쿼츠 결정이 계속 자라므로 원래 위치에 두고 파밍하세요.

### 서투스 쿼츠 파밍

```
Budding Certus 블록 주변에 서투스 쿼츠 클러스터가 자랍니다.
  → 다 자란 클러스터 수확 → 서투스 쿼츠 결정 획득
  → 결정 → 다양한 AE2 소재 제작에 사용

효율적인 파밍:
  운석에서 Budding Certus 블록을 실크터치로 채취
  → 자신의 베이스에 설치
  → 퀄츠 결정 자동 성장 파밍장 구축
```

## 전체 진행 로드맵

```
1. 운석 탐색 → 서투스 쿼츠 확보
2. Fluix Crystal 합성
3. 첫 ME 네트워크 구축 (Ad-Hoc, 컨트롤러 없음)
   → ME Drive + Storage Cell + Terminal
4. ME Controller 추가 → 채널 확장
5. Import/Export Bus로 자동 아이템 수입/수출
6. Pattern Encoding Terminal + Pattern Provider
   → 자동 제작(Autocrafting) 구축
7. 다른 기계(Mekanism, Create 등)와 연동
```
