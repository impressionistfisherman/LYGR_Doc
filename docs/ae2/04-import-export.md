# 자동 아이템 수입/수출

ME 네트워크를 상자, 기계 등 외부 인벤토리와 자동으로 연결하는 방법입니다.

---

## Import Bus (수입 버스)

외부 인벤토리에서 ME 네트워크로 아이템을 자동 수입합니다.

### 설치 방법

```
Import Bus를 상자 또는 기계 출력 면에 설치
→ 해당 인벤토리의 아이템을 자동으로 ME 네트워크로 수입
```

### 설정

```
Import Bus 우클릭 → GUI 열기

필터 슬롯:
  비워두면: 인벤토리의 모든 아이템 수입
  아이템 설정: 해당 아이템만 수입

속도 업그레이드:
  Speed Upgrade 카드 삽입 → 수입 속도 증가 (최대 4개)
```

### 활용 예시

```
Mekanism 광석 처리 라인 끝 상자
  ← [Import Bus] ← 처리된 주괴 자동 수입
  → ME 네트워크에 주괴 자동 저장
```

---

## Export Bus (수출 버스)

ME 네트워크에서 외부 인벤토리로 아이템을 자동 수출합니다.

### 설치 방법

```
Export Bus를 상자 또는 기계 입력 면에 설치
→ ME 네트워크에서 설정된 아이템을 자동으로 수출
```

### 설정

```
Export Bus 우클릭 → GUI 열기

필터 슬롯 (필수 설정!):
  수출할 아이템 종류를 지정해야 합니다
  비워두면 아무것도 수출하지 않습니다

Crafting Card 삽입:
  ME 네트워크에 해당 아이템이 없으면 자동 제작 요청
  → 자동 보충 시스템에 필수
```

### 활용 예시

```
용광로 연료 상자
  ← [Export Bus, 석탄 필터] ← ME 네트워크에서 석탄 자동 공급
  → 용광로 연료 자동 충전
```

## Storage Bus (저장 버스)

외부 인벤토리를 ME 네트워크의 **가상 저장소**로 편입시킵니다.

### Import/Export Bus와의 차이

```
Import Bus: 아이템을 ME 네트워크로 이동시킴
Export Bus: 아이템을 ME 네트워크에서 꺼냄
Storage Bus: 외부 인벤토리를 네트워크에서 바로 접근 가능하게 연결
             → 아이템이 이동하지 않음! 그냥 "창문" 역할
```

### 설치 방법

```
Storage Bus를 상자 면에 설치
→ 해당 상자의 내용이 ME Terminal에서 보임
→ Terminal에서 직접 넣고 뺄 수 있음
```

### 우선순위 (Priority) 설정

```
Storage Bus 또는 ME Drive GUI에서 우선순위 설정 가능

우선순위 높음: 아이템 저장 시 먼저 채워짐, 꺼낼 때 나중에 비워짐
우선순위 낮음: 아이템 저장 시 나중에 채워짐, 꺼낼 때 먼저 비워짐

활용:
  특정 상자(배럴 등)를 특정 아이템 전용 저장소로 지정할 때
  → 해당 Storage Bus에 높은 우선순위 + 필터 설정
```

## Interface (인터페이스)

가장 다목적 연결 블록입니다.

```
역할:
  - ME 네트워크에서 인접 인벤토리로 아이템 자동 공급
  - 인접 인벤토리에서 ME 네트워크로 아이템 자동 반환
  - Autocrafting 시스템의 핵심 블록
```

### 주요 활용

```
자동 보충:
  Interface 내 슬롯에 아이템과 원하는 수량 설정
  → ME 네트워크가 자동으로 해당 수량 유지
  → 부족하면 자동으로 채움, 넘치면 네트워크로 반환
```

## Level Emitter (레벨 에미터)

특정 아이템의 보유량에 따라 레드스톤 신호를 발생시킵니다.

```
설정:
  "철 주괴가 100개 이하면 레드스톤 ON"

활용:
  레드스톤 신호 → 기계 ON/OFF 제어
  → 철이 부족할 때만 Mekanism 광석 처리 라인 가동
  → 불필요한 에너지 소모 방지
```

## 실전 자동화 구성 예시

### 광석 처리 자동화

```
[Silent Gear 해머 or Digital Miner]
          ↓
     [상자 - 광석 버퍼]
          ↓
[Import Bus] → ME 네트워크
          ↓
[Export Bus, 광석 필터] → Mekanism Enrichment Chamber
          ↓
     [광석 처리 라인]
          ↓
[Import Bus] → ME 네트워크 (주괴 자동 수입)
```

### 용광로 자동화

```
ME 네트워크
  ↓
[Export Bus, 제련 재료 필터] → 용광로 입력
용광로 출력 →
[Import Bus] → ME 네트워크
```
