# 초반 보스 공략

Cataclysm의 첫 번째 보스들입니다.
이 두 보스를 처치하면 이후 진행에 필요한 핵심 소재를 얻을 수 있습니다.

---

## 🔥 Netherite Monstrosity (네더라이트 몬스트로시티)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | Soul Forge (네더) |
| **체력** | 500 HP |
| **방어력** | 10 |
| **권장 장비** | Apotheosis Rare 풀세트 + 내화 포션 |

### 던전 구조 — Soul Forge

```
네더에 생성되는 용광로 형태의 대형 구조물
→ 내부에 네더 관련 몬스터 다수 배치
→ 최심부에 Netherite Monstrosity 대기 중
```

**찾는 방법:**
JEI에서 **Eye of Monstrous**를 검색해 제작한 뒤, 네더에서 사용하면 Soul Forge 방향을 확인할 수 있습니다.

**주의 사항:**
- 던전 전체가 **네더** 환경이므로 내화 포션을 준비하는 것이 좋습니다
- 진입 전 네더 몬스터(Blaze, Zombie Piglin 등)를 정리하고 진행 권장

### 보스 패턴

| 공격 | 피해량 | 대처법 |
|------|--------|--------|
| **근접 강타** | 21 + 최대 HP의 8% | 측면으로 회피 |
| **용암 투척** | 근거리 전용 | 원거리 유지 |
| **분노 상태 (HP 33% 이하)** | 모든 공격 강화 | 포션 아끼지 말고 집중 |

**분노 상태 주의:**
HP 33% 이하에서 몬스트로시티가 불타오르며 공격 속도와 데미지가 대폭 증가합니다.
이 구간에서 가장 많은 플레이어가 사망하므로 회복 포션을 미리 준비하세요.

### 공략 전략

```
1. 내화 포션 음용 → 용암 피해 무력화
2. 원거리 공격 주력으로 HP를 50%까지 감소
3. HP 50% 이하부터 좀 더 적극적으로 근접 공격
4. HP 33% (분노) 돌입 시 회복 포션 즉시 사용
5. 집중 공격으로 빠르게 마무리
```

### 드롭 아이템

| 아이템 | 용도 |
|--------|------|
| **Infernal Forge** | 강력한 도구/무기 (Cataclysm 전용) |
| **Monstrous Horn** | 조합 재료 |
| **음악 디스크** | 낮은 확률 |

## 👁️ Ender Guardian (엔더 가디언)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | Ruined Citadel 최하층 (엔드) |
| **체력** | 150 HP |
| **방어력** | 12 |
| **권장 장비** | Apotheosis Rare\~Epic |
| **특이사항** | 엔더 드래곤 처치 후 엔드 외곽 섬 탐험 필요 |

### 던전 구조 — Ruined Citadel

```
엔드 외곽 섬에 생성되는 3층 구조의 대형 폐허

1\~2층: 미로형 구조 + 함정 블록
  → Ender Stone 함정: 걸어가면 속도 감소 + 가시 공격
  → Purpur 함정: 접촉 시 데미지
  → Endermapteras(곤충형 적) 다수

3층 (최하층): 숨겨진 보스 방
  → 중앙 제단에 접근 → Ender Guardian 소환
  → 2페이즈에서 바닥이 붕괴되어 더 큰 아레나 등장
```

**Eye of Void:**
Ruined Citadel을 찾으려면 **Eye of Void**가 필요합니다.
JEI에서 조합법을 확인하고 엔더의 눈처럼 던지면 방향을 알려줍니다.

### 함정 주의사항

```
엔드 스톤 색 블록: 걸으면 속도 감소 + 가시 발사
→ 조심히 이동하거나 점프로 통과

Purpur 함정 블록: 접촉 시 즉각 피해
→ 밟지 말고 우회
```

### 보스 패턴

| 공격 | 특징 | 대처법 |
|------|------|--------|
| **근접 강타** | 높은 단일 피해 | 백대시 후 반격 |
| **진동파 (Earthquake)** | 주변 범위 피해 | 점프로 회피 |
| **공허 룬 소환** | 지면에 공허 룬 생성, 초당 7 피해 | 룬 위치 파악 후 회피 |
| **HP 절반 — 헬멧 파괴** | 방어력 절반으로 감소 | 이 시점에 공격 집중 |

**2페이즈 (HP 50% 이하):**
바닥이 붕괴되어 더 넓은 아레나로 진입합니다.
낙하 피해 주의 — Free Runners 또는 Feather Falling 인챈트를 준비하면 안전합니다.

### 공략 전략

```
1. 입장 전 함정 블록 위치 파악
2. 제단 접근 → 보스 소환
3. 공허 룬을 밟지 않도록 이동 경로 확보
4. HP 50% 돌파 시 바닥 붕괴 대비
5. 2페이즈 낙하 후 즉시 체력 확인
6. 헬멧 파괴 후 방어력 감소 구간에서 집중 딜링
```

### 드롭 아이템

| 아이템 | 용도 |
|--------|------|
| **Gauntlet of Guard** | 강력한 장갑 무기 (공격 + 보호막 기능) |
| **음악 디스크 "The Citadel's Last Stand"** | 낮은 확률 |

> **참고: Void Core는 매우 중요**
>
> Void Core는 Ruined Citadel 내부의 **Ender Golem** 미니보스에게서 얻습니다.
> 이후 Void Forge, Void Assault Weapon 등 최종 무기 조합에 필요하므로
> Ender Guardian 공략 전에 성채 내부를 충분히 탐색하세요.
