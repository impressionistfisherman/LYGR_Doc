# 후반 보스 공략

Cataclysm의 최종 단계 보스들입니다.  
이 보스들은 이전 보스 드롭 장비가 없으면 생존이 매우 어렵습니다.

---

## 🦑 Scylla (스킬라)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | Acropolis (오버월드) |
| **권장 장비** | Apotheosis Epic + Cataclysm 장비 혼합 |

### 던전 구조 — Acropolis

```
그리스/고대 신전 테마의 대형 구조물
→ 다수의 기둥과 열린 아레나 구조
→ 다양한 종류의 몬스터 배치
```

### 보스 패턴

| 공격 | 특징 | 대처법 |
|------|------|--------|
| **다중 촉수 공격** | 여러 방향 동시 공격 | 촉수 패턴 확인 후 사이로 이동 |
| **잉크 분사** | 시야 방해 + 이동 둔화 | 잉크 범위 벗어나기 |
| **먹이 잡기** | 플레이어 포획 후 연속 피해 | 즉시 이탈 |
| **다리 소환** | 지면에 다리를 내려 광범위 공격 | 다리 사이 안전 구역으로 |

### 드롭 아이템

| 아이템 | 용도 |
|--------|------|
| **Scylla 전용 장비** | 강력한 수중/독 계열 무기 |
| **음악 디스크** | 10% 확률 |

## 👻 Maledictus (말레딕투스)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | Frosted Prison 최하층 (오버월드 설원) |
| **소환 방법** | 무덤 우클릭 → 자동 소환 |
| **권장 장비** | Apotheosis Epic + Cataclysm 장비 |

### 던전 구조 — Frosted Prison

```
설원 바이옴에 생성되는 대형 얼음 요새
→ 여러 층 감옥 구조
→ Draugr(드라우그) 언데드 적 집중
→ 최하층 보스 방: Maledictus 무덤
```

**Frosted Prison 몬스터:**
- **Draugr:** 검은 강철 무기를 든 언데드 전사, 다양한 변종 존재
- **Aptrgangr (미니 보스):** 대형 도끼를 든 거대 언데드, 매우 강력

**Aptrgangr 주의:**  
Aptrgangr는 미니보스급으로 매우 강력합니다.  
보스 방으로 내려가기 전에 반드시 먼저 처치하고 진행하세요.

### 보스 패턴

말레딕투스는 **분노 게이지**를 가진 보스입니다.  
플레이어에게 공격이 **적중할수록** 분노 게이지가 차올라 더 강해집니다.

| 공격 | 특징 | 대처법 |
|------|------|--------|
| **다중 무기 공격** | 여러 무기로 복합 공격 | 패턴 읽고 회피 |
| **영혼 공격** | 영혼 에너지 원거리 투사 | 좌우 이동으로 회피 |
| **분노 상태** | 피격 시 게이지 증가 → 공격 강화 | 방어보다 적극 회피 전략 |
| **얼음 공격** | 범위 냉기 피해 + 이동 감소 | 범위 밖으로 이탈 |

**공략 핵심:**  
말레딕투스의 분노 게이지는 **피격 횟수**에 따라 오릅니다.  
많이 맞을수록 보스가 강해지므로, 회피를 최우선으로 하고 안전할 때만 공격하세요.

### 드롭 아이템

| 아이템 | 용도 |
|--------|------|
| **Cursium Ingot** | Cataclysm 최종 장비 제작 핵심 소재 |
| **Soul Render** | 강력한 할버드형 무기 |
| **Annihilator (2개)** | 쌍둥이 메이스 무기 |
| **Cursed Bow** | 강력한 저주 활 |
| **음악 디스크 "Maledictus"** | 10% 확률 |

## 🔥 Ignis (이그니스) — 최종 보스

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | Burning Arena (네더) |
| **소환 방법** | Burning Ashes를 최상층 Altar of Fire에 배치 |
| **권장 장비** | Apotheosis Epic 풀 + 내화 포션 + Cataclysm 장비 |
| **체력** | 450 HP (2페이즈 포함) |

### 던전 구조 — Burning Arena

```
네더에 생성되는 거대 원형 콜로세움
→ 여러 층이 위로 쌓인 구조
→ 각 층을 올라가며 전투
→ 최상층: Altar of Fire
```

**Ignited Revenant (점화된 복수자):**  
Burning Arena 내부를 지키는 강력한 적입니다.

- **체력:** 80, **방어력:** 12
- **방패 사용:** 방패를 들고 막을 수 있음 (도끼의 치명타 공격으로 방패 부수기)
- **드롭:** Burning Ashes — **Ignis 소환에 필수!**
- 같은 층에 **2마리**씩 배치

**핵심:** Ignited Revenant 2마리를 처치해 Burning Ashes를 모은 뒤 Altar of Fire에 배치하면 Ignis가 소환됩니다.

### 보스 패턴

Ignis는 **3페이즈 보스**로 Cataclysm에서 가장 어렵습니다.

**1페이즈:**

| 공격 | 특징 | 대처법 |
|------|------|--------|
| **검 휘두르기** | 14 + 최대 HP의 5% 피해 | 백대시 |
| **방패 스매시** | 141.5 + 최대 HP의 15% 피해 | **절대 방패로 막지 말 것** |
| **검 + 방패 콤보** | 연속 공격 | 측면으로 회피 |

**2페이즈 (HP 50% 이하):**

```
Ignis가 파란색으로 변하며 모든 공격이 강화됩니다.
→ 추가로 불꽃 질주 공격, 광역 폭발 패턴 추가
→ 방패 스매시 피해가 더욱 증가
```

**방패 스매시 주의:**  
Ignis의 방패 스매시는 플레이어 장비의 **내구도를 무시하고 녹입니다.**  
이 공격을 맞으면 장비가 빠르게 망가집니다. **절대 정면에 서 있지 마세요.**

### 공략 전략

```
준비:
  - 내화 포션 필수
  - Apotheosis Epic 풀세트 + 최고 성능 무기 준비
  - 방어구 내구도 여분 확보 (스미스 모루 수리 재료 준비)

전투:
  1. Burning Arena 각 층 올라가며 Ignited Revenant 처치
  2. Burning Ashes 모두 수집 → 최상층 Altar of Fire에 배치
  3. Ignis 소환 → 전투 시작
  4. 방패 스매시 패턴 절대 정면으로 맞지 않기
  5. 검 공격 후 반격 타이밍 찾기
  6. 2페이즈 전환 시 즉시 회복 포션
  7. 2페이즈에서 공격 패턴 더욱 신중히 읽기
```

### 드롭 아이템

| 아이템 | 용도 |
|--------|------|
| **Ignitium Ingot** | Ignitium 장비 세트 제작 핵심 |
| **Incinerator (검)** | Ignis가 사용한 불꽃 검 |
| **Bulwark of the Flame (방패)** | Ignis가 사용한 불꽃 방패 |
| **Blazing Grips** | 특수 장갑 아이템 |
| **음악 디스크** | 10% 확률 |

### Ignitium 장비 세트

Ignis를 처치하면 얻는 **Ignitium Ingot**으로 강력한 장비를 제작할 수 있습니다.

```
제작 방법:
  Ignitium Ingot + 네더라이트 방어구 → 스미싱 테이블에서 조합
  → 네더라이트보다 강력한 Ignitium 장비 완성

Fusion Anvil 조합:
  Ignitium Chestplate + Elytra = Ignitium Elytra Chestplate
  (비행 + 최강 방어력 통합!)
```
