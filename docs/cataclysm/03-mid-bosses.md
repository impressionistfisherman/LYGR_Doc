# 중반 보스 공략

중반 보스들은 각각 독특한 기믹을 가지고 있습니다.
공략을 모르면 기믹을 이해하기 전까지 반복 사망할 수 있으니 이 가이드를 꼼꼼히 읽으세요.

---

## ⚙️ The Harbinger (하비저)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | Ancient Factory (오버월드 지하) |
| **체력** | 매우 높음 |
| **소환 방법** | Nether Star를 하비저 흉부에 우클릭 |
| **권장 장비** | Apotheosis Epic 풀세트 |

### 던전 구조 — Ancient Factory

```
오버월드 지하 깊숙이 생성되는 대형 기계 공장
→ 던전 블록으로 둘러싸여 있어 주 입구로만 진입 가능
→ 내부에 기계형 몬스터 다수
→ 핵심 기믹: EMP 블록
```

**찾는 방법:**
JEI에서 **Eye of Mech**를 검색해 제작한 뒤, 오버월드에서 사용하면 Ancient Factory 방향을 확인할 수 있습니다.

### 핵심 기믹 — EMP 블록

> **정보: EMP 블록은 반드시 활용하세요**
>
> Ancient Factory 내부 곳곳에 **EMP 블록**이 배치되어 있습니다.
> EMP를 하비저에게 사용하면 **스턴 + 소량 피해**를 줄 수 있습니다.
> 단, 하비저가 데스 레이저를 발사 중일 때는 EMP 효과가 없습니다.

### 보스 패턴

| 공격 | 특징 | 대처법 |
|------|------|--------|
| **미사일 발사** | 복수의 미사일 발사 | 이동으로 회피 |
| **데스 레이저** | 대형 레이저 빔 | 측면으로 빠르게 회피 |
| **돌진 공격** | 고속 돌진 | 타이밍 맞춰 회피 |
| **HP 50% (2페이즈)** | 화살 면역 + 모든 기술 강화 | 원거리 대신 근접 위주 전환 |

**2페이즈 주의:**
HP 50% 이하에서 하비저는 **화살에 면역**이 됩니다.
원거리 활/석궁을 사용하는 플레이어는 이 시점부터 전략을 바꿔야 합니다.

### 공략 전략

```
1. 던전 입장 전 내부 몬스터 먼저 정리
2. EMP 블록 위치 파악 후 기억
3. Nether Star 우클릭 → 보스 소환
4. 레이저 패턴은 발사 전 모션을 보고 좌우 회피
5. EMP를 타이밍 맞춰 사용 → 스턴 구간 집중 딜
6. HP 50% 이하 → 원거리 플레이어는 근접으로 전환
7. 지속적인 이동으로 미사일 회피
```

### 드롭 아이템

| 아이템 | 용도 |
|--------|------|
| **Witherite Block** | 분해 → Witherite Ingot (Fusion Anvil 제작 핵심) |
| **Wither Assault Shoulder Weapon** | 위더 해골 발사 대형 무기 |
| **Laser Gatling** | 레이저 총 |
| **Meat Shredder** | 강력한 근접 무기 |
| **음악 디스크 "Withering Meltdown"** | 낮은 확률 |

> **참고: Witherite Block → Mechanical Fusion Anvil**
>
> Witherite Block을 Witherite Ingot으로 분해하면
> **Mechanical Fusion Anvil** 제작이 가능합니다.
> 이 모루가 있어야 이후 보스 무기 조합을 할 수 있으므로 최우선으로 제작하세요.

## 🦴 Ancient Remnant (에인션트 레므넌트)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | Cursed Pyramid (오버월드 사막) |
| **소환 방법** | Desert Necklace를 뼈대에 우클릭 |
| **권장 장비** | Apotheosis Epic 풀세트 |
| **특이사항** | Desert Necklace를 던전 내부에서 먼저 수집해야 함 |

### 던전 구조 — Cursed Pyramid

```
사막에 생성되는 거대 피라미드
→ 여러 층으로 구성 (도서관, 보물방, 함정 복도)
→ Suspicious Sand에서 귀중품 발굴 가능
→ 핵심 아이템: Desert Necklace (내부 상자 또는 발굴)
```

**찾는 방법:**
JEI에서 **Eye of Desert**를 검색해 제작한 뒤, 사막 탐색 중 사용하면 Cursed Pyramid 방향을 확인할 수 있습니다.

**사막 피라미드 몬스터:**
- **Koboleton:** 작은 공룡 해골, 아이템 훔치기 공격 (슬라임 장갑으로 방어 가능)
- **Kobolediator (미니 보스):** 강화 Koboleton, 처치 시 Ancient Metal Ingot 드롭
- **Wadjet (미니 보스):** 뱀 형태, 모래 소환 공격

### 보스 패턴

| 공격 | 특징 | 대처법 |
|------|------|--------|
| **꼬리 휘두르기** | 넓은 범위 근접 공격 | 거리 유지 |
| **모래 폭풍** | 범위 모래 공격 | 측면 회피 |
| **돌진** | 고속 돌진 | 타이밍 맞춰 회피 |
| **분노 게이지** | 공격이 빗나갈수록 게이지 증가 | 공격은 정확하게, 무빙 과도 금지 |

**분노 게이지 주의:**
Ancient Remnant는 플레이어가 공격을 너무 많이 피할수록 **분노 게이지가 차오릅니다.**
게이지가 가득 차면 모든 공격력이 증가하므로, 회피와 공격을 균형있게 해야 합니다.

### 공략 전략

```
1. 피라미드 내부 탐색 → Desert Necklace 수집
2. Koboleton/Kobolediator 정리 (아이템 훔치기 방지)
3. 보스 방에서 Desert Necklace 우클릭 → 소환
4. 근접 공격 위주로 분노 게이지 억제
5. 꼬리/돌진은 타이밍 맞춰 회피
6. 모래 폭풍은 측면으로 이동
```

### 드롭 아이템

| 아이템 | 용도 |
|--------|------|
| **Ancient Metal Block** | 분해 → Ancient Metal Ingot (Ancient Metal 장비 제작) |
| **Sandstorm in a Bottle** | 모래 폭풍 소환 아이템 + Wrath of the Desert 조합 재료 |
| **Remnant Skull** | 새끼 Modern Remnant 길들이기 아이템 (펫!) |
| **음악 디스크 "Sands of Dominion"** | 낮은 확률 |

## 🌊 The Leviathan (레비아탄)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | Sunken City (오버월드 심해) |
| **소환 방법** | Abyssal Sacrifice를 Altar of Abyss에 배치 |
| **권장 장비** | Apotheosis Epic + 수중 호흡 포션 |
| **특이사항** | 전투의 일부가 **수중**에서 진행 |

### 던전 구조 — Sunken City

```
심해(Deep Ocean) 바이옴에 생성되는 거대 수중 신전
→ 여러 방으로 구성 (도서관, 감옥, 광석 구역)
→ 내부 몬스터가 매우 위험
→ 핵심: 중앙 방의 Altar of Abyss
```

**찾는 방법:**
JEI에서 **Eye of Abyss**를 검색해 제작한 뒤, 심해 탐색 중 사용하면 Sunken City 방향을 확인할 수 있습니다.

**수중 도시 몬스터:**
- **Deepling:** 산호 창 투척, 원거리 고피해
- **Deepling Brute:** 산호 미늘창, 근접 매우 위험
- **Deepling Warlock/Priest:** 마법 공격 (도시 내부 전용)
- **Coralssus:** Deepling이 탑승하는 대형 골렘
- **Coral Golem:** 결정화 산호 드롭

**Amethyst Crab (미니 보스):**
레비아탄 소환 전에 Amethyst Crab을 처치하면
**Blessed Amethyst Crab Meat**을 얻을 수 있습니다.
이 고기가 레비아탄 전투에서 생존에 **매우 중요합니다.**

### 보스 패턴

| 공격 | 특징 | 대처법 |
|------|------|--------|
| **발톱 공격** | 강력한 근접 피해 | 거리 유지 |
| **물 흐름 공격** | 플레이어를 당기거나 밀어냄 | 흐름 방향 반대로 이동 |
| **바닥 균열 레이저** | 바닥에서 레이저 발사, 광범위 | 레이저 사이 안전 구역으로 이동 |
| **포획** | 레비아탄이 플레이어를 잡아먹으려 함 | Blessed Amethyst Crab Meat로 탈출 |

**Blessed Amethyst Crab Meat가 중요한 이유:**
레비아탄이 플레이어를 포획했을 때, Blessed Amethyst Crab Meat를 사용하면 탈출할 수 있습니다.
이것 없이 포획되면 탈출 불가능해서 사망할 수 있습니다.

### 공략 전략

```
1. 수중 호흡 포션 준비 (전투 시간 길어짐)
2. Sunken City 내부 탐색 → Amethyst Crab 처치 → Blessed Crab Meat 획득
3. Altar of Abyss에 Abyssal Sacrifice 배치 → 보스 소환
4. Blessed Crab Meat를 핫바 접근 쉬운 위치에 배치
5. 바닥 균열 레이저 패턴 시 레이저 사이 안전 구역 찾기
6. 포획 시 즉시 Blessed Crab Meat 사용
7. 원거리 공격으로 거리 유지하며 지속 딜
```

### 드롭 아이템

| 아이템 | 용도 |
|--------|------|
| **Abyssal Claw** | 레비아탄의 잡기 공격 기반 갈고리 무기 |
| **Baby Leviathan Egg** | 새끼 레비아탄 부화 → 펫으로 키우기 |
| **음악 디스크** | 낮은 확률 |
