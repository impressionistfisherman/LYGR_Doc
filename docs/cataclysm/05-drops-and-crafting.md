## Mechanical Fusion Anvil 전체 조합 레시피

Mechanical Fusion Anvil은 **보스 아이템들을 더 강한 아이템으로 합성**하는 특수 제작대입니다.

### Anvil 제작 방법

```
Witherite Ingot × 6  (Harbinger 처치 후 얻는 Witherite Block 분해)
+ Redstone Block × 2
+ Anvil × 1
= Mechanical Fusion Anvil
```

### 전체 조합 목록

| 재료 1 | 재료 2 | 결과 | 필요 보스 처치 |
|--------|--------|------|---------------|
| Gauntlet of Guard | Bulwark of the Flame | **Gauntlet of Bulwark** | Ender Guardian + Ignis |
| Infernal Forge | Void Core | **Void Forge** | Monstrosity + Ender Golem |
| Ignitium Chestplate | Elytra | **Ignitium Elytra Chestplate** | Ignis |
| Wither Assault Shoulder Weapon | Void Core | **Void Assault Shoulder Weapon** | Harbinger + Ender Golem |
| Cursed Bow | Sandstorm in a Bottle | **Wrath of the Desert** | Maledictus + Ancient Remnant |

## 주요 아이템 상세

### ⚔️ Wither Assault Shoulder Weapon

```
획득: Harbinger 처치 드롭
효과: 위더 해골 발사 대형 무기 (로켓 런처 형태)
업그레이드: Fusion Anvil + Void Core = Void Assault Shoulder Weapon
```

### 🔫 Laser Gatling

```
획득: Harbinger 처치 드롭
효과: 레이저 연사 무기
재장전: 레드스톤 1개
주의: 블록에 불을 붙이는 부작용 있음 (기본 설정에서 비활성화)
```

### 🌪️ Sandstorm in a Bottle

```
획득: Ancient Remnant 처치 드롭
효과: 사용 시 모래 폭풍 소환
조합: Wrath of the Desert 재료
또한: Curio 슬롯에 착용 → 모래 회오리 변신 이동할 수 있습니다
```

### ⛏️ Infernal Forge

```
획득: Netherite Monstrosity 처치 드롭
효과: 강력한 도구 기능
조합: Fusion Anvil + Void Core = Void Forge
```

### 🗡️ Soul Render

```
획득: Maledictus 처치 드롭
효과: 강력한 할버드형 무기 (광범위 공격)
특성: 영혼 계열 추가 피해
```

### 🛡️ Gauntlet of Guard

```
획득: Ender Guardian 처치 드롭
효과: 장갑 형태의 무기 + 보호막 기능
조합: Fusion Anvil + Bulwark of the Flame = Gauntlet of Bulwark
```

### 🟣 Void Core

```
획득: Ruined Citadel 내부 Ender Golem 처치 드롭
효과: 공허 룬 소환
조합: Infernal Forge와 합성해 Void Forge 제작
      Wither Assault Shoulder Weapon과 합성해 Void Assault Shoulder Weapon 제작
```

### 🔥 Ignitium Armor Set

```
제작: Ignitium Ingot + 네더라이트 방어구 → 스미싱 테이블
  → Ignitium Helmet, Chestplate, Leggings, Boots

Elytra 합성:
  Ignitium Chestplate + Elytra → Fusion Anvil
  = Ignitium Elytra Chestplate
  (비행 + 최강 방어력 통합)

특성:
  - 네더라이트보다 높은 방어력
  - 화염 면역
  - 착용 시 주변에 불꽃 파티클
```

## 펫 시스템

Cataclysm은 보스 처치 보상으로 **특수 펫**을 얻을 수 있습니다.

### Modern Remnant (모던 레므넌트)

```
획득: Ancient Remnant 처치 → Remnant Skull 드롭
사용: Remnant Skull을 아기 공룡(Ancient Remnant 베이비)에게 사용
효과: 길들여진 공룡 동반자
```

### Baby Leviathan (아기 레비아탄)

```
획득: The Leviathan 처치 → Baby Leviathan Egg 드롭
사용: 알을 부화시키면 아기 레비아탄 등장
효과: 수중 동반자
```
