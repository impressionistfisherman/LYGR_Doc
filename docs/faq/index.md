## 🚀 시작 & 설치 관련

**Q. 게임이 실행이 안 돼요.**  
A. 아래 순서대로 확인해봐:
1. Java 21이 설치되어 있는지 확인 (`java -version` 명령어로)
2. RAM 할당량이 최소 6GB 이상인지 확인
3. CurseForge 앱을 관리자 권한으로 실행해보기
4. 모드팩 재설치 시도

**Q. 로딩이 너무 오래 걸려요.**  
A. 정상이야! 모드 178개를 처음 로딩하면 5~15분 걸려. 두 번째부터는 빨라져.

**Q. 로딩 중에 멈춰버렸어요.**  
A. 로딩 바가 10분 이상 멈춰있으면 강제 종료 후 재시작. 그래도 안 되면 모드팩 재설치.

**Q. 게임이 실행되자마자 꺼져요.**  
A. RAM 할당량 부족이 원인인 경우가 많아. CurseForge에서 RAM을 최소 6GB로 늘려봐. [설정 가이드 참고 →](../setup-guide/01-performance.md)

**Q. 한국어 설정은 어떻게 해요?**  
A. `ESC → Options → Language → 한국어` 선택. 일부 모드는 번역이 없어서 영어로 나오는 경우 있어.

## ⚙️ 게임 작동 관련

**Q. FPS가 너무 낮아서 못 하겠어요.**  
A. [성능 최적화 가이드](../setup-guide/01-performance.md) 보고 설정해봐. Render Distance 낮추는 것만으로도 많이 올라가.

**Q. 게임이 주기적으로 1~2초씩 멈춰요 (프리징).**  
A. RAM 부족으로 인한 가비지 컬렉션(GC) 현상이야. RAM 할당량 늘리고, Java 인수 최적화 설정 적용해봐. [성능 최적화 가이드](../setup-guide/01-performance.md) 참고.

**Q. 소리가 안 나와요.**  
A. `ESC → Options → Music & Sounds → Master Volume` 확인. 0이면 올리기.

**Q. 화면이 너무 어두워요.**  
A. `ESC → Options → Video Settings → Brightness` 올리기. 또는 야시경 물약 마시기.

**Q. 마우스 감도가 너무 빠르거나 느려요.**  
A. `ESC → Options → Mouse Settings → Sensitivity` 조절.

**Q. 키가 안 먹혀요.**  
A. 키 충돌일 가능성 높아. `ESC → Options → Controls` 에서 빨간색 표시된 키 확인하고 변경해. [키 바인드 가이드](../setup-guide/02-keybinds.md) 참고.

## ⚔️ 전투 & 직업 관련

**Q. 직업은 어떻게 선택해요?**  
A. `C키`를 누르면 직업 창이 열려. 다이아몬드 갑옷을 갖추고 나서 선택하는 걸 추천해.

**Q. 직업을 바꾸고 싶어요.**  
A. `C키` → 직업 창 → 다른 직업 선택 가능. 단, 기존 직업 스킬 포인트 초기화될 수 있어.

**Q. 스킬 포인트가 어디 갔어요?**  
A. `K키`로 스킬 트리 열면 포인트 확인 가능. 경험치로 레벨업하면 포인트가 생겨.

**Q. 공격해도 데미지가 너무 낮아요.**  
A. 공격 쿨다운 때문이야. 화면 하단 칼 아이콘이 꽉 찰 때까지 기다렸다가 공격해. 무작정 빠르게 클릭하면 데미지가 오히려 낮아져.

**Q. 몬스터가 너무 강해요.**  
A. Apotheosis **세계 티어(World Tier)**가 올라가 있을 가능성 있어. 세계 티어가 높을수록 몬스터가 강해지고 드롭도 좋아져. 세계 티어 확인하고 감당 가능한 티어로 조정해봐.

**Q. Affix 아이템이 뭐예요?**  
A. Apotheosis 모드로 추가된 특수 능력이 달린 장비야. 몬스터를 잡다 보면 파란색/노란색/분홍색 이름의 아이템이 드롭되는데 그게 Affix 아이템이야. 등급이 높을수록 능력이 강해.

**Q. 보스를 찾을 수가 없어요.**  
A. `/locate structure [구조물 이름]` 명령어 사용. 예: `/locate structure cataclysm:netherite_monstrosity`

## 🏗️ 아이템 & 제작 관련

**Q. 레시피를 모르겠어요.**  
A. 인벤토리(`E`)를 열면 오른쪽에 JEI 아이템 목록이 있어. 아이템에 마우스 올리고 `R` 누르면 레시피 나와.

**Q. 도구가 갑자기 사라졌어요.**  
A. 내구도가 0이 돼서 부서진 거야. 아이템 아이콘 아래 빨간 바가 뜨면 수리해줘야 해. [수리 방법 →](../user-guide/beginners-guide/04-gathering-crafting.md#아이템-수리-방법-4가지)

**Q. 모루 수리 비용이 너무 높다고 나와요.**  
A. 같은 아이템에 모루를 너무 많이 쓰면 비용이 무한정 올라가. 그라인딩스톤으로 인챈트 초기화하거나 새로 제작하는 게 나을 수 있어.

**Q. 작업대에 레시피 올려도 결과물이 안 나와요.**  
A. 모드 아이템은 모드 전용 작업대가 따로 있는 경우가 있어. JEI에서 레시피 확인할 때 어떤 작업대인지 같이 확인해봐.

**Q. 아이템 이름이 빨간색이에요.**  
A. 저주(Curse) 인챈트가 달린 아이템이야. 저주 종류:
- **사라짐의 저주**: 죽으면 아이템이 드롭 안 되고 사라짐
- **결합의 저주**: 갑옷 슬롯에서 뺄 수 없음
그라인딩스톤으로 인챈트 제거 가능.

**Q. 인챈트 테이블에서 원하는 인챈트가 안 나와요.**  
A. 인챈트는 랜덤이야. 원하는 게 나올 때까지 반복하거나, 마을 사서 주민에게서 인챈트 책을 구매하는 게 안정적이야.

**Q. Silent Gear 장비를 어떻게 만들어요?**  
A. Silent Gear 전용 작업대가 필요해. 자세한 건 [Silent Gear 가이드](../silent-gear/01-overview.md) 참고.

## 🌍 세계 & 탐험 관련

**Q. 집을 못 찾겠어요.**  
A. `F3`으로 현재 좌표 확인. 집 좌표를 미리 메모해두는 습관이 중요해. 나침반은 최초 스폰 위치를 가리켜 (침대 위치 아님).

**Q. 특정 구조물을 찾고 싶어요.**  
A. `/locate structure [구조물 ID]` 명령어 사용. 예:
```
/locate structure minecraft:village_plains  ← 평원 마을
/locate structure minecraft:stronghold      ← 요새 (엔드 포탈)
/locate structure whendungeonsarise:large_dungeon  ← WDA 대형 던전
```

**Q. 네더 포탈이 어디 연결될지 모르겠어요.**  
A. 네더 좌표 × 8 = 오버월드 좌표야. 예: 네더 X=100이면 오버월드 X=800 위치와 연결됨.

**Q. 엔드 포탈을 못 찾겠어요.**  
A. 엔더 아이를 던지면 요새 방향으로 날아가. 블레이즈 파우더 + 엔더 진주 → 엔더 아이. 땅으로 떨어지는 지점 아래를 파면 요새가 있어.

**Q. 맵이 어두워서 아무것도 안 보여요.**  
A. `ESC → Options → Video Settings → Brightness` 최대로 올리기. 또는 야시경 물약 사용.

**Q. 청크가 제대로 안 로드돼요.**  
A. `F3 + A`를 누르면 청크 강제 리로드돼.

## 🔧 기술 모드 관련

**Q. Create 기계가 안 돌아가요.**  
A. 회전력(SU)이 부족하거나 연결이 안 된 거야. 발전기(워터휠/풍차)가 연결되어 있는지, 회전축이 끊기지 않았는지 확인해봐.

**Q. Mekanism 기계가 작동을 안 해요.**  
A. 에너지가 없는 거야. 발전기 설치하고 에너지 케이블로 연결해줘야 해. 기계 우클릭하면 에너지 충전 상태 확인 가능.

**Q. AE2 네트워크가 작동을 안 해요.**  
A. 채널 초과가 원인인 경우가 많아. 연결된 기계 수가 케이블 채널 한도를 초과하면 작동 안 해. 밀도 케이블(Dense Cable)로 교체하거나 네트워크 구조 재설계 필요.

**Q. AE2 아이템이 네트워크에 안 들어가요.**  
A. ME 드라이브에 스토리지 셀이 들어있는지, 드라이브가 컨트롤러에 연결되어 있는지 확인해봐.

**Q. Create 기차가 움직이지 않아요.**  
A. 레일이 올바르게 연결되어 있는지, 기차 스케줄이 설정되어 있는지 확인. [Create 가이드](../create/01-overview.md) 참고.

## ✨ 마법 & 스킬 관련

**Q. 마법을 어떻게 배워요?**  
A. 던전, 마법사 탑, 구조물에서 마법 스크롤을 찾아서 주문서에 등록하면 돼. `Z키`로 마법 목록 확인.

**Q. 마법을 시전했는데 아무것도 안 나와요.**  
A. 마나가 부족한 거야. 마나 게이지 확인하고, 마나가 회복될 때까지 기다려.

**Q. 스킬 포인트를 잘못 투자했어요. 되돌릴 수 있나요?**  
A. `K키` → 스킬 트리에서 초기화 옵션 확인. 경험치나 특정 아이템을 소모하고 초기화 가능한 경우 있어.

**Q. 직업 고유 스킬이 안 써져요.**  
A. 스킬마다 쿨다운과 조건이 있어. 해당 직업 가이드에서 스킬 사용 조건 확인해봐. [RPG Classes 가이드](../classes/01-class-overview.md) 참고.

## 💥 오류 & 버그 관련

**Q. "Failed to synchronize registry data" 오류가 나요.**  
A. 서버와 클라이언트 모드팩 버전이 다를 때 나오는 오류야. 같은 버전의 모드팩을 사용하고 있는지 확인해봐.

**Q. 게임이 갑자기 꺼지고 Crash Report가 생겼어요.**  
A. 크래시 리포트 파일 위치: `%appdata%\.minecraft\crash-reports\`  
파일 열어서 `Caused by:` 부분을 찾으면 원인이 나와.  
주요 원인들:
- `OutOfMemoryError` → RAM 할당량 늘리기
- `NullPointerException` → 특정 모드 버그, 모드팩 재설치
- `ConcurrentModificationException` → 동시 접근 오류, 재시작

**Q. 특정 아이템을 들면 게임이 꺼져요.**  
A. 모드 버그일 가능성 높아. 그 아이템을 상자에 넣고 보관하거나, `/kill @e[type=item]` 명령어로 바닥 아이템 전체 제거 시도.

**Q. 청크 오류로 특정 위치에서 계속 튕겨요.**  
A. 해당 좌표 청크 파일이 손상된 거야.  
```
세계 폴더 → region 폴더 → 해당 좌표 .mca 파일 삭제
→ 게임 실행하면 해당 청크 새로 생성됨
⚠️ 그 청크에 있던 건축물/아이템은 사라져!
```

**Q. 몬스터가 스폰이 아예 안 돼요.**  
A. 게임 규칙에서 몬스터 스폰이 꺼진 거야. 치트 허용 상태에서:
```
/gamerule doMobSpawning true
```

**Q. 날씨가 계속 비가 와요.**  
A. 치트 허용 상태에서:
```
/weather clear 1000000  ← 맑은 날씨 유지
```

**Q. 모드 아이템의 한국어 번역이 이상해요 / 없어요.**  
A. 해당 모드가 한국어 번역을 지원하지 않거나, 번역이 미완성인 경우야. 영어로 보이는 건 정상이야.
