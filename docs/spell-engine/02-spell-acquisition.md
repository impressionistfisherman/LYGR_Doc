# 주문 습득 방법

주문을 사용하려면 세 단계를 거쳐야 합니다.

```
1. 스펠 스크롤 획득
      ↓
2. Spell Binding Table 제작
      ↓
3. 스펠북에 주문 등록
      ↓
   사용할 수 있습니다!
```

---

## Step 1 — 스펠 스크롤 (Spell Scroll) 획득

스펠 스크롤 하나에 주문 하나가 담겨 있습니다.

### 획득 경로

#### 던전 & 구조물 상자
- 마법사 타워, 위자드 던전, 일반 던전 상자 등에서 발견
- 가장 안정적인 초반 획득 경로
- 탐험 중 구조물을 발견하면 상자를 반드시 확인하세요

#### 마법 관련 몬스터 드롭
- 마법사 계열 몬스터(Witch, Evoker, 모드팩 추가 마법사 몹 등)가 드롭
- 직업 관련 스크롤을 목표로 특정 몬스터 파밍 가능

#### Wandering Trader / 마을 주민 거래
- 일부 주민이 스펠 스크롤을 판매
- 원하는 계열을 빠르게 구하고 싶을 때 유용

#### Apotheosis Invader 드롭
- Invader 처치 시 가끔 스펠 스크롤 드롭
- 고티어 Invader일수록 더 좋은 주문 스크롤 가능성 증가

---

## Step 2 — Spell Binding Table 제작

스펠 스크롤을 스펠북에 등록하는 전용 작업대입니다.

### 제작 방법

JEI에서 `Spell Binding Table`을 검색해 조합법을 확인하세요.  
초반 소재로 만들 수 있으므로 직업 선택 후 바로 제작을 권장합니다.

### 위치

Spell Binding Table은 설치형 블록입니다.  
작업 공간에 설치 후 우클릭으로 GUI를 열어 사용합니다.

---

## Step 3 — 스펠북에 주문 등록

### 스펠북(Spell Book) 이란?

주문을 담는 아이템입니다. 직업에 따라 전용 스펠북이 있습니다.

| 스펠북 종류 | 담을 수 있는 계열 |
|------------|-----------------|
| Wizard's Tome | Arcane, Frost |
| Elemental Grimoire | Fire, Frost, Lightning |
| Paladin's Bible | Holy, Healing |
| 범용 스펠북 | 계열 제한 없음 (효율 낮음) |

:::warning 스펠북과 계열이 맞지 않으면 등록 불가
예를 들어 Wizard's Tome에는 Holy 계열 주문을 등록할 수 없습니다.  
스크롤을 줍기 전에 내 직업 스펠북과 계열이 맞는지 확인하세요.
:::

### 등록 방법

1. Spell Binding Table GUI 열기
2. 왼쪽 슬롯에 **스펠북** 투입
3. 오른쪽 슬롯에 **스펠 스크롤** 투입
4. 경험치 소모 → 주문이 책에 추가됨

### 스펠북 용량

스펠북 하나에 등록할 수 있는 주문 수는 **최대 9개**입니다.  
9개를 초과하면 새 스펠북이 필요합니다.

---

## 직업 기본 스펠북

일부 직업은 직업 아이템을 습득할 때 **기본 주문이 이미 등록된 스펠북**을 받습니다.

- 별도 스크롤 없이 바로 사용할 수 있습니다
- 직업의 핵심 주문 1\~2개가 미리 등록되어 있음
- 추가 주문은 스크롤을 찾아서 등록해야 함

---

## 주문 업그레이드 — 상위 스크롤

같은 종류의 주문도 **티어(Tier)가 있습니다.**

```
Fireball I  →  Fireball II  →  Fireball III
  (기본)          (강화)          (최강화)
```

- 높은 티어 스크롤일수록 희귀하고 데미지/효과가 강함
- 낮은 티어 스크롤로 시작하고, 탐험하며 상위 티어로 교체
- 같은 주문이 이미 등록된 스펠북에 상위 티어 스크롤을 등록하면 업그레이드됨

---

## 스크롤 정리 팁

스펠 스크롤이 쌓이기 시작하면 내 직업에 맞는 것을 선별해야 합니다.

```
✅ 내 직업 주력 계열 스크롤 → 스펠북에 등록
✅ 좋은 계열이지만 지금 당장 필요 없는 것 → 보관
❌ 내 직업과 전혀 맞지 않는 계열 → 다른 플레이어와 교환 또는 판매
```

:::info 스크롤은 버리지 마세요
다른 직업으로 전환하거나, 파티원이 사용할 수 있으므로  
맞지 않는 스크롤도 일단 보관해두는 것이 좋습니다.
:::
