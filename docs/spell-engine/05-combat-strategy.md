# 속성 상성 & 전투 전략

적마다 특정 마법 계열에 강하거나 약한 속성이 있습니다.
상성을 파악하면 같은 Spell Power로도 훨씬 높은 데미지를 입힐 수 있습니다.

---

## 속성 상성표

### 바닐라 몬스터

| 몬스터 | 약점 계열 | 저항 계열 | 메모 |
|--------|-----------|-----------|------|
| 좀비, 스켈레톤 등 **언데드** | Holy ✨ | — | Holy 주문으로 추가 데미지 |
| **블레이즈** | Frost ❄️ | Fire 🔥 | 냉기 주문이 매우 효과적 |
| **가스트** | Frost ❄️ | Fire 🔥 | — |
| **엔더맨** | — | Arcane 🔮 | 비전 주문 효율 낮음 |
| **크리퍼** | Lightning ⚡ | — | 번개 주문 권장 |
| **위더** | — | — | 모든 계열 가능, Holy 추천 |
| **엔더 드래곤** | — | — | 특정 상성 없음 |

### 모드팩 추가 몬스터

모드팩에 포함된 추가 몬스터들은 몬스터 툴팁이나 Chronicles of Shadows에서 속성 정보를 확인하세요.
일반적으로:

- **용/드래곤 계열** — Frost 약점이 많음
- **빙결 계열 몬스터** — Fire 약점
- **어둠/저주 계열** — Holy 약점
- **기계/골렘 계열** — Lightning 약점 많음

> **참고: 모르면 일단 시도해보세요**
>
> 주문을 시전했을 때 데미지 숫자가 **초록색**으로 표시되면 약점 공격입니다.
> **빨간색** 숫자가 작게 표시되면 저항이 있다는 신호입니다.

## 전투 전략

### 마법사 (Wizard / Elemental Wizard) 기본 전투 패턴

마법 직업의 핵심은 **거리 유지**와 **쿨다운 관리**입니다.

```
기본 패턴:
  1. 안전 거리 확보 (이동 속도 중요)
  2. 주력 주문 시전
  3. 쿨다운 중 → 이동 or 보조 주문
  4. 적이 근접하면 → 냉기/넉백 주문으로 거리 벌리기
  5. 주력 주문 재시전
```

### 팔라딘 (Paladin) 기본 전투 패턴

근접 공격과 마법을 혼합하는 하이브리드 전투입니다.

```
기본 패턴:
  1. Holy/방어막 버프 선시전
  2. 근접 돌진 + 근접 공격
  3. 틈틈이 Holy 공격 주문 시전
  4. 체력이 낮아지면 Healing 주문 시전
  5. 아군 있을 때 → 적 처치보다 힐/버프 우선
```

## 상황별 주문 활용법

### 보스 전투

```
준비:
  - 버프 주문(방어막, 속도 증가 등) 전투 전에 선시전
  - 포션 충분히 준비

전투 중:
  - 차징 주문 → 안전 위치에서 사용
  - 보스 공격 패턴 파악 후 공격 타이밍 맞추기
  - 체력 50% 이하 → 즉시 힐/포션 사용 우선

피해야 할 것:
  - 보스 앞에서 차징 (피격으로 취소됨)
  - 쿨다운 기다리며 가만히 서 있기
```

### 다수 몬스터 처리

```
우선순위:
  1. 광역 주문(Lightning, Fire 폭발) 으로 먼저 약화
  2. 냉기 주문으로 둔화 → 접근 차단
  3. 개별 처치

실수 패턴:
  - 단일 타겟 주문으로 한 마리씩 처치 (비효율)
  - 포위당한 후에 차징 시전 (취소됨)
```

### 원거리 적 처리

```
  - Arcane/Fire 원거리 주문으로 선제 공격
  - 적이 이동하는 방향을 예측해서 주문 조준
  - 벽이나 장애물을 이용한 은엄폐 + 선타
```

## 오프핸드 스펠북 활용 전략

오프핸드 스펠북에는 **쿨다운이 다른 유틸리티 주문**을 넣는 것이 효율적입니다.

### 추천 오프핸드 주문 조합

| 상황 | 오프핸드 추천 주문 |
|------|-------------------|
| 생존 우선 | 즉시 힐 주문 |
| 기동성 확보 | 텔레포트, 대시 주문 |
| 적 제어 | 냉기 슬로우, 넉백 주문 |
| 파티 지원 | 버프, 실드 주문 |

## 마법 빌드 공통 주의사항

```
❌ 이렇게 하면 안 됩니다:
  - 계열이 다른 Spell Power 장비 착용 (데미지 증가 없음)
  - 차징 주문을 적 바로 앞에서 사용 (취소됨)
  - 쿨다운 중 멈춰서 기다리기 (기본 공격 + 이동으로 채우기)
  - Spell Power 낮은 상태에서 Spell Haste만 올리기

✅ 이렇게 하세요:
  - 주력 계열 장비 우선 수집
  - 이동 속도 어느 정도 확보 (전투 생존성)
  - 쿨다운 구성이 다른 주문 2\~3개 세트로 운용
  - 속성 상성 확인 후 전투
```
