# 주문 시전 방법

스펠북을 등록했다면 이제 전투에서 써야 합니다.
처음엔 UI가 낯설 수 있으니 조작법을 먼저 익히세요.

---

## 스펠 핫바 (Spell Hotbar)

스펠북 또는 직업 무기를 **손에 든 상태**에서 화면 하단에 스펠 핫바가 나타납니다.

- 현재 선택된 주문이 강조 표시됩니다
- 쿨다운 중인 주문은 아이콘이 어두워지며 타이머가 표시됩니다
- 사용 조건 미충족 시 아이콘이 빨갛게 표시됩니다

---

## 기본 조작법

| 조작 | 행동 |
|------|------|
| **마우스 스크롤** | 스펠 핫바 주문 전환 |
| **우클릭** | 현재 선택된 주문 시전 |
| **우클릭 길게 누르기** | 차징(Charging) 주문 시전 |
| **Shift + 스크롤** | 스펠북 간 전환 (여러 권 소지 시) |

### 단축키 커스터마이징

기본 단축키가 다른 모드와 충돌할 경우 **설정 → 키 바인딩**에서 변경하세요.

---

## 시전 방식 두 가지

### 즉시 시전 (Instant Cast)

우클릭 한 번으로 바로 발동되는 주문입니다.

```
특징:
- 시전 시간 없음
- 이동 중에도 시전 가능
- 쿨다운만 기다리면 연속 사용할 수 있습니다

예시: 단일 타격 주문, 텔레포트, 즉시 힐
```

### 차징 시전 (Charged Cast)

우클릭을 길게 누르는 동안 차징하다가 놓으면 발동하는 주문입니다.

```
특징:
- 차징 중 이동 불가 또는 속도 감소
- 차징이 완료되어야 최대 효과 발동
- 차징 중 피격 시 시전 취소될 수 있음
- 일반적으로 즉시 시전보다 강력함

예시: 대형 파이어볼, 광역 폭발, 집중 광선
```

> **주의: 차징 주문 사용 시 주의**
>
> 차징 중에는 이동이 제한됩니다.
> 적이 근접하면 시전이 취소될 수 있으므로 안전한 거리에서 사용하거나
> 아군이 탱킹해주는 상황에서 활용하세요.

## 쿨다운 (Cooldown) 관리

각 주문에는 사용 후 재사용까지의 쿨다운이 있습니다.

### 쿨다운 확인

- 스펠 핫바 아이콘 위에 쿨다운 타이머 표시
- 숫자가 사라지면 재사용할 수 있습니다

### 쿨다운 단축하는 방법

| 방법 | 효과 |
|------|------|
| **Spell Haste 스탯** | 모든 주문 쿨다운 % 단축 |
| **스킬 트리 노드** | 특정 주문 쿨다운 단축 |
| **특정 Affix 장비** | 주문 쿨다운 감소 효과 |

### 쿨다운 중 전투 방법

마법 직업이라도 쿨다운 중에는 기본 공격을 활용하세요.

```
권장 전투 패턴:
  주력 주문 시전 → 쿨다운 중 기본 공격 또는 보조 주문 → 쿨다운 완료 → 주력 주문
```

## 오프핸드 활용

스펠북을 **오프핸드(왼손 슬롯)** 에 넣으면 무기와 병행 사용할 수 있습니다.

```
메인핸드: 직업 무기 (근접 공격 가능)
오프핸드: 스펠북 (보조 주문 시전 가능)
```

> **정보: 오프핸드 스펠북 제한**
>
> 일부 주문은 **메인핸드 전용**으로 오프핸드 스펠북에서 시전 불가합니다.
> 주문 툴팁에서 시전 가능 손 정보를 확인하세요.

### 오프핸드 활용이 유용한 경우

- 근접 직업(Paladin, Berserker)이 보조 버프/힐 주문 추가 장착
- 원거리 직업이 긴급 텔레포트/회피 주문 상시 대기
- 쿨다운 다른 두 가지 주문을 번갈아 사용

---

## 주문 시전 조건

일부 주문에는 시전 조건이 있습니다. 조건 미충족 시 주문이 빨갛게 표시되며 시전 실패합니다.

### 조건 유형

| 조건 유형 | 설명 |
|-----------|------|
| **마나/에너지** | 직업에 따라 자원을 소모하는 주문 |
| **재료 소모** | 특정 아이템이 인벤토리에 있어야 시전 가능 |
| **특수 상태** | 특정 상태(스텔스, 분노 등)에서만 사용할 수 있습니다 |

### 조건 확인 방법

스펠 핫바의 주문 아이콘 위에 마우스를 올리면 시전 조건이 표시됩니다.

---

## 자주 발생하는 문제

**주문이 시전되지 않아요.**
- 쿨다운 중인지 확인
- 시전 조건(마나/재료 등)이 충족되어 있는지 확인
- 주문이 스펠북에 제대로 등록되어 있는지 확인

**우클릭이 블록 설치로 동작해요.**
- 스펠북이 메인핸드에 있는지 확인
- 빈 공간을 향해 우클릭하면 주문이 시전됩니다

**차징 주문이 도중에 취소돼요.**
- 피격 시 차징이 취소되는 주문입니다
- 안전한 위치에서 사용하거나 보호막 주문을 먼저 시전하세요
