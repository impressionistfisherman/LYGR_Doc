# 6단계 — 전투 & 성장

> 효과적인 전투 방법과 장비 강화 시스템을 익혀서 더 강해집시다!

---

## 바닐라 전투 완전 가이드

### 공격 쿨다운 시스템

마인크래프트는 **공격 쿨다운**이 있습니다.  
공격하고 나면 잠깐 기다려야 다음 공격이 최대 데미지가 됩니다.

```
화면 하단 핫바 옆에 작은 칼 아이콘이 있습니다
│▓▓▓▓▓░░│ ← 이렇게 채워지는 게 쿨다운 게이지
칼이 꽉 찼을 때 (100%) 공격 → 최대 데미지
칼이 안 찼을 때 공격 → 데미지 크게 감소!
```

**빠르게 클릭하는 게 나쁜 이유:**
- 클릭 속도 > 쿨다운 속도 → 공격이 최대 데미지의 20%만 나옴
- 천천히 기다렸다가 공격하는 것이 DPS(초당 데미지)가 더 높습니다

### 크리티컬 히트

```
공중에 있을 때 공격 → 데미지 1.5배 크리티컬!

실전 사용법:
1. 점프키 누르기
2. 올라가는 중에 공격 (내려오는 중 아님!)
3. 크리티컬 파티클(별) 뜨면 성공

크리티컬 불가 조건:
- 달리기 중 (Sprint)
- 웅크리기 중 (Shift)
- 사다리, 덩굴 위에 있을 때
- 상태 이상 중 (맹목, 둔화 등)
```

### 넉백 (밀쳐내기)

공격하면 적이 뒤로 밀려납니다.

```
기본 넉백 거리: 약 0.4블록
질주 공격 (달리기 중 공격): 넉백 대폭 증가 (약 3블록!)
인챈트 '넉백': 추가 거리
```

**절벽 전술:**
```
절벽 끝에 적을 몰고 → 질주 공격 → 적 낙사!
몬스터 그룹에서 유용
```

### 방패 전투

**방패 사용법:**
```
오프핸드(F키)에 장착
→ 우클릭 꾹 누르기 → 방어 자세
→ 정면 화살, 근거리 공격 차단!
```

**방패 효과:**
- 대부분의 화살 100% 차단
- 근거리 공격 데미지 대폭 감소
- 크리퍼 폭발도 방패 방향이면 피해 줄어듦

**방패 비활성화:**
적이 **도끼**로 공격하면 방패가 1.6초 동안 비활성화됩니다.  
→ 그 순간 반격 타이밍!

**방패 만들기:**
```
[나무 판자]
[나무 판자][철 주괴][나무 판자]
[나무 판자][나무 판자][나무 판자]
→ 방패 (나무 판자 6개 + 철 주괴 1개)
```

방법 3: 지그재그로 달리며 접근 (화살 회피율 높음)

### 💣 크리퍼 (Creeper) — 가장 위험!

| 항목 | 정보 |
|------|------|
| HP | 20 |
| 폭발 피해 | 거리에 따라 최대 49 (즉사 가능!) |
| 폭발 예고 | 쉿 소리 + 하얗게 번쩍임 |
| 폭발 시간 | 소리 후 약 1.5초 |
| 드롭 | 화약, 낙뢰 맞으면 음악 디스크 |

**공략 (가장 중요!):**
```
방법 1 — 안전 처치:
활로 멀리서 쏘기 (가장 안전)

방법 2 — 검 처치:
1. 크리퍼 발견
2. 빠르게 접근해서 1타
3. 즉시 뒤로 달리기 (3~4칸)
4. 크리퍼가 멈추면 다시 접근 1타
5. 반복 → 쉿 소리 나기 전에 처치

방법 3 — 쉿 소리 났을 때:
→ 쉿 소리 나면 즉시 뒤로 달리기
→ 5칸 이상 떨어지면 폭발 피해 없음
→ 9칸 이상이면 거의 안전
```

**크리퍼 폭발 피해 범위:**
```
즉사 범위: 1~2칸 이내
큰 피해: 3~4칸
약간 피해: 5~6칸
안전: 7칸 이상
```

## 인챈트 시스템 완전 가이드

### 인챈트 테이블 만들기

```
[다이아][    ][다이아]
[흑요석][책  ][흑요석]
[흑요석][흑요석][흑요석]
→ 인챈트 테이블 (다이아 2 + 흑요석 4 + 책 1)
```

### 책장으로 인챈트 레벨 높이기

인챈트 테이블 주변에 책장을 놓으면 더 높은 레벨의 인챈트가 가능합니다.

```
책장 최대 15개 → 레벨 30 인챈트 가능 (최대)
```

**책장 설치 규칙:**
```
인챈트 테이블 기준으로
→ 2칸 이내 (직접 붙이면 안 됨, 1칸 띄어야 함!)
→ 같은 높이 또는 1칸 높게
→ 사이에 아무 블록도 없어야 함 (공기여야 함)
```

**최적 배치 (U자형):**
```
📚📚📚📚📚
📚         📚
📚  📖테이블  📚
📚         📚
📚📚📚📚📚
```

**책장 만들기:**
```
[판자][판자][판자]
[책 ][책 ][책 ]    →  책장 (판자 6개 + 책 3개)
[판자][판자][판자]

책 만들기: 가죽 1개 + 종이 3개 → 책 1개
종이: 사탕수수 3개 → 종이 3개
```

### 인챈트 방법

1. 인챈트 테이블 우클릭 → UI 열림
2. 왼쪽에 장비 넣기
3. 오른쪽에 청금석(Lapis Lazuli) 넣기
4. 오른쪽에 3가지 인챈트 옵션 표시됩니다
5. 원하는 레벨 클릭 → 경험치 + 청금석 소모

> ⚠️ **인챈트는 무작위입니다!** 원하는 인챈트가 안 나올 수 있습니다.  
> 원하는 게 나올 때까지 반복하거나, 인챈트 책을 구해서 모루에서 합치는 방법도 있습니다.

### 필수 인챈트 목록

**무기 인챈트:**

| 인챈트 | 최대 레벨 | 효과 |
|--------|-----------|------|
| 날카로움 (Sharpness) | V | 데미지 +3 (V 기준) |
| 화염 (Fire Aspect) | II | 적에게 불 붙임 |
| 약탈 (Looting) | III | 아이템 드롭 증가 |
| 넉백 (Knockback) | II | 더 멀리 밀쳐냄 |
| 쓸기 (Sweeping Edge) | III | 범위 공격 강화 |

**갑옷 인챈트:**

| 인챈트 | 최대 레벨 | 효과 |
|--------|-----------|------|
| 보호 (Protection) | IV | 모든 피해 감소 |
| 화염 보호 (Fire Protection) | IV | 화염 피해 감소 |
| 폭발 보호 (Blast Protection) | IV | 폭발 피해 감소 |
| 투사체 보호 (Projectile Protection) | IV | 화살 피해 감소 |
| 가시 (Thorns) | III | 공격 당하면 반격 |
| 수중 호흡 (Respiration) | III | 수중에서 산소 감소 느림 |
| 수생 친화 (Aqua Affinity) | I | 수중 채굴 속도 정상화 |
| 비단 발걸음 (Feather Falling) | IV | 낙하 피해 감소 |

**도구 인챈트:**

| 인챈트 | 최대 레벨 | 효과 |
|--------|-----------|------|
| 효율 (Efficiency) | V | 채굴 속도 증가 |
| 행운 (Fortune) | III | 광석 드롭량 증가 |
| 정밀 채굴 (Silk Touch) | I | 블록 그대로 드롭 |
| 내구성 (Unbreaking) | III | 내구도 감소 속도 낮춤 |
| **수리 (Mending)** | I | 경험치로 내구도 회복 |

> 💡 **수리(Mending)는 모든 장비에 필수!**  
> 수리 인챈트가 있으면 경험치를 얻을 때마다 장비가 자동으로 수리됩니다.  
> 테이블에서 나오기 어려우니까 마을 사서에게서 인챈트 책으로 구매 추천!

### 모루 (Anvil) 사용법

```
만들기: 철 블록 3개 + 철 주괴 4개 → 모루
```

**모루로 할 수 있는 것들:**
- 인챈트 책 + 장비 → 장비에 인챈트 적용
- 장비 + 같은 재료 → 내구도 수리
- 장비 이름 바꾸기
- 인챈트된 장비 2개 합치기

> ⚠️ **모루는 사용할수록 비용이 증가합니다!**  
> 같은 장비에 모루를 너무 많이 쓰면 나중에 "수리 비용 초과"가 돼서 사용할 수 없습니다.

### ⚔️ RPG Classes (직업 시스템)

다이아몬드 장비를 갖추면 **직업 선택**이 가능합니다.  
직업을 선택하면 패시브 능력치와 고유 기술이 생깁니다.

**직업 선택하는 법:**
```
설정 → 키 바인딩 → class 또는 rpg 검색
→ 직업 창 열기 → 원하는 직업 선택
```

**직업별 상세 설명:**

| 직업 | 주 스타일 | 장점 | 단점 |
|------|-----------|------|------|
| **Berserker** | 근거리 돌격 | HP 많음, 강한 공격 | 방어력 낮음 |
| **Rogue** | 기습, 독 | 빠름, 백어택 강함 | HP 낮음 |
| **Warrior** | 균형형 전사 | 방패 시너지, 안정적 | 특출난 점 없음 |
| **Paladin** | 성기사 | 치유 + 방어 | 공격력 낮음 |
| **Priest** | 서포터 | 아군 힐, 버프 | 혼자선 약함 |
| **Wizard** | 마법 딜러 | 원거리, 광역 | HP 낮음, 마나 관리 필요 |
| **Witcher** | 몬스터 사냥꾼 | 특정 몬스터에 특화 | 만능이 아님 |

**초보자 추천 직업:**
- **Berserker** — 그냥 가서 공격하면 됩니다. 가장 직관적
- **Warrior** — 방패 + 검 스타일, 안정적인 생존

**자세한 내용:** [RPG Classes 가이드 →](../../classes/01-class-overview.md)

### ✨ Apotheosis (강화 & 인챈트 확장)

바닐라 인챈트 시스템을 크게 확장하는 모드입니다.

**핵심 기능:**
- **Affix 아이템**: 랜덤 특수 능력이 붙은 장비. 몬스터/보스 처치시 드롭
- **젬 & 소켓**: 장비에 소켓을 뚫어서 젬을 박으면 능력치 추가
- **세계 티어**: 전체 몬스터 강도와 드롭 품질을 올릴 수 있습니다

**Affix 아이템 등급:**
```
일반(흰색) < 마법(파란색) < 희귀(노란색) < 영웅(분홍) < 신화(빨강)
```

**자세한 내용:** [Apotheosis 가이드 →](../../apotheosis/01-overview.md)

## 전투 팁 모음

### 상황별 전투 팁

| 상황 | 팁 |
|------|-----|
| 적이 많을 때 | 좁은 통로로 유도해서 한 마리씩 |
| 체력 낮을 때 | 도망치면서 음식 먹기 (먹는 중에도 이동할 수 있습니다) |
| 동굴에서 | 횃불로 시야 확보 후 전투 |
| 보스전 | 환경 이용 (절벽, 물, 높은 곳) |
| 크리퍼 폭발 | 방패 정면으로 피해를 줄일 수 있습니다 |

### 경험치 효율적으로 모으기

경험치는 인챈트, 수리 등에 필요합니다.

**경험치 많이 주는 것들:**
- 몬스터 처치 (일반: 5, 보스: 수십~수백)
- 채굴 (광석별로 다름, 다이아가 제일 많음)
- 양조 (물약 만들기)
- 동물 번식 (가끔)
- 용광로 결과물 수거

**경험치 팜 만들기:**
```
던전 스포너 발견
→ 스포너 주변 공간 만들기
→ 몬스터가 한 곳에 모이게 유도
→ 한 방에 처치하면서 경험치 수집
```
