# 7단계 — 마법 & 스킬

> 바닐라 마법 요소를 익히고, 이 모드팩의 강력한 마법 & 스킬 시스템으로 넘어가봅시다.

---

## 바닐라 마법 요소

### 물약 양조 시스템

물약은 전투에서 매우 유용합니다. 특히 보스전에서 필수입니다!

**물약 양조기 만들기:**
```
[    ][블레이즈 막대][    ]
[조약돌][조약돌][조약돌]
[조약돌][조약돌][조약돌]
→ 양조기 (블레이즈 막대 1개 + 조약돌 3개)
```

**유리병 만들기:**
```
[유리][    ][유리]
[    ][유리][    ]
→ 유리병 3개 (유리 3개)
```

**물 채우기:**
```
유리병 들고 물 우클릭 → 물병
```

### 물약 제조 과정

```
1단계: 물병 → + 네더 사마귀 → 어색한 물약 (기본 물약)
2단계: 어색한 물약 → + 재료 → 효과 물약
3단계 (선택): 화약 → 투척 물약, 용의 숨결 → 잔류 물약
```

**기본 필수 물약들:**

| 물약 | 2단계 재료 | 효과 | 지속 시간 |
|------|-----------|------|-----------|
| 치유 물약 | 빛나는 수박 | 즉시 체력 +4 | 즉시 |
| 재생 물약 | 유령 눈물 | 체력 서서히 회복 | 45초 |
| 힘 물약 | 블레이즈 파우더 | 공격력 +3 | 3분 |
| 속도 물약 | 설탕 | 이동 속도 +20% | 3분 |
| 내화 물약 | 마그마 크림 | 화염 피해 무효 | 3분 |
| 야시경 물약 | 황금 당근 | 어두운 곳도 밝게 | 3분 |
| 수중 호흡 물약 | 복어 | 물속 산소 감소 안 함 | 3분 |
| 느린 낙하 물약 | 유령 솜털 | 낙하 속도 감소 | 1분 30초 |

**물약 강화 (레드스톤/형석):**
- 레드스톤 추가 → 지속 시간 8분으로 연장
- 형석 추가 → 효과 강화 (II로 업그레이드), 지속 시간 단축

**물약 원재료 획득처:**

| 재료 | 획득 방법 |
|------|-----------|
| 네더 사마귀 | 네더 요새 농장 |
| 빛나는 수박 | 황금 너깃 8개 + 수박 |
| 블레이즈 파우더 | 블레이즈 막대 가공 |
| 마그마 크림 | 마그마 큐브 처치 |
| 황금 당근 | 황금 너깃 8개 + 당근 |
| 유령 눈물 | 유령 처치 |
| 설탕 | 사탕수수 가공 |
| 복어 | 낚시 or 물속 처치 |

### 인챈트된 황금 사과

엄청나게 강력한 아이템입니다!

**일반 황금 사과:**
```
황금 너깃 8개 + 사과 1개 → 황금 사과
효과: 흡수 2하트(30초), 재생 II(5초)
```

**인챈트 황금 사과 (God Apple):**
```
황금 블록 8개 + 사과 1개 → 인챈트 황금 사과
효과: 흡수 4하트(2분), 재생 V(20초), 내화(5분), 저항(5분)
→ 보스전 직전에 먹으면 거의 무적!
→ 제작 불가, 구조물 체스트에서만 획득
```

## 🎮 이제 이 모드 써보세요! — 마법 시스템

### ✨ Spell Engine (마법 주문 시스템)

이 모드팩의 핵심 마법 시스템입니다.  
마법 스크롤을 수집해서 강력한 주문을 시전할 수 있습니다.

**기본 개념:**

| 개념 | 설명 |
|------|------|
| 마나 | 마법 시전에 필요한 자원. 시간이 지나면 회복 |
| 스크롤 | 마법을 배우는 재료. 던전, 마법사 탑에서 획득 |
| 주문서 | 스크롤을 넣어서 마법 사용. 무기처럼 착용 |
| 시전 방식 | 원하는 주문을 선택 후 우클릭으로 시전 |

**마법 배우는 방법:**
```
1. 던전, 구조물, 마법사 탑 탐험 → 마법 스크롤 획득
2. 주문서(Spell Book) 제작 또는 획득
3. 인벤토리에서 스크롤 → 주문서에 적용
4. `Controls`에서 `spell`, `cast`, `book` 검색 → 마법 선택/시전 키 확인
5. 주문서 들고 우클릭 → 마법 시전!
```

**원소별 마법 종류:**

| 원소 | 특징 | 대표 마법 |
|------|------|-----------|
| 🔥 화염 | 지속 데미지, 범위 | 화염구, 화염 폭발 |
| ❄️ 얼음 | 둔화, 동결 | 얼음 창, 눈보라 |
| ⚡ 번개 | 높은 단타, 마비 | 번개 소환, 전기 충격 |
| 💚 자연 | 치유, 버프 | 치유 파동, 재생 토템 |
| 🌑 암흑 | 디버프, 흡수 | 생명 흡수, 저주 |
| 💨 바람 | 이동, 밀쳐내기 | 돌풍, 비행 |

**마나 관리 팁:**
```
마나 회복 속도: 천천히 자동 회복
마나 증가 방법:
→ Puffish Skills 마법 트리에 포인트 투자
→ 마법 관련 장비 착용
→ 마나 회복 물약 사용
```

**자세한 내용:** [Spell Engine 가이드 →](../../spell-engine/01-overview.md)

### 🌳 Puffish Skills (RPG 스킬 트리)

레벨업해서 **스킬 포인트**를 얻고 캐릭터를 성장시키는 시스템입니다.

**경험치 획득 방법:**
- 몬스터 처치
- 광석 채굴
- 아이템 제작
- 낚시
- 동물 번식

**스킬 포인트 얻는 법:**
```
경험치 바가 꽉 차면 레벨업
레벨업마다 스킬 포인트 1~3개 획득
Controls에서 `skill` 또는 `puffish` 검색 → 스킬 트리 열기 → 포인트 투자
```

**스킬 트리 상세:**

**전투 트리 (Combat):**

- 체력 증가 (HP +2~10)
- 공격력 증가 (ATK +0.5~3)
- 크리티컬 확률 증가
- 크리티컬 데미지 증가
- 방어력 증가
- 넉백 저항

**채굴 트리 (Mining):**

- 채굴 속도 증가
- 행운 효과 증가 (더 많은 드롭)
- 특수 광석 발견 확률 증가
- 인벤토리 확장 (추가 슬롯)

**마법 트리 (Magic):**

- 최대 마나 증가
- 마나 회복 속도 증가
- 마법 데미지 증가
- 마법 지속 시간 증가
- 마법 범위 증가

**생존 트리 (Survival):**

- 허기 감소 속도 낮춤
- 낙하 피해 감소
- 독/화염 저항
- 야간 시야
- 수중 이동 속도 증가

**빌드 추천:**

| 플레이 스타일 | 트리 분배 추천 |
|--------------|--------------|
| 근거리 전사 | 전투 70% + 생존 30% |
| 마법사 | 마법 60% + 전투 20% + 생존 20% |
| 탐험가 | 생존 40% + 채굴 40% + 전투 20% |
| 올라운더 | 4개 트리 균등 분배 |

**자세한 내용:** [Puffish Skills 가이드 →](../../skill-system/01-overview.md)

## 바닐라 마법 vs 모드 마법 비교

| 항목 | 바닐라 인챈트/물약 | Spell Engine |
|------|-----------------|--------------|
| 사용 방법 | 패시브 (자동 적용) | 능동적 (버튼 누르기) |
| 자원 소모 | 없음 | 마나 소모 |
| 다양성 | 인챈트 30여종 | 마법 60+ 종 |
| 획득 방법 | 테이블, 교역 | 스크롤 수집 |
| 전투 스타일 | 큰 변화 없음 | 마법사 스타일 가능 |

## 마법 & 스킬 시너지 빌드

직업(RPG Classes) + 스킬 트리(Puffish) + 마법(Spell Engine) 조합이 중요합니다!

**마법사 빌드 예시:**
```
직업: Wizard (마법 능력치 보너스)
+ Puffish 마법 트리 최대 투자 (마나 +100, 마법 뎀 +30%)
+ Spell Engine 번개 계열 마법 특화
= 엄청난 원거리 마법 딜러
```

**근거리 마법검사 빌드:**
```
직업: Warrior
+ Puffish 전투 + 마법 균등 투자
+ Spell Engine 자기 버프 마법 (힘, 속도, 방어)
= 근거리 전투 + 자기 강화 마법 조합
```
