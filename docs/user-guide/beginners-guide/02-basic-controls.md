# 2단계 — 기본 조작법

> 마인크래프트의 기본 조작을 정리했습니다.  
> 이미 마인크래프트를 해봤으면 넘어가도 되고, 처음이면 꼭 읽어보세요!

---

## 이동 조작

### 기본 이동

| 키 | 동작 | 설명 |
|----|------|------|
| `W` | 앞으로 이동 | 마우스 방향 기준 |
| `S` | 뒤로 이동 | |
| `A` | 왼쪽으로 이동 | |
| `D` | 오른쪽으로 이동 | |
| `Space` | 점프 | 1칸 높이 점프 |
| `Shift` | 살금살금 걷기 | 절벽에서 안 떨어짐! |
| `Ctrl` | 달리기 | W와 함께 누르기 |
| `마우스 이동` | 시점 회전 | 좌우상하 시점 |

> 💡 **Shift(살금살금)는 정말 중요합니다!**  
> 절벽 가장자리나 높은 곳에서 Shift를 누르고 있으면 절대 떨어지지 않습니다.  
> 위험한 곳에서 습관적으로 Shift를 눌러두는 것이 좋습니다.

### 달리기 (Sprint)

`Ctrl + W`를 누르거나, `W`를 빠르게 두 번 눌러도 달리기가 됩니다.  
달리기를 하면 이동 속도가 약 1.3배 빨라지고 **허기가 더 빨리 닳습니다**.  
가까운 거리는 그냥 걷고, 먼 거리를 이동할 때는 달리세요.

### 수영

물에 들어가면 자동으로 수영 모드가 됩니다.
- `Space` — 위로 올라가기
- `Shift` — 아래로 내려가기
- `W/A/S/D` — 수평 이동

> ⚠️ 물속에 있으면 **산소(공기 방울)**가 줄어듭니다. 다 떨어지면 익사할 수 있습니다!  
> 물속에서 산소 방울 아이콘이 줄어드는 게 보이면 빨리 올라오세요.

### 웅크리기 vs 살금살금

Shift를 누르면 캐릭터가 낮아지면서 살금살금 걷습니다. 이걸 **스니킹(Sneaking)**이라고 합니다.
- 블록 끝에서 안 떨어짐
- NPC 이름표 숨기기
- 일부 모드에서 스텔스 효과

## 마우스 조작

| 동작 | 기능 |
|------|------|
| 마우스 이동 | 시점 회전 |
| **왼쪽 클릭** | 블록 채굴 / 몬스터 공격 |
| **오른쪽 클릭** | 블록 설치 / 아이템 사용 / NPC 대화 |
| **마우스 휠** | 핫바 슬롯 변경 |
| 가운데 클릭 | 보고 있는 블록을 핫바에 복사 (크리에이티브 모드) |

### 왼쪽 클릭 — 채굴

- **꾹 누르고 있어야 합니다!** 한 번만 톡 치면 채굴이 안 됩니다.
- 올바른 도구를 사용해야 빨리 캘 수 있습니다 (아래 채굴 섹션 참고)
- 채굴 중에 마우스를 블록에서 벗어나면 취소됩니다

### 오른쪽 클릭 — 상호작용

오른쪽 클릭은 상황에 따라 다르게 작동합니다:
- 빈 손으로 블록 우클릭 → 블록과 상호작용 (문 열기, 상자 열기, 버튼 누르기)
- 아이템 들고 블록 우클릭 → 아이템 설치
- 음식 들고 허공에 우클릭 → 먹기
- 활 들고 우클릭 → 조준

> 💡 **아이템 설치하려는데 계속 상호작용만 되는 경우:**  
> Shift를 누르면서 우클릭하면 강제로 설치됩니다!

## 인벤토리 조작

### 인벤토리 열기/닫기

`E` 키를 누르면 인벤토리가 열립니다. 다시 `E`나 `ESC`를 누르면 닫힙니다.

### 인벤토리 구조

| 영역 | 슬롯 | 용도 |
|------|------|------|
| 갑옷 슬롯 | 투구, 흉갑, 레깅스, 부츠 | 방어구 장착 |
| 오프핸드 | 방패/보조 아이템 1칸 | 방패, 토템, 횃불, 스펠북 등 |
| 인벤토리 | 27칸 | 들고 다니는 아이템 보관 |
| 핫바 | 9칸 | 숫자키 `1`\~`9`로 바로 사용하는 아이템 |

핫바는 화면 아래에 항상 보이는 9칸입니다. 전투, 채굴, 음식처럼 자주 쓰는 아이템은 핫바에 넣어두세요.

### 아이템 이동하기

| 방법 | 동작 |
|------|------|
| **클릭** | 아이템 집기 (들고 있는 중에 다시 클릭하면 놓기) |
| **Shift + 클릭** | 인벤토리 ↔ 핫바 간 즉시 이동 |
| **드래그** | 들고 있는 아이템을 여러 칸에 분배 |
| **우클릭 (스택에서)** | 절반만 집기 |
| **우클릭 (들고 있는 중)** | 한 개씩 놓기 |
| **더블클릭** | 같은 아이템 전부 모으기 |
| **Q** | 아이템 하나 바닥에 버리기 |
| **Ctrl + Q** | 아이템 전체 바닥에 버리기 |

> 💡 **Shift + 클릭을 정말 많이 쓰게 될 것입니다.** 이것 하나만 익혀도 인벤토리 관리가 훨씬 편합니다.

### 핫바 (Hotbar)

화면 맨 아래에 항상 보이는 **9칸**이 핫바입니다.  
`1\~9` 키를 누르거나 마우스 휠을 굴려서 슬롯을 바꿀 수 있습니다.

**핫바에 넣어두면 좋은 것들:**
- 1번: 주무기 (검, 도끼)
- 2번: 곡괭이
- 3번: 도끼 (나무 채집용)
- 4번: 삽 (흙, 모래 채집용)
- 5번: 음식
- 6번: 횃불
- 7번: 블록 (건축 재료)
- 8번: 활 or 보조 무기
- 9번: 여유 슬롯

## 기본 조작 키 전체 정리

### 게임 내 기본 키

| 키 | 기능 |
|----|------|
| `E` | 인벤토리 열기 |
| `F` | 양손 아이템 교체 |
| `Q` | 아이템 버리기 |
| `T` | 채팅창 열기 |
| `/` | 명령어 입력 |
| `F1` | HUD 숨기기/보이기 |
| `F2` | 스크린샷 |
| `F3` | 디버그 화면 (좌표, FPS 등) |
| `F5` | 3인칭 시점 전환 |
| `ESC` | 게임 메뉴 |

### F3 디버그 화면 — 중요!

`F3`을 누르면 화면에 정보가 잔뜩 나옵니다. 처음엔 복잡해 보이지만 몇 가지만 알면 됩니다.

```
XYZ: 123.4 / 64.0 / -56.7
      ↑         ↑      ↑
      X좌표    Y좌표   Z좌표
      (동서)  (높이)  (남북)
```

- **Y 좌표**: 현재 높이. 광석 찾을 때 중요합니다!
  - Y=64: 지표면 기준
  - Y=0\~16: 다이아몬드 있는 깊이
  - Y=-57: 다이아 + 레드스톤 최적 채굴 위치

## 채굴 기초

### 채굴 방법

1. 캐고 싶은 블록에 마우스 커서 올리기
2. 왼쪽 버튼 **꾹 누르기**
3. 블록에 금이 가면서 부서짐 → 아이템 드롭
4. 가까이 가면 자동으로 줍기

### 올바른 도구 사용

도구가 없거나 잘못된 도구를 쓰면 채굴이 느리거나 아이템이 안 드롭됩니다.

| 블록 종류 | 필요 도구 | 잘못된 도구로 캐면? |
|-----------|-----------|-------------------|
| 나무, 흙, 모래 | 없어도 가능 (도끼/삽이면 더 빠름) | 느리지만 드롭됩니다 |
| 돌, 조약돌 | 나무 곡괭이 이상 | 드롭되지 않음! |
| 철 원석 | 돌 곡괭이 이상 | 드롭되지 않음! |
| 금 원석 | 철 곡괭이 이상 | 드롭되지 않음! |
| 다이아몬드 원석 | 철 곡괭이 이상 | 드롭되지 않음! |
| 흑요석 | 다이아 곡괭이 이상 | 드롭되지 않음! |

> ⚠️ **중요!** 다이아몬드를 나무 곡괭이로 캐면 다이아몬드가 드롭되지 않고 블록만 사라집니다.  
> 반드시 올바른 도구 확인하고 채굴하세요!

### 도구 종류와 용도

| 도구 | 주 용도 |
|------|---------|
| **곡괭이 (Pickaxe)** | 돌, 광석, 금속 블록 |
| **도끼 (Axe)** | 나무, 목재 블록 |
| **삽 (Shovel)** | 흙, 모래, 자갈, 눈 |
| **괭이 (Hoe)** | 흙을 경작지로 변환 (농사용) |
| **검 (Sword)** | 전투, 거미줄 제거 |

## 제작 (크래프팅) 기초

### 인벤토리 제작 (2x2)

인벤토리를 열면 오른쪽 위에 **2x2 제작 칸**이 있습니다.  
간단한 아이템 (판자, 막대기, 상자 등)은 여기서 만들 수 있습니다.

### 작업대 제작 (3x3)

더 복잡한 아이템은 **작업대**가 필요합니다. 작업대는 3x3 제작 칸을 제공합니다.

**작업대 만들기:**
```
[판자][판자]    →    [작업대]
[판자][판자]
```
나무 판자 4개를 2x2로 놓으면 됩니다.

**나무 판자 만들기:**
```
인벤토리 2x2에 나무 원목 1개 → 나무 판자 4개
```

### JEI — 레시피 검색 도구 (모드)

이 모드팩에는 **JEI (Just Enough Items)**가 설치되어 있습니다.  
인벤토리를 열면 오른쪽에 아이템 목록이 보입니다.

| 동작 | 방법 |
|------|------|
| 아이템 레시피 보기 | 아이템에 마우스 올리고 `R` |
| 아이템 사용처 보기 | 아이템에 마우스 올리고 `U` |
| 아이템 검색 | 아래 검색창에 이름 입력 |
| 창고로 아이템 이동 | 아이템 클릭 (상자 열었을 때) |

> 💡 **JEI는 정말 자주 쓰게 될 것입니다.** 레시피를 모르면 바로 R 눌러서 확인하세요!

## 모드팩 전용 추가 키

모드팩에는 바닐라에 없는 키가 많이 추가됩니다. 다만 모드 키는 버전과 설정에 따라 달라질 수 있으므로, 아래처럼 확인해서 사용하세요.

| 기능 | 관련 모드 | 확인 방법 |
|------|-----------|-----------|
| 스킬 트리 | Puffish Skills | `Controls`에서 `skill` 또는 `puffish` 검색 |
| 직업/클래스 창 | RPG Classes | `Controls`에서 `class` 또는 `rpg` 검색 |
| 마법 선택/시전 | Spell Engine 계열 | `Controls`에서 `spell`, `cast`, `book` 검색 |
| 배낭 열기 | Traveler's Backpack | 기본값은 보통 `B`, 충돌 시 `backpack` 검색 |
| 지도/웨이포인트 | 지도 모드 | `map`, `waypoint`, `journey`, `xaero` 검색 |
| 퀘스트북 | 퀘스트 모드 | `quest`, `ftb` 검색 |

확실하게 자주 쓰는 키는 다음과 같습니다.

| 키 | 기능 | 관련 모드 |
|----|------|-----------|
| `R` | 아이템 레시피 보기 | JEI |
| `U` | 아이템 사용처 보기 | JEI |
| `W` 길게 | Create Ponder 설명 보기 | Create |
| `Ctrl + T` | World Tier 화면 열기 | Apotheosis |
| `Ctrl` | 재료 스탯/특성 확인 | Silent Gear |

> 💡 키가 겹치거나 변경하고 싶으면 `ESC → Options → Controls → Key Binds`에서 기능명을 검색해 변경하세요.

## 생존 관련 기초 지식

### 허기 게이지

화면 아래 오른쪽에 🍗 아이콘이 10개 있습니다. 이것이 **허기 게이지**입니다.

```
🍗🍗🍗🍗🍗🍗🍗🍗🍗🍗  ← 배부름 (달리기 가능, 체력 회복)
🍗🍗🍗🍗🍗🍗░░░░░░  ← 허기 있음 (달리기 불가)
🍗░░░░░░░░░░░░  ← 배고픔 (체력 닳기 시작)
░░░░░░░░░░░░░░  ← 굶주림 (체력이 1칸만 남음)
```

- 허기가 **6칸 이상**이면 체력이 자동 회복됩니다
- 허기가 **3칸 이하**면 달리기 불가
- 허기가 **0**이면 일반 난이도에서 체력이 0.5칸까지 깎입니다

### 체력 게이지

화면 아래 왼쪽에 ❤️ 아이콘이 10개 있습니다. 기본 최대 체력은 20 (하트 10개).

- 모드로 최대 체력이 늘어날 수 있습니다
- 허기가 충분하면 자동 회복 (느림)
- 치유 물약, 음식으로 즉시 회복 가능

### 산소 게이지

물속에 들어가면 하트 위에 방울이 나타납니다. **10개**가 다 없어지면 익사 피해를 받습니다.  
물 위로 올라오면 바로 회복됩니다.
