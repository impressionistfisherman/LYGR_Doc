# 4단계 — 채집 & 제작

> 자원을 체계적으로 모으고, 철 → 다이아 장비까지 업그레이드하는 법을 알아봅시다.

---

## 지하 탐험 준비

### 탐험 전 체크리스트

지하에 내려가기 전에 이것들을 준비해:

- [ ] 돌 곡괭이 (또는 철 곡괭이) 2~3개 (내구도 소모되니까 여러 개!)
- [ ] 돌 검
- [ ] 횃불 64개 이상 (지하는 어두움!)
- [ ] 음식 32개 이상
- [ ] 인벤토리 빈 공간 (자원 담을 자리)

> ⚠️ **죽으면 아이템 다 잃어!**  
> 좋은 아이템은 집에 상자를 만들어 보관하고, 채굴용 아이템만 챙겨가세요.

### 상자 만들기

아이템을 보관할 상자를 먼저 집에 만들어두세요.

```
[판자][판자][판자]
[판자]      [판자]
[판자][판자][판자]  →  상자 (가운데 비워!)
```

상자 2개를 옆에 나란히 설치하면 **대형 상자**가 됩니다 (2배 용량).  
집에 여러 개 만들어두고 종류별로 분류해두면 편합니다.

## 광석 완전 가이드

### 모든 광석 목록

| 광석 | 색 | 채굴 가능 Y범위 | 최적 Y좌표 | 필요 곡괭이 |
|------|-----|--------------|-----------|------------|
| 석탄 | 검은 점 | 0~320 | Y=96 부근 | 나무 이상 |
| 구리 | 주황 점 | -16~112 | Y=48 부근 | 돌 이상 |
| 철 | 베이지/황금 점 | -64~320 | Y=16 부근 | 돌 이상 |
| 금 | 노란 점 | -64~32 | Y=-16 부근 | 철 이상 |
| 레드스톤 | 빨간 점 | -64~16 | Y=-59 부근 | 철 이상 |
| 에메랄드 | 초록 점 | -16~320 | 산 바이옴 | 철 이상 |
| 다이아몬드 | 파란 점 | -64~16 | Y=-57 부근 | 철 이상 |
| 흑요석 | 보라/검정 | 용암+물 접촉 | 지하 | 다이아 이상 |
| 고대 잔해 | 갈색 | -16~119 | Y=15 부근 | 다이아 이상 |

> 💡 **F3 누르면 현재 Y좌표 확인할 수 있습니다!**  
> 다이아 찾을 거면 Y=-57까지 내려가야 합니다.

### 효율적인 채굴 방법

**방법 1: 계단식 내려가기 (안전)**
```
앞으로 2칸 파고 → 1칸 내려가고 → 앞으로 2칸 파고 → 반복
이렇게 하면 안전하게 깊이 내려갈 수 있습니다
```

**방법 2: 수직 갱도 파기 (빠름, 위험)**
```
2x1 크기로 수직으로 아래로 파기
한 칸씩 내려가면서 옆에 발판 만들기
⚠️ 용암이 나올 수 있어서 위험합니다!
```

**방법 3: 동굴 탐험 (효율 최고)**
```
자연 생성된 동굴 찾기
동굴 벽에 이미 광석이 노출되어 있음
→ 광석 발견 즉시 채굴
⚠️ 몬스터가 많아서 위험함!
```

**방법 4: 지층 채굴 (Branch Mining, 가장 효율적)**
```
목표 깊이까지 내려가기 (예: Y=-57)
→ 중심 통로 만들기 (2칸 높이, 1칸 넓이)
→ 좌우로 2칸 간격으로 옆 통로 파기
→ 각 통로를 16~24칸씩 파기
→ 광석 발견하면 채굴
```

## 제련 시스템

### 용광로 사용법

```
용광로 설치 → 우클릭으로 열기

위쪽 칸: 제련할 재료
아래쪽 칸: 연료
오른쪽 칸: 결과물
```

### 연료 종류와 효율

| 연료 | 아이템 1개당 제련 횟수 |
|------|---------------------|
| 나무 판자 | 1.5개 |
| 나무 원목 | 1.5개 |
| 석탄 | 8개 |
| 목탄 | 8개 |
| 석탄 블록 | 80개 |
| 용암 양동이 | 100개 |

> 💡 **석탄 블록이 가장 효율적!** 석탄 9개로 블록 1개 만들 수 있습니다.

### 주요 제련 레시피

| 재료 | 결과물 | 용도 |
|------|--------|------|
| 철 원석 | 철 주괴 | 도구, 갑옷, 기계 |
| 금 원석 | 금 주괴 | 황금 도구, 장식 |
| 구리 원석 | 구리 주괴 | Create 모드 재료 |
| 모래 | 유리 | 창문, 병 |
| 조약돌 | 돌 | 건축 재료 |
| 나무 원목 | 목탄 | 석탄 대용 연료 |
| 날것 소고기 | 익힌 소고기 | 음식 |
| 날것 닭고기 | 익힌 닭고기 | 음식 |
| 날것 돼지고기 | 익힌 돼지고기 | 음식 |
| 날것 양고기 | 익힌 양고기 | 음식 |

### 용광로 여러 개 쓰기

한 개만 쓰면 느려. **여러 개를 병렬로** 사용하면 빨라!

```
용광로 5개 설치 → 각각에 철 원석 나눠서 넣기
→ 동시에 제련됩니다!
```

## 장비 업그레이드 로드맵

### 단계 1: 나무 → 돌 (첫날)

```
나무 원목 채집 → 작업대 제작 → 나무 도구 제작
→ 돌 채굴 → 돌 도구 제작
```

### 단계 2: 돌 → 철 (2~3일차)

철 주괴를 충분히 모아서 업그레이드하세요.

**철 장비 재료 수:**
```
철 곡괭이: 철 주괴 3개 + 막대기 2개
철 도끼: 철 주괴 3개 + 막대기 2개
철 검: 철 주괴 2개 + 막대기 1개
철 삽: 철 주괴 1개 + 막대기 2개

철 투구: 철 주괴 5개
철 흉갑: 철 주괴 8개
철 레깅스: 철 주괴 7개
철 부츠: 철 주괴 4개
── 갑옷 풀세트 합계: 24개
```

> 💡 철 곡괭이가 있으면 금, 레드스톤, 다이아몬드도 채굴할 수 있습니다!

### 단계 3: 철 → 다이아몬드

다이아몬드는 Y=-57 근처에서 나옵니다.

**다이아몬드 장비 재료 수:**
```
다이아 곡괭이: 다이아 3개
다이아 검: 다이아 2개

다이아 투구: 다이아 5개
다이아 흉갑: 다이아 8개
다이아 레깅스: 다이아 7개
다이아 부츠: 다이아 4개
── 갑옷 풀세트 합계: 24개
```

> ⚠️ **다이아몬드는 비싸서 함부로 사용하지 마세요!**  
> 인챈트 테이블도 다이아 2개 필요하고, 모드에서도 많이 써.

## 음식 확보 & 농사

### 동물 사냥으로 음식 얻기

| 동물 | 드롭 음식 | 익혔을 때 허기 회복 |
|------|-----------|-------------------|
| 소 (Cow) | 날것 소고기 | ❤️❤️❤️❤️ (높음) |
| 돼지 (Pig) | 날것 돼지고기 | ❤️❤️❤️❤️ |
| 양 (Sheep) | 날것 양고기 | ❤️❤️❤️ |
| 닭 (Chicken) | 날것 닭고기 | ❤️❤️ (낮음) |

> 💡 동물을 죽이지 않고 **번식**시키면 무한 공급 가능!  
> 소에게 밀, 돼지에게 당근, 양에게 밀, 닭에게 씨앗을 주면 번식합니다.

### 기본 농사

음식을 직접 길러서 안정적으로 공급할 수 있습니다.

**밀 농사 시작하기:**
```
1. 마을에서 씨앗 훔치거나 풀 부수면 씨앗 드롭
2. 물 근처 흙에 괭이 우클릭 → 경작지(갈색 흙)로 변환
3. 씨앗을 경작지에 우클릭으로 심기
4. 자라는데 약 15~20분 (밤에도 자람)
5. 다 자라면 좌클릭으로 수확 (씨앗 + 밀 드롭)
```

**밀로 빵 만들기:**
```
[밀][밀][밀]  →  빵 (허기 2.5칸 회복)
```

**다른 작물들:**
| 작물 | 씨앗/재료 | 획득처 |
|------|-----------|--------|
| 감자 | 감자 자체 심기 | 마을 농장, 좀비 드롭 |
| 당근 | 당근 자체 심기 | 마을 농장, 좀비 드롭 |
| 수박 | 수박 씨앗 | 마을 상자, 던전 상자 |
| 호박 | 호박 씨앗 | 마을 상자 |

## 기본 건축 — 집 짓기

임시 은신처에서 제대로 된 집으로 업그레이드해봅시다.

### 최소한의 집 구성

| 방 | 필요한 이유 |
|----|------------|
| 침대 방 | 스폰 포인트 설정, 밤 스킵 |
| 작업대 + 용광로 방 | 제작, 제련 |
| 창고 방 | 아이템 보관 (상자 최소 10개) |
| 채굴 입구 | 지하로 안전하게 내려가는 통로 |

### 집 조명 중요성

집 안팎이 어두우면 몬스터가 스폰됩니다!

**몬스터 스폰 조건:** 빛 레벨 0인 곳 (완전 어둠)

**조명 설치 기준:**
- 횃불은 4칸마다 하나씩 설치하면 충분
- 바닥, 벽, 천장 어디에 설치해도 됨
- 집 주변 외부도 밝혀야 집 근처 몬스터 없어짐

## 🎮 이제 이 모드 써보세요! — Silent Gear & Silent's Gems

> 철 갑옷을 갖추고 지하 탐험이 안정적으로 됐으면, 이제 훨씬 강한 장비를 만들 시간입니다!

### Silent Gear란?

바닐라 장비 시스템을 **완전히 대체**하는 모드입니다.  
도구와 무기를 **파트(부품)**로 나눠서 재료를 선택해서 조합할 수 있습니다.

```
예시: 곡괭이 만들기
날(Head): 다이아몬드 → 채굴 속도 높음
막대(Rod): 끝끝나무 → 내구도 증가
장식(Tip): 에메랄드 → 행운 증가
→ 결과: 세 재료 특성이 합쳐진 강력한 곡괭이!
```

**바닐라 장비 vs Silent Gear:**

| 항목 | 바닐라 | Silent Gear |
|------|--------|-------------|
| 재료 선택 | 정해진 재료만 | 수십 가지 재료 조합 |
| 특성 | 없음 | 재료마다 다른 특성 |
| 수리 | 같은 재료로만 | 수리 키트로 언제든지 |
| 업그레이드 | 불가능 | 파트 교체로 업그레이드 |

**언제 시작할까?**
- 다이아몬드를 10개 이상 모았을 때
- 지하 탐험이 안정적으로 가능할 때

## 내구도 & 아이템 수리

### 내구도가 무엇인가요?

모든 도구, 무기, 갑옷에는 **내구도**가 있습니다. 사용할수록 닳고, 0이 되면 **아이템이 부서져서 사라집니다!**

```
아이템 아이콘 아래 색깔을 바로 확인하세요:

[🗡️] ████████  ← 초록 = 아직 여유 있음
[🗡️] ████░░░░  ← 노랑 = 절반 남음, 슬슬 주의
[🗡️] ██░░░░░░  ← 빨강 = 얼마 안 남음! 수리해야 합니다
[🗡️]           ← 바가 아예 없어지면 곧 부서짐 (위험!)
```

> ⚠️ **빨간 바가 뜨면 즉시 수리하거나 여분 아이템을 꺼내세요!**  
> 경고 없이 갑자기 부서지는 경우도 있습니다.

**재료별 도구 내구도:**

| 재료 | 내구도 |
|------|--------|
| 나무 | 59 |
| 돌 | 131 |
| 철 | 250 |
| 금 | 32 (매우 낮음!) |
| 다이아몬드 | 1561 |
| 네더라이트 | 2031 |

> 💡 금 도구는 채굴 속도는 빠른데 내구도가 32밖에 안 됩니다. 금방 부서지니까 조심하세요!

---

### 아이템 수리 방법 4가지

#### 방법 1: 같은 아이템 2개 합치기 (가장 기본)

인벤토리 제작칸에서 **같은 아이템 2개를 합치면** 내구도가 합산됩니다.

```
[닳은 철 검] + [닳은 철 검] → [철 검 (내구도 합산 + 5% 보너스)]
```

**단점:** 인챈트가 사라질 수 있습니다. 인챈트가 없는 아이템에만 사용하세요.

---

#### 방법 2: 모루(Anvil)로 수리 ✅ 추천

**모루 만들기:**
```
[철 블록][    ][철 블록]
[       ][철 블록][       ]
[철 주괴][철 주괴][철 주괴]
→ 모루 (철 블록 3개 + 철 주괴 4개)
```

**사용법:**
```
모루 우클릭 → 열기
왼쪽 칸: 수리할 아이템
오른쪽 칸: 해당 아이템의 원재료
→ 경험치 소모 후 수리됩니다 (인챈트 유지!)

예시:
다이아 검 → 다이아몬드
철 갑옷   → 철 주괴
나무 곡괭이 → 나무 판자
```

> ⚠️ **모루는 사용 횟수 제한이 있습니다!**  
> 같은 아이템에 너무 많이 쓰면 "수리 비용이 너무 높음" 이 떠서 사용할 수 없습니다.  
> → 이럴 때는 아래 그라인딩스톤으로 초기화하거나 새로 제작하세요.

---

#### 방법 3: 수리(Mending) 인챈트 ✅ 최고 추천

**수리 인챈트**가 달려 있으면 경험치를 얻을 때마다 **자동으로 내구도 회복!**

```
몬스터 처치, 채굴 → 경험치 획득 → 수리 인챈트 장비 자동 회복
→ 따로 수리 안 해도 항상 새것 상태 유지!
```

**수리 인챈트 얻는 법:**
- 마을 **사서(Librarian) 주민** 교역에서 인챈트 책 구매 (가장 안정적)
- 인챈트 테이블에서 운 좋으면 나옴
- 던전 상자에서 드롭

**수리 달아야 할 우선순위:**
```
1순위: 다이아 갑옷 풀세트
2순위: 주무기 (검/도끼)
3순위: 곡괭이
→ 이것들만 수리 달려 있으면 장비 걱정 거의 없습니다!
```

---

#### 방법 4: 그라인딩스톤 (Grindstone)

인챈트를 **제거**하면서 내구도를 일부 회복하는 기구입니다.

```
만들기:
[막대기][돌 슬래브][막대기]
[판자  ][         ][판자  ]
→ 그라인딩스톤

사용법: 우클릭 → 아이템 넣기
→ 인챈트 제거 + 경험치 환급 + 내구도 약간 회복
```
