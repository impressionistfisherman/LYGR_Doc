# 5단계 — 탐험 & 구조물

> 세계를 탐험하고 구조물에서 좋은 아이템을 얻는 방법을 배워봅시다.
> 마인크래프트 세계는 **사실상 무한대**입니다. 갈 곳이 무궁무진합니다!

---

## 탐험 전 준비

### 탐험 기본 장비

| 아이템 | 수량 | 이유 |
|--------|------|------|
| 음식 | 64개 | 허기 보충 |
| 횃불 | 64개 | 길 표시, 조명 |
| 무기 (검/도끼) | 1개 | 몬스터 처치 |
| 갑옷 | 풀세트 | 피해 감소 |
| 곡괭이 | 2개 | 막힌 길 뚫기 |
| 여분 블록 | 64개 | 다리 만들기, 막기 |
| 물 양동이 | 1개 | 용암 처리, 낙하 방지 |

### 길 잃지 않는 법

마인크래프트 세계는 커서 쉽게 길을 잃기 쉽습니다.

**방법 1: F3으로 좌표 기억**
```
F3 누르기 → XYZ 좌표 확인
집 좌표를 메모장에 적어두기!
예: 집 위치 X=142, Y=68, Z=-89
→ 나침반 방향 보면서 좌표 찾아가기
```

**방법 2: 횃불로 길 표시**
```
갈 때: 왼쪽 벽에 횃불 설치
올 때: 횃불이 오른쪽에 있도록 따라오기
→ 항상 일관된 방향으로 설치하면 길을 잃을 확률이 줄어듭니다
```

**방법 3: 지도 만들기**
```
종이 8개 + 나침반 1개 → 빈 지도
지도 들고 우클릭 → 현재 위치 중심으로 지도 생성
돌아다니면 자동으로 채워짐
```

**방법 4: 집에 신호탑 만들기**
```
집 지붕에 유리 + 횃불로 높은 탑 만들기
멀리서도 보여서 방향 파악 가능
→ 비콘 모드의 빛기둥 기능을 활용할 수 있습니다
```

## 바닐라 구조물 완전 가이드

### 🏘️ 마을 (Village)

초반에 가장 중요한 구조물 중 하나입니다.

**찾는 방법:**
- 평원, 초원, 타이가, 사막, 설원 바이옴에 생성
- 지형이 트인 곳에서 멀리 보면 집들이 보임

**마을에서 얻을 수 있는 것들:**

| 장소 | 얻을 수 있는 것 |
|------|----------------|
| 농가 상자 | 씨앗, 빵, 음식 |
| 무기상 상자 | 무기, 갑옷 |
| 도서관 상자 | 책, 인챈트 책 |
| 대장장이 상자 | 철 장비, 재료 |
| 교회 상자 | 부패한 살덩이, 레드스톤, 금 |
| 농장 | 밀, 감자, 당근, 비트뿌리 |

**주민 교역:**
마을 주민(NPC)들과 **에메랄드**로 아이템을 교역할 수 있습니다.

```
주민 우클릭 → 교역 UI 열림
에메랄드 주기 → 좋은 아이템 받기
교역할수록 주민 레벨업 → 더 좋은 아이템 나옴
```

주민 직업별 교역:

| 직업 | 교역 아이템 |
|------|------------|
| 무기상 (검 표시) | 검, 도끼, 인챈트 장비 |
| 방어구상 (갑옷 표시) | 갑옷, 말 갑옷 |
| 도구상 (도끼 표시) | 곡괭이, 삽, 괭이 |
| 사서 (책 표시) | 인챈트 책 (매우 유용!) |
| 농부 (밀 표시) | 음식, 씨앗 |
| 사제 (성배 표시) | 레드스톤, 글로우스톤 |

> ⚠️ **주민을 죽이거나 마을을 불태우지 마세요!**
> 주민이 줄어들면 교역이 제한되고, 약탈자 공격을 받을 수 있습니다.

**약탈자 습격 (Raid) 주의:**
마을 근처에서 약탈자를 처치하면 **불길한 징조** 효과를 얻을 수 있습니다.
그 상태로 마을에 들어가면 **습격(Raid)**이 시작됩니다.
습격을 막으면 영웅의 마을 효과(교역 할인)를 얻습니다.

### 🪨 던전 (Dungeon)

지하에 있는 작은 방. **스포너와 체스트** 2개가 있습니다.

**스포너란?**
몬스터를 계속 소환하는 블록입니다.
- 좀비 스포너, 스켈레톤 스포너, 거미 스포너가 있습니다
- 주변 8칸 이내가 밝아지면 작동하지 않습니다

**던전 공략:**
```
1. 스포너 발견
2. 빠르게 주변에 횃불 6\~8개 설치 (밝게!)
3. 몬스터 처치
4. 안전해지면 상자 열기
```

**던전 상자 주요 드롭:**
- 인챈트 책 (황금 사과, 새들, 이름표)
- 음식
- 무기, 갑옷

> **참고:** 스포너는 되도록 부수지 마세요.
> 스포너는 나중에 몬스터 팜(자동 경험치 농장)으로 활용할 수 있습니다.

### ⚒️ 폐광 (Abandoned Mineshaft)

지하에 자연 생성된 광산입니다.

**특징:**
- 레일, 광산 카트, 상자 수레가 있습니다
- 거미 스포너가 자주 있음
- 길이 복잡하게 얽혀 있어서 **길 잃기 쉬움**

**탐험 팁:**
```
횃불을 한 방향 벽에만 설치
→ 갈 때는 오른쪽 벽에, 올 때는 왼쪽에 오게 따라가기
교차로에서 방향 표시 남기기
```

**폐광 상자 드롭:**
- 레일, 광산 카트
- 금 사과, 음식
- 다이아몬드, 레드스톤, 금

### 🏰 사막 사원 (Desert Temple)

사막 바이옴에 있는 피라미드 모양 건물.

**주의!** 바닥에 **숨겨진 폭발 함정**이 있습니다.

```
1. 사원 중앙에 내려가면 색 블록 패턴 바닥이 있습니다
2. 그 아래에 상자 4개와 TNT 9개가 있음
3. 색 블록 밟으면 TNT 폭발!

안전하게 여는 법:
→ 주변 벽에서 아래로 내려가기
→ 색 블록 건드리지 않고 주변 흙 제거
→ 상자 열기
```

### 🌊 해저 신전 (Ocean Monument)

바다 깊은 곳에 있는 거대한 건물.

**위험 요소:**
- 가디언, 엘더 가디언이 지키고 있음
- 마이닝 피로 효과 → 채굴 속도 매우 느려짐
- 물속이라 호흡 관리 필요

**필요 준비물:**
- 수중 호흡 물약
- 야시경 물약 (어두움)
- 내구도 좋은 장비

**보상:** 해면 블록, 골드 블록 8개

### 🏯 삼림 대저택 (Woodland Mansion)

숲 바이옴에 있는 거대한 저택.

**위험도: 매우 높음!** 철 갑옷으로는 힘듭니다. 최소 다이아 갑옷 권장.

**내부 특징:**
- 3층 구조, 방이 엄청나게 많음
- 주민우두머리(Evoker), 약탈자 지도자(Vindicator)가 있음
- 토템 드롭 (불사의 토템 — 한 번 죽는 걸 막아줌!)

**불사의 토템:**
```
Evoker(주민우두머리) 처치 → 불사의 토템 드롭
오프핸드(F키)에 장착 → 죽을 때 자동 발동
→ 체력 1로 부활 + 재생, 흡수 효과
```

## 네더 탐험

철 갑옷 이상을 갖추면 도전할 수 있는 차원입니다.

### 네더 포탈 만들기

```
흑요석 10개 필요 (테두리만)
   ██
  █  █
  █  █
   ██

중앙 빈 공간에 부싯돌과 부시로 불 붙이기
→ 보라색 공간이 생기면 들어가기!
```

**흑요석 얻는 법:**
```
방법 1: 용암 위에 물 붓기 → 흑요석으로 굳음
방법 2: 자연 생성된 흑요석 채굴 (다이아 곡괭이 필요)
```

### 네더 바이옴들

| 바이옴 | 특징 | 주요 자원 |
|--------|------|-----------|
| 네더 황무지 | 기본 네더 | 네더 석영, 황금 |
| 크림슨 숲 | 빨간 버섯 숲 | 크림슨 목재 |
| 뒤틀린 숲 | 파란 버섯 숲 | 뒤틀린 목재, 엔더맨 |
| 영혼 모래 계곡 | 하얀 연기 | 영혼 모래, 영혼 흙 |
| 황갈색 암석 고원 | 산 지형 | 황갈색 암석 |

### 네더에서 얻어야 할 것들

| 아이템 | 획득 방법 | 용도 |
|--------|-----------|------|
| 블레이즈 막대 | 블레이즈 처치 | 양조기, 엔더 아이 |
| 블레이즈 파우더 | 블레이즈 막대 가공 | 물약 재료, 연료 |
| 네더 사마귀 | 네더 요새 농장 | 기본 물약 재료 |
| 엔더 진주 | 엔더맨 처치 | 엔더 아이 재료 |
| 황금 검 | 피글린 교역 | 네더라이트 파편 교역 |

**엔더 아이 만들기:**
```
블레이즈 파우더 + 엔더 진주 → 엔더 아이
→ 던지면 엔드 요새 방향을 알려줍니다.
```

### 네더 위험 요소

| 위험 | 설명 | 대처법 |
|------|------|--------|
| 용암 | 바닥과 천장에 용암 많음 | 물 양동이를 준비 |
| 가스트 | 원거리 불덩이 공격 | 방패로 막거나 도망 |
| 피글린 | 금 아이템 있으면 공격 안 함 | 황금 갑옷 한 조각이라도 착용 |
| 좀비 피글린 | 건드리면 주변 전부 공격 | 건드리지 않기 |
| 마그마 큐브 | 용암 위에서 점프 | 가장자리 조심 |

## 이제 이 모드도 확인해보세요 — 탐험 모드

### 🏰 When Dungeons Arise & Taverns

바닐라보다 훨씬 거대하고 화려한 구조물들이 세계 곳곳에 생성됩니다.

**세계에서 발견할 수 있는 대표 구조물들:**

| 구조물 | 규모 | 권장 장비 |
|--------|------|-----------|
| 작은 탑 | 소형 | 철 갑옷 |
| 마법사 탑 | 중형 | 철\~다이아 |
| 버려진 성 | 대형 | 다이아 갑옷 |
| 나는 섬 | 대형 | 다이아 갑옷 |
| 해적선 | 대형 | 다이아 갑옷 |
| 거대 탑 | 초대형 | 최상급 장비 |

**구조물 찾는 법:**
```
/locate structure whendungeonsarise:[구조물 이름]
예: /locate structure whendungeonsarise:small_dungeon
→ 가장 가까운 좌표를 알려줌
```

**여관(Tavern) 시스템:**
마을 같은 NPC 여관이 세계에 생성됩니다.
- 여관주인한테서 방을 구매할 수 있습니다
- 각종 퀘스트 NPC가 있음
- 상인에게서 특별 아이템을 구매할 수 있습니다

**자세한 내용:** [When Dungeons Arise 가이드 →](/docs/wiki-reference/dungeons-arise)

### 🗺️ IDAS (Integrated Dungeons and Structures)

스토리가 담긴 **대형 던전**들이 추가됩니다. 바닐라 던전보다 훨씬 크고 복잡합니다.

**IDAS 구조물 특징:**
- 각 구조물마다 테마가 있습니다 (지하묘지, 마법 연구소, 자연 신전 등)
- 퍼즐과 함정이 있습니다
- 보스 몬스터가 지키고 있음
- 특별한 보상 아이템

**단계별 구조물 추천:**

| 단계 | 장비 기준 | 추천 구조물 |
|------|-----------|------------|
| 초반 | 철 갑옷 | 작은 지하 동굴, 버려진 탑 |
| 중반 | 다이아 갑옷 | 지하묘지, 마법 탑 |
| 후반 | 강화 장비 | 대형 요새, 엔드게임 던전 |

**자세한 내용:** [IDAS 가이드 →](/docs/idas/overview)

## 탐험 중 아이템 관리

### 인벤토리 꽉 찼을 때

탐험하다 보면 인벤토리가 꽉 차는 경우가 많습니다.

**우선순위 결정:**
```
버려도 되는 것:
→ 흙, 자갈, 모래 (집에 충분히 있을 때)
→ 나무 & 돌 도구 (이미 철/다이아 도구 있을 때)
→ 흔한 음식 (더 좋은 음식 있을 때)

절대 버리면 안 되는 것:
→ 다이아몬드, 에메랄드
→ 인챈트된 아이템
→ 희귀 드롭 아이템
```

### 탐험 배낭 모드

이 모드팩에는 **Traveler's Backpack** 모드가 있습니다.
배낭을 만들어서 인벤토리를 크게 늘릴 수 있습니다.

```
가죽 7개 + 구리 주괴 2개 → 기본 배낭
배낭 착용 → `Controls`에서 `backpack` 검색 후 열기 → 추가 27칸!
```

## 엔더 드래곤 도전 준비

바닐라 최종 보스인 **엔더 드래곤**에 도전하려면 준비가 필요합니다.

### 엔드 포탈 찾기

```
1. 엔더 아이 던지기 (블레이즈 파우더 + 엔더 진주)
2. 엔더 아이가 날아가는 방향으로 이동
3. 엔더 아이가 땅으로 떨어지는 곳 = 요새 위치
4. 요새를 파고 들어가서 엔드 포탈 방 찾기
5. 포탈 틀에 엔더 아이 채우기 (12개 필요)
```

### 엔더 드래곤 도전 권장 장비

| 장비 | 최소 | 권장 |
|------|------|------|
| 갑옷 | 다이아 풀세트 | 다이아 + 좋은 인챈트 |
| 무기 | 다이아 검 | 날카로움 V 다이아 검 |
| 활 | 다이아 활 | 화력 V + 무한 |
| 음식 | 황금 당근 | 황금 당근 64개 |
| 기타 | 물약 | 재생, 힘, 속도 물약 |
