# 8단계 — 기술 발전 & 자동화

> 손으로 하는 모든 반복 작업을 기계로 자동화하자.  
> Create → Mekanism → AE2 순서로 배우면 자연스럽게 연결돼!

---

## 자동화가 필요한 이유

중반 이후가 되면 자원 소모가 엄청나게 늘어나.

```
예시: Silent Gear 최상급 장비 하나 만들기
→ 다이아몬드 30개 + 특수 광석 20개 + 가공 재료 50개 필요
→ 손으로 일일이 채굴하면 몇 시간

자동화 후:
→ 광산이 알아서 채굴
→ 용광로 100개가 동시에 제련
→ 아이템이 알아서 창고로 이동
→ 내가 할 일: 버튼 누르기
```

---

## 바닐라 자동화 요소

모드 없이도 기본적인 자동화가 가능해. 이걸 먼저 익혀두면 모드 자동화를 이해하기 쉬워.

### 호퍼 (Hopper)

아이템을 자동으로 이동시켜주는 기본 장치야.

```
만들기: 철 주괴 5개 + 상자 1개 → 호퍼

[철][  ][철]
[철][상자][철]
[  ][철][  ]
```

**사용법:**
```
호퍼를 상자 위에 설치 → 호퍼로 들어온 아이템이 상자로 이동
용광로 위에 호퍼 설치 → 아이템 자동 투입
용광로 옆에 호퍼 설치 → 결과물 자동 수거
```

**자동 제련 라인:**
```
[상자 (원료)] → [호퍼] → [용광로] → [호퍼] → [상자 (완성품)]
연료도 같은 방식으로 옆면에 연결!
```

### 발사기 (Dispenser) & 관찰자 (Observer)

```
발사기: 내부 아이템을 앞으로 발사
관찰자: 앞 블록 변화 감지 → 신호 출력
```

**자동 수확 농장:**
```
관찰자가 작물 성장 감지
→ 발사기에 물 양동이 발사
→ 작물 다 쓸려내려가면서 수확
→ 호퍼로 자동 수거
```

## 🎮 이제 이 모드 써봐! — 기술 모드

### ⚙️ 1단계: Create 모드

**가장 먼저 배워야 할 기술 모드.** 직관적이고 시각적으로 작동하는 게 보여서 재밌어!

**핵심 개념:**

| 개념 | 설명 |
|------|------|
| SU (Stress Units) | 기계가 소비하는 힘의 단위 |
| RPM | 분당 회전수. 높을수록 기계가 빠름 |
| 회전력 | 기어, 축, 벨트로 전달되는 동력 |
| 발전기 | 회전력을 생성하는 장치 |
| 소비 장치 | 회전력을 소비해서 작업하는 기계 |

**동력 발생 → 전달 → 소비 구조:**
```
[풍차/워터휠] → [회전축] → [기어박스] → [기계들]
  (발전)         (전달)      (방향변환)   (소비)
```

**초반 시작 순서:**

**Step 1: 워터휠 (Water Wheel) 설치**
```
재료: 나무 판자 8개 + 막대기 1개 → 워터휠
설치 조건: 흐르는 물 옆에 설치
→ 물의 흐름으로 회전력 생성 (16 SU)
```

물 흐름 만드는 법:
```
물 소스 블록 위에서 아래로 흘리기
→ 워터휠 주변에 흐르는 물 만들기
→ 많을수록 회전력 증가
```

**Step 2: 회전축 (Shaft) 연결**
```
워터휠 → 회전축 → 기계
축을 이어붙이면 회전력이 전달됨
```

**Step 3: 기어박스 (Gearbox)로 방향 변환**
```
회전 방향을 꺾을 때 사용
L자, T자 방향 변환 가능
```

**Step 4: 첫 번째 기계 — 압착기 (Press)**
```
재료: 철 주괴 5개 + 기타 재료 → 압착기
용도: 판, 기어, 링 제작
→ 이 모드의 핵심 재료들 생산
```

**Step 5: 풀무 (Blaze Burner) — 열 발생**
```
재료: 블레이즈 번창 블록 + 철 조각들
→ 믹서(Mixer)와 배합 가마(Basin)에 열 공급
```

**자주 쓰는 Create 기계들:**

| 기계 | 용도 | 필요 SU |
|------|------|---------|
| 압착기 (Press) | 판, 기어 제작 | 8 |
| 믹서 (Mixer) | 합금, 혼합물 제작 | 8 |
| 분쇄기 (Millstone) | 가루 생산 | 4 |
| 절단기 (Saw) | 나무 자르기, 판자→막대기 | 4 |
| 팬 (Fan) | 바람 생성, 요리, 스모크 처리 | 8 |
| 베어링 (Bearing) | 회전 구조물 만들기 | - |
| 컨베이어 (Belt) | 아이템 자동 이동 | 4/칸 |
| 박스 (Vault/Chest) | 대용량 창고 | - |

**Create 자동 제련 라인 예시:**
```
[상자] → [깔때기(Funnel)] → [컨베이어] → [압착기] → [컨베이어] → [상자]
원료 투입             이동               가공              완성품
```

**자세한 내용:** [Create 가이드 →](../create-mod/)

### ⚡ 2단계: Mekanism 모드

Create보다 더 강력한 **산업용 자동화 모드**야.  
광석을 최대 **5배**로 처리할 수 있어!

**핵심 재료: 오스뮴 (Osmium)**
```
오스뮴 원석: Y=0~48에서 채굴
→ 모든 Mekanism 기계의 기본 재료
→ 충분히 모아두기! (최소 64개)
```

**에너지 시스템:**

Mekanism 기계는 **에너지(Joule)**를 소비해.  
먼저 발전기를 만들어야 기계가 작동해!

**초반 발전기들:**

| 발전기 | 연료 | 발전량 |
|--------|------|--------|
| 열 발전기 (Heat Gen.) | 용암 | 40 J/t (쉬운 설치) |
| 기체 발전기 (Gas Gen.) | 수소 등 기체 | 변동 |
| 바이오 발전기 (Bio Gen.) | 유기물 | 소량 |
| 풍력 발전기 | 없음 (높이 영향) | 변동 |
| 태양 발전기 | 없음 (낮에만) | 소량 |

**에너지 저장: 에너지 큐브**
```
발전기 → 에너지 큐브 (저장) → 기계들
→ 발전량이 소비량보다 많으면 저장됨
→ 밤이나 연료 부족해도 동작 가능
```

**광석 처리 단계 (중요!):**

| 단계 | 배율 | 필요 기계 | 시작 시기 |
|------|------|-----------|---------|
| 기본 제련 | x1 | 용광로 | 처음부터 |
| Crusher | x2 | Crusher | Mekanism 시작 |
| Enrichment Chamber | x3 | Crusher + Enrichment | 중반 |
| Purification Chamber | x4 | 추가 기계 체인 | 중후반 |
| Chemical Injection | x5 | 전체 체인 완성 | 후반 |

**x2 처리 설치법 (처음 목표):**
```
1. Crusher 설치 + 에너지 연결
2. 광석 → Crusher → 분말 2개 획득
3. 분말 → 용광로 → 주괴 2개!
```

**x5까지 가면:**
```
다이아몬드 원석 1개 → 다이아몬드 5개!
에메랄드 원석 1개 → 에메랄드 5개!
→ 자원 5배 증폭
```

**Mekanism 고급 장비:**

| 장비 | 특징 |
|------|------|
| 나노 갑옷 | 에너지 충전식, 피해 흡수 |
| 퀀텀 갑옷 | 최상급 방어구, 거의 무적 |
| 디지털 채굴기 | 3x3 or 5x5 범위 채굴 |
| 전기 활 | 에너지 충전식 강력한 활 |
| 핵융합 반응로 | 엄청난 양의 에너지 생산 |

**자세한 내용:** [Mekanism 가이드 →](../mekanism-mod/)

### 📦 3단계: Applied Energistics 2 (AE2)

모든 아이템을 **디지털 네트워크 하나**로 관리하는 최종 자동화 모드야.

**핵심 개념:**

```
ME 네트워크 = 디지털 창고 시스템
→ 수천 종류 아이템을 하나의 인터페이스로 관리
→ 아이템 꺼내기: 검색창에 이름 입력 → 클릭
→ 자동 제작: 패턴 등록 → 버튼 하나로 제작
```

**AE2 시작 재료: 쿼츠**
```
Certus Quartz: AE2 전용 광석 (Y=0~80에서 채굴)
규석(Quartz): 일반 규석 (사막 모래 아래)
→ 두 가지 모두 필요
```

**기본 구성 요소:**

| 부품 | 역할 |
|------|------|
| ME 컨트롤러 | 네트워크의 중심 |
| ME 드라이브 | 아이템 저장 (스토리지 셀 필요) |
| ME 터미널 | 아이템 검색 & 꺼내기 UI |
| ME 케이블 | 기계들 연결 |
| 인터페이스 | 다른 기계와 연결 |
| 패턴 프로바이더 | 자동 제작 명령 전달 |

**ME 네트워크 구축 순서:**
```
Step 1: ME 컨트롤러 설치 (전력 연결 필요)
Step 2: ME 드라이브 연결 + 스토리지 셀 삽입
Step 3: ME 케이블로 기계들 연결
Step 4: ME 터미널 설치
Step 5: 기존 상자 아이템 → ME 네트워크로 이전
```

**채널 시스템 (중요!):**

AE2는 **채널(Channel)** 시스템으로 기계 연결 개수를 제한해.

```
일반 케이블: 최대 8채널 (8개 기계만 연결 가능)
밀도 케이블: 최대 32채널
컨트롤러 인접 케이블: 최대 8채널/면
```

> ⚠️ 채널 초과하면 기계가 작동 안 해!  
> 네트워크가 커지면 케이블 설계가 중요해져.

**자동 제작 설정:**
```
1. 패턴 인코더에서 레시피 등록
2. 패턴 → 패턴 프로바이더에 넣기
3. ME 터미널에서 아이템 이름 검색
4. 만들기 버튼 클릭 → 자동으로 재료 수집 후 제작!
```

**자세한 내용:** [Applied Energistics 2 가이드 →](../applied-energistics-2/)

## 기술 모드 진행 순서 (전체 로드맵)

```
[바닐라] 호퍼 + 발사기로 기초 자동화 배우기
    ↓
[Create] 워터휠 → 압착기 → 간단한 공장 라인
    ↓
[Create] 대형 농장, 자동 채굴, 컨트랩션 열차
    ↓
[Mekanism] 오스뮴 채굴 → 열 발전기 → Crusher x2
    ↓
[Mekanism] 에너지 인프라 구축 → x3, x4 라인
    ↓
[Mekanism] 핵융합 발전 → x5 완성 라인
    ↓
[AE2] Certus Quartz 채굴 → ME 네트워크 첫 구축
    ↓
[AE2] 자동 제작 설정 → 모든 아이템 통합 관리
    ↓
[통합] Create + Mekanism + AE2 메가베이스 완성
```

## 초보자가 자주 하는 실수

| 실수 | 원인 | 해결책 |
|------|------|--------|
| 기계가 안 돌아가요 | 에너지/회전력 없음 | 발전기 먼저! |
| Create 기계가 안 움직여 | 회전력이 안 닿음 | 축/벨트로 연결 확인 |
| Mekanism 기계가 꺼짐 | 에너지 부족 | 발전량 > 소비량 확인 |
| AE2 기계가 작동 안 함 | 채널 초과 | 밀도 케이블 사용 |
| 아이템이 증발함 | 컨베이어에서 떨어짐 | 테두리 블록으로 막기 |
| 제련이 안 됨 | 연료 없음 | 연료 슬롯 확인 |
