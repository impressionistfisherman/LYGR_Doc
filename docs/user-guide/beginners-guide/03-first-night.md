# 3단계 — 첫날밤 생존

> 세계에 스폰되는 순간부터 **딱 10분**이 있습니다.
> 그 안에 첫날밤을 버틸 준비를 마쳐야 합니다!
> 침착하게 이 순서대로 하면 죽을 가능성이 낮습니다.

---

## 시간 개념 이해하기

마인크래프트 하루는 현실 시간으로 **20분**입니다.

```
낮 (약 10분) → 일몰 (약 1.5분) → 밤 (약 7분) → 새벽 (약 1.5분)
```

- **낮**: 몬스터가 햇빛에 불타 야외가 비교적 안전합니다
- **일몰**: 몬스터가 스폰되기 시작하므로 이때부터 위험해집니다
- **밤**: 야외에 좀비, 스켈레톤, 거미, 크리퍼 등이 돌아다닙니다
- **새벽**: 다시 햇빛, 몬스터가 불타기 시작

> 게임을 시작하면 낮입니다. **10분 안에 은신처와 기본 도구**를 준비하세요.

## 첫날 완벽 타임라인

### ⏱️ 0\~1분: 스폰 직후 주변 파악

세계에 처음 스폰되면 잠깐 멈추고 주변을 둘러보세요.

**확인할 것들:**
- 근처에 나무가 있는가? (강력 권장)
- 근처에 동물(소, 돼지, 양, 닭)이 있는가?
- 근처에 마을이 있는가? (있으면 대박!)
- 바이옴이 어디인가? (사막, 설원은 생존 어려움)

**만약 스폰이 좋지 않은 경우:**
- 사막: 나무가 없어서 어렵습니다. 선인장이 보이면 멀리 이동해서 나무를 찾으세요
- 설원: 나무는 있지만 음식을 구하기 어렵습니다
- 바다 한가운데: 헤엄쳐서 육지로 이동

### ⏱️ 1\~3분: 나무 채집

가장 먼저 해야 할 일은 **나무 채집**입니다.

**목표: 원목 최소 16개, 여유 있으면 32개**

```
나무에 가까이 가기
→ 나무 원목 블록에 마우스 커서 올리기
→ 왼쪽 클릭 꾹 누르기
→ 블록 부서지면서 원목 드롭
→ 가까이 가서 자동으로 줍기
```

> **나무를 캘 때 팁:**
> - 나무 밑동부터 캐면 위 블록들도 같이 드롭됩니다 (아님, 각각 캐야 합니다)
> - 잎 블록은 시간이 지나면 자동으로 사라지면서 묘목이나 사과가 드롭됩니다
> - 나무 종류(참나무, 자작나무 등)는 달라도 다 쓸 수 있습니다

**인벤토리에서 판자로 변환:**
1. `E`로 인벤토리 열기
2. 원목을 2x2 제작 칸에 놓기 (어느 칸이든 1개만)
3. 결과 슬롯에 나무 판자 4개 클릭해서 가져오기
4. 원목 전부 판자로 변환해두기

### ⏱️ 3\~4분: 작업대 & 기본 도구 제작

**작업대 만들기:**
```
[판자][판자]
[판자][판자]  →  작업대 1개
```

**작업대 설치:**
작업대를 핫바에 놓고 우클릭으로 바닥에 설치.

**막대기 만들기 (도구 재료):**
```
[판자]
[판자]  →  막대기 4개
```

**나무 곡괭이 만들기:**
```
[판자][판자][판자]
      [막대]
      [막대]      →  나무 곡괭이
```

**나무 도끼 만들기 (나무 채집 속도 2배):**
```
[판자][판자]
[판자][막대]
      [막대]      →  나무 도끼
```

**나무 검 만들기 (방어용):**
```
[판자]
[판자]
[막대]            →  나무 검
```

### ⏱️ 4\~6분: 돌 확보 & 업그레이드

나무 도구는 너무 약합니다. 빨리 돌 도구로 업그레이드해야 합니다.

**조약돌 얻기:**
- 주변 돌 지형 (바위, 동굴 입구, 언덕)에서 나무 곡괭이로 채굴
- **조약돌 목표: 24개 이상**

> **참고:** 동굴 입구에서 캐면 빠르게 조약돌을 확보할 수 있습니다.
> 단, 동굴 안으로 깊이 들어가지 마세요. 몬스터가 있습니다.

**돌 도구 제작:**
```
나무 곡괭이 → 돌 곡괭이 (판자 대신 조약돌)
나무 검 → 돌 검
나무 도끼 → 돌 도끼
```

레시피는 나무 도구와 같고 재료만 조약돌로 바꾸면 됩니다.

### ⏱️ 6\~8분: 용광로 & 횃불 제작

**용광로 만들기:**
```
[조약돌][조약돌][조약돌]
[조약돌][    ][조약돌]
[조약돌][조약돌][조약돌]  →  용광로 (가운데 비우기)
```

**횃불 만들기:**
먼저 석탄을 찾거나 목탄을 만들어야 합니다.

방법 1 — **석탄 채굴**: 돌 속에 검은 점이 있는 블록이 석탄입니다. 돌 곡괭이로 캐기.

방법 2 — **목탄 만들기**: 석탄 없을 때 사용
```
용광로 설치 → 위 칸에 나무 원목, 아래 칸에 나무 판자(연료) → 결과물: 목탄
```

**횃불 제작:**
```
[석탄 or 목탄]
[막대기]         →  횃불 4개
```

### ⏱️ 8\~10분: 은신처 만들기

밤이 오기 전에 잘 곳이 필요합니다.

**방법 1: 언덕/벽면 파고 들어가기 (가장 빠름)**
```
1. 언덕이나 산 벽면 찾기
2. 3칸 넓이, 2칸 높이, 3칸 깊이로 파기
3. 입구에 블록 막기 (1칸만 열어두고)
4. 안에 횃불 설치
5. 나중에 문 만들어서 달기
```

**방법 2: 땅을 파고 들어가기**
```
1. 땅에 2x2 구멍 파기 (3칸 깊이)
2. 내려가서 가로로 공간 만들기
3. 올라올 때 블록으로 막기
```

**방법 3: 나무 위로 올라가기 (임시)**
```
좀비는 나무를 타지 못합니다
나무 위에서 밤을 보냅니다
아침에 내려와서 제대로 된 집을 짓습니다
```

**문 만들기 (선택):**
```
[판자][판자]
[판자][판자]
[판자][판자]  →  나무 문 3개
```

> **참고:** 어려움 난이도에서는 좀비가 닫힌 문을 부수려고 할 수 있습니다.
> 일반 난이도에서는 문을 부수지 않습니다.

## 첫날밤을 버티는 3가지 방법

### 방법 1: 은신처 안에서 대기 (추천!)

가장 안전한 방법입니다.

- 은신처 입구 막기
- 안에 횃불 설치 (어두우면 몬스터 스폰!)
- 밤 동안 **작업대, 용광로 작업** 하기
- 낮이 될 때까지 기다리기

**은신처 안에서 할 것들:**
- 더 많은 도구 제작
- 용광로로 철 제련 (동굴에서 얻은 게 있으면)
- 나무 판자 더 만들기
- 침대 재료 구하기

### 방법 2: 침대 구해서 밤 건너뛰기

양(🐑)을 찾으면 밤을 건너뛸 수 있습니다.

**침대 만들기:**
```
[양털][양털][양털]
[판자][판자][판자]  →  침대
```

- 양을 검으로 처치하면 양털 1개 드롭
- 양털 3개가 필요하므로 양 3마리를 처치하거나 가위로 양털을 얻습니다
- 침대 설치하고 우클릭 → 자기 → 밤 건너뛰기!

> ⚠️ 근처에 몬스터가 있으면 "지금 자기 위험하다" 메시지가 나옵니다.
> 그럼 몬스터 제거하거나 안전한 실내에서 자야 합니다.

### 방법 3: 어쩔 수 없이 밖에 있다면

은신처 못 만들고 밤이 됐을 때:

| 상황 | 대처법 |
|------|--------|
| 좀비 발견 | 거리 유지하며 검으로 공격 |
| 스켈레톤 발견 | 뒤로 피하면서 접근해서 검 사용 |
| 크리퍼 발견 | 즉시 도망치기 (아래 크리퍼 섹션 참고) |
| 어두운 곳 | 횃불 설치해서 밝히기 |
| 고지대 | 위로 올라가기 (좀비는 못 올라옴) |

## 위험한 몬스터들

### 좀비 (Zombie) 🧟
- **특징**: 느리게 움직이고 근거리 공격
- **낮에는**: 햇빛에 불탐
- **공략**: 뒤로 물러서면서 칼로 공격. 크리티컬(점프 후 공격)이 효과적

### 스켈레톤 (Skeleton) 🦴
- **특징**: 활로 원거리 공격. 멀수록 위험
- **공략**: 빠르게 접근해서 근거리 공격. 방패로 화살 막기
- **주의**: 언덕 뒤나 나무 뒤에서 쏘는 경우가 있습니다

### ⚠️ 크리퍼 (Creeper) 💣
- **특징**: 초록색 몸, 가까이 가면 **폭발**해서 큰 피해를 줍니다.
- **경고음**: "치지지지지" 하는 소리가 나면 2초 뒤 폭발
- **공략**:
  1. 소리가 나면 **즉시 뒤로 달리기**
  2. 폭발하기 전에 2\~3칸 이상 떨어지기
  3. 멀리서 활로 처치하는 게 가장 안전
  4. 검으로 잡을 때는 타격 후 바로 뒤로 달리기 반복

> **주의:** 크리퍼는 초반에 특히 위험한 몬스터입니다.
> 폭발하면 땅도 파이고 근처 아이템도 모두 날아갑니다.
> 집 근처에서는 특히 조심하세요.

### 거미 (Spider) 🕷️
- **특징**: 빠르게 달리고 벽 타기 가능
- **낮에는**: 공격적이지 않음 (중립)
- **공략**: 몸이 커서 때리기 쉬움. 여러 마리 동시에 오면 위험

### 엔더맨 (Enderman) 🟣
- **특징**: 키가 크고 검은 몸
- **주의사항**: **눈을 마주치면 공격해옴!** 마우스 커서가 몸에 닿으면 돌변
- **공략**: 발 아래쪽을 공격 (키가 커서 머리 치기 어려움)
- **엔더 진주** 드롭 → 나중에 엔더 포탈에 필요

## 첫날밤 이후 — 2일차 목표

첫날밤을 넘겼다면 다음 목표를 잡아야 합니다.

| 우선순위 | 목표 | 방법 |
|----------|------|------|
| ⭐ 매우 중요 | 철 갑옷 제작 | 지하에서 철 채굴 → 용광로 제련 → 갑옷 제작 |
| ⭐ 매우 중요 | 음식 확보 | 동물 사냥 → 용광로에 구워서 저장 |
| ⭐ 중요 | 침대 제작 | 양 3마리 사냥 → 양털 3개 + 판자 3개 |
| 권장 | 집 제대로 짓기 | 임시 은신처를 더 넓게 |
| 권장 | 광물 채굴 | 지하 탐험 시작 |

## 이제 이 모드도 확인해보세요 — 첫날부터 알아두면 좋은 것들

### 바닐라에서 첫날 돌아다니다 만나는 모드 콘텐츠

이 모드팩에는 세계 생성 단계부터 모드 콘텐츠가 가득합니다.
아직 상대하기는 이르지만 **눈에 익혀두면** 나중에 도움이 됩니다.

**발견할 수 있는 것들:**

| 발견 | 설명 | 지금 해야 할지? |
|------|------|----------------|
| 거대한 성이나 탑 | When Dungeons Arise 구조물 | 나중에 도전 권장 |
| 여관 건물 | Taverns 모드 | 들어가도 됩니다. NPC가 퀘스트를 줄 수 있습니다 |
| 특이한 나무나 식물 | 바이옴 모드 | 자원으로 사용해도 됩니다 |
| 빛나는 광석 | Silent Gems | 캐두면 나중에 장비 재료로 쓸 수 있습니다 |
| 이상한 색깔의 동물 | 동물 추가 모드 | 가끔 유용한 아이템을 드롭합니다 |

> **주의:** 거대한 구조물은 초반에는 피하는 것이 좋습니다.
> 첫날에 던전에 들어갔다가 강한 몬스터에게 죽으면 모든 아이템을 잃을 수 있습니다.

## 죽었을 때 어떻게 해야 하나요?

### 죽으면 일어나는 일

```
죽음 → 들고 있던 모든 아이템 + 경험치 일부가 바닥에 드롭됩니다
     → 침대 설정했으면 침대 위치에서 부활
     → 침대 없으면 세계 최초 스폰 위치에서 부활
     → 드롭된 아이템은 5분 후 사라집니다.
```

> **주의:** 죽으면 경험치도 일부 사라집니다. 레벨이 높을수록 손실이 큽니다.

### 죽은 직후 해야 할 것

**Step 1: 죽은 위치 파악**
```
죽기 직전 F3으로 좌표 기억해두는 습관 들이기
채팅창에 "○○ 이(가) ○○에게 죽었습니다" + 좌표 표시되는 경우도 있습니다
```

**Step 2: 여분 장비 준비하기**
```
집 상자에 여분 장비 세트를 보관해두세요
→ 죽어도 바로 다시 출발할 수 있습니다.
```

**Step 3: 빠르게 회수하러 가기**
```
5분 안에 회수해야 합니다.
낮이면 달려서 바로 줍기
밤이면 몬스터 피하면서 조심히
동굴 안이면 횃불을 챙겨서 불을 밝히면서
```

### 아이템 잃지 않는 습관

| 습관 | 효과 |
|------|------|
| 좋은 아이템은 집 상자에 보관 | 죽어도 집에 남아있음 |
| 탐험 전 여분 장비 세트 준비 | 죽어도 바로 재출발 |
| 침대로 스폰 포인트 설정 | 집 근처에서 부활 |
| 불사의 토템 오프핸드 착용 | 한 번 죽어도 자동 부활 |

### 불사의 토템 (Totem of Undying)

```
획득처: 삼림 대저택 → 주민우두머리(Evoker) 처치 → 드롭
사용법: F키로 오프핸드에 장착
효과:   체력 0이 될 때 자동 발동
        → 체력 1로 부활 + 재생 II + 흡수 II
```

> **권장:** 보스전에 갈 때는 불사의 토템을 오프핸드에 장착해두면 안전합니다.

## 상태이상 가이드

### 해로운 상태이상 (Debuff)

화면 오른쪽 상단에 아이콘으로 표시됩니다.

| 상태이상 | 효과 | 대처법 |
|----------|------|--------|
| **독 (Poison)** | 체력이 서서히 닳음 (최소 반 하트 남음) | 우유 마시기 or 기다리기 |
| **맹독 (Wither)** | 체력이 0까지 닳음! 매우 위험 | 즉시 우유 마시기 |
| **화염 (Fire)** | 불 붙어서 지속 피해 | 물에 들어가기 |
| **배고픔 (Hunger)** | 허기 빠르게 감소 | 음식 먹기 |
| **둔화 (Slowness)** | 이동 속도 감소 | 우유 or 기다리기 |
| **광산 피로** | 채굴 속도 대폭 감소 | 우유 마시기 |
| **약화 (Weakness)** | 공격력 감소 | 우유 or 기다리기 |
| **실명 (Blindness)** | 시야가 좁아짐 | 기다리기 |
| **불운 (Bad Omen)** | 마을 들어가면 습격 발생 | 우유 마시기 |

> **참고:** 우유는 대부분의 상태이상을 제거합니다.
> 소(Cow) 우클릭 + 양동이 → 우유 양동이
> 들고 우클릭 → 마시기 → 상태이상 제거
> 단, 좋은 효과(힘 물약 등)도 같이 사라지므로 주의하세요.

### 독 vs 맹독 차이

```
독(Poison):   체력이 반 하트까지만 깎임 → 혼자서는 죽지 않습니다
맹독(Wither): 체력이 0까지 깎임 → 방치하면 죽을 수 있습니다! 즉시 우유를 마시세요
```

맹독은 Wither 보스, 위더 스켈레톤, 일부 모드 몬스터가 부여합니다.

## 낙하 피해

```
3칸 이하 낙하:  피해 없음
4칸 낙하:       반 하트 피해
10칸 낙하:      3.5 하트 피해
23칸 이상 낙하: 즉사 (기본 체력 기준)
```

**낙하 피해 방지법:**

| 방법 | 설명 |
|------|------|
| 물에 떨어지기 | 물은 낙하 피해를 막아줍니다. |
| 물 양동이 테크닉 | 떨어지는 중 발 아래 물 붓기 → 피해 없음 |
| 건초 더미 위 착지 | 낙하 피해 80% 감소 |
| 비단 발걸음 인챈트 | 부츠에 달면 낙하 피해 대폭 감소 |
| 느린 낙하 물약 | 천천히 떨어짐 |

**물 양동이 테크닉:**
```
높은 곳에서 떨어지는 중
→ 땅에 닿기 직전에 물 양동이 우클릭 (발 아래 블록에)
→ 물이 생기면서 피해 없이 착지!
타이밍이 중요하므로 연습이 필요합니다
```

## 자주 겪는 초보 실수 & 해결책

| 실수 | 해결책 |
|------|--------|
| 도구가 갑자기 사라집니다 | 내구도 바 항상 확인, 수리 인챈트 달기 |
| 죽어서 아이템 다 잃음 | 집에 여분 세트 보관, 토템 오프핸드 |
| 집을 찾지 못함 | F3으로 집 좌표 메모해두기 |
| 용암에 아이템 빠트림 | 용암 근처에서 인벤토리 열지 않기 |
| 크리퍼한테 집이 폭발 | 집 주변 조명 충분히 설치하기 |
| 독으로 죽음 | 우유를 준비해두기 |
| 경험치 없어서 인챈트 못 함 | 몬스터 팜 만들기, 채굴 많이 하기 |
| 모루 수리 비용이 너무 높음 | 수리 인챈트로 예방, 그라인딩스톤 초기화 |
