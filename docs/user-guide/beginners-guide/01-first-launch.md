# 1단계 — 설치 & 첫 실행

> 모드팩을 처음 설치하고 게임을 실행하는 전 과정을 알려줄게.  
> 컴퓨터를 잘 모르는 사람도 따라할 수 있도록 최대한 자세히 썼어.

---

## 필요한 것들 먼저 확인하세요

### 컴퓨터 사양 확인

모드가 178개나 있어서 컴퓨터 사양이 어느 정도 필요합니다.

| 항목 | 최소 사양 | 권장 사양 |
|------|-----------|-----------|
| **운영체제** | Windows 10 64비트 | Windows 10/11 64비트 |
| **RAM (메모리)** | 12GB | 16GB 이상 |
| **그래픽카드** | GTX 1060 이상 | RTX 2060 이상 |
| **저장공간** | 10GB 이상 여유 | 20GB 이상 여유 |
| **인터넷** | 모드 다운로드용 필요 | - |

> ❗ **RAM이 8GB 이하면 게임이 자꾸 튕길 수 있습니다.**  
> 컴퓨터 RAM 확인하는 법: `Windows키 + Pause/Break` → 시스템 정보에서 확인

### 마인크래프트 정품 계정

마인크래프트는 유료 게임입니다. **Microsoft 계정**으로 구매해야 합니다.
- 구매 사이트: [minecraft.net](https://minecraft.net)
- Java Edition 구매 필요 (Bedrock Edition 아님!)

---

## Step 1 — Java 설치

마인크래프트 Java Edition은 Java로 만들어진 게임이라 Java가 필요합니다.

### Java 21 설치 방법

1. [https://adoptium.net](https://adoptium.net) 접속
2. **"Latest LTS Release"** 버튼 클릭 (Java 21)
3. 운영체제에 맞는 설치 파일 다운로드
   - Windows: `.msi` 파일
4. 다운로드된 파일 실행 → **Next → Next → Install** 쭉 눌러서 설치
5. 설치 완료 후 **컴퓨터 재시작**

> 💡 **이미 Java가 있습니다도** 버전이 낮으면 문제가 생길 수 있습니다. Java 21 새로 설치 추천.

### Java 설치 확인
```
1. Windows키 + R 누르기
2. "cmd" 입력 후 Enter
3. 검은 창에 java -version 입력 후 Enter
4. "version 21..." 같은 문구가 나오면 성공!
```

---

## Step 2 — CurseForge 앱 설치

CurseForge는 모드팩을 쉽게 설치하고 관리해주는 앱입니다.

1. [https://curseforge.com/download/app](https://curseforge.com/download/app) 접속
2. **"Download for Windows"** 클릭
3. 다운로드된 `.exe` 파일 실행
4. 설치 진행 (기본 설정으로 Next → Install)
5. 설치 완료 후 CurseForge 앱 실행

### CurseForge 첫 실행 설정

1. 앱 실행하면 게임 선택 화면이 나와
2. **Minecraft** 선택
3. 마인크래프트 Java 경로 자동 감지됩니다 (안 되면 수동으로 찾기)

---

## Step 3 — 모드팩 설치

### 방법 A: CurseForge에서 직접 검색

1. CurseForge 앱 → Minecraft → **Browse Modpacks**
2. 검색창에 모드팩 이름 입력
3. 모드팩 찾으면 **Install** 클릭
4. 버전 선택 → Install
5. 모드 178개 다운로드 시작 (인터넷 속도에 따라 10~30분)

### 방법 B: zip 파일 직접 임포트

플레이어한테서 모드팩 zip 파일을 받은 경우:

1. CurseForge 앱 → Minecraft
2. 오른쪽 위 **"Create Custom Profile"** 또는 **"Import"** 클릭
3. **"Import"** → zip 파일 선택
4. 자동으로 설치 시작

> 💡 설치 중 오류가 나도 당황하지 마세요. 대부분 인터넷 문제라 다시 시도하면 됩니다.

---

## Step 4 — RAM 할당량 늘리기 ⚠️ 매우 중요!

이게 가장 중요한 설정입니다. **이 설정 안 하면 게임이 자꾸 튕길 수 있습니다.**

모드가 178개나 있어서 기본 RAM 할당량으로는 부족합니다.

### RAM 설정 방법

1. CurseForge 앱에서 설치된 모드팩 찾기
2. 모드팩 이름 오른쪽 **점 세 개 (...)** 또는 **톱니바퀴** 클릭
3. **"Profile Options"** 또는 **"설정"** 클릭
4. **"Java Settings"** 탭 찾기
5. **"Allocated Memory"** 슬라이더를 조정

| 내 PC RAM | 권장 할당량 |
|-----------|-------------|
| 12GB | 6~7GB (6144~7168MB) |
| 16GB | 8~10GB (8192~10240MB) |
| 32GB | 12GB (12288MB) |

> ⚠️ **PC 전체 RAM의 절반 이상은 할당하지 마세요.** 운영체제도 RAM이 필요합니다.  
> 예: 16GB PC → 최대 10GB까지만

---

## Step 5 — 첫 실행

설정이 끝났으면 이제 게임을 켜볼 시간입니다.

### 게임 실행

1. CurseForge → 모드팩 → **Play** 버튼 클릭
2. 마인크래프트 런처가 열리면서 게임 시작

### 첫 실행 시 정상적인 현상들

처음 실행하면 이상해 보일 수 있는데 전부 정상입니다.

| 현상 | 이유 | 대처법 |
|------|------|--------|
| 로딩이 5~15분 걸림 | 178개 모드 로딩 중 | 기다리기 |
| 검은 화면에 글씨가 잔뜩 | 모드 초기화 로그 | 무시하기 |
| 메인 화면 배경이 달라짐 | 모드팩 커스텀 UI | 정상 |
| 메모리 사용량이 높음 | 모드팩 특성 | 정상 (설정대로면 OK) |

> ❗ **로딩 중에 강제 종료하지 마세요!** 파일이 손상될 수 있습니다.

### 게임이 실행 안 되면?

**오류 메시지가 나오는 경우:**
- `OutOfMemoryError` → RAM 할당량이 너무 낮음. Step 4 다시 하기
- `Java version` 관련 오류 → Java 21 설치 확인
- `모드 충돌` 관련 오류 → 모드팩 재설치 시도

---

## Step 6 — 새 세계 만들기

게임이 켜졌으면 새 세계를 만들어야 합니다.

### 세계 만드는 순서

1. 메인 화면에서 **"싱글플레이어"** 클릭
2. **"새 세계 만들기"** 클릭
3. 아래 표대로 설정

### 추천 세계 설정

| 설정 항목 | 추천값 | 설명 |
|-----------|--------|------|
| **세계 이름** | 원하는 이름 아무거나 | 나중에 못 바꿈 |
| **게임 모드** | 생존 | 정상 플레이 |
| **난이도** | 일반 (Normal) | 처음엔 이 정도가 적당합니다 |
| **세계 유형** | 기본값 | 일반 지형 |
| **치트 허용** | 켜기 | 막힐 때 도움이 됩니다 |
| **보너스 상자** | 켜기 (선택) | 초반 아이템 조금 줌 |

> 💡 **난이도 추천:**
> - 마인크래프트 처음이면: **쉬움 (Easy)**
> - 어느 정도 알면: **일반 (Normal)**
> - 도전하고 싶으면: **어려움 (Hard)**

4. **"세계 만들기"** 클릭
5. 세계 생성까지 1~2분 기다리기

---

## 자주 겪는 문제들

### 게임이 너무 느립니다 (프레임 낮음)
1. 게임 내 `Options → Video Settings`
2. **Render Distance** 를 **8~10** 으로 낮추기
3. **Graphics** 를 **Fast** 로 변경
4. **Smooth Lighting** 끄기

### 게임이 가끔 멈췄다가 다시 움직입니다 (렉)
- RAM 할당량 늘리기 (Step 4 다시 확인)
- 백그라운드 프로그램 종료 (크롬, Discord 등)

### 소리가 안 남
1. `Options → Music & Sounds`
2. **Master Volume** 확인 (0이면 올리기)
