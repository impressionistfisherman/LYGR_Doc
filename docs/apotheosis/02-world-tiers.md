# 월드 티어 (World Tier)

월드 티어는 Apotheosis의 **RPG 진행 시스템**입니다.
티어를 올릴수록 몹이 강해지지만 더 좋은 등급의 Affix 아이템이 드롭됩니다.
티어는 언제든 **Ctrl + T** 로 올리거나 내릴 수 있습니다.

> **주의: 가장 먼저 해야 할 것**
>
> 첫 Affix 아이템을 주웠다면 즉시 **Ctrl + T** 를 눌러 World Tier: Haven을 활성화하세요.
> 이것을 하지 않으면 아이템의 Affix 효과가 **잠긴 채로** 유지됩니다.

---

## 티어 상세

### 🏕️ Tier 1 — Haven (안식처)

| 항목 | 내용 |
|------|------|
| **해금 조건** | 첫 Affix 아이템 획득 후 Ctrl+T로 수동 활성화 |
| **주요 변화** | Affix 아이템 능력치 잠금 해제 |
| **드롭 특징** | Common \~ Uncommon 등급 위주 |

Haven은 Apotheosis 시스템이 **실제로 켜지는** 단계입니다.
이 단계 전까지는 Affix 아이템을 들고 있어도 효과가 발동하지 않습니다.

---

### 🌄 Tier 2 — Frontier (개척지)

| 항목 | 내용 |
|------|------|
| **해금 조건** | Uncommon 등급 장비 풀세트 착용 (무기 포함) |
| **주요 변화** | Apothic Invader 등장 시작 |
| **드롭 특징** | Rare 등급 아이템 출현 빈도 증가 |

Frontier에 진입하면 하늘에서 **빛기둥**이 내리고 경고음이 울립니다.
이것이 Apothic Invader의 스폰 신호입니다.
처음엔 무서울 수 있지만, 인베이더 처치가 다음 티어 해금의 필수 조건입니다.

**Invader 조우 시 행동 요령:**
- 빛기둥 위치를 확인하고 거리를 둔 채 장비 상태 점검
- 준비가 안 됐으면 일단 도망 (시간이 지나면 디스폰)
- 처치 시 Affix 아이템 **100% 드롭** 보장

---

### ⛰️ Tier 3 — Ascent (등반)

| 항목 | 내용 |
|------|------|
| **해금 조건** | Rare 등급 풀세트 착용 + Invader 1마리 처치 |
| **주요 변화** | 몹 체력·공격력 대폭 상승 |
| **드롭 특징** | Epic 등급 아이템 드롭 시작 |

이 단계부터 기존 바닐라 플레이 방식으로는 생존이 어려워집니다.
스킬 트리, 스펠 시스템, Apotheosis 장비 강화를 적극 활용해야 합니다.

---

### 🗻 Tier 4 — Summit (정상)

| 항목 | 내용 |
|------|------|
| **해금 조건** | Epic 등급 풀세트 착용 + 위더(Wither) 처치 |
| **주요 변화** | — |
| **드롭 특징** | Mythic 등급 아이템 본격 등장 |

위더는 이 모드팩에 추가된 강화 몬스터들과 함께 상당히 강력합니다.
Epic 장비로 충분히 강화한 후 도전하세요.

---

### 👑 Tier 5 — Pinnacle (절정)

| 항목 | 내용 |
|------|------|
| **해금 조건** | Epic 등급 풀세트 착용 + 엔더 드래곤 처치 |
| **주요 변화** | 최고 난이도 적용 |
| **드롭 특징** | Mythic 아이템 최고 확률 |

최종 티어입니다. 이 단계에서 Mythic 장비 파밍을 목표로 합니다.

---

## 티어별 장비 목표 요약

```
Haven    → 아무 Affix 아이템 일단 모으기
Frontier → Uncommon 풀세트 완성
Ascent   → Rare 풀세트 완성 + Invader 처치
Summit   → Epic 풀세트 완성 + 위더 처치
Pinnacle → Mythic 파밍 시작
```

## Apothic Invader & Elite

### Apothic Invader (침략자)

Frontier 티어 이상에서 등장하는 강력한 특수 몹입니다.

- **스폰 신호:** 하늘에서 빛기둥 + 채팅창 경고 메시지
- **특징:** 스스로도 Affix 장비를 착용하고 있어 매우 강력
- **드롭:** Affix 아이템 1개 **100% 드롭** + 추가 보너스 아이템
- **중요:** 다음 티어 해금에 Invader 처치가 필요합니다

**전투 팁:**
- 원거리 공격이 유리합니다
- 인베이더는 스폰 후 플레이어를 추적합니다
- 빛기둥이 보이면 이동하기 전에 HP와 장비를 먼저 확인하세요

### Apothic Elite (정예)

인베이더보다 조용하게 등장하는 강화 몬스터입니다.

- 별도 스폰 연출 없이 일반 몹처럼 등장
- 처치 시 추가 드롭 보너스
- 마주치면 그냥 잡는 것이 이득

### Invader Dungeon (침략자 던전)

월드에 생성되는 특수 구조물입니다.

- 내부에 상자 + Invader가 보스처럼 배치
- 좋은 Affix 아이템을 안정적으로 파밍할 수 있는 장소
- 탐험 중 발견하면 표시해두세요

## 자주 묻는 질문

**Q: 티어를 내리면 어떻게 되나요?**
A: 난이도가 낮아지고 드롭 등급도 낮아집니다. 언제든 다시 올릴 수 있으므로 부담 없이 내려도 됩니다.

**Q: Invader가 너무 강해서 못 잡겠어요.**
A: 일정 시간이 지나면 자연스럽게 디스폰됩니다. 억지로 잡으려다 죽는 것보다 장비를 더 맞춘 뒤 도전하는 것이 현명합니다.

**Q: 티어 해금 장비 조건이 정확히 뭔가요?**
A: 무기 슬롯 포함 모든 장비 슬롯에 해당 등급 이상의 Affix 아이템이 착용되어 있어야 합니다. 한 슬롯이라도 비어 있거나 등급 미달이면 해금되지 않습니다.
