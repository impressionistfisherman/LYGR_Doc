# 스포너 시스템 (ApothicSpawners)

ApothicSpawners는 바닐라의 몬스터 스포너를 **완전히 새로 만들어줍니다.**
이제 스포너를 채취하고, 종류를 바꾸고, 성능을 업그레이드할 수 있습니다.

---

## 바닐라와의 차이점

| 항목 | 바닐라 | ApothicSpawners |
|------|--------|-----------------|
| 스포너 채취 | 불가 | **실크터치로 채취 가능** |
| 소환 몬스터 변경 | 불가 | **스폰 에그로 변경할 수 있습니다** |
| 스폰 속도 | 고정 | **업그레이드 아이템으로 조정** |
| 스폰 범위 | 고정 | **업그레이드로 확장 가능** |
| 레드스톤 제어 | 불가 | **비교기 삽입으로 ON/OFF 가능** |

---

## 스포너 채취

### 필요 도구
- **실크터치(Silk Touch)** 인챈트가 적용된 곡괭이

### 방법
실크터치 곡괭이로 스포너를 그냥 캐면 됩니다.
바닐라처럼 파괴되지 않고 아이템으로 드롭됩니다.

> **참고: 어떤 스포너를 채취하면 좋을까요?**
>
> 초반 경험치/드롭 파밍 효율 기준 추천 순서:
>
> 1. **스켈레톤** — 화살, 뼈 수급 + 경험치 효율 우수
> 2. **좀비** — 철 조각, 썩은 고기 수급
> 3. **블레이즈** (네더) — 블레이즈 파우더 수급 필수
> 4. **거미** — 실, 거미 눈 수급

---

## 스포너 설치 및 종류 변경

### 설치
원하는 위치에 스포너를 그냥 설치하면 됩니다.
설치 후에는 어둠이 있어야 스폰이 진행됩니다.

### 소환 몬스터 종류 변경

**스폰 에그 획득 방법:**
1. `Capturing` 인챈트가 적용된 무기 제작 (JEI에서 조합법 확인)
2. 원하는 몬스터를 Capturing 무기로 처치
3. 일정 확률로 해당 몬스터의 스폰 에그 드롭

**변경 방법:**
스폰 에그를 손에 들고 스포너에 우클릭 → 소환 몬스터 변경 완료

## 스포너 업그레이드 아이템

업그레이드 아이템을 스포너에 우클릭하면 해당 스포너에 적용됩니다.
적용된 업그레이드는 스포너 GUI에서 확인할 수 있습니다.

### 업그레이드 목록

| 아이템 | 효과 |
|--------|------|
| 🍬 **설탕** | 최소 스폰 딜레이 -5틱 (더 빠르게 스폰) |
| ⏰ **시계** | 최대 스폰 딜레이 -5틱 |
| 🕷️ **발효된 거미 눈** | 스폰 수 +1 |
| 😭 **가스트의 눈물** | 스폰 수 ×2 |
| 💎 **프리즈마린 결정** | 스폰 범위 확장 |
| 🔥 **불꽃 막대기** | 스폰 범위 +1 블록 |
| 🍎 **황금 사과** | 스폰 수 무제한 |
| 📡 **비교기** | 레드스톤 신호로 스포너 ON/OFF 제어 |

> **주의: 황금 사과는 매우 비쌉니다**
>
> 황금 사과 업그레이드는 스폰 수를 무제한으로 만들어주지만,
> 재료가 비쌉니다. 초반엔 설탕 + 시계 + 발효된 거미 눈부터 시작하세요.

## 기본 파밍 구조 만들기

### 스텝 1 — 스포너 설치

```
목표: 스켈레톤 스포너 1개 설치
위치: 지하 또는 밀폐된 어두운 공간
크기: 최소 9×9×5 (스폰 공간 확보)
```

### 스텝 2 — 스포너 업그레이드

```
우선순위:
1. 시계 × 여러 개 → 스폰 속도 증가
2. 발효된 거미 눈 → 한 번에 스폰 수 증가
3. 나중에 황금 사과 → 최종 업그레이드
```

### 스텝 3 — 몹 처리 구조

**낙하 데미지 방식 (추천):**
```
스포너 → 워터 슬라이드 → 낙하 20\~22블록 → 체력 1 상태
→ 플레이어 1타 처치 → 경험치 + 드롭 수거
```

**칼날 트랩 방식:**
```
스포너 → 물로 한 곳에 집합 → 칼날/유리 트랩 설치
→ 자동 처리 가능 (경험치 수급 불가)
```

### 스텝 4 — 드롭 수거 자동화 (선택)

- 호퍼로 드롭 아이템을 상자로 자동 수집
- Create 모드의 벨트 + 퍼널 조합으로 대규모 자동화 가능

## AFK 파밍 팁

- **비교기 업그레이드**를 삽입하면 레드스톤으로 스포너를 끌 수 있습니다
- 자리를 비울 때 레버로 스포너를 끄고, 돌아오면 다시 켜는 방식으로 활용
- 인벤토리가 꽉 차면 드롭이 소멸하므로 호퍼 + 상자 연결 필수

---

## 경험치 파밍 최적화

경험치 파밍에서 중요한 것은 **플레이어가 직접 처치**하는 것입니다.

- 완전 자동(칼날 트랩 등)으로 처치하면 경험치가 감소합니다
- 낙하 트랩으로 체력 1을 만든 뒤 플레이어가 1타로 처치하는 방식이 경험치 효율 최대
- ApothicEnchanting 인퓨전, 스킬 레벨업에 경험치가 많이 필요하므로 초반에 경험치 팜을 만들어두는 것을 강력 추천
