# 인챈트 시스템 (ApothicEnchanting)

ApothicEnchanting은 바닐라 인챈트 시스템을 **완전히 재설계**합니다.
레벨 30 제한이 사라지고, 인챈트 테이블 주변에 다양한 전용 블록을 배치해
원하는 인챈트를 유도하고 레벨을 극한까지 올릴 수 있습니다.

---

## 바닐라와의 주요 차이점

| 항목 | 바닐라 | ApothicEnchanting |
|------|--------|-------------------|
| 인챈트 레벨 상한 | 30레벨 | **제거됨** (재료에 따라 무한 상승 가능) |
| 모루 수리 횟수 제한 | 있음 (점점 비싸짐) | **제거됨** |
| 인챈트 테이블 주변 블록 | 책장만 허용 | **전용 블록 다수 추가** |
| 특정 인챈트 유도 | 불가 | **블록 배치로 유도 가능** |

---

## 인챈트 테이블 주변 블록 (Shelf)

책장(Bookshelf) 대신 다양한 전용 Shelf 블록을 인챈트 테이블 주변에 배치할 수 있습니다.
배치한 블록 종류에 따라 **활성화되는 인챈트 종류와 파워**가 달라집니다.

### 블록 확인 방법

JEI에서 `shelf` 또는 `apotheosis`로 검색하면 전체 목록이 나옵니다.
각 Shelf 블록의 툴팁에 **어떤 인챈트를 강화하는지** 표시됩니다.

### 주요 Shelf 종류 (초반 → 후반 순서)

| 블록 | 단계 | 특징 |
|------|------|------|
| **Seashelf** | 초중반 | 수중 계열 인챈트 강화, 구하기 쉬움 |
| **Deepshelf** | 중반 | 도구/무기 계열 강화, 심층암 필요 |
| **Melonshelf** | 중반 | 다목적, 제작 간단 |
| **Infused 시리즈** | 후반 | 각 Shelf의 강화 버전, 인퓨전 과정 필요 |
| **Draconic Endshelf** | 최종 | 최대 인챈트 파워 제공 |

---

## 인퓨전 (Infusion)

일부 Shelf는 인챈트 테이블 자체에 넣어 **더 강한 버전(Infused)으로 업그레이드** 할 수 있습니다.

### 인퓨전 방법

1. 인챈트 테이블 GUI에서 Shelf 아이템을 넣기
2. 경험치 소모 → Infused 버전으로 변환
3. Infused Shelf를 테이블 주변에 다시 배치

### 주의사항

- 일부 Shelf는 **레벨 60 이상**이 필요합니다
- 인퓨전에는 상당한 경험치가 소모됩니다
- Apotheosis 스포너 팜으로 경험치를 충분히 확보하고 진행하세요

---

## 인챈트 테이블 배치 전략

### 원하는 인챈트가 안 뜰 때

인챈트는 테이블 주변 Shelf 구성에 따라 활성화 여부가 달라집니다.

1. 인챈트 테이블 GUI의 **미리보기 창** 확인
2. 원하는 인챈트 계열에 해당하는 Shelf를 추가 배치
3. 반대 계열 Shelf를 너무 많이 두면 원치 않는 인챈트가 섞일 수 있음

> **참고: 인챈트 계열별 Shelf 추천**
>
> - **공격력/무기 계열:** Deepshelf, Infused Deepshelf
> - **방어구 계열:** Seashelf, Crystalline Seashelf
> - **도구 계열:** Melonshelf, Infused Deepshelf
> - **최대 파워 구성:** Draconic Endshelf + Echoing Deepshelf 조합

## 최대 인챈트 셋업 로드맵

초반부터 최종 셋업까지의 단계별 진행입니다.

### 1단계 — 기본 셋업 (초반)
```
일반 책장 × 15 배치
→ 기본 레벨 30 인챈트 가능
```

### 2단계 — Seashelf 전환 (중반 초)
```
Seashelf × 16 배치
→ 인챈트 파워 상승, 더 높은 레벨 인챈트 가능
```

### 3단계 — Infused Seashelf (중반)
```
Seashelf → 인챈트 테이블에 넣어 인퓨전
→ Infused Seashelf로 변환 후 재배치
→ 인챈트 파워 대폭 상승
```

### 4단계 — 고급 Shelf 혼합 (중후반)
```
Heart-Forged Seashelf + Crystalline Seashelf + Soul-Touched Deepshelf 등
→ 목적에 맞는 인챈트 계열 특화 구성
```

### 5단계 — 최종 셋업 (후반)
```
Draconic Endshelf × 5 + Echoing Deepshelf × 다수
→ 최대 인챈트 파워 달성
→ 거의 모든 인챈트 최고 레벨 가능
```

## 추가 기능

### Altar of the Sea (바다의 제단)

쓸모없는 인챈트 장비를 새 인챈트 책으로 변환하는 블록입니다.

- 장비 4개 투입 → 경험치 소모 → 인챈트 책 생성
- 투입한 장비의 인챈트가 강할수록 더 좋은 책이 나옴
- 내구도 상관없이 모든 인챈트 장비 사용할 수 있습니다
- 잡다한 인챈트 아이템을 처리하는 가장 효율적인 방법

### Tome (특수 책)

특정 인챈트만 담긴 전용 책 아이템입니다.

- 원하는 인챈트를 정확히 겨냥해서 적용 가능
- 일반 인챈트 풀의 랜덤성을 피하고 싶을 때 유용
- 드롭 및 특수 제작으로 획득

### Anvil 개선

ApothicEnchanting이 적용된 환경에서 모루는:
- 수리 횟수 제한이 **사라집니다**
- 기존 "너무 비싸다" 오류가 발생하지 않습니다
- Apotheosis Affix 아이템과 인챈트를 자유롭게 조합 가능
