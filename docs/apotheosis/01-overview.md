# Apotheosis 개요

Apotheosis는 단순한 인챈트 모드가 아닙니다.
장비에 랜덤 특수 효과(Affix)가 붙고, 젬을 소켓에 박고, 월드 티어를 올려가며 점점 강해지는 **RPG 성장 시스템**입니다.

> **정보: 인게임 가이드북 먼저 만드세요**
>
> **책 + 금괴 → 조합 → 📕 Chronicles of Shadows**
>
> 이 책이 Apotheosis의 공식 인게임 가이드북입니다.
> Patchouli 모드가 설치되어 있어야 열립니다. 이 모드팩에는 이미 포함되어 있으니 바로 만들어두세요.

---

## 모듈 구조

Apotheosis는 4개의 독립 모듈로 구성됩니다. 이 모드팩에는 전부 설치되어 있습니다.

| 모듈 | 역할 |
|------|------|
| **Apotheosis** (본체) | Affix 장비, 젬, 월드 티어, 보스, 인베이더 |
| **ApothicEnchanting** | 인챈트 테이블 완전 개편, 신규 인챈트 다수 추가 |
| **ApothicSpawners** | 스포너 채취·편집·업그레이드 |
| **ApothicAttributes** | 캐릭터 능력치 확장 (공속, 크리티컬, 행운 등) |

각 모듈은 독립적으로 작동하지만, 함께 사용할 때 시너지가 극대화됩니다.

## 핵심 시스템 요약

Apotheosis를 처음 접하면 동시에 너무 많은 것이 쏟아집니다.
우선 아래 다섯 가지 시스템만 이해하면 나머지는 자연스럽게 따라옵니다.

### 1. Affix 아이템
몬스터 처치나 던전 상자에서 얻는 **특수 효과가 붙은 장비**입니다.
이름 색상이 등급을 나타내며, 등급이 높을수록 더 강력한 효과가 붙습니다.
→ [Affix 시스템 상세 가이드](/docs/apotheosis/affix-system)

### 2. 월드 티어 (World Tier)
첫 Affix 아이템을 주운 뒤 **Ctrl + T** 를 눌러 활성화하는 난이도/보상 단계입니다.
티어가 높을수록 더 강한 적과 더 좋은 장비가 등장합니다.
→ [월드 티어 상세 가이드](/docs/apotheosis/world-tiers)

### 3. 젬 & 소켓
Affix 아이템의 소켓에 젬을 박아 능력치를 추가로 강화하는 시스템입니다.
→ [젬 & 소켓 상세 가이드](/docs/apotheosis/gems-and-sockets)

### 4. 작업대 시스템
분해, 리포징, 증강 등 장비를 강화하는 전용 작업대 5종입니다.
→ [작업대 상세 가이드](/docs/apotheosis/workstations)

### 5. 인챈트 시스템
ApothicEnchanting이 바닐라 인챈트를 완전히 개편합니다.
→ [인챈트 시스템 상세 가이드](/docs/apotheosis/enchanting)

## 시작 순서 권장 로드맵

처음 Apotheosis를 접하는 플레이어를 위한 권장 순서입니다.

```
1. Chronicles of Shadows 가이드북 제작
2. 첫 Affix 아이템 획득 (오버월드 타워 구조물 탐색)
3. Ctrl + T → World Tier: Haven 활성화
4. Affix 장비 풀세트 맞추기 (Uncommon 이상 목표)
5. World Tier: Frontier 해금 → Invader 도전
6. Gem Cutting Table 제작 → 소켓 활용 시작
7. Reforging Table 제작 → Affix 리롤 시작
8. ApothicEnchanting 고급 인챈트 셋업 구축
9. World Tier 순차 상승 (Ascent → Summit → Pinnacle)
```

## 이 가이드에서 다루지 않는 것

- 세부 수치 계산 (메타 분석 수준의 최적화)
- 모드팩별 커스텀 설정값 (서버 설정에 따라 다를 수 있음)
- Chronicles of Shadows 원문 번역 (인게임에서 직접 확인 권장)
