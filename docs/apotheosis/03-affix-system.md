# Affix 시스템

Affix는 Apotheosis의 **핵심 장비 시스템**입니다.  
무기, 방어구, 도구에 랜덤으로 붙는 특수 효과로, 같은 철 검이라도 Affix에 따라 성능이 완전히 달라집니다.

---

## 아이템 등급 (Rarity)

Affix 아이템은 이름 색상으로 등급을 구분합니다.

| 색상 | 등급 | 특징 |
|------|------|------|
| ⬜ 회색 | **Common** | 기본 수치 소폭 향상, 소켓 없음 |
| 🟩 초록 | **Uncommon** | 수치 향상 + 소켓 1개 가능성 |
| 🟦 파랑 | **Rare** | 수치 대폭 향상 + 특수 효과 1~2개 |
| 🟪 보라 | **Epic** | 강력한 특수 효과 복수 + 소켓 다수 |
| 🟧 주황 | **Mythic** | 최고 등급, 게임 흐름을 바꾸는 효과 |

:::tip 등급 확인 방법
아이템을 집어 들면 이름 색상이 바로 보입니다.  
툴팁에는 해당 아이템의 Affix 효과 목록이 전부 표시됩니다.
:::

---

## Affix 효과 종류

Affix 효과는 아이템 종류(무기/방어구/도구)에 따라 다른 풀에서 뽑힙니다.

### 근접 무기 주요 Affix

| Affix 이름 | 효과 |
|-----------|------|
| **Cleaving** | 공격 시 전방 범위 적에게 동시 피해 |
| **Life Steal** | 피해를 가할 때 체력 회복 |
| **Execute** | 체력이 일정 % 이하인 적에게 추가 피해 |
| **Berserking** | 체력이 낮을수록 공격력 증가 |
| **Celerity** | 공격 속도 증가 |

### 원거리 무기 주요 Affix

| Affix 이름 | 효과 |
|-----------|------|
| **Spectral** | 화살이 무한 (내구도 소모 없음) |
| **Bouncing** | 화살이 벽에 반사됨 |
| **Piercing** | 화살이 여러 적을 관통 |
| **Explosive** | 화살 착탄 시 폭발 |

### 방어구 주요 Affix

| Affix 이름 | 효과 |
|-----------|------|
| **Resilient** | 받는 피해 % 감소 |
| **Thorns** | 받은 피해의 일부를 공격자에게 반사 |
| **Feather Falling** | 낙하 피해 감소 |
| **Speed** | 이동 속도 증가 |
| **Health** | 최대 체력 증가 |

### 도구 주요 Affix

| Affix 이름 | 효과 |
|-----------|------|
| **Super Massive** | 3×3 범위 채굴 |
| **Greedy** | 광석 추가 드롭 |
| **Magnetic** | 드롭 아이템 자동 수집 |

## Affix 아이템 획득 방법

### 1. 오버월드 타워 구조물

Apotheosis가 생성하는 전용 타워가 오버월드 곳곳에 있습니다.

- 사막, 설원, 일반 등 여러 바이옴 변형 존재
- 내부 상자에 **Starter Affix 아이템 보장** 포함
- 타워 주변에 TNT가 있으니 조심하세요

### 2. 몬스터 드롭

- 모든 적대 몹이 Affix 아이템을 드롭할 수 있습니다
- 월드 티어가 높을수록 드롭 확률과 등급이 올라갑니다

### 3. Apothic Invader 처치

- Affix 아이템 **100% 드롭** 보장
- 고티어에서 좋은 등급을 안정적으로 얻는 가장 확실한 방법

### 4. 던전 상자

- 일반 던전, Apotheosis 전용 구조물 상자에서 출현
- Invader Dungeon 내부 상자에서 특히 좋은 아이템이 나옵니다

### 5. Wandering Trader 구매

- 가끔 Affix 아이템을 판매하는 경우 있음
- 비용 대비 효율은 낮지만 특정 등급이 필요할 때 활용할 수 있습니다

## 아이템 비교 방법

Affix 아이템을 비교할 때는 **수치 + 효과** 두 가지를 모두 봐야 합니다.

```
예시 비교:

[Rare] 철 검
  - 공격력 +4
  - Cleaving (전방 범위 공격)
  - Life Steal 2%

[Uncommon] 다이아몬드 검
  - 공격력 +7
  - (효과 없음)
```

단순히 기본 수치만 높다고 좋은 것이 아닙니다.  
Cleaving + Life Steal 조합이 순수 공격력 차이를 상회하는 경우가 많습니다.
