# 젬 & 소켓

젬(Gem)은 Affix 아이템의 빈 소켓에 박아
**추가 능력치와 효과를 부여**하는 강화 시스템입니다.

---

## 젬 획득 방법

### 몬스터 드롭
- **플레이어가 직접 처치한** 모든 적대 몹에서 드롭 가능합니다
- 자동 파밍(스포너 트랩 등)으로 얻은 몬스터에게서는 **드롭되지 않습니다**
- 월드 티어가 높을수록 드롭 확률과 등급 상승

### 던전 상자
- 일반 던전 및 Apotheosis 전용 구조물 상자에서 발견

### Apothic Invader 처치
- 아이템 드롭과 함께 젬을 추가로 떨어뜨리는 경우 있음

---

## 젬 등급 (Purity)

젬에도 Affix 아이템처럼 등급이 존재합니다.
같은 종류의 젬이라도 **순도(Purity)** 가 높을수록 효과가 더 강합니다.

| 등급 | 설명 |
|------|------|
| Common | 기본 효과, 가장 흔함 |
| Uncommon | 효과 소폭 강화 |
| Rare | 효과 중간 강화 |
| Epic | 효과 대폭 강화 |
| Mythic | 최대 효과, 매우 희귀 |

---

## 소켓 확인 방법

아이템 툴팁 하단에 소켓 정보가 표시됩니다.

```
[ ○ Empty Socket ]   → 빈 소켓 (젬 삽입 가능)
[ ● Ruby ]           → 루비 젬이 삽입된 상태
```

> **정보: 소켓 개수와 등급의 관계**
>
> - Common 아이템: 소켓 없음
> - Uncommon: 소켓 0\~1개
> - Rare: 소켓 1\~2개
> - Epic: 소켓 2\~4개
> - Mythic: 소켓 최대
>
> 소켓 수가 많은 Epic 이상 아이템을 획득하면 젬 강화 효과를 극대화할 수 있습니다.

---

## 젬 삽입 & 제거

### 젬 삽입 방법

1. **Gem Cutting Table** 제작 (JEI에서 조합법 확인)
2. 테이블 GUI에 Affix 아이템과 삽입할 젬을 함께 투입
3. 원하는 소켓에 젬 선택 후 삽입

### 젬 제거 시 주의사항

> **위험: 젬 제거 시 젬이 파괴됩니다**
>
> 젬을 소켓에서 빼면 **젬이 영구적으로 파괴**됩니다.
> 아이템 자체는 보존되고 소켓이 빈 상태로 돌아오지만,
> 비싼 Epic/Mythic 젬은 제거 전에 반드시 신중하게 결정하세요.

---

## 젬 종류별 효과

젬 효과는 **박히는 아이템 종류**에 따라 다르게 적용됩니다.
(무기에 박으면 공격 관련 효과, 방어구에 박으면 방어/체력 관련 효과)

### 핵심 젬 목록

| 젬 이름 | 무기 효과 | 방어구 효과 |
|---------|-----------|-------------|
| **Ruby** | 공격력 증가 | 최대 체력 증가 |
| **Sapphire** | 공격 속도 증가 | 이동 속도 증가 |
| **Emerald** | 행운 증가 | 행운 증가 |
| **Amethyst** | 크리티컬 확률 증가 | 크리티컬 저항 증가 |
| **Topaz** | 화염 피해 추가 | 화염 저항 증가 |
| **Opal** | — | 방어력 증가 |

> **참고: 젬 효과 정확한 수치 확인**
>
> JEI에서 젬 아이템 위에 커서를 올리면
> 삽입 가능한 아이템 종류별 효과와 수치를 확인할 수 있습니다.

### 차원별 전용 젬

일부 젬은 특정 차원(오버월드/네더/엔드)의 몬스터에서만 드롭됩니다.

- **오버월드:** 기본 젬 대부분
- **네더:** 화염/저항 관련 고성능 젬
- **엔드:** 텔레포트/포스 관련 고성능 젬

## 젬 업그레이드 — Gem Dust & 순도 향상

낮은 순도의 젬을 버리지 말고 **Gem Dust**로 분쇄해서 활용하세요.

### 방법

1. Gem Cutting Table에서 젬 → Gem Dust로 분쇄
2. Gem Dust 여러 개를 조합해 더 높은 순도의 젬으로 합성
3. JEI에서 "Gem Dust" 검색 → 조합 비율 확인

### 언제 분쇄하면 좋을까요?

- Common/Uncommon 젬이 너무 많이 쌓였을 때
- 원하는 종류의 고순도 젬이 필요할 때
- 중반 이후 Common 젬은 대부분 분쇄해서 업그레이드 재료로 활용하는 것이 효율적

## 초반 젬 전략

처음 소켓에 어떤 젬을 박아야 할지 모를 때의 기준입니다.

**전투 위주 플레이:**
- 무기: Ruby (공격력) 또는 Amethyst (크리티컬)
- 방어구: Ruby (체력) 우선

**생존 위주 플레이:**
- 방어구 전 슬롯: Ruby (체력) + Opal (방어력) 조합

**후반 최적화:**
- 직업과 스킬 트리 방향을 결정한 후 특화된 젬 세팅으로 전환
