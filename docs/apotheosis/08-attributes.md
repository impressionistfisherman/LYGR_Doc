# 능력치 시스템 (ApothicAttributes)

ApothicAttributes는 바닐라의 단순한 능력치 체계를 **RPG 스타일로 확장**합니다.  
공격 속도, 크리티컬, 시전 속도 등 다양한 세부 스탯이 추가됩니다.

:::info 다른 모드와의 연동
ApothicAttributes의 능력치는 다음 모드들과 연동됩니다:
- **Apotheosis** Affix 아이템의 스탯 수치
- **puffish_skills / skill_tree** 스킬 포인트 투자 효과
- **Spell Power** 마법 데미지 계산
:::

## 추가 능력치 목록

### 공격 관련

| 능력치 | 설명 | 주요 획득처 |
|--------|------|-------------|
| **공격력 (Attack Damage)** | 근접 공격 기본 데미지 | Affix 장비, 스킬 트리 |
| **공격 속도 (Attack Speed)** | 초당 공격 횟수 | Affix 장비, 스킬 트리 |
| **크리티컬 확률 (Crit Chance)** | 치명타 발동 확률 | Affix 장비, 젬, 스킬 트리 |
| **크리티컬 데미지 (Crit Damage)** | 치명타 시 데미지 배율 | Affix 장비, 젬, 스킬 트리 |

### 방어 관련

| 능력치 | 설명 | 주요 획득처 |
|--------|------|-------------|
| **최대 체력 (Max Health)** | 최대 HP | Affix 장비, 젬, 스킬 트리 |
| **방어력 (Armor)** | 받는 피해 감소 | Affix 장비, 젬 |
| **넉백 저항 (Knockback Resistance)** | 밀려남 저항 | 방어구 Affix |

### 이동/유틸 관련

| 능력치 | 설명 | 주요 획득처 |
|--------|------|-------------|
| **이동 속도 (Move Speed)** | 걷기/달리기 속도 | Affix 장비, 스킬 트리 |
| **행운 (Luck)** | 드롭/인챈트 보너스 | Affix 장비, 스킬 트리 |

## 능력치 확인 방법

### 인벤토리 스탯 창

인벤토리를 열면 캐릭터 탭에서 **현재 전체 능력치**를 확인할 수 있습니다.

- 항목별 현재 수치와 기본값 비교 가능
- 장비를 바꾸면 실시간으로 변화량 표시

### 아이템 툴팁

장비 아이템에 마우스를 올리면:
```
+5 공격력
+1.2 공격 속도
+3% 크리티컬 확률
```
형식으로 착용 시 변화 수치가 표시됩니다.

:::tip 비교 모드
인벤토리를 연 상태에서 다른 장비에 마우스를 올리면  
현재 착용 중인 장비와의 차이가 `+/-` 로 표시됩니다.  
장비 교체 결정에 적극 활용하세요.
:::

## 능력치 우선순위 가이드

처음에는 어떤 능력치부터 올려야 할지 막막할 수 있습니다.  
플레이 스타일에 따른 우선순위입니다.

### 근접 전투 위주

```
1순위: 최대 체력 (생존 기반)
2순위: 공격력 (기본 딜)
3순위: 크리티컬 확률 + 크리티컬 데미지
4순위: 공격 속도
```

### 원거리/마법 위주

```
1순위: 최대 체력 (생존 기반)
2순위: Spell Power 계열 (Spell Engine 연동)
3순위: 크리티컬 확률
```

### 탐험/서바이벌 위주

```
1순위: 최대 체력
2순위: 이동 속도
3순위: 행운 (드롭 보너스)
```

## 능력치 상한선

일부 능력치는 최대값이 존재합니다.

| 능력치 | 기본값 | 실질적 상한 |
|--------|--------|-------------|
| 크리티컬 확률 | 5% | 100% (이 이상은 의미 없음) |
| 공격 속도 | 4.0 | 20.0 (초당 20회) |
| 이동 속도 | 0.1 | 약 0.4\~0.5 (그 이상은 조작 어려움) |

크리티컬 확률 100%에 도달하면 모든 공격이 치명타가 되므로,  
이 이후엔 **크리티컬 데미지** 배율을 높이는 것이 더 효율적입니다.

## 자주 묻는 질문

**Q: 능력치가 너무 많아서 무엇을 봐야 할지 모르겠습니다.**  
A: 초반엔 체력 / 공격력 / 방어력 세 가지만 보면 충분합니다.  
크리티컬이나 공격 속도는 중반 이후에 신경 쓰는 것이 자연스럽습니다.

**Q: 능력치 수치가 소수점으로 나오는 이유는 뭔가요?**  
A: Affix, 젬, 스킬 포인트, 인챈트 등 여러 소스의 수치가 더해지기 때문입니다.  
정확한 수치보다는 **어느 방향으로 올라가고 있는지** 추세를 보는 것이 중요합니다.
