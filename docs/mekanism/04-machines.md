# 핵심 기계 목록

Mekanism의 주요 기계들을 역할별로 정리합니다.
모든 기계의 조합법은 JEI에서 확인하세요.

---

## 기본 가공 기계

### Enrichment Chamber (농축 챔버)

| 항목 | 내용 |
|------|------|
| **역할** | 광석 → 분말 2개, 석탄 → 농축 석탄, 레드스톤 농축 등 |
| **에너지** | 중간 소비 |
| **특이사항** | 석탄/레드스톤을 항상 먼저 농축해서 사용 권장 |

---

### Energized Smelter (에너지 제련로)

| 항목 | 내용 |
|------|------|
| **역할** | 바닐라 용광로 역할 + 에너지로 작동 |
| **장점** | 속도 빠름, 업그레이드 가능 |
| **업그레이드** | Speed Upgrade, Energy Upgrade 슬롯 보유 |

---

### Crusher (분쇄기)

| 항목 | 내용 |
|------|------|
| **역할** | 덩어리(Clump) → 더러운 분말, 돌 → 자갈, 자갈 → 모래 등 |
| **용도** | 광석 처리 라인의 중간 단계 |

---

### Metallurgic Infuser (금속 주입기)

| 항목 | 내용 |
|------|------|
| **역할** | 재료에 특정 물질 주입 → 합금 제작 |
| **주요 레시피** | 철 + 탄소 → 강철, 각종 합금 제작 |
| **주입재** | 탄소(석탄), 레드스톤, 다이아몬드, 정제 흑요석 등 |

---

### Purification Chamber (정화 챔버)

| 항목 | 내용 |
|------|------|
| **역할** | 광석 + 산소 → 덩어리(Clump) |
| **가스 필요** | 산소(Oxygen) 파이프 연결 필수 |
| **용도** | Tier 2 이상 광석 처리 |

---

### Osmium Compressor (오스뮴 압축기)

| 항목 | 내용 |
|------|------|
| **역할** | 오스뮴 분말 → 압축 오스뮴, 각종 압축 소재 제작 |
| **용도** | 고급 소재 제작에 필요 |

---

### Precision Sawmill (정밀 톱)

| 항목 | 내용 |
|------|------|
| **역할** | 통나무 → 6개 판자 (바닐라는 4개), 나무 계열 정밀 처리 |
| **장점** | 목재 수율 1.5배 |

---

## 화학 처리 기계

### Electrolytic Separator (전해 분리기)

| 항목 | 내용 |
|------|------|
| **역할** | 액체(물, 브라인 등)를 가스로 분리 |
| **물 → 수소 + 산소** | Tier 2 광석 처리의 핵심 |
| **브라인 → 나트륨 + 염소** | Tier 3 광석 처리에 필요 |

---

### Chemical Infuser (화학 주입기)

| 항목 | 내용 |
|------|------|
| **역할** | 두 가지 가스를 결합해 새 가스 생성 |
| **주요 레시피** | 수소 + 염소 → 염화수소(HCl) |

---

### Chemical Injection Chamber (화학 주입 챔버)

| 항목 | 내용 |
|------|------|
| **역할** | 광석 + 염화수소 → 파편(Shard) 4개 |
| **용도** | Tier 3 광석 처리 핵심 |

---

### Chemical Dissolution Chamber (화학 용해 챔버)

| 항목 | 내용 |
|------|------|
| **역할** | 광석 + 황산 → 더러운 슬러리 |
| **용도** | Tier 4 광석 처리 첫 단계 |

---

### Chemical Washer (화학 세척기)

| 항목 | 내용 |
|------|------|
| **역할** | 더러운 슬러리 + 물 → 깨끗한 슬러리 |
| **용도** | Tier 4 광석 처리 |

---

### Chemical Crystallizer (화학 결정화기)

| 항목 | 내용 |
|------|------|
| **역할** | 깨끗한 슬러리 → 결정 5개 |
| **용도** | Tier 4 광석 처리 마지막 화학 단계 |

## 물류/자동화 기계

### Logistical Transporter (물류 수송기)

| 항목 | 내용 |
|------|------|
| **역할** | 기계 간 아이템 자동 수송 |
| **설정** | Configurator로 우선순위, 필터 설정 가능 |
| **특징** | 색상 코드로 라우팅 제어 가능 |

---

### Logistical Sorter (물류 분류기)

| 항목 | 내용 |
|------|------|
| **역할** | 상자에서 특정 아이템만 꺼내 수송기에 투입 |
| **설정** | 아이템 필터, 투입 속도 조절 |
| **추천** | 여러 종류 광석을 하나의 상자에서 자동 분류 처리 |

---

### Digital Miner (디지털 채굴기)

| 항목 | 내용 |
|------|------|
| **역할** | 지정 범위의 블록을 필터 기반으로 자동 채굴 |
| **설정** | OreDict 필터, 아이템 필터, 채굴 반경 설정 |
| **에너지** | 매우 높은 에너지 소모 |

**필터 설정 예시:**
```
ore* 필터: 이름이 "ore"로 시작하는 모든 블록 채굴
oreDiamond: 다이아몬드 광석만 채굴
oreiron, oregold: 철+금만 채굴
```

> **주의: 디지털 채굴기는 에너지를 많이 먹습니다**
>
> Ultimate 에너지 발전 시스템 없이는 디지털 채굴기가 금방 멈춥니다.
> 중후반 이후 충분한 에너지 생산이 확보된 뒤 사용하세요.

## Factory (팩토리) — 기계 다중 처리

팩토리는 기본 기계의 **병렬 처리 버전**입니다.

| 팩토리 티어 | 동시 처리 슬롯 |
|------------|---------------|
| **Basic Factory** | 3슬롯 동시 처리 |
| **Advanced Factory** | 5슬롯 동시 처리 |
| **Elite Factory** | 7슬롯 동시 처리 |
| **Ultimate Factory** | 9슬롯 동시 처리 |

### 팩토리 전환 방법

기존 기계를 제작대에 놓고 팩토리 업그레이드 키트를 조합하면
해당 기계의 팩토리 버전으로 변환됩니다.

```
Enrichment Chamber → Basic Enrichment Factory → Advanced Enrichment Factory ...
```

> **참고: 팩토리 전환 시점**
>
> 광석 종류가 많아져서 처리 병목이 생기기 시작하면
> 기계를 Factory로 전환하는 것이 좋습니다.
> Elite Factory 1개 = 기계 7개 분량이므로 공간도 절약됩니다.
