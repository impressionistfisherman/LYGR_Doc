# 고급 장비 — 제트팩, MekaSuit, Mekatool

Mekanism은 전투부터 탐험, 채굴까지 모든 것을 커버하는 **최강의 개인 장비**를 제공합니다.

---

## 제트팩 (Jetpack)

Mekanism의 상징적인 아이템입니다.

### 기능

- **비행:** 수소(Hydrogen)를 소모해 공중 비행
- **호버 모드:** Shift + 비행키로 공중에서 제자리 유지

### 연료 — 수소(Hydrogen)

```
수소 충전 방법:
  Electrolytic Separator로 물 → 수소 생산
  수소 Gas Tank에 저장
  Gas Tank를 제트팩에 우클릭 충전
  또는
  Gas Filling Station 설치 → 자동 충전
```

### 제트팩 티어

| 티어 | 연료 용량 | 비행 속도 |
|------|-----------|-----------|
| **Basic Jetpack** | 24,000 mB | 보통 |
| **Advanced Jetpack** | 48,000 mB | 빠름 |
| **Elite Jetpack** | 96,000 mB | 더 빠름 |
| **Ultimate Jetpack** | 192,000 mB | 최고속 |

> **참고: 제트팩을 먼저 만드세요**
>
> Mekanism에서 제트팩은 단순한 이동 수단이 아닙니다.
> 탐험, 전투, 채굴 모든 상황에서 생존성이 극적으로 올라갑니다.
> Tier 2 광석 처리 시스템이 갖춰지면 수소를 충분히 확보하고 제트팩을 바로 제작하세요.

## Free Runners (프리 러너)

부츠 대신 착용하는 이동 보조 장비입니다.

### 기능

- **낙하 피해 면역:** 어떤 높이에서 떨어져도 피해 없음
- **계단 자동 오르기:** 1블록 단차를 점프 없이 자동으로 오름

### 필수 이유

제트팩과 함께 사용 시 추락 사고가 없어집니다.
제트팩 초보 시절 추락사를 막는 생명선입니다.

---

## Atomic Disassembler (원자 분해기)

Mekanism의 올인원 도구입니다.

### 기능

- **모드 전환:** 한 아이템으로 곡괭이/삽/도끼/검 모두 기능
- **AOE 모드:** 3×3 또는 5×5 범위 채굴
- **Vein Mine 모드:** 같은 종류 블록 연속 채굴
- **고속 모드:** 에너지를 더 소모하지만 채굴 속도 극대화

### 에너지 소모

- 에너지로 작동 → 배터리(Energy Tablet 등) 필수
- 에너지가 없으면 일반 도구와 동일 성능으로 저하
- 에너지가 충분할 때 성능이 극대화됨

### 전투 성능

Atomic Disassembler는 무기로도 강력합니다.

- 기본 데미지가 매우 높음
- 에너지가 있을 때 추가 데미지 발동

## MekaSuit (메카슈트)

Mekanism 최종 방어구입니다. 각 부위별로 고유 기능이 있습니다.

### MekaSuit Helmet (헬멧)

- **자동 산소 공급:** 수중/진공에서 산소 자동 공급
- **야간 투시:** 야간 자동 활성화
- **자동 급식:** 배고픔 자동 채우기 (식량 필요)
- **HUD 표시:** 체력, 에너지, 가스 상태 HUD 표시

### MekaSuit Bodyarmor (흉갑)

- **제트팩 내장:** 별도 제트팩 없이 비행 가능
- **수소 내장 탱크:** 비행 연료 내장

### MekaSuit Pants (레깅스)

- **고속 이동:** 이동 속도 대폭 증가
- **수영 속도 증가**

### MekaSuit Boots (부츠)

- **Free Runners 내장:** 낙하 피해 면역 + 계단 자동 오르기

### MekaSuit 업그레이드 모듈

MekaSuit에 모듈을 추가해 기능을 더 확장할 수 있습니다.

| 모듈 | 효과 |
|------|------|
| **Energy Shield** | 에너지로 작동하는 방어막 |
| **Blasting Unit** | 채굴 범위 확장 |
| **Gyroscopic Stabilizer** | 공중 이동 안정화 |
| **Charge Distribution** | 팀원에게 에너지 공유 |

## Mekatool (메카툴)

Atomic Disassembler의 후속 최강 도구입니다.

### 기능

- Atomic Disassembler의 모든 기능 포함
- **더 넓은 AOE:** 최대 7×7 범위 채굴
- **Vein Mine 강화:** 더 빠르고 넓은 광맥 채굴
- **자동 제련:** 채굴과 동시에 자동 제련 가능

### 제작 조건

- Atomic Disassembler 포함
- Ultimate 티어 소재 다수 필요
- 중후반 이후 목표 아이템

---

## Robit (로빗)

자동으로 아이템을 수집하고 도와주는 소형 로봇입니다.

### 기능

- 주변 아이템 자동 수집
- 인벤토리 역할 (추가 저장 공간)
- 이동하면서 따라다님
