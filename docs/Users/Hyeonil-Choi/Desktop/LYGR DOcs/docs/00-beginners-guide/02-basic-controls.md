# 2단계 — 기본 조작법

> 마인크래프트의 기본 조작을 익혀보자. 이미 알고 있으면 넘어가도 돼!

---

## 기본 키 배열

### 이동 & 시점

| 키 | 동작 |
|----|------|
| `W` `A` `S` `D` | 앞뒤좌우 이동 |
| `마우스` | 시점 회전 |
| `Space` | 점프 |
| `Space` × 2 | 날기 (크리에이티브 모드) |
| `Shift` | 살금살금 걷기 (절벽에서 안 떨어짐!) |
| `Ctrl` | 달리기 (누른 채로 W) |

> 💡 **Shift는 꼭 기억해!** 절벽 가장자리에서 Shift 누르면 안 떨어져.

### 인벤토리 & 상호작용

| 키 | 동작 |
|----|------|
| `E` | 인벤토리 열기/닫기 |
| `1~9` | 핫바 슬롯 선택 |
| `마우스 휠` | 핫바 스크롤 |
| `우클릭` | 블록 설치 / NPC 대화 / 아이템 사용 |
| `좌클릭` | 블록 채굴 / 공격 |
| `F` | 들고 있는 아이템 버리기 |
| `Q` | 아이템 하나 버리기 |

### 채굴

- 나무는 **빈손**으로도 캘 수 있어
- 돌은 **나무 곡괭이 이상** 필요
- 철은 **돌 곡괭이 이상** 필요
- 금·다이아는 **철 곡괭이 이상** 필요

> 💡 좌클릭을 **꾹 눌러서** 채굴해. 한 번만 누르면 안 깎여!

## 인벤토리 이해하기

```
┌────────────────────────────────────┐
│  [아이템 칸 × 27]   ← 가방 (메인 인벤토리)│
│  [아이템 칸 × 9 ]   ← 핫바 (빠른 접근)   │
│  [갑옷 슬롯 × 4]    ← 머리/상체/하체/발  │
└────────────────────────────────────┘
```

- **핫바**: 화면 아래쪽에 항상 보이는 9칸. 자주 쓰는 아이템을 여기 넣어둬.
- **인벤토리**: `E`로 열면 나오는 전체 가방.
- **아이템 이동**: 클릭해서 집고, 다른 칸에서 클릭하면 이동. `Shift+클릭`하면 바로 이동!

## 제작 (크래프팅) 기초

### 인벤토리에서 제작
`E`로 인벤토리를 열면 오른쪽 위에 **2×2 제작 칸**이 있어.  
간단한 아이템은 여기서 바로 만들 수 있어.

### 작업대 제작 & 사용
더 복잡한 아이템은 **작업대(3×3 칸)**가 필요해.

```
1. 나무 원목 채집 → E 열기 → 2x2 칸에 원목 놓기 → 나무 판자 4개 획득
2. 나무 판자 4개를 2x2로 놓기 → 작업대 획득
3. 작업대를 바닥에 설치 → 우클릭으로 열기
```

> 📌 **레시피 찾기 팁:** `E` 키로 인벤토리를 열면 오른쪽에 **JEI(Just Enough Items)** 아이템 목록이 보여. 아이템에 마우스를 올리고 `R`을 누르면 레시피가 나와! `U`를 누르면 그 아이템을 재료로 쓰는 레시피가 나와.

## 모드팩 전용 추가 키

모드팩에 모드가 많아서 키가 추가됐어. 처음엔 헷갈릴 수 있으니까 주요한 것만 알아두자.

| 키 | 기능 |
|----|------|
| `K` | 스킬 트리 열기 (Puffish Skills) |
| `Z` | 마법 목록 열기 (Spell Engine) |
| `C` | 직업 정보 창 (RPG Classes) |
| `Tab` | 퀘스트북 열기 |
| `B` | 배낭 열기 (있을 경우) |

> 💡 키 충돌이 나면 **Options → Controls** 에서 바꿀 수 있어.

## 꼭 알아야 할 생존 기초

### 허기(배고픔) 관리
- 화면 아래 ❤️ 옆에 있는 🍗 아이콘이 **배고픔 게이지**야
- 게이지가 비면 **달리기 불가**, 더 비면 **체력이 깎여**
- 음식을 들고 `우클릭`으로 먹어

### 체력 회복
- **배고픔이 충분하면** 자동으로 체력 회복돼
- 음식 먹기, 침대에서 자기, 일부 물약으로 즉시 회복 가능

### 밤과 낮
- 낮: 약 10분 / 밤: 약 7분
- **밤에는 좀비, 스켈레톤, 거미** 등 몬스터가 스폰됨
- 첫날밤 전에 은신처를 꼭 만들어야 해!
