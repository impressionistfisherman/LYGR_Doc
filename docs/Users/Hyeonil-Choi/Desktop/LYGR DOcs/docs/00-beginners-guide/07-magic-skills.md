# 7단계 — 마법 & 스킬

> 이 모드팩에는 강력한 마법 시스템과 스킬 트리가 있어.  
> 바닐라에 익숙해졌으면 이제 본격적으로 활용해보자!

---

## 바닐라 마법 요소 기초

마인크래프트 바닐라에도 기본적인 마법 요소가 있어.

### 물약 (Potion)
몬스터를 잡거나 탐험할 때 큰 도움이 돼.

**물약 제조 과정:**
```
1. 양조기 제작: 블레이즈 막대 1개 + 조약돌 3개
2. 기본 물약(어색한 물약) → 네더 사마귀 + 물병
3. 기본 물약 + 재료 추가 → 효과 물약
```

**필수 물약들:**

| 물약 | 재료 | 효과 |
|------|------|------|
| 치유 물약 | 반딧불 + 기본 물약 | 즉시 체력 회복 |
| 힘 물약 | 블레이즈 파우더 | 공격력 증가 |
| 속도 물약 | 설탕 | 이동 속도 증가 |
| 내화 물약 | 마그마 크림 | 화염 데미지 무효 |
| 야시경 물약 | 황금 당근 | 어둠에서 잘 보임 |

### 마법 부여된 책
인챈트 테이블이나 던전 체스트에서 얻을 수 있어.
모루(anvil)에서 장비 + 인챈트 책을 합치면 인챈트 적용!

## 🎮 이제 이 모드 써봐! — 마법 & 스킬 모드들

### ✨ Spell Engine (마법 시스템)

마법 스크롤을 모아서 강력한 주문을 쓸 수 있어.

**입문 방법:**
```
1. 탐험하다 보면 마법 스크롤 발견
2. 마법 책에 스크롤 등록 (주문서 UI 활용)
3. Z 키로 마법 목록 열기
4. 마나를 소비해서 주문 시전!
```

**초반에 배울 수 있는 마법들:**
- 🔥 화염구 — 원거리 데미지
- ❄️ 얼음 창 — 적 둔화
- ⚡ 번개 부르기 — 강력 단타
- 💚 치유 주문 — 자신 or 아군 회복

> 💡 마법사/위치 계열 직업이 마법 효율이 높아. 하지만 다른 직업도 마법 쓸 수 있어!

**자세한 내용:** [Spell Engine 가이드 →](../spell-engine-spell-power/)

### 🌳 Puffish Skills (스킬 트리)

경험치를 모아서 **스킬 포인트**를 얻고, 캐릭터를 원하는 방향으로 성장시킬 수 있어.

**기본 작동 방식:**
```
몬스터 처치, 채굴, 제작 → 경험치 획득
경험치로 레벨업 → 스킬 포인트 획득
K 키로 스킬 트리 열기 → 포인트 투자
```

**스킬 트리 종류:**

| 트리 | 특화 |
|------|------|
| **전투** | 공격력, 방어력, HP 증가 |
| **채굴** | 채굴 속도, 행운 증가 |
| **마법** | 마나, 주문 효과 증가 |
| **생존** | 허기 효율, 낙하 저항 |

> 💡 처음엔 **전투 트리**에 집중하면 생존이 쉬워져.

**자세한 내용:** [Puffish Skills 가이드 →](../puffish-skills-skill-tree-rpg/)

## 마법 & 스킬 활용 팁

### 마나 관리
- 마법을 쓰면 마나 소모
- 마나는 시간이 지나면 회복
- Puffish 스킬 트리로 최대 마나 늘릴 수 있어

### 어떤 빌드가 좋을까?

| 플레이 스타일 | 추천 빌드 |
|--------------|-----------|
| 막 돌격하는 스타일 | Berserker + 전투 스킬 트리 |
| 멀리서 마법 쏘는 스타일 | Wizard + 마법 스킬 트리 |
| 아군 서포트 스타일 | Priest + 생존 스킬 트리 |
| 은밀하게 기습 스타일 | Rogue + 전투 스킬 트리 |
