# 4단계 — 채집 & 제작

> 자원을 체계적으로 모으고, 좋은 장비를 만드는 법을 알아보자.

---

## 바닐라 자원 수집 기초

### 중요한 광석 목록

| 광석 | 채굴 깊이 | 필요 곡괭이 | 용도 |
|------|-----------|------------|------|
| 석탄 | 어디서나 | 나무 이상 | 횃불, 연료 |
| 철 | Y 16~112 | 돌 이상 | 도구, 갑옷, 기계 |
| 구리 | Y 0~96 | 돌 이상 | Create 모드 재료 |
| 금 | Y -64~32 | 철 이상 | 레시피, 교역 |
| 레드스톤 | Y -64~16 | 철 이상 | 회로, 기계 |
| 다이아몬드 | Y -64~16 | 철 이상 | 최상위 도구, 갑옷 |
| 에메랄드 | 산 바이옴 | 철 이상 | 마을 주민 교역 |
| 흑요석 | 용암+물 접촉 | 다이아 이상 | 네더 포탈 |

> 💡 **좋은 채굴 높이:** Y=-57 부근에서 채굴하면 다이아와 레드스톤을 동시에 많이 얻을 수 있어.  
> `F3`키로 현재 좌표를 확인할 수 있어.

---

## 용광로 활용법

```
용광로 우클릭 → 열기
위쪽 칸: 제련할 재료
아래쪽 칸: 연료 (석탄, 목탄, 나무 등)
결과물이 오른쪽 칸에 나와
```

**주요 제련 레시피:**

| 재료 | 결과 |
|------|------|
| 철 원석 | 철 주괴 |
| 금 원석 | 금 주괴 |
| 모래 | 유리 |
| 나무 원목 | 목탄 |
| 날것 고기 | 익힌 고기 |
| 조약돌 | 돌 |

---

## 바닐라 장비 티어

```
나무 → 돌 → 철 → 금 → 다이아몬드 → 네더라이트
(약함)                              (강함)
```

> 💡 **금 장비는 함정!** 금 도구는 채굴이 빠르지만 내구도가 너무 낮아. 인챈트용으로만 써.

### 철 갑옷 세트 (초반 목표)

```
철 주괴 24개 필요
- 투구:   철 5개
- 흉갑:   철 8개
- 레깅스: 철 7개
- 부츠:   철 4개
```

---

## 음식 & 농사 기초

### 빠른 음식 확보
- 🐮 소, 🐷 돼지, 🐔 닭 사냥 → 날것 고기 → 용광로에 익히기
- 익힌 소고기가 허기 회복량이 가장 높아

### 밀 농사
```
1. 마을에서 씨앗 획득 or 풀 부수기 (가끔 씨앗 드롭)
2. 물 근처 흙을 괭이로 경작지 만들기
3. 씨앗 심기 → 다 자라면 좌클릭으로 수확
4. 밀 3개 → 빵 (간단한 음식)
```

## 🎮 이제 이 모드 써봐! — Silent Gear

> 철 장비에 익숙해졌으면, 이제 **훨씬 강한 커스텀 장비**를 만들 시간이야!

### Silent Gear & Silent's Gems 소개

바닐라 장비 시스템을 완전히 대체하는 모드야.  
**재료를 조합해서** 도구와 무기의 성능을 커스터마이즈할 수 있어.

```
예시:
- 날 재료: 다이아몬드 → 높은 채굴 속도
- 손잡이: 금 → 채굴 속도 보너스
- 장식재: 에메랄드 → 내구도 증가
= 세 재료가 합쳐진 고성능 곡괭이 완성!
```

**언제 시작할까?**
- 철 갑옷을 갖추고 지하 탐험이 편해졌을 때
- 다이아몬드를 10개 이상 모았을 때

**자세한 내용:** [Silent Gear 가이드 →](../silent-gear-silents-gems/)

## 제작 팁 모음

| 팁 | 설명 |
|----|------|
| **JEI 활용** | `E` 열기 → 우측 아이템 목록 → `R` 레시피, `U` 사용처 |

| **우클릭 반분** | 스택에서 우클릭 → 반만 집기 |
