# 6단계 — 전투 & 성장

> 바닐라 전투 기초를 익히고, 모드팩의 강화 시스템을 활용해 더 강해지자!

---

## 바닐라 전투 기초

### 공격 시스템
마인크래프트는 **쿨다운 시스템**이 있어.

```
공격 후 잠깐 기다려야 다음 공격이 최대 데미지 가능
→ 화면 하단 칼 아이콘이 꽉 찼을 때 공격하기
→ 무작정 빠르게 클릭하면 오히려 데미지가 낮아짐!
```

### 기본 전투 요령

| 기술 | 방법 | 효과 |
|------|------|------|
| **크리티컬** | 공중에 있을 때 (점프 후) 공격 | 데미지 1.5배 |
| **백 어택** | 뒤에서 공격 | 기습 효과 (일부 모드) |
| **방패 막기** | 방패 들고 우클릭 | 원거리 공격 차단 |
| **물러서기** | S키 + 공격 | 몬스터 공격 피하기 |

### 방패 만들기 (강력 추천!)

```
나무 판자 6개 + 철 주괴 1개 → 방패
우클릭으로 막기 → 대부분의 원거리 공격을 막아줌
```

## 바닐라 인챈트 기초

**인챈트**는 마법 부여라고도 해. 장비를 더 강하게 만들어줘.

### 인챈트 테이블 만들기

```
다이아몬드 2개 + 흑요석 4개 + 책 1개 → 인챈트 테이블
```

**주변에 책장을 놓으면** 더 높은 레벨 인챈트가 가능해:
```
책장 최대 15개 → 레벨 30 인챈트 가능 (최대치)
```

### 필수 인챈트 목록

| 인챈트 | 대상 | 효과 |
|--------|------|------|
| **날카로움 V** | 검 | 데미지 대폭 증가 |
| **보호 IV** | 갑옷 전체 | 전체 피해 감소 |
| **수리** | 모든 장비 | 경험치로 수리됨 (강력 추천!) |
| **내구성 III** | 모든 장비 | 내구도 감소 속도 낮춤 |
| **가시 III** | 갑옷 | 공격받으면 반격 데미지 |
| **효율 V** | 곡괭이 | 채굴 속도 증가 |
| **비단 발걸음** | 부츠 | 낙하 데미지 없음 |

## 🎮 이제 이 모드 써봐! — RPG 전투 시스템

> 바닐라 전투에 익숙해졌으면, 이제 모드팩의 본격 RPG 시스템을 활용할 시간이야!

### ⚔️ RPG Classes (직업 시스템)

다이아몬드 갑옷을 갖추고 나면 **직업을 선택**할 수 있어.

**6가지 직업 종류:**

| 직업 | 스타일 | 특징 |
|------|--------|------|
| **Berserker** | 근거리 돌격 | 높은 HP, 강한 공격 |
| **Rogue** | 기습, 독 | 빠른 이동, 백어택 |
| **Warrior** | 균형 전사 | 방패 + 공격 |
| **Paladin** | 성기사 | 치유 + 방어 |
| **Priest** | 서포터 | 아군 버프, 힐 |
| **Wizard** | 마법사 | 원거리 마법 딜 |
| **Witcher** | 사냥꾼 | 몬스터 특화 |

> 💡 처음엔 **Berserker나 Warrior** 추천. 직관적이고 근거리 전투 위주라 쉬워.

**자세한 내용:** [RPG Classes 가이드 →](../rpg-classes-berserker-rogues-wizards-witcher/)

### ✨ Apotheosis (강화 & 인챈트 확장)

바닐라 인챈트보다 훨씬 강력한 **Apotheosis 인챈트 시스템**이 있어.

**초반에 알아야 할 것:**
- 보스를 잡으면 **Affix 아이템** 드롭 (특수 능력 달린 장비)
- 젬을 소켓에 박아서 장비 능력치 추가 가능
- 바닐라 인챈트 레벨 제한을 돌파할 수 있어

**자세한 내용:** [Apotheosis 가이드 →](../apotheosis-apothic-enchanting-spawners/)

## 보스 공략 기초

**크리퍼 ⚠️**
```
가장 위험한 몬스터! 
→ 쉿 소리 나면 즉시 도망
→ 활로 원거리에서 처치 추천
→ 화약 드롭 → 폭탄, TNT 재료
```

**엔더맨**
```
눈 마주치면 공격해옴
→ 발 아래쪽 공격하기
→ 엔더 진주 드롭 → 엔더 포탈 재료
```
