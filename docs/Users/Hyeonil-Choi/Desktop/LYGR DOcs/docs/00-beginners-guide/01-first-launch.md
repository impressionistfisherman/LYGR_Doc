# 1단계 — 설치 & 첫 실행

> 모드팩을 처음 설치하고 게임을 시작하는 방법이야.

---

## 필요한 것들

| 항목 | 설명 |
|------|------|
| **Java** | 마인크래프트 실행에 필요 (Java 21 권장) |
| **런처** | CurseForge 또는 Modrinth App (모드팩 관리) |
| **마인크래프트 정품** | Microsoft 계정 필요 |
| **RAM** | 최소 8GB 할당 권장, 넉넉하면 10~12GB |

---

## 설치 순서

### 1. Java 설치
1. [Adoptium.net](https://adoptium.net) 에서 **Java 21** 다운로드
2. 설치 후 컴퓨터 재시작

### 2. CurseForge 앱 설치
1. [curseforge.com](https://curseforge.com) 에서 앱 다운로드 & 설치
2. 앱 실행 → Minecraft 선택

### 3. 모드팩 설치
1. CurseForge에서 모드팩 검색 or 직접 `.zip` 파일 임포트
2. 설치 완료까지 기다려 (모드 178개라 좀 걸려!)

### 4. RAM 할당 늘리기 ⚠️ 중요!
모드가 많아서 RAM을 충분히 줘야 해.

1. CurseForge → 모드팩 우클릭 → **프로필 설정**
2. **Java 설정** → 메모리 슬라이더를 **8192MB (8GB)** 이상으로
3. 저장 후 실행

> ❗ RAM이 부족하면 게임이 자꾸 튕겨! 꼭 설정해줘.

---

## 첫 실행 시 정상적인 현상들

- **로딩 시간이 길다** → 178개 모드 로딩 중, 첫 실행은 5~10분 걸릴 수 있어
- **화면에 글씨가 잔뜩 나온다** → 콘솔 로그 정상이야, 무시해
- **배경 화면이 바뀌었다** → 모드팩 커스텀 메인 메뉴 정상

---

## 새 세계 만들기

1. **싱글플레이어** 클릭
2. **새 세계 만들기** 클릭
3. 아래 설정 추천:

| 설정 | 추천값 | 이유 |
|------|--------|------|
| 세계 이름 | 원하는 이름 | - |
| 게임 모드 | **생존** | 정상 플레이 |
| 난이도 | **일반** | 처음엔 일반 추천 |
| 세계 유형 | **기본값** | - |
| 치트 허용 | **켜기** (선택) | 막히면 도움됨 |

4. **세계 만들기** → 생성까지 기다려
