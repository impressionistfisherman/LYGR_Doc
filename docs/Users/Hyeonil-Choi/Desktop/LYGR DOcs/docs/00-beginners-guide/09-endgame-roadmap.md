# 9단계 — 엔드게임 로드맵

> 여기까지 왔으면 진짜 고수야!  
> 이 모드팩의 최종 목표들을 한눈에 정리해줄게.

---

## 전체 성장 흐름 요약

```
초반 (1~3일)
  └ 첫날밤 생존 → 철 갑옷 → 기본 직업 선택

초중반 (3~7일)
  └ 다이아 장비 → 스킬 트리 투자 → Create 자동화 시작
  └ 마법 스크롤 수집 → Apotheosis Affix 아이템

중반 (1~2주)
  └ Silent Gear 커스텀 장비 → Mekanism x3 라인
  └ IDAS 중급 던전 탐험 → Cataclysm 초반 보스

중후반 (2~4주)
  └ AE2 ME 네트워크 구축 → Mekanism x5 라인
  └ Apotheosis 고급 인챈트 → RPG Classes 전직 완성

후반/엔드게임 (한 달 이상)
  └ Cataclysm 최종 보스 → 엔더 드래곤, 위더
  └ 모든 모드 통합 메가베이스 완성
```

---

## 엔드게임 목표 체크리스트

- [ ] **엔더 드래곤** 처치 (바닐라 최종 보스)

  - [ ] Ender Guardian
  - [ ] Ignis (불꽃 거인)
  - [ ] Leviathan (바다 괴물)
  - [ ] Harbinger
  - [ ] Ancient Remnant
  - [ ] Scorned Witch
  - [ ] Coral Golem

- [ ] Puffish Skills **모든 트리 최대 레벨**
- [ ] Spell Engine **전 원소 마법 습득**

- [ ] Create **대형 자동 농장 + 공장** 구축
- [ ] Mekanism **x5 광석 처리 라인** 완성
- [ ] AE2 **완전 자동 제작 네트워크** 구축
