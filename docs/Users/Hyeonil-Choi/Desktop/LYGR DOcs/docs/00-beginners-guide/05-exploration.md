# 5단계 — 탐험 & 구조물

> 세계를 돌아다니면서 구조물을 발견하고, 아이템을 얻는 법을 알아보자.

---

## 바닐라 구조물 기초

### 마을 (Village)
초반에 가장 유용한 구조물이야!

**마을에서 얻을 수 있는 것들:**
- 🌾 밀, 감자, 당근 (농사 재료)
- 🛏️ 침대 (밤 스킵 가능)
- 🏪 주민 교역 (에메랄드로 좋은 아이템 구매)
- 📦 체스트 전리품 (무기, 도구, 음식)

> 💡 **마을 주민을 죽이지 마!** 교역이 막히고, 밤에 약탈자가 습격해.

### 던전 (Dungeon)
지하에 있는 작은 방. 스포너와 체스트가 있어.

```
스포너: 몬스터를 계속 소환하는 블록
→ 횃불로 주변을 밝히면 스폰 중단
→ 체스트에 좋은 아이템 있을 수 있어
```

### 폐광 (Mineshaft)
지하에 있는 광산. 레일, 체스트, 거미 스포너가 있어.
길을 잃기 쉬우니까 횃불을 한 방향에만 설치해서 길 표시해.

### 요새 (Stronghold)
**엔더 드래곤으로 가는 엔드 포탈**이 있는 곳.
엔더 아이(Ender Eye)를 던지면 방향을 알려줘.

## 네더 가기

철 장비를 갖추면 **네더(지옥)**에 도전할 수 있어.

### 네더 포탈 만들기
```
흑요석 10개 필요 (14개면 꽉 찬 포탈)
   ███
  █   █
  █   █
   ███
→ 부싯돌과 부시로 불 붙이기
→ 보라색 공간이 생기면 들어가!
```

**흑요석 얻는 법:**
- 용암 위에 물 붓기 → 흑요석으로 굳음
- 다이아몬드 곡괭이로만 채굴 가능

### 네더에서 얻을 것들

| 아이템 | 용도 |
|--------|------|
| 네더 석영 | 장식, Observer 제작 |
| 네더 황금 | 피글린 교역 재료 |
| 블레이즈 막대 | 양조, 엔더 아이 제작 |
| 위더 스켈레톤 두개골 | 위더 소환 (보스) |
| 네더라이트 파편 | 최강 장비 업그레이드 재료 |

> ⚠️ **네더는 위험해!** 반드시 내화성 물약 가져가고, 용암 주의.

## 🎮 이제 이 모드 써봐! — 구조물 모드들

> 바닐라 구조물에 익숙해졌으면, 모드팩의 엄청난 구조물들을 탐험해봐!

### 🏰 When Dungeons Arise
바닐라보다 10배 이상 큰 던전과 성채가 세계에 생성돼.

**초반에 볼 수 있는 구조물:**
- 작은 마법사 탑, 버려진 광산, 숲 속 야영지
- 체스트에 좋은 인챈트 장비가 들어 있어

> ⚠️ 규모가 클수록 몬스터도 강해! 철 갑옷 이상 갖추고 도전해.

**자세한 내용:** [When Dungeons Arise 가이드 →](../wiki-when-dungeons-arise-and-taverns/)

### 🗺️ IDAS (Integrated Dungeons and Structures)
스토리가 있는 대형 던전들이 추가돼.

**구조물 발견 팁:**
```
/locate structure [구조물 이름]
예: /locate structure idas:catacombs
→ 가장 가까운 구조물 좌표를 알려줌
```

**자세한 내용:** [IDAS 가이드 →](../integrated-dungeons-and-structures-idas/)

## 탐험할 때 꼭 챙길 것들

### 탐험 체크리스트

- [ ] 🍖 음식 충분히 (64개 이상)
- [ ] 🕯️ 횃불 여러 개 (길 표시용)
- [ ] ⚔️ 무기 & 갑옷 (철 이상)
- [ ] 💊 물약 (치유 물약 몇 개)
- [ ] 🗺️ 지도 (현재 위치 파악)
- [ ] 📦 빈 인벤토리 공간 (전리품 담을 자리)

### 죽지 않으려면

1. **절대 아이템 꽉 찬 채로 탐험하지 마** → 죽으면 아이템 다 잃어
2. **좋은 아이템은 집에 두고 출발** → 탐험용 아이템만 챙기기
3. **침대를 스폰 포인트로 설정** → 죽어도 집에서 부활
