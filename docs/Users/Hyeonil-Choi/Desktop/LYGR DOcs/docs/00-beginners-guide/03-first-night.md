# 3단계 — 첫날밤 생존

> 세계에 스폰되면 딱 **10분** 안에 첫날밤을 버텨야 해.  
> 침착하게 이 순서대로 하면 돼!

---

## 첫날 타임라인 (10분 안에!)

### ⏱️ 0~2분: 나무 채집
스폰 근처 나무를 찾아서 **원목 16개 이상** 채집해.

```
나무 좌클릭 꾹 누르기 → 원목 채집
원목 → 인벤토리에서 나무 판자로 변환
나무 판자 4개 → 작업대 제작
```

> 💡 나무 종류는 달라도 상관없어! 색깔만 다름.

### ⏱️ 2~4분: 도구 제작
작업대를 땅에 설치하고 기본 도구를 만들어.

**제작 순서:**
1. **막대기** (나무 판자 2개를 세로로)
2. **나무 곡괭이** (판자 3개 + 막대기 2개)
3. **나무 도끼** (나무 자르기 더 빠르게)
4. **나무 검** (몬스터 방어용)

### ⏱️ 4~6분: 돌 채집 & 업그레이드
근처 돌 지형(산, 절벽, 동굴 입구)에서 **조약돌 16개** 이상 채집.

```
나무 곡괭이로 돌 채굴 → 조약돌 획득
조약돌로 → 돌 곡괭이, 돌 검, 돌 도끼 제작
```

> 💡 **도구는 재료가 좋을수록 빠르고 강해.** 돌 > 나무

### ⏱️ 6~8분: 은신처 & 횃불
밤이 오기 전에 잘 곳을 만들어야 해.

**간단한 은신처 만들기:**
```
산이나 언덕 벽면에 3×3×2 크기로 파고 들어가기
입구를 문이나 블록으로 막기
```

**횃불 제작:**
```
나무 막대기 1개 + 석탄(또는 목탄) 1개 → 횃불 4개
```

> 💡 **목탄 만드는 법:** 용광로(원석 8개로 제작)에 나무를 태우면 목탄이 나와. 석탄 대신 쓸 수 있어!

### ⏱️ 8~10분: 용광로 & 철 제련 준비

**용광로 제작:**
```
작업대에서 조약돌 8개 (가운데 빈칸) → 용광로
```

용광로를 설치하고 돌 채굴하면서 나온 **철 원석**이 있으면 제련 시작해.

---

## 첫날밤을 버티는 방법

### 방법 1: 은신처 안에서 기다리기 (추천!)
- 입구를 막고 안에서 횃불 설치 후 대기
- 낮이 될 때까지 기다리기
- 이때 **작업대, 용광로, 침대** 작업 하기

### 방법 2: 침대 구하기
양(🐑)을 찾으면 빨리 해결 가능!

```
양털 3개 + 나무 판자 3개 → 침대
침대 설치 → 우클릭으로 자기 → 밤 스킵!
```

> 💡 초원 바이옴에 양이 많아. 스폰 근처 초원을 찾아봐.

### 방법 3: 어쩔 수 없이 밖에 있다면
- **달리기**로 몬스터 피하기
- 창문 없는 집 찾아 숨기
- 나무 위로 올라가기 (좀비는 못 올라와)
- 절대 동굴 들어가지 마! (더 많은 몬스터)

## 첫날 밤 이후: 2일차 목표

| 목표 | 방법 |
|------|------|
| 🛡️ 갑옷 제작 | 철 원석 제련 → 철 갑옷 (투구, 흉갑, 레깅스, 부츠) |
| 🍖 음식 확보 | 동물(소, 돼지, 닭) 사냥 또는 밀 농사 시작 |
| ⛏️ 광석 채굴 | 지하로 내려가서 철, 석탄 채굴 |
| 🏠 기지 확장 | 임시 은신처를 좀 더 넓고 안전하게 |
