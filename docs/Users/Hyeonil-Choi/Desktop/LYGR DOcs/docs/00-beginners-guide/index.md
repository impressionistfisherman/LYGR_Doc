# 🌟 초보자 완전 가이드

> **마인크래프트가 처음이거나, 모드팩이 처음인 친구들을 위한 안내서야.**  
> 이 가이드만 따라가면 어느새 모드팩을 자유롭게 즐기고 있을 거야!

---

## 이 모드팩은 어떤 게임이야?

**LYGR 모드팩**은 기본 마인크래프트에 수백 개의 모드를 더해서 만든 **RPG 어드벤처 + 공학 + 마법** 복합 게임이야.

바닐라(기본) 마인크래프트를 알고 있으면 훨씬 쉽지만, **완전 처음이어도 이 가이드가 있으니까 걱정하지 마.**

### 이 모드팩의 큰 방향 세 가지

| 방향 | 설명 | 대표 모드 |
|------|------|-----------|
| ⚔️ **RPG 전투** | 직업, 스킬, 강화 장비로 강력한 보스 처치 | RPG Classes, Apotheosis, Cataclysm |
| ✨ **마법** | 스크롤과 마법 무기로 원소 마법 사용 | Spell Engine, Puffish Skills |
| ⚙️ **기술/자동화** | 기계로 자원을 자동으로 처리하고 기지 구축 | Create, Mekanism, AE2 |

> 💡 세 방향 모두 즐길 수 있고, 한 가지에 집중해도 재미있어!

---

## 이 가이드를 읽는 순서

```
1단계 → 설치 & 첫 실행         (처음 시작할 때)
2단계 → 기본 조작법            (조작이 헷갈리면)
3단계 → 첫날밤 생존            (게임 시작 직후)
4단계 → 채집 & 제작            (자원 모으고 도구 만들기)
5단계 → 탐험 & 구조물          (세계 돌아다니기)
6단계 → 전투 & 성장            (더 강해지기)
7단계 → 마법 & 스킬            (마법 시스템 입문)
8단계 → 기술 발전              (자동화 & 공학)
9단계 → 엔드게임 로드맵        (최종 목표)
```

> 📌 **팁:** 각 단계 끝에 "이제 이 모드 써봐!" 코너가 있어. 자연스럽게 모드팩에 적응할 수 있도록 도와줄 거야.
