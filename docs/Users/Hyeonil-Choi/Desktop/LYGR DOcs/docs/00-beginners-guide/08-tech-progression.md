# 8단계 — 기술 발전 & 자동화

> 손으로 하나하나 채굴하고 제작하는 건 한계가 있어.  
> 기계로 모든 걸 자동화해보자!

---

## 왜 자동화가 필요할까?

| 수동 | 자동화 |
|------|--------|
| 철 1개 얻는데 1~2분 | 철 100개가 알아서 제련되어 있음 |
| 레시피 일일이 손으로 제작 | 자동 제작기가 처리 |
| 인벤토리 관리 손수 | 창고 시스템이 자동 정리 |
| 탐험 중 자원 고갈 | 자원이 기지에서 계속 생산 |

---

## 🎮 이제 이 모드 써봐! — 기술 모드 입문 순서

> 처음엔 압도될 수 있어. 순서대로 하나씩 배워가면 돼!

---

### ⚙️ 1순위: Create (기계 & 발전기)

**게임 내에서 가장 직관적인 기술 모드**야.  
기어, 벨트, 축으로 연결되는 물리 기반 기계들이 진짜 돌아가는 걸 볼 수 있어!

**처음 만들어야 할 것:**
```
1. 풍차 베어링 + 돛 → 회전력(SU) 생산
2. 압착기 → 밀판, 기어 제작
3. 믹서 → 합금 재료 혼합
4. 보울트 + 체스트 → 자동 아이템 수송
```

**Create로 할 수 있는 것들:**
- 🌾 자동 농장
- ⚒️ 자동 채굴기
- 🏭 자동 제련 라인
- 🚂 기차 & 운반 시스템

> 💡 Create는 복잡해 보여도 하나씩 배우면 정말 재밌어. 처음엔 간단한 **자동 제련 라인**부터 도전해봐.

**자세한 내용:** [Create 가이드 →](../create-mod/)

---

### ⚡ 2순위: Mekanism (고급 기계)

**Create보다 더 강력한 산업용 기계 모드**야.  
철 1개를 최대 5개로 뻥튀기할 수 있어!

**첫 목표:**
```
1. 오스뮴 채굴 (Y=0~48)
2. 에너지 발전기 설치 (열 발전기)
3. 분쇄기 → 광석 2배 처리
4. 점점 기계 체인 확장
```

**Mekanism 광석 처리 티어:**

| 티어 | 배율 | 필요 기계 |
|------|------|-----------|
| 기본 제련 | x1 | 용광로 |
| 분쇄기 | x2 | Crusher |
| 정제 라인 | x3 | Enrichment + Smelter |
| 추출 라인 | x4 | Purification + 추가 |
| 완전 라인 | x5 | 전체 체인 |

> 💡 x5 라인 구축 전까지는 다이아, 에메랄드 등 귀한 광석 아껴놔!

**자세한 내용:** [Mekanism 가이드 →](../mekanism-mod/)

---

### 📦 3순위: Applied Energistics 2 (창고 & 자동 제작)

**마지막 단계의 자동화 모드**야.  
모든 아이템을 하나의 네트워크로 관리하고, 자동으로 제작까지 해줘.

**핵심 기능:**
```
ME 네트워크 = 거대한 디지털 창고
→ 아이템 수천 종류를 하나의 창고에 저장
→ 검색창에서 아이템 이름 입력 → 바로 꺼내기
→ 제작 패턴 등록 → 버튼 누르면 자동 제작!
```

> ⚠️ AE2는 처음엔 어렵게 느껴져. Create → Mekanism → AE2 순서로 배우는 걸 추천해.

**자세한 내용:** [Applied Energistics 2 가이드 →](../applied-energistics-2/)

## 기술 발전 로드맵

```
[초반] 손으로 채굴 & 제작
    ↓
[초중반] Create 풍차 + 기본 자동화
    ↓
[중반] Mekanism 발전기 + 광석 2배 처리
    ↓
[중후반] Mekanism x5 라인 + Create 대형 공장
    ↓
[후반] AE2 ME 네트워크 + 완전 자동 제작
    ↓
[엔드게임] 모든 시스템 통합 메가베이스
```

---

## 초보자 자주 하는 실수

| 실수 | 해결책 |
|------|--------|
| 에너지 없어서 기계 작동 안 함 | 발전기 먼저 설치! |
| 아이템 수송 방향 헷갈림 | 화살표 방향 잘 보기 |
| 채널 초과로 AE2 작동 안 함 | 케이블 종류 확인 |
| Create 회전 방향 불일치 | 기어박스로 방향 바꾸기 |
