# 악세서리 & 장신구 가이드

---

## 💍 Accessories (by Shedaniel)

### 개요

Curios API와 함께 **악세서리 슬롯 시스템**을 제공하는 라이브러리 모드입니다.  
이 모드 자체보다는 다른 악세서리 모드들이 이 모드를 기반으로 작동합니다.

### Accessories 슬롯 열기

```
설정 → 키 바인딩 → accessory, curios, inventory 검색
또는 인벤토리에서 별도 탭 확인
슬롯 종류: 반지, 목걸이, 벨트, 등, 발목, 얼굴 등
```

## 💎 Jewelry (RPG Series)

### 개요

RPG Series 직업 모드와 연동되는 **보석 장신구 시스템**입니다.  
보석을 가공해 직접 만들 수 있는 반지와 목걸이 등을 추가합니다.

### 보석 획득

```
주요 보석 종류:
  - Ruby (루비) — 빨강
  - Sapphire (사파이어) — 파랑
  - Emerald Gem (에메랄드 보석) — 초록
  - Topaz (토파즈) — 노랑
  - Amethyst Gem (자수정) — 보라

획득 방법:
  - 오버월드 채굴 (각 보석별 생성 Y레벨 다름)
  - 던전 상자 드롭
  - 마을 보석상 주민 거래
```

### 장신구 제작 방법

```
1. 보석 원석 → 가공 (제작대 또는 Gem Cutting Table)
   → 가공된 보석 획득

2. 가공된 보석 + 금/은 소재 → 조합
   → 반지, 목걸이 등 완성

조합법: JEI에서 "Ring", "Necklace", "Jewelry" 검색
```

### 주요 장신구 효과

**반지 계열:**
| 반지 | 주요 효과 |
|------|----------|
| Ruby Ring | 공격력 증가 |
| Sapphire Ring | 이동 속도 증가 |
| Emerald Ring | 행운 증가 |
| Topaz Ring | 화염 저항 |
| Amethyst Ring | 마법 적성 증가 |

**목걸이 계열:**
| 목걸이 | 주요 효과 |
|--------|----------|
| 각 보석 목걸이 | 해당 보석 특성의 방어적 효과 |
| 특수 목걸이 | 직업 시너지 효과 |

### 장신구 강화

일부 장신구는 **더 강한 소재**로 업그레이드가 가능합니다.

```
일반 Ruby Ring (금 + 루비)
  → 스미싱 테이블 업그레이드
  → Enhanced Ruby Ring (더 강한 공격력 보너스)
```

## ✨ Additional RPG Jewelry

### 개요

RPG Jewelry의 **확장판**으로, 더 많은 종류의 보석 장신구와  
직업 특화 악세서리를 추가합니다.

### 추가 보석 종류

기본 Jewelry 보석 외에 희귀 보석이 추가됩니다.

```
추가 보석:
  - Opal (오팔) — 무지개빛, 다목적 보너스
  - Pearl (진주) — 수중 관련 특수 효과
  - Onyx (오닉스) — 야간/어둠 환경 보너스
  - Moonstone (문스톤) — 야간 능력치 강화
```

### 직업 전용 악세서리

RPG Series 직업별 전용 악세서리가 추가됩니다.

```
예시:
  Wizard's Focus Ring: 주문 크리티컬 확률 증가
  Paladin's Holy Pendant: 치유 효과 증가
  Rogue's Shadow Earring: 은신 지속 시간 증가
  Berserker's Rage Torc: Rage 속성 강화
  Archer's Precision Charm: 원거리 피해와 명중 보조
```

## 악세서리 빌드 전략

### 최대 악세서리 슬롯 확보

```
Leather Belt (Relics): Charm 슬롯 +8
악세서리 슬롯 확장 아이템들

목표: 최대한 많은 슬롯 확보 → 더 많은 장신구 장착
```

### 직업별 추천 악세서리 조합

**근접 딜러 (Berserker/Rogue/Witcher):**
```
반지: Ruby Ring × 2 (공격력)
목걸이: Amethyst Necklace (마법 적성)
악세서리: Golden Hook (경험치) + 직업 전용 악세서리
```

**마법사 (Wizard/Elemental Wizard):**
```
반지: Amethyst Ring × 2 (마법 적성)
목걸이: 주문 강화 목걸이
악세서리: Relics (RPG) 주문 강화 트링켓
```

**원거리 딜러 (Archer):**
```
반지: Ruby Ring + Opal Ring (공격력 + 범용 보너스)
목걸이: 원거리 피해 또는 치명타 보조 목걸이
악세서리: Archer 전용 악세서리 + 이동 보조 트링켓
```

**지원/탱커 (Paladin):**
```
반지: Topaz Ring (화염 저항) + Opal Ring (다목적)
목걸이: Holy 강화 목걸이
악세서리: 체력 + 회복 관련 악세서리
```

## Curios 슬롯 관리 팁

```
슬롯 우선순위:
  1. 효과가 가장 강한 아이템 먼저 장착
  2. 비어있는 슬롯은 Relics/Artifacts로 채우기
  3. 직업 시너지 악세서리 우선

착용 확인:
  키 바인딩에서 accessory/curios 검색 → 악세서리 창에서 현재 장착 현황 확인
  아이템 툴팁에서 어떤 슬롯에 장착되는지 확인 가능
```
