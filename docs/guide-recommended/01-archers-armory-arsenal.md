# 활 & 원거리 장비 가이드

RPG Series에서 원거리 전투를 담당하는 모드들입니다.

---

## 🏹 Archers (RPG Series)

### 개요

기본 활/석궁 전투를 **RPG 스킬 시스템으로 확장**하는 모드입니다.  
Rogues의 활 특화 버전이라고 생각하면 됩니다.

### 시작 방법

```
1. 마을에서 Archery Range(사격장) 구조물 탐색
   → 내부에 활 관련 장비/화살 구매 가능 주민
   → Spell Binding Table 배치

2. Archery Manual 제작 (JEI 검색)
   → 또는 마을 가제보에서 Spell Binding Table 활용

3. 스펠북에 활 스킬 등록
```

### 핵심 기능 — 화살통 (Quiver)

활 모드의 핵심 편의 기능입니다.

| 화살통 종류 | 보관 용량 |
|------------|-----------|
| **Quiver** | 화살 4×64개 (256개) |
| **Hunting Quiver** | 볼트(크로스볼 화살) 8×64개 |
| **Battle Quiver** | 화살 12×64개 (768개) |

```
사용법:
  화살통을 Curios 슬롯에 장착
  → 인벤토리의 화살을 자동으로 사용
  → 인벤토리 화살 칸 절약
```

### 주요 활 스킬

| 스킬 | 효과 |
|------|------|
| **Power Shot** | 충전된 강력한 단일 화살 |
| **Barrage** | 연속 화살 발사 (다수 대상) |
| **Entangling Roots** | 화살이 적을 뿌리로 묶음 (이동 불가) |
| **Hunter's Mark** | 대상에게 표시 → 추가 피해 |
| **Multishot** | 한 번에 여러 화살 발사 |

### Auto-Fire Hook (자동 발사 갈고리)

```
기능: 활/석궁에 부착 → 우클릭 길게 누르면 완전 충전 상태로 자동 연사
부착: 활/석궁 + Auto-Fire Hook → 그라인드스톤으로 제거 가능
효과: 충전 대기 없이 빠른 연속 발사 가능
```

### 밸런스 변경 (바닐라 인챈트)

Archers 모드는 바닐라 활 인챈트를 재조정합니다.

```
Power 인챈트: 레벨당 +50% → +8% 데미지 (너무 강했던 것 조정)
Quick Charge: 레벨당 -20% → -10% 장전 시간
Infinity: 석궁에도 사용할 수 있습니다하게 변경
```

## 🎯 Archers Expansion (More RPG Classes)

### 개요

Archers 기본 모드를 **3가지 서브클래스**로 확장합니다.

### 3가지 서브클래스

**🗡️ Bounty Hunter (현상금 사냥꾼)**

```
특징: 빠른 연사 + 상태이상 화살
주요 스킬:
  - Toxic Arrow: 독 상태이상 화살
  - Concussive Shot: 기절 화살
  - Rapid Fire: 매우 빠른 연사
장점: 다수 적 처리, 지속 피해
```

**❄️ Tundra Archer (툰드라 궁수)**

```
특징: 냉기 화살 + 광역 빙결
영감: 리그 오브 레전드 Ashe 스킬 기반
주요 스킬:
  - Arctic Volley: 냉기 화살 다수 발사
  - Frozen Shot: 빙결 화살
  - Crystal Arrow: 광역 빙결 화살 (궁극기)
장점: 적 이동 제어, 안전한 원거리 전투
주의: 빙결 면역 적에게 효과 없음
```

**💣 War Archer (전쟁 궁수)**

```
특징: 폭발 화살 + 중장갑
주요 스킬:
  - Explosive Arrow: 착탄 시 폭발
  - Dual Shot: 동시에 2발 발사
  - Smoldering Arrow: 화염 지속 피해 화살
장점: 높은 단발 화력, 광역 피해
```

### 서브클래스 전용 방어구

각 서브클래스에 맞는 방어구 세트가 추가됩니다.  
JEI에서 해당 서브클래스 이름으로 검색하면 조합법 확인 가능.

## 👑 Armory (RPG Series)

### 개요

RPG Series 직업(Paladin, Rogue, Archer, Wizard 등)의 **최종 방어구 세트**를 추가합니다.
기존 직업 방어구를 특수 소재로 업그레이드하는 방식입니다.

### 업그레이드 방법

```
필요 소재:
  Superior Armor Upgrade — 엔드 시티 상자 또는 고대 도시 상자에서 획득
  Upgrade Crystals — 엔드게임 보스 드롭

업그레이드 과정:
  기존 직업 방어구 + Superior Armor Upgrade + Upgrade Crystals
  → 스미싱 테이블에서 조합
  → 에픽 버전 방어구 완성
```

### 직업별 에픽 방어구 효과

각 직업의 방어구는 **해당 직업 특화 보너스**가 붙습니다.

```
예시:
  Epic Wizard Robe: 주문 피해 + 주문 가속 대폭 증가
  Epic Paladin Armor: Holy Power + 방어력 동시 증가
  Epic Rogue Vest: 크리티컬 + 이동 속도 극대화
  Epic Archer Armor: 원거리 피해 + 화살 운용 강화
```

:::tip 언제 Armory를 신경 써야 하나요?
엔드 시티에 도달할 수 있는 시점, 즉 엔더 드래곤 처치 후부터  
Superior Armor Upgrade 수급이 가능해집니다.  
그 전까지는 Apotheosis Affix 장비로 충분합니다.
:::

## ⚔️ Arsenal (RPG Series)

### 개요

**제작 불가** — 오직 **보스 처치 또는 엔드게임 던전 상자**에서만 획득 가능한  
전설급 무기를 추가합니다.

### 핵심 특징

```
모든 Arsenal 무기의 공통점:
  - 조합법 없음 — 오직 드롭/상자에서만 획득
  - 모든 무기에 Spell Engine 패시브 스킬 내장
  - Minecraft Dungeons 스타일의 특수 효과
  - Tier 5 (최고 등급) 무기 태그
```

### 주요 무기 특수 효과 종류

| 효과 | 설명 |
|------|------|
| **Swirling** | 공격 시 주변 적에게 소용돌이 피해 |
| **Shockwave** | 강타 후 전방 충격파 발사 |
| **Radiance** | 타격 시 체력 회복 광역 효과 |
| **Chains** | 적을 끌어당기는 효과 |
| **Dynamo** | 연속 공격 시 데미지 누적 증가 |

### 획득 방법

```
보스 드롭:
  - 엔더 드래곤
  - 위더
  - 워든 (Warden)
  - 서드파티 모드 보스 (Cataclysm 보스 등)

던전 상자:
  - 엔드 시티 상자 (메인 수급처)
  - 기타 엔드게임 던전

루팅 인챈트 활용:
  보스 처치 시 Looting III로 드롭 확률 향상
```

:::info Arsenal 무기와 직업 연동
Arsenal 무기의 패시브 스킬은 Spell Engine 기반이므로  
현재 직업의 Spell Power 스탯이 높을수록 효과가 강해집니다.  
직업에 맞는 계열 스킬이 내장된 무기를 찾으세요.
:::
