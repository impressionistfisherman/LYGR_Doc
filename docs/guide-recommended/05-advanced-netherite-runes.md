# 장비 강화 가이드

---

## 🔱 Advanced Netherite

### 개요

기본 네더라이트를 **4단계 더 강화**할 수 있는 모드입니다.
각 단계는 고유한 특수 효과를 가집니다.

### 4가지 강화 단계

```
기본 네더라이트
  ↓ 1단계
Netherite-Iron (철 강화 네더라이트)
  ↓ 2단계
Netherite-Gold (금 강화 네더라이트)
  ↓ 3단계
Netherite-Emerald (에메랄드 강화 네더라이트)
  ↓ 4단계
Netherite-Diamond (다이아몬드 강화 네더라이트) ← 최강
```

> **주의: 단계별 순서 강제**
>
> Netherite-Diamond를 바로 만들 수 없습니다.
> 반드시 Iron → Gold → Emerald → Diamond 순서로 업그레이드해야 합니다.

### 단계별 재료

| 단계 | 필요 재료 |
|------|-----------|
| **Netherite-Iron** | 네더라이트 주괴 + 철 주괴 4개 |
| **Netherite-Gold** | Netherite-Iron 주괴 + 금 주괴 4개 |
| **Netherite-Emerald** | Netherite-Gold 주괴 + 에메랄드 4개 |
| **Netherite-Diamond** | Netherite-Emerald 주괴 + 다이아몬드 4개 |

### 강화별 특수 효과

각 단계는 갑옷 착용 시 **고유한 패시브 효과**를 부여합니다.

| 단계 | 갑옷 효과 |
|------|----------|
| **Netherite-Iron** | 팬텀이 플레이어에게 중립 (공격 안 함) |
| **Netherite-Gold** | 피글린이 플레이어에게 중립 |
| **Netherite-Emerald** | 엔더맨이 플레이어에게 중립 |
| **Netherite-Diamond** | 피글린 + 엔더맨 모두 중립 |

> **참고: 네더 탐험과 엔드 탐험이 편해집니다**
>
> Netherite-Gold 갑옷을 입으면 네더에서 피글린이 공격하지 않아
> 편하게 탐험할 수 있습니다.
> Netherite-Diamond는 피글린과 엔더맨 모두 무시해서 최고의 탐험 편의성을 제공합니다.

### 도구/무기 강화

갑옷뿐 아니라 **도구와 무기**도 동일하게 강화 가능합니다.

```
강화 가능 도구:
  곡괭이, 삽, 도끼, 괭이, 검 모두 강화 가능

도구 성능 변화:
  내구도 대폭 증가
  채굴 속도 증가
  공격력 증가

최상위 Netherite-Diamond 도끼: 공격력 13 (바닐라 네더라이트 도끼: 9)
```

### 수리 방법

각 단계의 장비는 **해당 단계의 주괴**로 수리합니다.

```
Netherite-Gold 흉갑 수리 → Netherite-Gold 주괴 사용
Netherite-Diamond 검 수리 → Netherite-Diamond 주괴 사용
```

### 진행 추천 순서

```
초반: 일반 네더라이트 장비 완성
중반: Netherite-Iron → Netherite-Gold 순차 업그레이드
      (네더 탐험 편의를 위해 Gold까지는 빠르게 올리기 추천)
후반: Netherite-Emerald → Netherite-Diamond 완성
      (자원이 많이 필요하므로 Mekanism/AE2 자원 라인 구축 후 진행)
```

## 🔮 Runes (RPG Series)

### 개요

무기나 도구에 **룬(Rune)을 각인**해 추가 효과를 부여하는 시스템입니다.
Paladins 직업 모드와 연계해 신성한 무기를 만들 수도 있습니다.

### 룬 종류

룬은 Curios 악세서리처럼 장비에 부여하는 마법 문자입니다.

```
기본 룬 종류:
  ⚔️ Attack Rune   — 공격력 증가
  🛡️ Defense Rune  — 방어력 증가
  💨 Speed Rune    — 공격/이동 속도 증가
  ✨ Holy Rune     — 성스러운 피해 추가 (언데드 특효)
  🔥 Fire Rune     — 화염 피해 추가
  ❄️ Frost Rune    — 냉기 피해 + 슬로우 추가
  ⚡ Lightning Rune — 번개 피해 추가
```

### 룬 획득 방법

```
조합: JEI에서 각 룬 이름 검색 → 직접 제작 가능
던전 상자: 일부 고급 룬은 던전에서만 획득 가능
주민 거래: 일부 주민이 룬 판매
```

### 룬 각인 방법

```
방법 1 — 직접 적용:
  룬을 손에 들고 무기/방어구에 우클릭
  → 해당 아이템에 룬 효과 각인

방법 2 — Spell Binding Table 활용:
  Spell Binding Table에 무기 + 룬 투입
  → 더 정밀한 각인 가능
```

### 주요 활용 시나리오

**언데드 사냥 특화:**
```
Silver Sword (Witcher) + Holy Rune 각인
→ 언데드/좀비/스켈레톤에게 막대한 추가 피해
→ IDAS 던전, 유령의 집 공략에 최적
```

**Paladin 특화:**
```
Holy Rune 장비 + Paladin의 Holy Power 스탯
→ 시너지로 성스러운 피해 극대화
→ 어떤 무기라도 성스러운 무기로 변환
```

**원소 마법사 보조:**
```
Fire/Frost/Lightning Rune을 무기에 각인
→ 근접 공격 시 해당 원소 추가 피해
→ 원소 마법사가 근접 공격을 하더라도 원소 피해 유지
```

### 룬 제거

```
그라인드스톤(Grindstone)에 아이템을 넣으면
각인된 룬이 제거됩니다.
(인챈트 제거와 동일한 방식)
```
