# 중세 무기 가이드

---

## ⚔️ Epic Knights

### 개요

중세 유럽 역사에서 영감을 받은 **16개 갑옷 세트**와 **26종 이상의 무기**를 추가합니다.
고증에 충실한 디자인으로, Better Combat 모드와 함께 사용하면 전투 애니메이션이 크게 향상됩니다.

### 핵심 소재 — 강철 (Steel)

Epic Knights의 모든 갑옷은 **강철(Steel)** 이 필요합니다.

```
강철 제조법:
  철 주괴 → 용광로에 넣기 → 강철 주괴
  (용광로 아무거나 사용할 수 있습니다)

주의: 다른 모드의 강철과 충돌하지 않는 독립적인 강철입니다.
```

### 갑옷 세트 목록

Epic Knights는 16개의 역사적 갑옷 세트를 추가합니다.
각 세트는 JEI에서 이름으로 검색해 조합법을 확인하세요.

| 세트 이름 | 특징 | 재료 |
|-----------|------|------|
| **Gambeson** | 천 기반 경갑옷, 낮은 방어력 | 양모, 가죽 |
| **Chain Mail** | 사슬 갑옷, 중간 방어력 | 철, 사슬 |
| **Gothic Armor** | 고딕 판금 갑옷, 높은 방어력 | 강철 |
| **Plate Armor** | 풀 플레이트, 최고 방어력 | 강철 |
| **Samurai Armor** | 일본식 갑옷, 다이 가능 | 강철 |
| **Mongolian Lamellar** | 몽골 갑옷, 높은 방어력 | 강철 |
| **Jousting Armor** | 마상창시합용, 이동 속도 페널티 | 강철 |

> **정보: 갑옷 성능 비교**
>
> 방어력 순서 (낮음 → 높음):
> Gambeson < Chain Mail < Gothic < Plate < Jousting (최고 방어력 + 이동 페널티)

### 갑옷 커스터마이징

일부 갑옷은 **배너 패턴 적용**과 **염색**이 가능합니다.

```
배너 적용: 갑옷 + 배너 → 조합
염색 가능: 일부 갑옷 (Samurai 등)은 염료로 색상 변경할 수 있습니다
서코트/망토: 부츠 슬롯에 장착하는 장식용 의상 아이템
```

### 무기 종류 (26종)

**검 계열:**
- Longsword (장검) — 긴 공격 범위
- Rapier (레이피어) — 빠른 공격, 낮은 피해
- Saber (세이버) — 균형형

**폴암 계열:**
- Spear (창) — 긴 사거리
- Halberd (할버드) — 높은 피해 + 방패 무력화
- Glaive (글레이브) — 범위 공격

**둔기 계열:**
- Mace (메이스) — 방어구 관통
- Warhammer (전투 망치) — 매우 높은 피해, 느린 속도

**특수:**
- Lance (랜스) — 기마 전투 특화 (+100% 마운트 피해)
- Longbow / Heavy Crossbow — 강화 원거리 무기

## 🗡️ Spartan Weaponry (Unofficial)

### 개요

30종 이상의 무기를 추가하는 모드입니다.
각 무기에는 **전투 특성(Weapon Trait)** 이 있어 단순히 무기를 더하는 것을 넘어
전술적인 전투 선택이 가능합니다.

> **정보: 비공식 포트**
>
> 이 모드팩에 포함된 Spartan Weaponry는 커뮤니티가 1.21.1용으로 비공식 포팅한 버전입니다.

### 기본 제작 재료

모든 Spartan Weaponry 무기는 기본 재료가 필요합니다.

```
Handle (손잡이):
  스틱 + 실 → Handle 1개
  또는 양모/가죽 4개 + 스틱 4개 → Handle 4개

Pole (장대):
  스틱 3개 + 실 1개 (세로 배치)

이후 Handle/Pole + 원하는 재료 조합으로 무기 제작
```

### 무기 특성(Trait) 전체 목록

Spartan Weaponry의 핵심은 각 무기의 고유 특성입니다.

| 특성 | 효과 | 해당 무기 |
|------|------|-----------|
| **Sweep I** | 범위 공격 (1 피해) | 대검, 창, 할버드 |
| **Sweep II** | 범위 공격 (절반 피해) | 장검, 세이버, 글레이브 |
| **Sweep III** | 범위 공격 (전체 피해) | 전투 도끼 |
| **Two-Handed I** | 양손 무기 (강타 공격) | 대검, 창, 할버드, 랜스 |
| **Two-Handed II** | 더 강한 양손 | 파이크 |
| **Knockback** | 적을 멀리 밀쳐냄 | 글레이브, 쿼터스태프 |
| **Shield Breach** | 방패 일시 무력화 | 할버드, 도끼계 |
| **Riding Damage Bonus** | 마운트 탑승 시 +100% 피해 | 랜스 |
| **Nausea** | 피격 시 구역질 효과 | 망치류 |

### 무기별 전투 추천 상황

| 무기 | 추천 상황 |
|------|-----------|
| **Dagger** | 빠른 연속 공격, 높은 DPS |
| **Longsword** | 범용, 범위 공격 |
| **Greatsword** | 다수 몹 처리 (Sweep III) |
| **Spear** | 안전한 거리에서 공격 |
| **Halberd** | 방패 들고 있는 적 상대 |
| **Pike** | 복도/좁은 공간 방어 |
| **Lance** | 말 탑승 시 보스 전투 |
| **Glaive** | 여러 적에게 동시 피해 |
| **Warhammer** | 높은 단발 피해, 방어구 관통 |

### 전용 인챈트

Spartan Weaponry는 특수 인챈트를 추가합니다.

| 인챈트 | 효과 | 대상 |
|--------|------|------|
| **Propel** | 투척 무기 속도/거리 증가 | 투척 무기 |
| **Incendiary** | 투척 무기에 화염 효과 | 투척 무기 |
| **Razor's Edge** | 투척 무기 데미지 증가 | 투척 무기 |

### 투척 무기 (Throwing Weapons)

일부 무기는 투척이 가능합니다.

```
투척 가능 무기:
  - Throwing Knife (투척 단검)
  - Javelin (투창)
  - Boomerang (부메랑) — 되돌아옴

사용법: 우클릭으로 투척
         Propel 인챈트로 속도/거리 향상
```

## 두 모드 조합 추천

```
전사형 빌드:
  Gothic/Plate Armor (Epic Knights) + Halberd/Greatsword (Spartan Weaponry)
  → 강한 방어 + 범위 공격

기마형 빌드:
  Plate Armor (Epic Knights) + Lance (Spartan Weaponry)
  → 말 탑승 시 +100% 피해 극대화

원거리 혼합:
  Chain Mail (Epic Knights) + Javelin (Spartan Weaponry)
  → 가벼운 갑옷 + 원거리 투척
```
