## 🏺 Better Archaeology

### 개요

바닐라의 **고고학 시스템을 대폭 확장**합니다.  
더 많은 발굴 위치, 더 다양한 유물, 새로운 구조물이 추가됩니다.

### 핵심 기능 — Suspicious 블록 발굴

```
Suspicious Sand (의심스러운 모래)
Suspicious Gravel (의심스러운 자갈)
→ 솔로 브러시(Brush)로 발굴

새로 추가된 Suspicious Dirt, Clay 등도 발굴 가능
```

### 새로운 발굴 가능 위치

| 위치 | 드롭 특징 |
|------|-----------|
| **고대 사원 (새로운 구조물)** | 고대 유물, 희귀 도자기 파편 |
| **사막 우물 주변** | 이집트 테마 유물 |
| **강가 발굴지** | 낚시/수중 관련 유물 |
| **고대 묘지** | 언데드 테마 유물 |

### 새로운 도자기 파편 & 완성품

```
발굴 → 도자기 파편 수집
파편 4개 조합 → 장식용 항아리 완성

Better Archaeology 전용 파편:
  각 파편에 독특한 고대 문양이 새겨져 있음
  → 베이스 장식에 활용할 수 있습니다
```

### 고고학 특화 장비

```
Better Brush: 향상된 솔로 브러시
  → 발굴 속도 증가
  → 내구도 높음

JEI에서 "Better" 또는 "Archaeology" 검색으로 관련 아이템 확인
```

## 🌍 Regions Unexplored

### 개요

**수십 가지 새로운 바이옴**을 추가해 세계 생성을 풍부하게 만드는 모드입니다.  
각 바이옴에는 고유한 블록, 식물, 나무, 가끔 특수 구조물이 있습니다.

### 주요 바이옴 분류

**숲/수풀 계열:**
```
Blossom Forest — 벚꽃 나무 가득한 분홍빛 숲
Prismatic Forest — 다색 나뭇잎 숲
Maple Forest — 단풍나무 숲 (가을 분위기)
```

**사막/건조 계열:**
```
Scorched Badlands — 극단적으로 건조한 황무지
Dead Forest — 죽은 나무들의 숲
```

**설원/동토 계열:**
```
Frozen Wasteland — 극한의 얼음 황야
Boreal Forest — 한랭 침엽수 숲
```

**특수 계열:**
```
Glowing Caves — 빛나는 식물이 있는 동굴 바이옴
Crystal Cliffs — 거대 크리스탈 절벽
```

### 탐험 가치

```
각 바이옴 고유 특산물:
  - 고유 나무 종류 → 독특한 목재/판자
  - 고유 식물 → 제조/장식 재료
  - 일부 바이옴 전용 광석 (희귀)

바이옴 탐색 팁:
  Nature's Compass로 원하는 바이옴 방향 확인
  Xaero's World Map으로 발견한 바이옴 위치 기록
```

## 🗺️ Explorify

### 개요

다양한 **새로운 구조물**을 오버월드에 추가합니다.  
각 구조물에는 고유한 루트와 몬스터가 있습니다.

### 주요 추가 구조물

| 구조물 | 특징 |
|--------|------|
| **Abandoned Witch Hut** | 폐허 마녀 오두막, 포션 특화 루트 |
| **Mossy Cave** | 이끼 낀 동굴, 특수 돌 블록 |
| **Ruins** | 다양한 폐허 변형 |
| **Underground Ruins** | 지하 유적 |
| **Fallen Log** | 거대 쓰러진 나무 |
| **Desert Pyramid Variants** | 사막 피라미드 변형 |

### 탐험 팁

```
Explorers Compass로 특정 구조물 검색:
  우클릭 → "Explorify" 또는 구조물 이름 검색
  → 가장 가까운 방향 표시

구조물 내부 확인 사항:
  - 모든 상자 열기
  - 숨겨진 방/통로 탐색
  - 특수 블록(Suspicious Sand 등) 발굴
```

## ⚔️ Illager Invasion

### 개요

**강화된 습격(Raid) 시스템**을 추가합니다.  
기본 습격보다 훨씬 다양하고 강한 Illager들이 공격합니다.

### 새로운 습격 웨이브

```
기본 바닐라 습격보다 다음이 추가됩니다:

강화된 Illager 종류:
  - 중갑옷 착용 Pillager (높은 방어력)
  - 마법 사용 Illager (Spell Engine 주문 사용)
  - 특수 대장(Captain) Illager (강력한 미니보스)

웨이브 구성:
  초반 웨이브: 기본 Illager
  중간 웨이브: 강화 Illager
  최종 웨이브: 대장 + 에보커
```

### 습격 대비법

```
습격 시작 전:
  [ ] 마을 외곽에 방어 구조물 설치 권장
  [ ] 충분한 포션 비축
  [ ] Apotheosis Rare 이상 장비 착용
  [ ] 마을 주민들 실내로 유도

전투 전략:
  - 마을 입구를 좁혀 적이 한 번에 몰려오지 않게
  - Illager 마법사는 우선 처치
  - 에보커의 소환 팬텀은 즉시 처치 필요
```

### 습격 보상

```
기본 습격 보상:
  - 마을 주민 거래 할인 (영웅의 귀환 효과)
  - 에메랄드
  - Totem of Undying (에보커 드롭)

Illager Invasion 추가 보상:
  - 강화 Illager 전용 드롭 아이템
  - 강화 인챈트 장비
  - 경험치 대량 획득
```

## 탐험 효율 극대화 팁

```
탐험 필수 장비:
  [ ] Xaero's World Map — 탐험 지역 기록
  [ ] Xaero's Minimap — 현재 위치/방향
  [ ] Explorer's Compass — 구조물 탐색
  [ ] Nature's Compass — 바이옴 탐색
  [ ] Waystones — 귀환 포인트 설정

탐험 순서 권장:
  1. 초반: 가까운 마을/구조물 탐색 → 기본 자원 확보
  2. 중반: Regions Unexplored 특수 바이옴 탐색 → 고유 블록/재료 수집
  3. 후반: 깊은 탐험 → Explorify 구조물 + Better Archaeology 발굴
```
