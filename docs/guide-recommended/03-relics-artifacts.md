# 유물 & 아티팩트 가이드

이 모드팩에는 두 종류의 "유물" 모드가 포함되어 있습니다.

| 모드 | 성격 | 슬롯 |
|------|------|------|
| **Relics** (SSKirillSS) | 복잡한 메커니즘, 레벨업 시스템 | Curios 슬롯 |
| **Relics (RPG Series)** | 직업 시너지 특화 트링켓 40+ | Curios 슬롯 |
| **Artifacts** | 독특한 패시브 효과 | Curios 슬롯 |

---

## 💎 Relics (SSKirillSS)

### 개요

단순한 아이템 추가를 넘어서 **레벨업 시스템**과 **랜덤 스탯**을 가진 복잡한 유물 시스템입니다.  
같은 종류의 유물이라도 레벨과 스탯이 달라 수집 가치가 있습니다.

### 핵심 특징

```
레벨업 시스템:
  유물을 Curios 슬롯에 장착한 상태에서 사용/전투 → 경험치 축적
  → 레벨업 → 효과 강화

랜덤 스탯:
  같은 유물이라도 드롭될 때마다 세부 수치가 다름
  → 더 좋은 스탯의 유물을 찾을 이유 생김

자동 밸런스:
  모드팩의 체력/데미지 수치에 자동으로 맞춰 조정됨
  → 바닐라부터 고수치 모드팩까지 항상 유용
```

### Curios 슬롯 열기

```
기본 단축키: G
→ 반지, 목걸이, 벨트, 등 슬롯 등 표시
→ 해당 슬롯에 맞는 유물 장착
```

### 유물 획득 방법

유물은 **특정 위치의 상자**에서만 드롭됩니다.

```
유물별 획득 위치:
  - 버려진 광산 상자 전용 유물
  - 사막 피라미드 전용 유물
  - 정글 신전 전용 유물
  - 마을 농부/어부/사냥꾼 집 상자 유물
  - 요새 상자 유물
  - 엔드 시티 유물 (고레벨)

탐험을 많이 할수록 다양한 유물을 수집 가능합니다.
```

### 주요 유물 효과 예시

| 유물 이름 | 효과 |
|-----------|------|
| **Magma Walker** | 뜨거운 블록 면역 + 일시적 용암 위 걷기 |
| **Phantom Boot** | 발 아래 임시 블록 생성 (공중 이동할 수 있습니다) |
| **Ender Hand** | 타겟과 위치 교체 (텔레포트) |
| **Firefly Queen** | 어두운 곳에서 임시 광원 자동 생성 |
| **Cicada in a Bottle** | 피격 시 주변 적이 공격 대상을 다른 몹으로 전환 |
| **Leather Belt** | Curios 슬롯 최대 +8개 추가 |

### 유물 조합

일부 유물은 다른 유물과 **조합**해 더 강한 유물을 만들 수 있습니다.

```
조합 방법: JEI에서 유물 이름 검색 → 조합 레시피 확인
결과: 두 유물의 효과를 합친 강화 버전
```

## 🎖️ Relics (RPG Series)

### 개요

RPG Series 직업 모드와 **시너지**가 있는 트링켓 40+종을 추가합니다.  
주문 강화, 전투 스킬 보조 등 직업 플레이를 극대화합니다.

### 획득 방법

```
주요 획득처:
  - 던전 상자 (대부분)
  - 에보커(Evoker) 처치 드롭
  - 위더 처치 드롭
  - 엔더 드래곤 처치 드롭
  - 일부는 보석 재료로 조합 가능
```

### 주요 트링켓 효과 분류

**마법 강화 계열:**
```
주문 피해 증가 트링켓
주문 가속 증가 트링켓
특정 계열(비전/화염/냉기 등) 강화 트링켓
```

**전투 계열:**
```
크리티컬 관련 트링켓
공격 속도 보너스 트링켓
백스탭 피해 트링켓 (Rogue 전용 시너지)
```

**방어/생존 계열:**
```
피격 시 효과 (반사, 회피 등)
체력 회복 관련 트링켓
```

## 🏺 Artifacts

### 개요

**독특하고 창의적인 패시브 효과**를 가진 아이템들을 추가합니다.  
Relics와 다르게 레벨업 시스템 없이 장착만 해도 즉시 효과 발동.

### 주요 아티팩트 목록

| 아이템 | 획득 방법 | 효과 |
|--------|-----------|------|
| **Golden Hook** | 몹 상자 드롭 | 처치 시 획득 경험치 증가 |
| **Umbrella** | 상자 드롭 | 낙하 속도 감소 (우산처럼) |
| **Digging Claws** | 광부 상자 | 맨손 채굴 속도 대폭 증가 |
| **Scarf of Invisibility** | 던전 상자 | 장착 시 가끔 투명화 발동 |
| **Obsidian Skull** | 용광로 상자 | 화염/용암 면역 |
| **Crystal Heart** | 던전 상자 | 최대 체력 영구 +2 |
| **Aglet** | 상자 드롭 | 이동 속도 증가 |
| **Flippers** | 물가 상자 | 수중 이동 속도 대폭 증가 |
| **Pocket Piston** | 상자 드롭 | 밀치기 공격 강화 |

### Artifacts + Relics 시너지

Relics 모드와 Artifacts를 함께 사용하면  
Artifacts 아이템에도 레벨업 시스템이 적용됩니다.

```
효과:
  Artifacts 아이템 사용/전투 → 경험치 → 레벨업 → 효과 강화
  → Artifacts 수집 및 성장의 재미가 추가됨
```

## Curios 슬롯 최대화 팁

유물/아티팩트를 많이 장착하려면 슬롯을 늘려야 합니다.

```
Leather Belt (Relics): Charm 슬롯 최대 +8개
Accessories 모드: 추가 악세서리 슬롯 제공

슬롯 종류별 아이템 장착:
  Ring 슬롯: 반지 계열
  Necklace 슬롯: 목걸이 계열
  Belt 슬롯: 허리띠 계열
  Back 슬롯: 배낭/망토 계열
  Charm 슬롯: 부적 계열
```
