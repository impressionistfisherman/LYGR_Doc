# 강화 몬스터 가이드

---

## 🧟 Mutant Monsters

### 개요

바닐라 몬스터의 **거대하고 강력한 뮤턴트 버전**을 추가합니다.  
각 뮤턴트는 일반 몬스터의 미니보스급으로 처치 시 고유 아이템을 드롭합니다.

### 스폰 방법

```
자연 스폰:
  일반 몬스터와 같은 환경에서 낮은 확률로 자연 스폰
  어두운 동굴, 밤 오버월드 등

Chemical X 사용:
  Chemical X를 일반 몬스터에게 던지기
  → 해당 몬스터의 뮤턴트 버전으로 변이
  (단, Spider Pig는 발효된 거미 눈을 먹은 돼지에게 Chemical X 사용)
```

### Chemical X 제작

```
JEI에서 "Chemical X" 검색 → 조합법 확인
(포션 유사 아이템으로 던지면 몬스터에게 적용)
```

### 🧟‍♂️ Mutant Zombie (뮤턴트 좀비)

| 항목 | 내용 |
|------|------|
| **체력** | 150 HP |
| **주요 공격** | 강타(지면 충격), 투척(공중으로 띄움) |
| **특수 능력** | 쓰러진 후 5틱 내에 불로 태우지 않으면 40HP로 부활 |

**공략:**
```
1. 강타 공격은 지면 충격파가 발생 → 옆으로 회피
2. 투척 공격 후 공중에서 추가 피해 주의
3. 쓰러지면 즉시 불 도구로 시체 점화 → 완전 처치
   (라이터 또는 화염 화살로 불 붙이기)
```

**드롭:**
- **Hulk Hammer** — 땅에 강타 시 주변 적 모두 넉백하는 특수 망치

### 🏹 Mutant Skeleton (뮤턴트 스켈레톤)

| 항목 | 내용 |
|------|------|
| **체력** | 150 HP |
| **주요 공격** | 정확하고 빠른 화살, 원거리 타격 |
| **특징** | 죽으면 부위별로 분해되어 흩어짐 |

**공략:**
```
1. 거리 유지가 오히려 위험 — 원거리 화살 정확도가 높음
2. 근접 접근 후 방패로 화살 차단하면서 공격
3. 죽으면 뼈 조각들이 흩어지며 일부 공격
   → 흩어진 조각들 수집 필수 (뼈 조각 드롭)
```

**드롭:**
- **Mutant Skeleton Bones** — 다수 드롭
- **Mutant Skeleton Armor** — 뼈 조각으로 조합하는 특수 방어구 세트

### 💥 Mutant Creeper (뮤턴트 크리퍼)

| 항목 | 내용 |
|------|------|
| **주요 공격** | 연쇄 폭발 (일반 크리퍼보다 훨씬 강력) |
| **특수 능력** | 죽으면 Creeper Minion Egg 드롭 |

**공략:**
```
1. 폭발 범위가 매우 넓음 → 항상 최대 거리 유지
2. 원거리 공격만으로 처치 권장
3. 폭발 피해 감소 포션 미리 음용
```

**드롭:**
- **Creeper Minion Egg** — 폭발 시 Creeper Shard 획득 가능
- **Creeper Minion Tracker** — 크리퍼 미니언 조종 도구

**Creeper Minion:**
```
Creeper Minion Egg를 땅에 던지면 → 아기 크리퍼 소환
→ 플레이어 따라다니며 적에게 폭발 공격
→ Creeper Minion Tracker로 추적/제어
```

### 🌀 Mutant Enderman (뮤턴트 엔더맨)

| 항목 | 내용 |
|------|------|
| **주요 공격** | 텔레포트 순간이동 + 블록 투척, 지형 조작 |
| **특수 능력** | 엔더 마법으로 지형 변형 |
| **드롭** | Endersoul Hand (확률 드롭) |

**공략:**
```
1. 텔레포트로 위치 예측이 어려움
2. 블록 투척은 방패로 차단 가능
3. 물이 있는 환경에서 싸우면 이동 제한 가능
4. 멀리서 화살 공격 후 텔레포트하면 따라가지 말고 제자리 공격
```

**Endersoul Hand:**
```
기능: 손에 들고 블록/적에게 우클릭
  → 블록: 집어들었다 원하는 위치에 배치 (블록 이동)
  → 적: 적과 위치를 바꾸는 텔레포트
활용: 탐험 시 블록 이동, 적 위치 교란
```

### ☃️ Mutant Snow Golem (뮤턴트 눈 골렘) — 우호적

```

→ 플레이어 주변에서 적을 향해 거대 얼음 블록 투척
→ 자연 스폰하거나 일반 눈 골렘에게 Chemical X 사용
```

### 🕷️ Spider Pig (거미 돼지)

```
생성 방법:
  돼지에게 발효된 거미 눈 먹이기
  → 그 돼지에게 Chemical X 사용
  → Spider Pig 탄생

특성:
  평소에는 중립 (공격 안 함)
  화나면 거미줄 + 도약 공격
  천장/벽 이동 가능 (거미처럼)
```

## 🔥 Infernal Mobs

### 개요

일반 몬스터에게 **무작위 특수 능력**이 붙어 스폰되는 시스템입니다.  
이름에 접두사가 붙어 어떤 능력을 가졌는지 힌트를 줍니다.

### 접두사별 능력 예시

| 접두사 | 능력 |
|--------|------|
| **Venomous** | 독 공격 |
| **Fiery** | 화염 공격 |
| **Chilling** | 슬로우/냉기 공격 |
| **Warp** | 순간이동 |
| **Vampire** | 피격 시 플레이어 체력 흡수 |
| **Ghastly** | 가스트 화염구 발사 |
| **Berserking** | 분노 시 공격력/속도 급상승 |
| **Ninja** | 투명화 + 빠른 이동 |
| **Bulwark** | 높은 방어력, 거의 무적에 가까움 |
| **Regen** | 체력 빠르게 재생 |

### Infernal Mobs 주의사항

```
❌ 이름에 특수 능력이 붙은 몬스터를 발견하면:
  1. 어떤 접두사인지 먼저 확인
  2. Bulwark → 원거리/마법 공격으로 우회
  3. Vampire → 체력 빠르게 내려가므로 회복 포션 준비
  4. Warp → 텔레포트하므로 원거리보단 근접 권장
  5. Ninja → 투명화 상태에서 갑자기 공격 — 넓은 공간에서 전투

✅ 처치 보상:
  Infernal Mobs 처치 시 추가 경험치 + 더 좋은 아이템 드롭
```

### Infernal Mobs 드롭 활용

```
Infernal 몬스터는 일반 몬스터보다 더 좋은 드롭을 줍니다:
  - 경험치 대폭 증가
  - 인챈트된 장비 드롭 확률 증가
  - 희귀 아이템 드롭 가능

위험하지만 도전할 가치가 있습니다!
```
