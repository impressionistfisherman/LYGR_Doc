# 직업 선택 가이드

이 모드팩에는 여러 RPG 직업 모드가 포함되어 있습니다.
전투 스킬은 Spell Engine 계열 API와 연동되는 경우가 많고, 성장 요소는 **puffish_skills / skill_tree**와 함께 사용됩니다.

---

## 직업 모드 계열 구분

직업 모드는 크게 두 계열로 나뉩니다.

### RPG Series (daedelus_dev 제작)
- Wizards (위자드)
- Rogues & Warriors (로그 & 워리어)
- Paladins & Priests (팔라딘 & 프리스트)
- Archers (아처) → 이 모드팩에 포함

### More RPG Classes (Fichteee 제작)
- Berserker (버서커)
- Elemental Wizards (원소 마법사)
- Forcemaster (포스마스터)
- Witcher (위처)

두 계열은 같은 전투/스킬 생태계 안에서 함께 사용할 수 있도록 구성되어 있습니다.

## 직업 시작 방법

직업마다 시작 흐름이 조금 다릅니다. 마법/주문 직업은 스펠북과 스크롤 관리가 중요하고, 무기 중심 직업은 직업 아이템과 전용 무기 세팅이 먼저입니다.

```
1. 직업 전용 아이템 또는 구조물 확인
   → 마을 구조물, 직업 관련 NPC, JEI 조합법을 먼저 확인

2. 직업 선택 후 전용 장비 방향 정하기
   → Archers는 활/석궁, Warriors는 방패와 근접 무기처럼 주력 장비가 다릅니다

3. 주문 직업이라면 Spell Binding Table 준비
   → 스펠북과 스크롤을 등록해 사용할 주문을 구성

4. 무기 중심 직업이라면 전용 무기와 패시브/액티브 스킬 확인
   → 스킬 트리와 장비 옵션을 먼저 맞추는 편이 효율적입니다

5. `Controls`에서 `skill` 또는 `puffish`를 검색해 스킬 창 키를 확인한 뒤 직업 전용 스킬 트리 확인
```

## 공통 자원 — 룬 (Rune)

Wizards 계열과 일부 주문 직업은 주문 시전에 **룬(Rune)** 이 필요할 수 있습니다.
실제 소모 여부와 룬 종류는 주문 툴팁, 스펠북 툴팁, JEI를 우선해서 확인하세요.

```
룬 조합법: JEI에서 "Rune" 검색
종류: Arcane Rune, Fire Rune, Frost Rune, Holy Rune 등
     직업과 주문 계열에 맞는 룬을 준비하세요
```

## 직업 전환

직업 전환 방식은 모드와 서버 설정에 따라 달라질 수 있습니다.
다른 직업 아이템을 장착하기 전에 기존 스킬 트리 포인트, 장착 조건, 리셋 여부를 먼저 확인하세요.
