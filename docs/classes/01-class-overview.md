# 직업 선택 가이드

이 모드팩에는 **7개의 직업 모드**가 포함되어 있습니다.  
모든 직업은 **Spell Engine**을 기반으로 작동하며, **puffish_skills / skill_tree**와 연동됩니다.

---

## 직업 모드 계열 구분

직업 모드는 크게 두 계열로 나뉩니다.

### RPG Series (daedelus_dev 제작)
- Wizards (위자드)
- Rogues & Warriors (로그 & 워리어)
- Paladins & Priests (팔라딘 & 프리스트)
- Archers (아처) → 이 모드팩에 포함

### More RPG Classes (Fichteee 제작)
- Berserker (버서커)
- Elemental Wizards (원소 마법사)
- Forcemaster (포스마스터)
- Witcher (위처)

두 계열 모두 Spell Engine API를 사용해 **완벽히 호환**됩니다.

## 직업 시작 방법 (공통)

모든 직업의 시작 방법은 동일합니다.

```
1. Spell Binding Table 제작 (JEI 검색)
   주변에 책장 배치 필수

2. 직업 전용 아이템 획득
   → 마을 구조물(가제보, 위자드 타워, 성소 등)에서 구매
   → 또는 JEI에서 조합법 확인 후 직접 제작

3. Spell Book에 스크롤 등록
   (Spell Binding Table에 스크롤 + 책 투입)

4. 스펠북을 인벤토리 전용 슬롯에 장착
   (일부 직업은 손에 들어야 함)

5. 스킬 창(K)에서 직업 전용 스킬 트리 확인
```

---

## 공통 자원 — 룬 (Rune)

Wizards 계열과 일부 직업은 주문 시전에 **룬(Rune)** 을 소모합니다.  
룬은 화살처럼 소모품이므로 항상 인벤토리에 여유분을 유지하세요.

```
룬 조합법: JEI에서 "Rune" 검색
종류: Arcane Rune, Fire Rune, Frost Rune, Holy Rune 등
     직업에 맞는 룬을 대량으로 준비하세요
```

## 직업 전환

직업을 바꾸고 싶다면 언제든지 다른 직업의 아이템을 제작해 장착하면 됩니다.  
단, 이전 직업의 **스킬 트리 포인트는 초기화**되며 새 직업의 트리를 처음부터 쌓아야 합니다.
