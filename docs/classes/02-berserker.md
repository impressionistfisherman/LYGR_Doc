# Berserker (버서커)

> *"체력이 낮을수록 강해진다. 죽기 직전이 가장 강하다."*

---

## 직업 개요

버서커는 **분노(Rage) 속성**을 핵심으로 하는 근접 딜러입니다.
체력이 낮아질수록 공격력이 상승하는 독특한 메커니즘 덕분에,
위험 상태를 유지하면서 극한의 데미지를 뽑아내는 스타일입니다.

| 항목 | 내용 |
|------|------|
| **주 무기** | 도끼 계열 (버서커 전용 도끼 포함) |
| **핵심 자원** | 분노(Rage) 스택 |
| **주요 스탯** | 근접 공격력, 크리티컬, Rage 속성 |
| **장비** | 버서커 전용 방어구 세트 |
| **난이도** | ★★★★☆ (체력 관리가 어려움) |

---

## 핵심 메커니즘 — Rage (분노)

### Rage 속성이란?

버서커 장비의 Rage 속성은 **현재 체력이 낮을수록 공격력을 증폭**시킵니다.

```
공식 (근사값):
  최대 체력에 가까울 때: 데미지 보너스 낮음
  체력 50% 이하: 데미지 보너스 상승
  체력 20% 이하: 데미지 보너스 최대
```

### Rage 버프 (상태 이상)

특정 스킬 사용 시 **Rage 버프**가 부여됩니다.
Rage 버프 레벨이 높을수록 크리티컬 확률과 데미지가 추가 상승합니다.

> **주의: 저체력 플레이는 위험합니다**
>
> 버서커의 강함은 체력이 낮을 때 나옵니다.
> 하지만 체력이 낮으면 당연히 쉽게 죽습니다.
> **Blood Reckoning 스킬**로 적시에 체력을 보충하는 것이 버서커 플레이의 핵심입니다.

## 핵심 스킬

### Wild Rage (와일드 레이지)

```
효과: Rage 버프 즉시 활성화
      Rage 스택이 높을수록 크리티컬 확률과 데미지 상승
용도: 전투 시작 or 피격 직후 즉시 사용
```

### Bloody Strike (블러디 스트라이크)

```
효과: 강력한 근접 일격
      체력을 소모하고 흡수 하트(Absorption) 부여
      Rage와 시너지: 저체력일수록 피해 증가
용도: 주력 딜 스킬, Rage 버프 중 사용 시 극대화
```

### Blood Reckoning (블러드 레크닝)

```
효과: 흡수 하트를 실제 체력으로 변환 + 치유
      현재 Rage 버프 지속 시간 2배 연장
용도: 저체력 상태에서 생존 유지
      Bloody Strike → Blood Reckoning 콤보가 핵심
```

### Outrage (아웃레이지)

```
효과: 일부 부정적 효과 해제 또는 전투 회복 보조
      Rage 상태와 연계해 생존 시간을 늘리는 용도
용도: 독/슬로우 등 디버프 대응
      CC기 해제 후 반격 시 사용
```

## 버서커 전용 장비

### 버서커 도끼

버서커 전용 도끼 무기가 추가됩니다.

- **Rage 속성** 내장 → 저체력 데미지 보너스 활성화
- 일반 도끼보다 더 높은 기본 데미지
- 일부 도끼는 조합보다 던전 루트나 전리품으로 얻는 편이 빠를 수 있음

> **참고: 던전에서 더 좋은 도끼를 찾으세요**
>
> 버서커용 고급 도끼는 JEI 조합법과 던전 전리품을 함께 확인하세요.
> IDAS 던전과 일반 던전을 탐험하며 Rage 속성 도끼를 수집하세요.

### 버서커 방어구 세트

| 부위 | 효과 |
|------|------|
| 헬멧 | Rage 속성 + 받는 피해 소폭 감소 |
| 흉갑 | Rage 속성 + 체력 증가 |
| 레깅스 | Rage 속성 + 이동 속도 |
| 부츠 | Rage 속성 + 낙하 피해 감소 |

## 스킬 트리 방향

버서커의 skill_tree 직업 트리는 두 방향으로 갈립니다.

### 공격 특화 (Fury Path)

```
- 크리티컬 확률 + 크리티컬 데미지 극대화
- Rage 스택 빠르게 쌓는 패시브
- 저체력 데미지 보너스 강화
- Bloody Strike 강화 노드
```

### 생존 특화 (Endurance Path)

```
- 체력 + 방어력 강화
- Blood Reckoning 치유량 증가
- 피격 시 Rage 스택 더 빠르게 쌓기
- 반격 패시브 노드
```

초보 버서커는 **생존 특화**로 시작해 버서커 플레이에 익숙해진 후 공격 특화로 전환 권장.

## 전투 기본 루프

```
1. Wild Rage → Rage 버프 활성화
2. 근접 전투 → 체력 소모
3. 체력 50% 이하 → Bloody Strike (흡수 하트 확보)
4. 흡수 하트 생겼을 때 → Blood Reckoning (체력 변환 + Rage 연장)
5. 반복 → 저체력 상태 유지하며 최대 데미지 유지
6. 위험하면 → Outrage (디버프 해제) + 빠른 회복
```

## 초보자 주의사항

- **절대 체력을 가득 채운 채로 싸우지 마세요.** Rage 속성이 발동하지 않습니다.
- **Blood Reckoning 타이밍이 핵심.** 흡수 하트가 쌓였을 때 즉시 실제 체력으로 변환하세요.
- 포션을 너무 자주 쓰면 오히려 데미지가 줄어듭니다. 전략적으로 아끼세요.
- Apotheosis의 **Life Steal Affix** 장비와 조합하면 시너지가 매우 강력합니다.
