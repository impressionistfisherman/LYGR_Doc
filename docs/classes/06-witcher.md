# Witcher (위처)

> *"마물은 두렵지 않다. 알면 됩니다."*

---

## 직업 개요

위처는 **5가지 징표(Sign) 마법**과 **검 전투**를 결합한 마법검사 직업입니다.
Witcher 3 게임에서 영감을 받아 제작되었으며,
포션 시스템과 전략적 전투를 즐기는 플레이어에게 적합합니다.

| 항목 | 내용 |
|------|------|
| **주 무기** | 검 (Steel Sword, Silver Sword) |
| **핵심 자원** | 체력, 포션 |
| **주요 스탯** | 공격력, Spell Power, 크리티컬 |
| **특이사항** | 몬스터 계열별 전용 검(강철검/은검) |
| **난이도** | ★★★★☆ |

---

## 핵심 시스템 — 5가지 징표 (Sign)

위처의 핵심은 상황에 맞는 징표를 선택해 전투를 유리하게 이끄는 것입니다.

### 🔥 Igni (이그니)

```
효과: 전방에 화염 분사 — 화염 피해 + 불 지속 피해
용도: 다수 적이 모여있을 때, 약한 적 빠른 처리
```

### 🌀 Aard (아르드)

```
효과: 강력한 충격파 — 적 넉백 + 스턴
용도: 근접 적이 너무 많이 달려올 때 거리 벌리기
```

### 🛡️ Quen (퀜)

```
효과: 에너지 보호막 생성 — 피해 일정량 흡수
용도: 강적과 전투 시작 전 선시전 필수
      보호막이 있는 동안 더 공격적으로 싸울 수 있음
```

### 🌿 Yrden (이르덴)

```
효과: 마법 함정 설치 — 함정 위 적 이동 속도 감소 + 지속 피해
용도: 빠른 적 제어, 도망치는 적 잡기, 장기전
```

### 💜 Axii (악시)

```
효과: 정신 지배 — 적 일시 무력화 또는 아군으로 전환
용도: 강한 적 1마리 잠시 무력화하여 다른 적 먼저 처리
```

## 두 가지 검

위처는 **강철검**과 **은검** 두 가지 검을 상황에 맞게 사용합니다.

| 검 종류 | 특화 대상 | 비고 |
|---------|-----------|------|
| **Steel Sword (강철검)** | 인간, 인간형 적 | 기본 근접 공격용 |
| **Silver Sword (은검)** | 몬스터, 언데드, 마법 생명체 | 몬스터에 추가 피해 |

```
사용법:
  강철검: 인간형 적(Illager, 좀비 플레이어 등) 전투 시
  은검:   Alex's Mobs, 언데드, Cataclysm 보스 등

올바른 검 선택 시 데미지가 크게 상승합니다.
```

## 포션 시스템

위처는 전투 전 포션을 마셔 능력을 강화합니다.
위처 전용 포션은 **강한 부작용**이 있지만 강력한 효과를 제공합니다.

### 주요 위처 포션

| 포션 | 효과 | 지속 시간 |
|------|------|-----------|
| **Swallow (제비)** | 체력 빠르게 재생 | 30초 |
| **Cat (고양이)** | 야간 투시 + 감지 범위 증가 | 60초 |
| **White Honey (하얀 꿀)** | 모든 독성 효과 즉시 제거 | 즉시 |
| **Thunderbolt (번개)** | 크리티컬 확률 + 공격 속도 대폭 증가 | 30초 |

> **참고: 포션 재료 수집**
>
> 위처 포션은 특수 재료가 필요합니다.
> 탐험 중 허브, 버섯, 몬스터 재료를 꾸준히 수집해두세요.
> Apotheosis 스포너 팜 옆에 작물 재배 공간을 만들어두면 좋습니다.

## 핵심 스킬

| 스킬 | 효과 |
|------|------|
| **Pirouette** | 회전 베기 — 전방 넓은 범위 피해 |
| **Riposte** | 카운터어택 — 적 공격 후 즉시 반격 |
| **Whirl** | 빠른 연속 베기 — 다수 적에게 효과적 |
| **Rend** | 강타 — 높은 단일 피해 |

## 전투 루프

```
전투 전:
  1. Quen 징표 → 보호막 활성화
  2. Thunderbolt 포션 → 전투력 버프

전투 시작:
  3. 적 계열 확인 → 올바른 검 선택
  4. 다수 적 → Aard로 넉백 + Igni로 화염
  5. 단일 강적 → Yrden 함정 설치 → 검 공격
  6. 위험 순간 → Axii로 잠시 무력화

체력 관리:
  7. 체력 낮으면 → Swallow 포션 + 후퇴
  8. Quen 보호막 깨지면 → 즉시 재시전
```

## 스킬 트리 방향

### ⚔️ 검사 특화

```
근접 공격력 증가 노드
Pirouette/Whirl 강화 노드
카운터어택 데미지 노드
```

### 🔮 징표 특화

```
징표 피해 증가 노드
징표 쿨다운 단축 노드
Quen 보호막 강화 노드
```

대부분의 플레이어는 검+징표 균형 빌드를 권장합니다.
