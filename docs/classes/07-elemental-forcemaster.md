# Elemental Wizards & Forcemaster

---

## Elemental Wizards (원소 마법사)

### 직업 개요

> *"불, 얼음, 번개 — 상황에 맞게 원소를 바꿉니다."*

Elemental Wizards는 **화염, 냉기, 번개** 세 가지 원소를 자유롭게 전환하며 싸우는 마법사입니다.
기본 Wizards가 하나의 계열에 집중한다면, Elemental Wizards는 상황에 따라 원소를 바꿔 유연하게 대처합니다.

| 항목 | 내용 |
|------|------|
| **주 무기** | 원소 마법 지팡이 |
| **핵심 자원** | 원소력 (원소별 별도 관리) |
| **주요 스탯** | Fire/Frost/Lightning Spell Power |
| **특징** | 3원소 자유 전환, 다양한 상황 대응 |
| **난이도** | ★★★★☆ (원소 관리가 복잡) |

### 원소 전환 시스템

```
기본 원소: 화염 (Fire)

전환 방법:
  특정 스킬 또는 키 바인딩으로 원소 전환
  → 실제 키는 `Controls`에서 elemental, spell, class 키워드로 확인
  → 전환된 원소에 맞는 주문 세트 활성화

전환 전략:
  화염  → 다수 적 광역 피해
  냉기  → 적 제어 + 둔화
  번개  → 연쇄 공격 + 광역
```

### 원소별 핵심 주문

**화염 원소 (Fire)**

| 주문 | 효과 |
|------|------|
| **Magma Orb** | 폭발 화염 투사체 |
| **Meteor Shower** | 광역 운석 낙하 |
| **Volcanic Eruption** | 지점 화염 폭발 |

**냉기 원소 (Frost)**

| 주문 | 효과 |
|------|------|
| **Glacial Lance** | 관통 빙하 투사체 |
| **Frozen Ground** | 범위 빙결 + 슬로우 |
| **Ice Barrier** | 냉기 방어막 |

**번개 원소 (Lightning)**

| 주문 | 효과 |
|------|------|
| **Chain Lightning** | 여러 적 연쇄 번개 |
| **Storm Surge** | 번개 폭풍 소환 |
| **Static Shield** | 피격 시 번개 반사 방어막 |

### 전투 전략

```
다수 몹: 화염 원소 → 광역기 연사
빠른 적: 냉기 원소 → Frozen Ground로 발목 잡기
군중전:  번개 원소 → Chain Lightning으로 연쇄 처리

보스:
  1단계: 냉기 → Frozen Ground로 이동 제한
  공격:  화염 또는 주력 원소 → 피해 집중
  위험:  냉기 → Ice Barrier로 방어
```

### 스킬 트리 방향

```
균형형 (추천):
  3원소 모두 소폭 강화 노드
  원소 전환 속도 증가 노드
  전환 시 보너스 패시브

특화형:
  주력 원소 하나를 극대화
  해당 원소 쿨다운 단축 + 데미지 증가
```

## Forcemaster (포스마스터)

### 직업 개요

> *"힘은 무기에서 오지 않는다. 내 두 손에서 온다."*

Forcemaster는 **주먹 무기(Knuckle)**를 사용하는 근접 마법격투가입니다.
MMORPG FLYFF(플라이 포 펀)에서 영감을 받은 독특한 직업입니다.
빠른 공격 속도와 스턴/공중 콤보가 특기입니다.

| 항목 | 내용 |
|------|------|
| **주 무기** | Knuckle (너클) — 주먹 무기 |
| **핵심 자원** | 기력 |
| **주요 스탯** | 공격 속도, 크리티컬, 근접 공격력 |
| **특징** | 스턴 + 공중 콤보, 빠른 근접 전투 |
| **난이도** | ★★★☆☆ |

### 전용 무기 — Knuckle (너클)

```
특징:
  양손 주먹 무기
  공격 속도 매우 빠름
  각 주먹에 마법 에너지 각인
  타격 시 마법 효과 발동

제작: JEI에서 "Knuckle" 검색
종류: 기본 너클, 원소 강화 너클, 보스 재료 너클
```

### 핵심 스킬

| 스킬 | 효과 |
|------|------|
| **Force Palm** | 강력한 일격 + 전방 충격파 |
| **Air Uppercut** | 적을 공중으로 날려 올림 |
| **Cyclone Punch** | 회전 연타 — 주변 범위 피해 |
| **Aerial Combo** | 공중에 뜬 적에게 추가 연속 공격 |
| **Force Barrier** | 에너지 방어막 생성 |
| **Gravity Slam** | 큰 충격파 — 범위 넉다운 |

### 콤보 시스템

Forcemaster의 핵심은 **공중 콤보**입니다.

```
기본 콤보:
  1. Air Uppercut → 적이 공중으로 올라감
  2. Aerial Combo → 공중의 적에게 연속 공격
  3. 적이 땅에 내려오면 → Force Palm으로 마무리

다수 적 콤보:
  1. Gravity Slam → 주변 적 전부 넉다운
  2. Cyclone Punch → 쓰러진 적에게 회전 연타
```

### 전투 루프

```
1. Force Barrier → 방어막 활성화
2. 빠른 근접 공격으로 접근
3. Air Uppercut → 적 공중으로 올리기
4. Aerial Combo → 공중 연속 공격
5. Force Palm → 마무리 일격
6. 다음 적으로 빠르게 이동
```

### 스킬 트리 방향

**콤보 특화**

```
Air Uppercut 높이 증가 노드
Aerial Combo 히트 수 증가 노드
콤보 공격 데미지 보너스 노드
```

**광역 특화**

```
Gravity Slam 범위 증가 노드
Cyclone Punch 지속 시간 증가 노드
범위 공격 데미지 증가 노드
```
