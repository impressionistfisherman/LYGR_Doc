# Rogues & Warriors

같은 모드에 두 직업이 포함되어 있습니다.  
Rogue와 Warrior는 **완전히 다른 플레이 스타일**을 가집니다.

---

## 🗡️ Rogue (로그)

### 직업 개요

> *"적이 눈치채기 전에 이미 쓰러져 있다."*

Rogue는 은신과 백스탭을 통해 **단숨에 적을 제압**하는 암살자 직업입니다.  
일대일 전투에서 최강의 순간 폭발력을 자랑하지만, 여러 적을 상대하기 어렵습니다.

| 항목 | 내용 |
|------|------|
| **주 무기** | 단검, 단검 계열 무기 |
| **핵심 자원** | 스태미나 |
| **주요 스탯** | 크리티컬, 공격 속도, 이동 속도 |
| **난이도** | ★★★☆☆ |

### 핵심 메커니즘 — Stealth (은신)

```
은신 발동: 특정 스킬 또는 특수 장비 착용 상태에서 활성화
은신 중:   몬스터에게 감지되지 않음
           이동 속도 소폭 감소
백스탭:    은신 상태에서 적 등 뒤 공격 시 추가 피해
```

### 핵심 스킬

| 스킬 | 효과 |
|------|------|
| **Shadowstep** | 짧은 거리 순간이동 + 은신 진입 |
| **Backstab** | 은신 상태에서 강력한 뒤치기 공격 |
| **Poison Strike** | 독 효과 부여 근접 공격 (지속 피해) |
| **Smoke Bomb** | 범위 연막 + 은신 지속 |
| **Fan of Blades** | 전방 부채꼴 다중 단검 투척 |

### 전투 루프

```
1. Shadowstep → 적 뒤로 순간이동 + 은신
2. Backstab → 높은 추가 피해
3. Poison Strike → 지속 피해 부여
4. 적이 공격하면 → Shadowstep으로 다시 뒤로
5. 반복
```

### 장비 방향

- **단검 2개 이중 장비** (Better Combat 연동 시)
- 크리티컬 확률 + 공격 속도 우선
- Apotheosis Affix: **Execute** (저체력 추가 피해), **Celerity** (공격 속도)

## 🛡️ Warrior (워리어)

### 직업 개요

> *"방패가 있는 한, 나는 쓰러지지 않는다."*

Warrior는 방패와 두꺼운 방어구로 적의 공격을 막아내는 **근접 탱커**입니다.  
생존력이 높아 초보자에게도 적합하고, 파티에서 탱킹 역할을 담당합니다.

| 항목 | 내용 |
|------|------|
| **주 무기** | 검 + 방패 조합 |
| **핵심 자원** | 스태미나, 체력 |
| **주요 스탯** | 방어력, 최대 체력, 넉백 저항 |
| **난이도** | ★★☆☆☆ (초보자 추천) |

### 핵심 메커니즘 — Shield Mastery (방패 숙련)

```
방패 막기: 우클릭으로 피해 크게 감소
Shield Bash: 방패로 적을 가격 → 스턴 효과
방패 내구도: 공격을 막을수록 소모
            → 워리어 전용 방패는 내구도가 높음
```

### 핵심 스킬

| 스킬 | 효과 |
|------|------|
| **Shield Bash** | 방패로 강타 → 스턴 + 넉백 |
| **War Cry** | 범위 공격력 버프 (파티 전체 적용) |
| **Heroic Strike** | 강력한 단일 근접 공격 |
| **Rallying Banner** | 아군 체력 회복 + 버프 배너 설치 |
| **Berserker Mode** | 일시적으로 방어 포기, 공격력 대폭 증가 |

### 전투 루프

```
1. War Cry → 공격력 버프 활성화
2. 근접 전투 → 방패 막기로 피해 경감
3. 적이 근접할 때 → Shield Bash → 스턴
4. 스턴 중 → Heroic Strike 연속
5. 체력 낮으면 → 후퇴 + 포션 + 다시 진입
```

### 장비 방향

- **검 + 방패** 세트 유지
- 방어력 + 최대 체력 우선
- Apotheosis Affix: **Resilient** (피해 감소), **Health** (최대 체력)

## Rogue vs Warrior — 어떤 직업을 선택할까?

| 상황 | 추천 |
|------|------|
| 처음 하는 플레이어 | **Warrior** |
| 빠르고 공격적인 플레이 선호 | **Rogue** |
| 파티에서 탱킹 역할 | **Warrior** |
| 솔로 플레이, 1:1 암살 스타일 | **Rogue** |
| 다수 몹 상대 | **Warrior** |
| 보스 클리어 효율 | **Rogue** (높은 폭발 데미지) |
