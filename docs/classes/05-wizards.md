# Wizards (위자드)

> *"마법은 지식이다. 지식은 힘이다."*

---

## 직업 개요

Wizards는 **3가지 마법 계열** 중 하나를 선택해 특화하는 마법사 직업입니다.  
강력한 원거리 주문으로 적을 압도하지만, 방어력이 낮아 위치 선정이 중요합니다.

| 항목 | 내용 |
|------|------|
| **주 무기** | 완드(Wand), 지팡이(Staff) |
| **핵심 자원** | 룬 (Arcane/Fire/Frost Rune) |
| **주요 스탯** | Spell Power (계열별), Spell Haste |
| **마을 구조물** | 위자드 타워 (Wizard Tower) |
| **난이도** | ★★★☆☆ |

---

## 세 가지 마법 경로

직업 시작 시 세 경로 중 하나를 선택합니다.  
각 경로는 전용 스펠북과 주문 세트를 제공합니다.

### 🔮 Arcane (비전)

```
특징: 단일 대상 집중 딜, 범용성 높음
강점: 모든 상황에서 안정적인 피해
추천: 처음 Wizards를 플레이하는 플레이어
주문 스타일: 빠른 투사체 연사, 집중 일격
```

**주요 주문:**
- Arcane Bolt — 빠른 비전 투사체
- Arcane Missiles — 연속 비전 투사체
- Arcane Burst — 근접 범위 폭발
- Blink — 순간이동

---

### 🔥 Fire (화염)

```
특징: 광역 피해 + 지속 화염 데미지
강점: 다수 몹 처리, 넓은 범위 공격
추천: 공격적인 플레이 선호
주문 스타일: 폭발, 화염 장판, 광역기
```

**주요 주문:**
- Fireball — 폭발 화염탄
- Flame Wave — 전방 화염 파동
- Fire Bomb — 투척형 화염 폭탄
- Meteor — 대형 운석 낙하 (차징)

---

### ❄️ Frost (냉기)

```
특징: 적 둔화 + 빙결 제어
강점: 적 이동 방해, 안전한 거리 유지
추천: 컨트롤 지향 플레이어
주문 스타일: 슬로우, 빙결, 방어막
```

**주요 주문:**
- Ice Bolt — 냉기 투사체 + 슬로우
- Frost Nova — 주변 적 빙결
- Blizzard — 광역 지속 냉기 피해
- Frost Shield — 냉기 보호막

## 공통 시스템 — 룬 & 완드

### 룬 (Rune) — 주문 연료

```
Wizard 계열 주문은 룬을 요구하는 경우가 많습니다.
필요한 룬은 주문 툴팁과 스펠북 툴팁에서 확인하세요.

종류:
  Arcane Rune → 비전 주문용
  Fire Rune   → 화염 주문용
  Frost Rune  → 냉기 주문용

조합법: JEI에서 각 룬 검색
주력 주문에 맞는 룬을 여유 있게 준비
```

### 완드 & 지팡이

```
완드(Wand): 한 손 무기, 빠른 시전
지팡이(Staff): 양손 무기, 더 강한 주문 파워

주문 시전 방법:
  완드/지팡이를 손에 들고 + 스펠북 보조 장비 상태에서
  `Controls`에서 확인한 주문 시전 키로 사용

위자드 타워나 마을에서 구매 가능
또는 JEI에서 조합법 확인
```

---

## 마을 구조물 — 위자드 타워 (Wizard Tower)

마을에 **위자드 타워**가 생성됩니다.

```
구조:
  마을 내 높은 탑 형태
  → 내부에 Wizard Merchant 주민
  → 완드, 지팡이, 마법사 로브, 스크롤 판매
  → Spell Binding Table 배치

추천:
  게임 시작 후 마을을 발견하면 위자드 타워부터 탐색
  → 초반 장비 빠르게 구매 가능
```

## 위자드 전용 장비

### 마법사 로브 (Wizard Robe) 세트

```
구성: 모자, 로브, 하의, 신발
효과: 각 부위마다 Spell Power + Spell Haste 보너스
     선택한 계열의 스펠 파워 특화 버전 존재
```

### 아이템 강화 순서

```
초반: 위자드 타워에서 기본 완드 + 로브 구매
중반: Apotheosis Affix 장비 + Spell Power 스탯 확보
후반: 고급 완드/지팡이 + ApothicEnchanting으로 인챈트 극대화
```

## 스킬 트리 방향

### 🔮 단일 딜러 (Arcane 경로 추천)

```
크리티컬 확률 + 크리티컬 데미지 노드 집중
단일 타겟 주문 강화 노드
Blink 쿨다운 단축 노드 (생존성 향상)
```

### 🔥 광역 딜러 (Fire 경로 추천)

```
광역 주문 피해 증가 노드
화염 지속 피해 강화 노드
Spell Haste 증가 노드 (쿨다운 단축)
```

### ❄️ 제어형 (Frost 경로 추천)

```
빙결 지속 시간 증가 노드
슬로우 효과 강화 노드
Frost Shield 보호막 강화 노드
```

## 전투 기본 전략

```
공통 원칙:
  1. 거리 유지 필수 (방어력 낮음)
  2. Frost 주문으로 적 둔화 → 안전 확보
  3. 쿨다운 관리 — 주력 주문 → 서브 주문 → 주력 주문
  4. 다수 적은 Fire 광역기로 처리
  5. 단일 강적(보스)은 Arcane 집중딜

생존 팁:
  - Blink를 항상 쿨다운 준비 상태로 유지
  - 차징 주문은 안전한 위치에서 시전
  - 포션을 아끼지 말고 적극 활용
```
