# Paladins & Priests

Holy Power를 중심으로 하는 성스러운 직업 모드입니다.  
팔라딘은 근접+마법 하이브리드, 프리스트는 원거리 지원/힐러입니다.

---

## ⚔️✨ Paladin (팔라딘)

### 직업 개요

> *"빛이 닿는 곳에 악은 설 자리가 없다."*

팔라딘은 **근접 전투 + 성스러운 마법**을 결합한 하이브리드 직업입니다.  
자신을 치유하면서 싸우고, 언데드에게 특히 강한 성스러운 피해를 줍니다.

| 항목 | 내용 |
|------|------|
| **주 무기** | 검 + 방패 또는 양손 메이스 |
| **핵심 자원** | Holy Power, 룬 |
| **주요 스탯** | 최대 체력, Holy Spell Power, 방어력 |
| **특이사항** | 언데드/악마 계열에 추가 피해 |
| **난이도** | ★★★☆☆ |

### 마을 구조물 — 성소(Sanctuary)

팔라딘 모드는 마을에 **성소(Sanctuary)** 구조물을 추가합니다.

```
Sanctuary:
  마을 중앙에 있는 성스러운 건물
  → 내부에 Monk 주민 (팔라딘/프리스트 장비 판매)
  → Spell Binding Table 배치
  → 팔라딘 전용 장비 초반 구매 가능
```

### 핵심 스킬

| 스킬 | 종류 | 효과 |
|------|------|------|
| **Holy Bolt** | 공격 | 성스러운 투사체, 언데드 추가 피해 |
| **Smite** | 공격 | 강력한 성스러운 일격 |
| **Consecration** | 광역 | 지면에 성스러운 장판 → 범위 지속 피해 |
| **Divine Shield** | 방어 | 일시적 피해 면역 방어막 |
| **Lay on Hands** | 힐 | 자신 또는 아군 대량 즉시 치유 |
| **Blessing** | 버프 | 아군 공격력 + 방어력 증가 |
| **Judgment** | 공격 | 강력한 성스러운 폭발 |

### 전투 루프

```
전투 전:
  1. Blessing → 전투력 버프 활성화
  2. Divine Shield → 안전하게 진입

전투 중:
  3. 근접 공격 + 틈틈이 Holy Bolt/Smite
  4. 다수 적 → Consecration 장판 설치
  5. 체력 위험 → Lay on Hands 즉시 치유

보스 전투:
  6. Consecration을 보스 발 아래 배치
  7. Divine Shield를 위험 순간에 아끼기
```

### 장비 방향

- **방어력 + Holy Spell Power** 균형
- Apotheosis Affix: **Resilient** + **Health**
- Holy Spell Power 스탯 있는 Affix 장비 우선 수집

## 🙏 Priest (프리스트)

### 직업 개요

> *"모든 생명은 성스럽다. 나는 그 생명을 지킨다."*

프리스트는 **원거리 지원 + 대량 치유** 전문 직업입니다.  
솔로 플레이보다는 **파티에서 빛을 발하는** 직업입니다.

| 항목 | 내용 |
|------|------|
| **주 무기** | 완드, 지팡이 |
| **핵심 자원** | Holy Power, 룬 |
| **주요 스탯** | Healing Power, Holy Spell Power |
| **난이도** | ★★★☆☆ (솔로 어려움, 파티 필수) |

### 핵심 스킬

| 스킬 | 종류 | 효과 |
|------|------|------|
| **Heal** | 힐 | 단일 대상 대량 치유 |
| **Renew** | 힐 | 대상에게 지속 회복 버프 |
| **Mass Heal** | 광역힐 | 주변 모든 아군 치유 |
| **Holy Light** | 공격 | 성스러운 빛 투사체 |
| **Shield of Faith** | 방어 | 대상에게 피해 흡수 보호막 |
| **Prayer of Mending** | 패시브힐 | 피격 시 자동 치유 발동 |

### 솔로 플레이 팁

프리스트는 솔로 플레이 시 화력이 부족할 수 있습니다.

```
솔로 플레이 전략:
  1. Holy Light로 원거리 지속 딜
  2. Shield of Faith로 자신에게 보호막
  3. Renew로 자신에게 지속 치유 유지
  4. 근접 접근 시 최대한 거리 유지
  5. 위험하면 Mass Heal로 즉시 회복
```

---

## 공통 — Holy Power와 룬 확인

팔라딘과 프리스트의 일부 주문은 **룬** 또는 성속성 자원이 필요할 수 있습니다.

```
룬 종류: Holy Rune (성스러운 룬)
조합법: JEI에서 "Holy Rune" 검색
확인 기준: 주문 툴팁, 스펠북 툴팁, Spell Binding Table 결과 슬롯
```

힐과 보호막을 자주 쓰는 플레이 스타일이라면 Holy Rune을 여유 있게 준비하세요.

## Paladin vs Priest — 어떤 직업을 선택할까?

| 상황 | 추천 |
|------|------|
| 솔로 + 전투 직접 하고 싶음 | **Paladin** |
| 파티에서 힐러 역할 | **Priest** |
| 언데드 던전 공략 특화 | **Paladin** |
| 파티 서포트 특화 | **Priest** |
