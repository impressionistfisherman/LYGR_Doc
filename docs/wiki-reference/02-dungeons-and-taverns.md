# Dungeons and Taverns — 구조물 전체 목록

Dungeons and Taverns는 **바닐라 스타일**로 제작된 100+ 구조물 모드입니다.
새로운 블록이나 아이템을 추가하지 않으며, **바닐라 완전 호환**이 특징입니다.
단, **17가지 전용 인챈트**가 구조물 내에서만 획득 가능합니다.

---

## 핵심 시스템 — 여관(Tavern)

### 여관이 왜 중요한가?

Dungeons and Taverns의 핵심은 **여관에서 시작**합니다.

```
여관 → 내부 제도사(Cartographer) 주민과 거래
        → 던전 지도(Chart) 구매
        → 희귀 구조물 위치 확인
```

### 여관 활용 방법

```
1. 여관 발견 → 내부 제도사 주민 찾기
2. 제도사와 거래해 경험치 레벨업 (견습생 이상으로)
3. 던전 지도(Chart) 구매 가능 상태 확인
4. 지도를 구매해 목표 구조물 위치 파악
5. 이후 지도를 원하는 위치로 이동할 수 있습니다
```

### 여관 종류

| 여관 | 생성 바이옴 |
|------|------------|
| 기본 여관 | 평원, 숲 등 일반 바이옴 |
| 사막 여관 | 사막 |
| 타이가 여관 | 타이가, 설원 |
| 열대 여관 | 정글, 맹그로브 |
| 창백한 여관 (Pale Tavern) | 창백한 정원 |

## 미니보스 4종

Dungeons and Taverns에는 4명의 미니보스가 있습니다.
4명 전부 처치하면 **"Dungeon Master" 업적** 달성.

| 미니보스 | 위치 | 특징 |
|----------|------|------|
| **Spider Queen** | Toxic Lair | 거미 소환, 독 공격 |
| **Freeze** | Lone Citadel | 냉기 공격, 빙결 |
| **Ancient Guardian** | Trident Trial Monument | 수중 전투 |
| **Haunting** | End Castle | 엔드 계열 공격 |

## 전용 인챈트 17종

구조물 내 상자에서만 획득 가능한 인챈트입니다.
일부는 Secret Librarian 시스템으로 반복 구매 가능합니다.

| 인챈트 | 효과 |
|--------|------|
| **Aerial's Bane** | 공중 대상(Breeze, Vex, Wither 등)에게 추가 피해 |
| **Illager's Bane** | Illager 계열 추가 피해 |
| **Might** | 석궁 전용 Power (활의 Power와 동일) |
| **Ghasted** | 석궁 화살 → 가스트 화염구로 변환 |
| **Traveler** | 달리기 속도 + 계단 자동 오르기 |
| **Photosynthesis** | 햇빛에서 아이템 자동 수리 (Mending과 비호환) |
| **Wax Wings** | 천천히 활강 시 Elytra 내구도 소모 없음 |
| **Antidote** | 독/위더 피해를 체력 대신 갑옷 내구도로 전환 |
| **Tri-Shot** | 석궁 3발 동시 발사 |
| **Blowthrough** | 화살 관통 (벽 관통 아님) |
| **Outreach** | 상호작용 범위 증가 |
| **Gravity** | 화살이 중력의 영향을 더 받음 |
| **Wither Coated** | 타격 시 Wither 상태이상 부여 |
| **Hydro Veil** | 수중 방어 강화 |
| **Swift Soar** | Elytra 빠른 활강 속도 향상 |
| **Spite** | 피격 시 공격자에게 반사 피해 |
| **Wind Burst** | 타격 시 폭발적 넉백 |

## Secret Librarian 시스템 — 인챈트 갱신

특정 구조물 내에 Librarian 주민을 배치하면 독점 인챈트를 반복 구매할 수 있습니다.

```
방법:
  1. Librarian 주민을 목표 구조물 내부로 이동
  2. 견습생(Apprentice) 레벨까지 육성
  3. 해당 구조물 전용 인챈트 거래 해금

구조물별 해금 인챈트:
  Desert Ruins → Outreach
  Jungle Ruins → Photosynthesis
  Illager Hideout/Manor → Illager's Bane
  Toxic Lair → Antidote, Blowthrough, Tri-Shot
  Lone Citadel → Wind Burst, Aerial's Bane, Might
  Nether Keep → Hydro Veil, Swift Soar, Wither Coated
  Sealing Halls → 5가지 인챈트
  End Castle → Aerial's Bane, Gravity, Wax Wings
```
