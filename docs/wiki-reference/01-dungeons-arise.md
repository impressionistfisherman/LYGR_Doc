# When Dungeons Arise — 구조물 전체 목록

When Dungeons Arise는 **40+ 개의 대형 구조물**을 오버월드/네더/엔드에 생성합니다.  
새로운 아이템이나 몬스터는 추가하지 않으며, **바닐라 몬스터와 아이템만 사용**합니다.  
단, 5가지 **전용 인챈트**가 던전 상자에서만 드롭됩니다.

---

## 탐험 시작 방법

### 방법 1 — 보물 지도 시스템 (권장)

```
소형 구조물(야영지, 어부 오두막 등) 탐색
→ 내부 상자에서 Explorer Map 발견
→ 지도가 가리키는 곳에 더 큰 구조물 존재
→ 큰 구조물에서 또 다른 지도 → 반복
```

### 방법 2 — /locate 명령어

```
/locate structure whendungeonsarise:<구조물 이름>

전체 목록 자동완성:
  채팅창에 /locate structure whendungeonsarise: 입력 후 Tab 키
```

## 탐험 팁

### 공중 구조물 접근

```
Heavenly Conqueror's Keep, Sky Airship 등 공중 구조물:
  방법 1: Mekanism 제트팩 또는 Elytra
  방법 2: 블록 쌓기
  방법 3: Wizard Blink 텔레포트 스킬

주의: 낙사 위험 — Free Runners 또는 Feather Falling 필수
```

### 지하 구조물 탐색

```
Foundry, Infested Temple 등 지하 구조물:
  - 지표면에서는 발견 불가능
  - Explorer Map 없이는 찾기 매우 어려움
  - 먼저 소형 구조물을 탐색해 지도 확보 권장
  - 또는 /locate 명령어 사용
```

### 베이스로 활용하기

```
적합한 베이스 구조물:
  Keep Kayra — 넓은 공간, 다층 구조
  Monastery — 마을 분위기, 주민 존재
  Heavenly Conqueror's Keep — 방어 유리한 공중 위치
  
베이스 전환 방법:
  1. 내부 몬스터 전부 처치
  2. 스포너 파괴 또는 조명 설치
  3. Waystones 설치 → 귀환 포인트 등록
```

### 루트 수거 우선순위

```
높은 가치 루트 위치:
  - 최상층 보스 방 (가장 좋은 상자)
  - 숨겨진 지하실 (벽면 우클릭 탐색)
  - 사이드 탑/별관 (본 건물 옆 작은 탑)
  - 도서관/서재 구역 (인챈트 책 집중)
```

## 난이도별 진입 가이드

```
소형 (야영지, 감시탑):
  → 철 장비 이상 충분

중형 (Shiraz Palace, Colosseum):
  → 다이아몬드 + Apotheosis Uncommon 이상

대형 (Keep Kayra, Heavenly Keep):
  → Apotheosis Rare~Epic 이상

공중/지하 (Airship, Foundry):
  → Apotheosis Epic + 이동 수단 확보 후 도전
```
