# 초반 구조물 가이드

초반 구조물은 **철 장비 이상**이면 충분히 클리어할 수 있습니다.  
중요한 것은 구조물 내 **보물 지도**를 찾아 다음 구조물로 이어지는 진행 연결고리를 만드는 것입니다.

---

## 🏠 Brick House (브릭 하우스)

| 항목 | 내용 |
|------|------|
| **생성 위치** | 오버월드 평원, 초원 바이옴 |
| **권장 장비** | 철 장비 |
| **난이도** | ★☆☆☆☆ |

**특징:**
- IDAS 구조물 중 가장 쉬운 입문 구조물
- 2~3층 규모의 작은 건물
- 지하에 소규모 저장 공간 존재
- 일반 좀비/스켈레톤 + 간혹 Alex's Mobs 몬스터

**주요 드롭:**
- 기본 식량, 철 도구
- 가끔 인챈트 책
- **보물 지도 → 다음 구조물 안내**

**팁:**
- 지하 비밀 공간을 꼭 확인하세요. 상자가 숨겨져 있는 경우가 많습니다.
- 1층 벽난로 주변에 숨겨진 공간이 있는 변형이 존재합니다.

## 초반 구조물 탐험 전략

### 보물 지도 우선 확보

모든 초반 구조물에서 가장 중요한 것은 **보물 지도**입니다.  
상자를 열면 지도를 먼저 인벤토리에 챙기세요.

```
보물 지도 사용법:
  지도를 손에 들면 → 가장 가까운 대상 구조물 방향 표시
  지도 위의 X 표시 = 목적지
```

### 지하 공간 탐색

IDAS 구조물의 가장 좋은 루트는 **지하 공간**에 숨겨진 경우가 많습니다.  
모든 방의 바닥과 벽을 두드려보고 숨겨진 통로나 지하 방을 찾으세요.

### Alex's Mobs 특수 몬스터 주의

초반 구조물이라도 Alex's Mobs의 특수 몬스터가 등장할 수 있습니다.

- **Cave Centipede (동굴 지네):** 독 공격, 천장 이동
- **Rocky Roller (바위 굴리기):** 구르기 공격, 충격파
- **Dropbear (드롭베어):** 나무/천장에서 낙하 공격

초반 구조물이라도 방심하지 말고 조심하세요.
