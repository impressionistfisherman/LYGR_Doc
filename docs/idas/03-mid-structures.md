# 중반 구조물 가이드

중반 구조물부터 본격적인 전투와 탐험이 시작됩니다.
**다이아몬드 장비 이상** + Apotheosis Uncommon 장비를 권장합니다.

---

## 🏰 Castle (성)

| 항목 | 내용 |
|------|------|
| **생성 위치** | 언덕, 고원 바이옴 |
| **권장 장비** | 다이아몬드 풀세트 |
| **난이도** | ★★★☆☆ |

**특징:**
- 중형\~대형 규모의 석재 성
- 다층 구조 (외벽, 안뜰, 주탑)
- 중무장한 Illager 계열 몬스터 다수
- 외벽 순찰 경비 + 내부 함정

**공략 팁:**
- 외벽 위 궁수를 먼저 처리하고 진입하세요
- 주탑 최상층에 가장 좋은 루트가 있습니다
- 지하 감옥 구역에 비밀 루트 확인 필수

**주요 드롭:**
- Apotheosis Rare 등급 장비
- 인챈트 책 다수
- 에메랄드, 골드

## 👻 Haunted House (귀신의 집)

| 항목 | 내용 |
|------|------|
| **생성 위치** | 평원, 수풀 바이옴 |
| **권장 장비** | 다이아몬드 장비 |
| **난이도** | ★★★☆☆ |

**특징:**
- 언데드 + Alex's Mobs 혼합 스폰
- 어둡고 미로처럼 구성된 내부
- 바닥, 천장 함정 존재
- 지하에 핵심 루트 집중

**주의 몬스터:**
- **Murmur (머머):** 시야 방해 공격
- **Bone Serpent (뼈 뱀):** 좁은 복도에서 위험
- 강화 Skeleton + Zombie Illager

**주요 드롭:**
- 언데드 관련 특수 아이템
- 저주 인챈트 책 (분해 가치 있음)
- Apotheosis Rare 장비

## 🗡️ Bandit Village (도적 마을)

| 항목 | 내용 |
|------|------|
| **생성 위치** | 다양한 바이옴 |
| **권장 장비** | 다이아몬드 장비 |
| **난이도** | ★★★☆☆ |

**특징:**
- 여러 건물로 구성된 마을 형태
- 근접형 도적 몬스터 집중
- 마을 중앙에 지하 창고 존재

**공략 팁:**
- 한 건물씩 차례로 클리어하세요
- 몬스터가 마을 전체에서 몰려오는 경우 있음
- 중앙 지하 창고에 최고 루트 집중

**주요 드롭:**
- 무기 특화 인챈트 (Sharpness, Knockback 등)
- 도적 테마 특수 아이템
- 식량, 자원

| 항목 | 내용 |
|------|------|
| **생성 위치** | 다양한 바이옴 |
| **권장 장비** | 다이아몬드 장비 |
| **난이도** | ★★★☆☆ |

**특징:**
- 마법사 테마의 고탑

- 원거리 주문 또는 방패 활용 권장

- 최상층 보스를 처치하면 상자가 열리는 구조

**주요 드롭:**
- 인챈트 특화 아이템 (인챈트 레벨 증가 관련)
- ApothicEnchanting 전용 블록
- Spell Engine 스크롤

## 중반 공략 핵심 팁

### 1. 몬스터 등장 패턴 파악

IDAS 중반 구조물은 몬스터가 **특정 방이나 층에 집중**되어 있는 경우가 많습니다.
무리하게 돌진하지 말고 문 앞에서 유인해서 처치하는 전략이 효과적입니다.

### 2. 횃불 설치로 리스폰 방지

클리어한 방에 횃불을 설치해 어둠을 없애면 몬스터 재스폰을 막을 수 있습니다.

### 3. 지하 공간 필수 탐색

IDAS 구조물의 최고 루트는 **지하에 집중**되는 경향이 있습니다.
모든 방의 바닥을 확인하고 지하 통로를 반드시 탐색하세요.

### 4. 포션 적극 활용

중반 구조물부터 전투가 길어질 수 있습니다.
회복 포션을 충분히 가져가고, 빠른 포션(속도 포션)도 도피용으로 준비하세요.
