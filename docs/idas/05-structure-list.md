# 전체 구조물 목록 & /locate 명령어

---

## /locate 명령어 사용법

```
채팅창에 입력:
  /locate idas:<구조물 이름>

탭 완성:
  /locate idas 입력 후 Tab → 전체 목록 자동완성 표시

예시:
  /locate idas:labyrinth
  /locate idas:ancient_mines
  /locate idas:pillager_fortress
  /locate idas:tinkersworkshop
  /locate idas:castle
  /locate idas:haunted_house
```

:::info 게임 내 명령어 권한
싱글플레이 또는 관리자 권한이 있는 서버에서 사용할 수 있습니다합니다.
:::

## 구조물 탐험 진행 순서 추천

```
[초반]
  Brick House → Cottage → Fisherman's Lodge → Hermit's Hollow
  → 보물 지도 획득 → 다음 구조물 추적

[중반]
  Castle → Haunted House → Bandit Village → Enchanting Tower
  → 장비 강화 → Apotheosis Rare 풀세트 목표

[엔드게임 준비]
  Bearclaw Inn, Necromancer's Spire 등으로 Epic 장비 수집
  → 직업 스킬 중반 해금

[엔드게임]
  Ancient Mines → Pillager Fortress → Tinker's Workshop → Labyrinth
  (순서는 장비 수준에 따라 조정)
```

## 구조물 리셋 (재생성)

한 번 클리어한 구조물은 기본적으로 재생성되지 않습니다.  
새 세계를 시작하거나 다른 위치의 같은 구조물을 /locate로 찾는 것이 현실적입니다.  

단, **Apotheosis 스포너 시스템**을 활용하면 구조물 몬스터를 간접적으로 파밍할 수 있습니다.
