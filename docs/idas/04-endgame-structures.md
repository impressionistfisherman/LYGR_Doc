# 엔드게임 구조물 공략

IDAS에서 가장 도전적이고 보람 있는 구조물들입니다.
**Apotheosis Epic 이상 장비** + **직업 스킬 중반 이상** 해금을 강력히 권장합니다.

> **위험: 준비 없이 진입하지 마세요**
>
> 엔드게임 구조물은 시작 장비로 진입하면 즉사에 가까운 상황이 연출됩니다.
> 특히 Tinker's Workshop의 Create 기계 함정은 장비와 무관하게 순식간에 사망할 수 있습니다.

---

## 🏺 Labyrinth (라비린스)

IDAS의 시그니처 구조물이자 **가장 어려운 던전**입니다.

| 항목 | 내용 |
|------|------|
| **생성 위치** | 사막 바이옴 |
| **권장 장비** | Apotheosis Epic 풀세트 |
| **규모** | 매우 대형 (지하 미로) |
| **난이도** | ★★★★★ |

### 구조 개요

| 구역 | 내용 |
|------|------|
| 지표 입구 | 라비린스 진입 지점 |
| 지하 대규모 미로 | 수많은 분기 복도, 함정, 몬스터 |
| 비밀 방 | 최고 루트 후보 |
| 중간 보상 방 | 회복과 보상 확보 지점 |
| 최종 방 | 파라오 보스 |

### 내부 구성

- **미로 복도:** 무수히 많은 갈림길과 막다른 길
- **함정:** 바닥 함정, 화살 함정, 낙하 구덩이
- **Create 장치:** 일부 구역에 회전 톱날, 컨베이어 함정
- **특수 몬스터:** 이집트 테마 몬스터, 강화 좀비 등

### 파라오 보스

미로 최심부에 **파라오 보스**가 있습니다.

```
파라오 보스 특징:
  - 높은 체력과 방어력
  - 원거리 주문 공격
  - 소환 능력 (하수인 몬스터 소환)
  - 처치 시 전용 드롭 + 방 개방

공략 방법:
  1. 하수인 먼저 처치하고 파라오에 집중
  2. 원거리 주문은 블록 뒤로 피하기
  3. 회복 포션 아끼지 말고 사용
```

### 주요 보상

- **전용 음악 디스크 "Slither"** (Disc Fragment 조합)
- Apotheosis Mythic 등급 장비 가능
- 고레벨 인챈트 책 다수
- 희귀 마법 재료

### 탐험 팁

```
1. 횃불을 지나온 길에 배치 → 길 잃음 방지
2. 지도(Map) 사용 → 탐색 경로 추적
3. 막다른 길에도 숨겨진 문이 있는 경우 있음 (밀거나 우클릭)
4. Xaero's Minimap의 웨이포인트 활용 강력 추천
5. 중간에 귀환해서 회복 후 재진입 전략 유효
```

## ⛏️ Ancient Mines (고대 광산)

드워프 테마의 거대한 지하 광산 던전입니다.

| 항목 | 내용 |
|------|------|
| **생성 위치** | 지하 깊은 곳 (지표 입구 있음) |
| **권장 장비** | Apotheosis Rare\~Epic |
| **규모** | 대형 다층 구조 |
| **난이도** | ★★★★☆ |

### 구조 개요

```
지표 입구 (작은 오두막 형태)
  ↓
지하 1층: 주 통로 + 채굴 구역
  ↓
지하 2층: 더 깊은 광산 + 강화 몬스터
  ↓
최하층: 보스 구역 + 최고 루트
```

### 내용

- **채굴 테마:** 실제 광석이 많이 노출되어 있음 (탐험하며 광석도 수집 가능)
- **드워프 골렘:** 특수 골렘 몬스터 스폰
- **Create 채굴 기계:** 일부 구역에 실제로 작동하는 드릴과 컨베이어 함정

### 주요 보상

- 광업 관련 고레벨 인챈트
- Silent Gear 고급 재료 (Black Diamond 포함 가능)
- Apotheosis Epic 장비
- 드워프 테마 장식 블록 (베이스 꾸미기 용)

## 🏯 Pillager Fortress (필라저 요새)

대규모 필라저 테마 요새입니다.

| 항목 | 내용 |
|------|------|
| **생성 위치** | 다양한 바이옴 |
| **권장 장비** | Apotheosis Epic |
| **규모** | 대형 (성벽 + 내부 건물 복합) |
| **난이도** | ★★★★☆ |

### 구조 개요

```
외벽 + 탑 (궁수 다수)
  ↓
안뜰 (근접 전투 몬스터)
  ↓
주 건물 (다층 구조)
  ↓
지하 금고 (최고 루트)
```

### 공략 방법

```
1. 원거리 공격으로 성벽 위 궁수 먼저 제거
2. 성문 주변 함정 확인 후 진입
3. 안뜰에서 몬스터 유인해서 분산 처치
4. 주 건물 외부부터 내부 순서로 클리어
5. 지하 금고로 진입 → 최고 루트 수거
```

### 주요 보상

- 전용 음악 디스크 "Calidum"
- 무기 인챈트 특화 아이템
- Apotheosis Epic\~Mythic 장비
- 레이드 관련 특수 아이템

## ⚙️ Tinker's Workshop (팅커의 작업장)

**Create 기계 함정**으로 가득한 가장 독특한 던전입니다.

| 항목 | 내용 |
|------|------|
| **생성 위치** | 다양한 바이옴 (지하) |
| **권장 장비** | Apotheosis Epic |
| **특이사항** | Create 기계 함정 작동 |
| **난이도** | ★★★★★ |

### 핵심 주의사항

> **위험: Create 함정 주의**
>
> Tinker's Workshop 내부에는 **실제로 작동하는 Create 기계 함정**이 있습니다.
>
> - **회전 톱날 (Mechanical Saw):** 접촉 즉시 대량 피해
> - **분쇄 바퀴 (Crushing Wheels):** 들어가면 즉사
> - **컨베이어 벨트:** 강제 이동 → 함정으로 이동
>
> 절대 서두르지 말고 각 방을 천천히 확인하세요.

### 구조 개요

| 구역 | 내용 |
|------|------|
| 기계 복도 | 함정이 밀집된 이동 구간 |
| 조립 구역 | Create 자동화 기계들이 배치된 구간 |
| 핵심 방 | 보스와 최고 루트가 있는 최종 구역 |

### 공략 전략

```
1. 레버/버튼을 먼저 찾아 기계를 정지시키기
   (일부 기계는 레드스톤으로 제어 가능)

2. 기계 사이 이동 경로 파악 → 안전한 경로 확인 후 이동

3. 커베이어에 올라타지 말기
   (강제로 함정 지역으로 이동됨)

4. 분쇄 바퀴 근처 절대 접근 금지

5. 기계 뒤나 측면으로 우회 경로 탐색
```

### 주요 보상

- Create 관련 희귀 블루프린트
- 기계 공학 테마 특수 아이템
- Apotheosis Epic\~Mythic 장비
- Create 희귀 소재

## 🔮 Nexus (넥서스)

마법 에너지가 응집된 신비로운 구조물입니다.

| 항목 | 내용 |
|------|------|
| **생성 위치** | 다양한 바이옴 |
| **권장 장비** | Apotheosis Rare\~Epic |
| **난이도** | ★★★★☆ |

**특징:**
- 마법 테마 + 강화 마법사 몬스터
- Spell Engine 주문 스크롤 특화 드롭
- 내부 마법 장치가 실제로 작동

**주요 보상:**
- Spell Engine 희귀 스크롤 (Tier 3)
- 마법 관련 Apotheosis 장비
- 특수 마법 재료

## 엔드게임 공략 공통 팁

### 죽음 대비 전략

```
던전 진입 전:
  1. 근처에 Waystones 활성화
  2. 소지품 최소화 (중요 아이템은 집 상자에)
  3. 회복 포션 10개 이상 준비

죽었을 때:
  Apotheosis의 Grave 시스템으로 아이템 보존
  → 죽은 위치로 돌아가 무덤에서 아이템 회수
```

### 재탐험 전략

엔드게임 던전은 한 번에 완전 클리어가 어려울 수 있습니다.
일부를 클리어하고 귀환 → 강화 후 재진입하는 전략이 유효합니다.

```
/locate idas:<구조물 이름>으로 위치 재확인 가능
```
