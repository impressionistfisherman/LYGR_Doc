# 키 바인드 — 확인된 키와 충돌 정리

> 모드 키는 모드 버전, 서버팩 설정, 개인 설정에 따라 달라질 수 있습니다.  
> 이 문서는 **기본값으로 확인 가능한 키**와 **인게임에서 직접 확인해야 하는 키**를 나눠서 정리합니다.

---

## 먼저 확인할 위치

```
ESC → Options → Controls → Key Binds
```

검색창에 모드명이나 기능명을 입력하면 해당 키를 바로 찾을 수 있습니다.

| 찾고 싶은 것 | 검색어 예시 |
|--------------|-------------|
| 스킬 트리 | `skill`, `puffish` |
| 직업 창 | `class`, `rpg` |
| 마법 시전/전환 | `spell`, `cast`, `book` |
| 배낭 | `backpack`, `traveler` |
| 지도 | `map`, `journey`, `xaero` |
| 퀘스트 | `quest`, `ftb` |
| 장신구/Relics | `relic`, `curio`, `trinket` |

## 기본값으로 확인 가능한 모드 키

아래 항목은 일반적으로 해당 모드에서 사용하는 기본 키입니다. 충돌이 있으면 인게임 Controls 화면의 값이 우선입니다.

| 키 | 기능 | 모드 |
|----|------|------|
| `R` | 아이템 레시피 보기 (아이템 위에서) | JEI |
| `U` | 아이템 사용처 보기 (아이템 위에서) | JEI |
| `W` 길게 | Ponder 설명 보기 (Create 아이템 위에서) | Create |
| `Ctrl + T` | World Tier 화면 열기 | Apotheosis |
| `B` | 착용한 배낭 열기 | Traveler's Backpack |
| `Ctrl` | 재료 스탯/특성 확인 (아이템 위에서) | Silent Gear |

---

## 인게임에서 확인해야 하는 모드 키

아래 기능은 문서 저장소에 실제 `options.txt`나 키 설정 파일이 없어서 기본값을 확정하지 않습니다.  
반드시 `Controls → Key Binds`에서 검색한 뒤 현재 클라이언트에 표시되는 키를 사용하세요.

| 기능 | 관련 모드 | Controls 검색어 |
|------|-----------|------------------|
| 스킬 트리 열기 | Puffish Skills | `skill`, `puffish` |
| 직업/클래스 창 | RPG Classes | `class`, `rpg` |
| 마법 선택/시전/스펠북 전환 | Spell Engine 계열 | `spell`, `cast`, `book` |
| 장신구/Relics 능력 | Relics, Artifacts, Curios | `relic`, `curio`, `ability` |
| 지도 열기/마커 추가 | 지도 모드 | `map`, `waypoint`, `journey`, `xaero` |
| 퀘스트북 | 퀘스트 모드 | `quest`, `ftb` |

## 키 충돌 해결하기

모드가 많으면 같은 키에 여러 기능이 겹칠 수 있습니다. 이 경우 키가 입력되지 않거나 엉뚱한 기능이 실행됩니다.

### 충돌 확인 방법

```
ESC → Options → Controls → Key Binds
→ 빨간색으로 표시된 키 = 충돌
→ 해당 키 클릭 → 다른 키로 변경
```

### 먼저 비워두면 좋은 키

| 키 | 추천 용도 |
|----|-----------|
| `Z` | 마법 관련 기능 |
| `X` | 마법 전환 또는 직업 스킬 |
| `C` | 직업/캐릭터 관련 창 |
| `V` | 능력치/장비 정보 창 |
| `G` | 장신구, Relics, 특수 능력 |
| `M` | 지도 |

위 표는 “해당 모드의 기본 키”가 아니라 충돌을 줄이기 위한 추천 배치입니다. 실제 기능명은 Controls에서 검색해 지정하세요.

## 추천 배치 방식

### 전투형

| 키 | 배치할 기능 |

| `G` | Relics/장신구 능력 |
| 마우스 버튼4 | 자주 쓰는 전투 스킬 |
| 마우스 버튼5 | 방어/회피/긴급 스킬 |

| `G` | 주력 주문 시전 보조 |
| 마우스 버튼4 | 가장 자주 쓰는 주문 |
| 마우스 버튼5 | 두 번째 주문 또는 회피기 |

### 탐험형

| `X` | 비상용 이동기 또는 음식 슬롯 |
| 마우스 버튼4 | 횃불 슬롯 선택 |
| 마우스 버튼5 | 물 양동이 슬롯 선택 |

### 기술형

| 키 | 배치할 기능 |
|----|-------------|
| `R` | JEI 레시피 |

## 마우스 버튼 활용

마우스 사이드 버튼은 모드 기능 자체가 아니라 “키 바인딩을 옮겨 담는 자리”로 쓰는 것이 안전합니다.

```
마우스 버튼4 → 자주 쓰는 스킬 또는 주문
마우스 버튼5 → 방어, 회피, 물 양동이, 횃불 등 긴급 행동
```

마우스 버튼이 없는 경우에는 핫바 `5`\~`9`번에 음식, 횃불, 물 양동이, 포션을 고정해두세요.
