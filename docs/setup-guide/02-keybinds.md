# 키 바인드 — 전체 키 정리

> 모드가 178개라 키가 엄청나게 많습니다.  
> 자주 쓰는 키만 외워두고, 나머지는 여기서 찾아보세요!

---

## 키 설정 열기

```
ESC → Options → Controls → Key Binds
→ 키 검색창에 이름 입력하면 빠르게 찾을 수 있습니다
```

## 바닐라 기본 키

### 이동

| 키 | 기능 |
|----|------|
| `W` `A` `S` `D` | 앞/왼/뒤/오른쪽 이동 |
| `Space` | 점프 |
| `Shift` | 살금살금 (절벽에서 안 떨어짐) |

| `Ctrl + Q` | 아이템 버리기 (전체) |

| 키 | 기능 |
|----|------|

## 키 충돌 해결하기

모드가 많으면 같은 키에 여러 기능이 겹치는 경우가 있습니다.  
이럴 때 키가 안 먹히거나 이상하게 작동합니다.

### 충돌 확인 방법

```
ESC → Options → Controls → Key Binds
→ 빨간색으로 표시된 키 = 충돌!
→ 해당 키 클릭 → 다른 키로 변경
```

### 자주 충돌하는 키들

| 충돌 키 | 원인 | 해결책 |
|---------|------|--------|
| `C` | RPG Classes + 줌 기능 충돌 | 줌을 `Z`로 변경, 마법을 `X`로 |
| `Z` | Spell Engine + 다른 모드 | 한쪽을 `X` or `G`로 변경 |
| `V` | 능력치 창 + 다른 모드 | 덜 쓰는 쪽을 다른 키로 |
| `B` | 배낭 + 다른 모드 | 충돌한 쪽 다른 키로 |
| `R` | JEI 레시피 + 달리기 토글 | JEI는 인벤토리 안에서만 작동이라 보통 안 충돌 |

### 추천 키 배열 (충돌 최소화)

처음 시작할 때 이렇게 설정해두면 편합니다:

```
C   → 직업 창 (RPG Classes)
K   → 스킬 트리 (Puffish)
Z   → 마법 목록 (Spell Engine)
X   → 마법 시전
B   → 배낭
Tab → 퀘스트북
M   → 지도
V   → 능력치
```

> 처음 세팅할 때 본인 스타일에 맞는 것을 선택해서 그대로 따라 설정해보세요!

### 🗡️ 근거리 전사 추천 바인딩 (Berserker / Warrior)

전투 중에 마우스에서 손을 최대한 안 떼는 구조입니다.

| 키 | 기능 | 변경 이유 |
|----|------|-----------|
| `C` | 직업 창 | 기본값 유지 |
| `K` | 스킬 트리 | 기본값 유지 |
| `F` | 양손 교체 (무기 ↔ 방패) | 전투 중 빠른 교체 |
| `G` | 특수 스킬 사용 | 엄지 손가락으로 누르기 쉬운 위치 |
| `X` | 직업 고유 스킬 | 왼손 새끼손가락으로 접근 용이 |
| `Z` | 마법 (보조용) | 가끔 쓸 때 |
| `B` | 배낭 | 기본값 유지 |
| `Tab` | 퀘스트북 | 기본값 유지 |
| 마우스 버튼4 | 포션 빠른 사용 | 전투 중 손 안 떼고 사용 |
| 마우스 버튼5 | 방어 스킬 | 즉각 반응 가능 |

```
💡 근거리 전사 핵심 팁:
→ 자주 쓰는 스킬은 마우스 사이드 버튼에 배치
→ F키로 무기↔방패 교체를 빠르게 할 수 있게 연습
→ G, X는 왼손이 WASD에서 벗어나지 않는 키로 설정
```

### 🧙 마법사 추천 바인딩 (Wizard / Spell Engine 특화)

마법을 빠르게 전환하고 시전하는 구조입니다.

| 키 | 기능 | 변경 이유 |
|----|------|-----------|
| `Z` | 마법 목록 열기 | 기본값 유지 |
| `X` | 다음 마법으로 전환 | Z 바로 옆, 빠른 전환 |
| `C` | 직업 창 | 기본값 유지 |
| `K` | 스킬 트리 | 기본값 유지 |
| `F` | 마법 즐겨찾기 슬롯 교체 | 전투 중 마법 세트 교체 |
| `G` | 마나 물약 빠른 사용 | 마나 부족 시 즉시 |
| `V` | 마법 강화 토글 | 버프 상태 확인 |
| `B` | 배낭 | 기본값 유지 |
| 마우스 버튼4 | 마법 1번 즉시 시전 | 가장 자주 쓰는 마법 |
| 마우스 버튼5 | 마법 2번 즉시 시전 | 두 번째로 자주 쓰는 마법 |

```
💡 마법사 핵심 팁:
→ 자주 쓰는 마법 TOP 2는 마우스 사이드 버튼에 배치
→ X키로 마법 전환을 빠르게 연습
→ 마나 게이지 항상 확인하는 습관 들이기
```

이동과 채집에 최적화된 구조입니다.

| 키 | 기능 | 변경 이유 |
|----|------|-----------|
| `M` | 지도 열기 | 기본값 유지 |
| `B` | 배낭 열기 | 기본값 유지 |
| `Tab` | 퀘스트북 | 기본값 유지 |
| `F` | 양손 교체 (횃불 ↔ 곡괭이) | 채굴 중 빠른 교체 |
| `G` | 좌표 마커 설정 | 중요 위치 빠른 표시 |
| `X` | 빠른 식사 | 탐험 중 허기 관리 |
| `C` | 직업 창 | 기본값 유지 |
| `Z` | 마법 (비상용) | 위험 시 탈출 마법 |
| 마우스 버튼4 | 횃불 빠른 설치 | 채굴 중 자주 씀 |

```
→ 물 양동이도 빠르게 꺼낼 수 있게 설정해두기
→ 지도(M) 자주 열어서 현재 위치 확인하는 습관
```

### ⚙️ 엔지니어 추천 바인딩 (Create / Mekanism 특화)

기계 설치와 인터페이스 조작에 최적화된 구조입니다.

| 키 | 기능 | 변경 이유 |
|----|------|-----------|
| `Tab` | 퀘스트북 | 제작 레시피 참고 |
| `B` | 배낭 | 부품 많이 들고 다님 |
| `R` | JEI 레시피 | 자주 레시피 확인 |
| `F` | 양손 교체 (렌치 ↔ 다른 도구) | Create 렌치 자주 교체 |
| `G` | 에너지 상태 확인 | Mekanism 에너지 모니터 |
| `X` | 기계 복사 (Create) | 배치 중 설정 복사 |
| `C` | 직업 창 | 기본값 유지 |
| `K` | 스킬 트리 | 기본값 유지 |
| 마우스 버튼4 | Create 렌치 사용 | 기계 방향 전환 자주 함 |
| 마우스 버튼5 | 청사진 도구 | 구조물 설계 시 |

```
💡 엔지니어 핵심 팁:
→ Create 렌치를 마우스 버튼에 배치하면 기계 설치가 훨씬 편해
→ JEI(R키)는 기술 모드 레시피 확인에 매우 자주 씀
→ 배낭(B)은 항상 빠르게 열 수 있게 유지
```

## 마우스 버튼 활용

마우스 사이드 버튼이 있으면 유용하게 쓸 수 있습니다.

```
마우스 버튼 4 (앞으로) → 마법 시전
마우스 버튼 5 (뒤로) → 스킬 사용
→ 전투 중에 키보드에서 손을 떼지 않아도 됩니다
```

Controls에서 직접 마우스 버튼을 키로 등록할 수 있습니다.
