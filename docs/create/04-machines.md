# 핵심 기계 완전 목록

Create의 모든 주요 기계를 기능별로 정리합니다.
각 기계의 자세한 사용법은 인게임에서 **W키(Ponder)** 로 확인하세요.

---

## 가공 기계

### Millstone (맷돌)

| 항목 | 내용 |
|------|------|
| **기능** | 아이템 분쇄 (광석 → 분말, 밀 → 밀가루 등) |
| **동력** | 아래 또는 옆 축에서 회전력 |
| **투입** | 위에서 아이템 던지기 또는 호퍼 |
| **출력** | 옆면 또는 아래 호퍼 |
| **특이사항** | Crushing Wheels보다 효율 낮음, 초반 임시 사용 |

---

### Crushing Wheels (분쇄 바퀴)

| 항목 | 내용 |
|------|------|
| **기능** | 광석 → 분말 2배 생산, 다양한 재료 분쇄 |
| **동력** | 두 바퀴가 서로 맞물리게 회전력 연결 |
| **설치** | 두 바퀴를 나란히 배치, 회전 방향 반대여야 함 |
| **투입** | 위에서 아이템 투입 |
| **출력** | 아래로 자동 출력 |
| **추천** | 광석 2배 처리 → 자원 효율 극대화 |

**설치 주의:**
```
바퀴 두 개의 회전 방향이 서로 반대여야 합니다.
같은 방향이면 바퀴 사이에 아이템이 끼임.
기어박스로 한쪽 방향을 반전시켜 맞추세요.
```

---

### Mechanical Press (기계 프레스)

| 항목 | 내용 |
|------|------|
| **기능** | 아이템 압착(Pressing), 패킹(Packing), 컴팩팅 |
| **동력** | 위에서 회전력 공급 |
| **설치** | Basin 또는 Depot 위 1블록 위치 |
| **Basin과 조합** | 합금 제작, 특수 레시피 처리 |
| **벨트와 조합** | 벨트 위 아이템 자동 압착 |

**주요 레시피:**
- 판금(Sheet): 철/금/구리 주괴 → 판금
- 안데사이트 합금: Basin에서 자동 생산
- Packing: 9개 재료 → 블록 자동 압축

---

### Mechanical Mixer (기계 믹서)

| 항목 | 내용 |
|------|------|
| **기능** | 재료 혼합, 특수 레시피 처리, 양조 |
| **동력** | 옆 또는 위에서 회전력 (최소 32 RPM) |
| **설치** | Basin 위 1블록 위치 |
| **가열** | Blaze Burner로 Basin 가열 시 고급 레시피 처리 가능 |

**주요 레시피:**
- 안데사이트 합금 (재료 절반 소모)
- 황동 (Brass): 구리 + 아연 → 황동 (가열 필요)
- 포션 자동 양조 (가열 + 물약 재료)

---

### Encased Fan (팬)

| 항목 | 내용 |
|------|------|
| **기능** | 바람 생성 → 아이템 이동, 세척, 제련, 냉각 |
| **동력** | 뒷면 축에서 회전력 |
| **방향** | 팬이 향하는 방향으로 바람 발생 |

**팬 활용 레시피:**
| 팬 앞에 설치 | 효과 |
|-------------|------|
| 불 (Campfire, 화로 등) | **벌크 제련** — 아이템 자동 제련 |
| 물 | **세척(Washing)** — 모래 → 금조각 등 |
| 용암 | **벌크 훈연** |
| 눈/얼음 | **냉각** |

**세척 활용 예시:**
```
모래 → [Fan + 물] → 금 조각, 철 조각, 에메랄드 등
비교적 쉽게 금 조각 대량 확보 가능
```

---

### Spout (스파우트)

| 항목 | 내용 |
|------|------|
| **기능** | Basin에서 유체를 아이템에 주입 |
| **설치** | Basin 위에 설치 |
| **활용** | 유리병에 물 주입 → 물병, 포션 재료 |

---

### Blaze Burner (블레이즈 버너)

| 항목 | 내용 |
|------|------|
| **기능** | Basin 가열 (고급 레시피 처리용) |
| **연료** | 블레이즈 케이크 또는 일반 연료 |
| **획득** | Blaze를 Cage로 포획 → Burner 제작 |
| **필요 레시피** | 황동 제작, 고급 합금, 포션 양조 등 |

> **주의: 블레이즈 버너 없이는 황동 제작 불가**
>
> 황동(Brass)은 Mechanical Mixer + Blaze Burner 가열이 필수입니다.
> 네더 탐험으로 블레이즈를 포획해두세요.

## 이송 기계

### Mechanical Belt (기계 벨트)

| 항목 | 내용 |
|------|------|
| **기능** | 아이템 자동 이송 + 동력 전달 |
| **설치** | 두 Shaft 사이에 연결 |
| **속도** | RPM에 비례해 이송 속도 증가 |
| **기울기** | 경사지게 설치 가능 (최대 45도) |

---

### Funnel (퍼널)

| 항목 | 내용 |
|------|------|
| **기능** | 벨트/기계/상자 간 아이템 이송 |
| **종류** | Andesite Funnel (기본), Brass Funnel (필터 가능) |
| **방향** | 설치 방향에 따라 투입/출력 결정 |

**Brass Funnel 특징:**
- 특정 아이템만 통과하도록 **필터** 설정 가능
- 자동 선별 시스템의 핵심 부품

---

### Mechanical Arm (기계 팔)

| 항목 | 내용 |
|------|------|
| **기능** | 여러 위치에서 아이템을 집어 다른 위치로 이동 |
| **동력** | 아래 축에서 회전력 |
| **설정** | 마우스 우클릭으로 투입/출력 위치 설정 |
| **필요 소재** | 황동 (중반 이후) |

**주요 활용:**
- 복잡한 제조 라인의 재료 자동 공급
- 여러 기계를 하나의 팔로 연결
- 정밀 순서 제어가 필요한 자동화

---

### Deployer (배포기)

| 항목 | 내용 |
|------|------|
| **기능** | 손처럼 아이템을 블록에 사용하거나 공격 |
| **동력** | 뒤에서 회전력 |
| **활용** | 자동 파종, 자동 수확, 자동 제련, 아이템 사용 |

**활용 예시:**
- 씨앗 + Deployer → 자동 파종 팜
- 모루 + Deployer → 자동 인챈트 처리
- Precision Mechanism 제조 라인

---

## 고급 기계

### Mechanical Crafter (기계 제작대)

| 항목 | 내용 |
|------|------|
| **기능** | 특수 Create 레시피 자동 제작 (Crushing Wheel 등) |
| **설치** | 여러 개를 그리드로 배치해 레시피 모양 구성 |
| **동력** | 각 Crafter에 회전력 연결 |
| **소재** | 안데사이트 케이싱 + 기타 |

> **정보: Crushing Wheels 제작에 필수**
>
> Crushing Wheels는 일반 제작대로 만들 수 없습니다.
> Mechanical Crafter 21개를 배열해야 합니다.
> Ponder에서 배치 방법을 꼭 확인하세요.

---

### Schematicannon (설계도 대포)

| 항목 | 내용 |
|------|------|
| **기능** | 설계도(Schematic)를 자동으로 건축 |
| **용도** | 대규모 구조물 자동 건설 |
| **재료** | 건설에 필요한 블록을 상자에 미리 준비 |

**사용 방법:**
1. Schematic & Quill로 설계도 작성 또는 외부 schematic 파일 불러오기
2. Schematicannon 설치 + 재료 상자 연결
3. 설계도 위치 지정 → 자동 건설 시작

---

### Rotation Speed Controller

| 항목 | 내용 |
|------|------|
| **기능** | 출력 RPM을 정밀하게 지정 |
| **소재** | 황동 (중반) |
| **활용** | 특정 RPM이 필요한 기계에 정확한 속도 공급 |

---

## 기계 활용 우선순위

```
초반:
  Millstone → Mechanical Press → Mechanical Belt → Mechanical Mixer

중반:
  Crushing Wheels → Encased Fan(세척) → Mechanical Arm → Deployer

후반:
  Mechanical Crafter → Schematicannon → Steam Engine + 대규모 공장
```
