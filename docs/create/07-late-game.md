# 후반 진행 — 증기 엔진과 대규모 자동화

황동 생산과 중반 자동화 라인을 완성했다면 이제 **대규모 공장**을 구축할 차례입니다.

---

## 증기 엔진 (Steam Engine)

증기 엔진은 Create 최강의 동력원입니다.

### 구성 요소

| 블록 | 역할 |
|------|------|
| **Boiler (보일러)** | 물을 가열해 증기 생성 |
| **Steam Engine** | 증기를 회전력으로 변환 |
| **Flywheel** | 회전력 출력 안정화 |
| **Fluid Pipe** | 물 공급 |
| **Blaze Burner / 용암** | 가열원 |

### 보일러 구조

보일러는 **크기를 키울수록 SU 생산량이 기하급수적으로 증가**합니다.

```
최소 보일러: 1×1×1 → 기본 SU
중형 보일러: 3×3×3 → SU 대폭 증가
대형 보일러: 5×5×5 → 수만 SU 생산
```

보일러 외벽은 **Copper Casing** 또는 **Steam Engine 블록**으로 구성합니다.

### 구축 방법

```
1. 보일러 외곽 블록 배치 (최소 3×3×3 권장)
2. 내부에 Fluid Tank 또는 빈 공간 확보
3. 외벽에 Steam Engine 블록 여러 개 설치
4. Steam Engine에 Flywheel 연결
5. 보일러 아래 Blaze Burner 또는 용암 설치
6. 보일러에 물 공급 (Fluid Pipe + Pump 활용)
7. 가열원 켜기 → 증기 엔진 작동!
```

### 물 공급 자동화

증기 엔진은 지속적으로 물을 소비합니다.

```
물 공급 방법:
  1. Fluid Pump + Pipe로 강/호수에서 무한 공급
  2. Mechanical Pump + Water Tank 조합
```

> **참고: 가장 효율적인 가열원**
>
> Blaze Burner를 여러 개 사용하거나
> 용암을 보일러 아래에 배치하면 연료 걱정 없이 가열 가능합니다.

## 기차 시스템 (Train System)

기차는 Create 중반\~후반의 대규모 이동 및 물류 시스템입니다.

### 필요 소재

- **Train Track:** 황동 소재 레일
- **Train Station:** 기차 정차/출발 지점
- **Train Controls:** Precision Mechanism 필요
- **Bogey (보기):** 기차 바퀴 역할

### 기차 제작 방법

```
1. Train Track 레일 설치 (직선, 곡선, 경사 모두 가능)
2. Train Station 양 끝에 설치
3. Cart Assembler로 기차 차체 조립
   → Train Carriage 블록 사용
   → 위에 원하는 블록 자유롭게 배치
4. Train Controls 설치 → 출발/정차 스케줄 설정
```

### 기차 활용

**자동 물류 기차:**
```
기차 차체에 상자 + Funnel 설치
→ Station A에서 아이템 자동 적재
→ Station B로 이동 후 자동 하역
→ 원거리 물류 자동화
```

**이동식 채굴 기차:**
```
기차 앞에 Drill 블록 대량 부착
→ 레일을 따라 전진하며 자동 터널 채굴
→ 채굴물 상자에 자동 수거
```

## 대규모 자동화 공장 설계 원칙

### 1. 동력 네트워크 분리

모든 기계를 하나의 네트워크에 연결하지 마세요.

```
권장 분리 구성:
  동력 네트워크 1: 광석 처리 라인
  동력 네트워크 2: 가공 라인 (Press, Mixer)
  동력 네트워크 3: 이송 라인 (Belt)
  동력 네트워크 4: 정밀 제조 라인 (Arm, Deployer)
```

### 2. 버퍼 상자 활용

기계 간 처리 속도가 다를 경우 **버퍼 상자**를 중간에 배치해 흐름을 조절하세요.

```
[빠른 기계] → [버퍼 상자] → [느린 기계]
```

### 3. Funnel 필터로 아이템 분리

벨트에서 원하는 아이템만 꺼내려면 Brass Funnel 필터를 활용하세요.

```
벨트 옆에 Brass Funnel 설치
→ 원하는 아이템만 상자로 분기
→ 나머지는 계속 벨트 이동
```

### 4. Redstone 제어로 ON/OFF 자동화

Clutch 블록을 레드스톤으로 제어하면 특정 라인을 켜고 끌 수 있습니다.

```
Clutch: 레드스톤 신호로 회전력 차단/연결
→ 상자가 가득 차면 자동으로 기계 정지
→ 비면 다시 자동 시작
```

## 후반 자동화 목표 체크리스트

```
동력:
  [ ] 증기 엔진 3×3 이상 보일러 구축
  [ ] 물 무한 공급 자동화
  [ ] 동력 네트워크 분리 완료

가공:
  [ ] Crushing Wheels → Fan 세척 → 자동 선별 라인
  [ ] 황동 자동 생산 라인
  [ ] Precision Mechanism 자동 생산 라인

이송:
  [ ] 주요 기계 간 Belt + Funnel 완전 연결
  [ ] Mechanical Arm 복합 라인 구축
  [ ] (선택) 기차 물류 시스템 구축

건축:
  [ ] Schematicannon으로 대규모 건설 자동화
```
