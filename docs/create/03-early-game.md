# 초반 진행 — 안데사이트 시대

Create를 처음 시작하면 가장 먼저 마주치는 소재가 **안데사이트 합금**입니다.
거의 모든 초반 부품이 이것으로 만들어집니다.

---

## 안데사이트 합금 (Andesite Alloy)

### 수동 조합법

```
제작대:
  안데사이트 × 1 + 철 조각(Iron Nugget) × 1
  또는
  안데사이트 × 1 + 아연 조각(Zinc Nugget) × 1
  = 안데사이트 합금 × 2
```

### 자동 생산 (Mechanical Mixer 활용)

Mechanical Mixer + Basin 조합으로 만들면 **재료가 절반**만 소모됩니다.
초반부터 Mixer를 만들어두면 합금 소모량이 크게 줄어듭니다.

### 안데사이트 케이싱

안데사이트 합금을 들고 **나무 통나무(Log)에 우클릭**
→ 안데사이트 케이싱으로 즉시 변환
대부분의 초반 기계에 필요한 핵심 소재입니다.

---

## 아연 (Zinc) 확보

아연은 안데사이트 합금의 핵심 재료 중 하나입니다.

- 오버월드 Y: 0\~60 구간에서 채굴
- 철보다 약간 희귀하지만 초반에 충분히 확보 가능
- **Crushing Wheels** 구축 후 광석 2배 처리 가능 (중반)

---

## 초반 제작 순서

### 1단계 — 엔지니어 고글 제작

SU/RPM 모니터링 필수 도구입니다. 가장 먼저 만드세요.

```
JEI에서 "Engineer's Goggles" 검색 → 조합법 확인
```

### 2단계 — 첫 동력 확보

물레방아(Water Wheel) 또는 풍차(Windmill) 중 선택합니다.

**물레방아:**
- 구하기 쉽고 설치 간단
- 물이 흐르는 곳 어디서든 설치 가능
- 단, SU가 낮아 초반 기계 2\~3개가 한계

**풍차:**
- 날개를 많이 달수록 강력
- 넓은 공간이 필요하지만 물 없이도 작동
- 초반 큰 투자지만 중반까지 버팀목이 됨

### 3단계 — Millstone (맷돌) 제작

첫 번째 가공 기계입니다.

```
기능:
  광석 → 분말 (제련 전 분쇄)
  밀 → 밀가루
  등 다양한 재료 분쇄 가능
```

**설치:**
- Millstone 아래에 회전하는 축 연결
- 위에서 아이템 투입 → 분말로 출력
- 출력은 옆면 또는 아래 호퍼로 수거

### 4단계 — Mechanical Press + Basin

안데사이트 합금 자동 생산 라인의 핵심입니다.

**Mechanical Press:**
- Basin 위 1블록 간격으로 설치
- 회전력 공급 → 자동으로 Basin의 재료 가공

**Basin:**
- 재료를 넣는 그릇
- Press, Mixer, Spout 등과 함께 사용

**안데사이트 합금 자동 생산 구성:**
```
[호퍼] → [Basin] ← [호퍼]
            ↑
     [Mechanical Press]
            ↑
         [동력원]

Basin에 안데사이트 + 철 조각 투입
→ Press가 자동으로 압착
→ 안데사이트 합금 생산
```

### 5단계 — Mechanical Belt (컨베이어)

아이템 이송 자동화의 기본입니다.

**설치:**
1. 두 개의 Shaft 사이에 벨트 연결
2. Shaft에 회전력 공급 → 벨트 작동

**활용:**
- 기계 간 아이템 자동 이송
- Funnel(퍼널)로 벨트에서 상자/기계로 자동 분배

---

## 첫 자동화 루프 예시

안데사이트 합금 자동 생산 루프 구성입니다.

```
[아연 광석 수동 투입]
         ↓
   [Millstone] → 아연 분말
         ↓
   [제련로(Furnace)] → 아연 주괴
         ↓
   [Nugget 변환 제작대] → 아연 조각
         ↓
   [Basin + Press] → 안데사이트 합금 생산!
```

이 루프를 완성하면 **안데사이트 합금을 반자동으로 생산**할 수 있습니다.

---

## 초반 자주 하는 실수

**기계가 작동하지 않아요.**
→ 회전력이 연결되어 있는지, 방향이 맞는지 확인하세요.
→ 엔지니어 고글로 SU 상태를 확인하세요.

**과부하가 자꾸 발생해요.**
→ 물레방아 1개로는 기계 2\~3개가 한계입니다. 발전기를 추가하거나 풍차로 전환하세요.

**Basin에서 아이템이 안 나와요.**
→ Basin 옆면 또는 아래에 호퍼를 설치해 수거하세요.
→ Funnel을 Basin 출구 방향에 설치하면 자동 수거됩니다.
