# 컨트랩션 (Contraption) 시스템

컨트랩션은 Create의 **움직이는 구조물** 시스템입니다.
블록 덩어리가 통째로 이동하면서 작업을 수행합니다.

---

## 컨트랩션이란?

회전력을 받으면 부착된 블록들이 **하나의 덩어리로 움직이는** 구조물입니다.

```
예시:
  - 앞뒤로 왔다갔다 하는 자동 채굴기
  - 회전하는 나무 벌채기
  - 움직이는 기차
  - 돌아가는 풍차 날개에 부착된 기계들
```

---

## 컨트랩션 구성 요소

### Movement Anchor (이동 앵커)

컨트랩션을 움직이게 하는 핵심 블록입니다.
앵커에 회전력을 공급하면 부착된 블록들이 함께 움직입니다.

| 앵커 종류 | 이동 방식 |
|-----------|-----------|
| **Mechanical Piston** | 직선 왕복 운동 |
| **Mechanical Bearing** | 회전 운동 |
| **Rope Pulley** | 위아래 이동 |
| **Cart Assembler** | 레일 위 이동 (기차) |
| **Gantry Carriage** | 레일 위 직선 이동 |

### 블록 부착 방법

컨트랩션에 여러 블록을 함께 이동시키려면 블록들을 **연결**해야 합니다.

| 방법 | 설명 |
|------|------|
| **Super Glue** | 두 블록을 고정 연결 (우클릭으로 붙이기/떼기) |
| **Linear Chassis** | 직선 방향으로 자동 연결 |
| **Radial Chassis** | 원형으로 자동 연결 |
| 슬라임 블록 / 꿀 블록 | 인접 블록 자동 연결 |

---

## 주요 컨트랩션 활용법

### 🔄 Mechanical Bearing — 회전 컨트랩션

Bearing에 부착된 구조물이 회전합니다.

**풍차 만들기:**
```
Windmill Bearing 설치
→ Bearing 앞면에 Sail 블록 부착 (최소 8개)
→ 회전력 공급 → 풍차 회전 시작
→ Bearing 뒤에서 동력 출력
```

**회전 채굴기:**
```
Mechanical Bearing + Mechanical Drill 여러 개 부착
→ 회전하면서 주변 블록 채굴
```

---

### ↔️ Mechanical Piston — 직선 왕복 컨트랩션

피스톤처럼 앞뒤로 이동합니다.

**자동 채굴 터널 뚫기:**
```
Mechanical Piston 앞에 Drill 블록 부착
→ 피스톤이 앞으로 이동 → Drill이 블록 채굴
→ Super Glue로 상자 연결 → 채굴 아이템 자동 수거
→ 레드스톤으로 왕복 자동화
```

**자동 농장:**
```
Deployer + Harvester를 피스톤에 부착
→ 왕복 이동하며 자동 파종 + 수확
```

---

### 🚂 Cart Assembler — 기차/이동 컨트랩션

레일 위에서 이동하는 컨트랩션입니다.

**자동 채굴기차:**
```
Cart Assembler 위에 구조물 조립
→ Drill, 상자, Funnel 등 부착
→ 레일 위에서 전진하며 자동 채굴
→ 상자에 채굴물 자동 수거
```

> **참고: 기차 시스템은 황동 이후**
>
> 본격적인 기차 시스템은 황동으로 만드는 Train Track과 Station이 필요합니다.
> Cart Assembler는 기초 이동 컨트랩션 전용으로 먼저 활용하고,
> 기차 시스템은 중반 이후 별도로 구축하는 것을 권장합니다.

---

### ⬆️ Rope Pulley — 수직 이동

줄처럼 위아래로 이동합니다.

**엘리베이터:**
```
Rope Pulley 위에 플랫폼 구조물 연결
→ 레드스톤으로 올리고 내리기
```

**자동 광산 엘리베이터:**
```
Rope Pulley에 플랫폼 + 상자 부착
→ 지하에서 채굴한 아이템을 지상으로 자동 운반
```

---

## 컨트랩션 제작 팁

### Super Glue 활용

Super Glue는 두 블록을 고정 연결합니다.

```
사용법:
  Super Glue 손에 들기
  → 연결할 두 블록 면을 순서대로 우클릭
  → 연결 완료 (주황색 표시)
  → 다시 우클릭으로 연결 해제
```

### 컨트랩션 크기 제한

컨트랩션 하나에 포함될 수 있는 블록 수에는 기본 제한이 있습니다.
(기본 설정 기준 최대 약 2,048블록)

### 무거운 컨트랩션 = 높은 SU 소비

컨트랩션이 클수록 이동에 필요한 SU가 증가합니다.
과부하가 발생하면 컨트랩션 크기를 줄이거나 동력을 증가시키세요.

---

## 자주 발생하는 문제

**컨트랩션이 조립되지 않아요.**
→ 모든 블록이 앵커와 연결(Glue/Chassis/슬라임)되어 있는지 확인하세요.

**컨트랩션이 이동 중 멈춰요.**
→ 이동 경로에 장애물 블록이 있습니다. 경로를 확인하세요.
→ SU 과부하 가능성도 확인하세요.

**컨트랩션이 분해되지 않아요.**
→ Bearing/Piston의 이동 모드(Movement Mode) 설정을 확인하세요.
→ 우클릭 길게 누르기 → Value Settings에서 이동 모드 변경할 수 있습니다.
