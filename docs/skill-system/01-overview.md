# 스킬 시스템 개요

이 모드팩의 스킬 성장 시스템은 세 개의 모드가 **층층이 연결된 구조**로 작동합니다.

---

## 구조 한눈에 보기

| 단계 | 역할 |
|------|------|
| puffish_skills | 경험치, 레벨, 스킬 포인트, 스킬 해금의 기반 엔진 |
| default_skill_trees | 모든 플레이어가 쓰는 전투/채굴 기본 트리 |
| skill_tree (RPG Series) | 직업 선택 후 해금되는 직업 전용 고급 트리 |
| puffish_attributes | 스킬 트리 투자 결과를 실제 스탯 수치로 반영 |

진행 흐름은 `경험치 획득 → 스킬 포인트 획득 → 기본/직업 트리 투자 → 스탯 반영` 순서입니다.

---

## 각 모드의 역할

### puffish_skills — 스킬 시스템의 기반 엔진

모든 스킬 시스템의 핵심입니다.

- 경험치를 쌓으면 스킬 레벨이 오르고, 레벨업 시 스킬 포인트 획득
- 스킬 포인트를 스킬 트리의 노드에 투자해서 능력 강화
- 스킬 창 키는 `설정 → 키 바인딩`에서 `skill` 또는 `puffish`로 검색해 확인

### default_skill_trees — 기본 스킬 트리

직업과 무관하게 **모든 플레이어**가 공통으로 사용하는 두 가지 트리입니다.

- **전투 트리:** 체력, 공격력, 방어력, 이동 속도 등
- **채굴 트리:** 채굴 속도, 행운

### skill_tree (RPG Series) — 직업 전용 스킬 트리

직업 모드(Paladins, Rogues, Wizards 등)를 선택한 후 해금되는 고급 트리입니다.

- 단순 스탯 증가가 아닌 **직업 고유 패시브, 주문 강화** 노드 포함
- 100개 이상의 노드로 구성
- 직업별로 **공격 특화 / 방어 특화** 두 방향의 갈림길 존재

### puffish_attributes — 스탯 수치 처리

스킬 트리에서 투자한 포인트가 **실제 수치로 반영**되도록 처리합니다.

- 크리티컬 확률, 공격 속도, 이동 속도 등 세분화된 스탯 추가
- Apotheosis Affix 장비의 스탯과 **합산**됩니다

---

## 스킬 창 열기

```
설정 → 키 바인딩 → skill 또는 puffish 검색
```

모드팩 설정이나 개인 설정에 따라 키가 달라질 수 있으므로, 인게임 키 바인딩 화면에 표시되는 값을 기준으로 사용하세요.

스킬 창을 열면:
- 왼쪽: 전투 트리 / 채굴 트리 탭
- 직업 선택 후: 직업 전용 트리 탭 추가

---

## 전체 진행 흐름

```
게임 시작
    ↓
전투 & 채굴로 경험치 획득
    ↓
스킬 레벨업 → 포인트 획득
    ↓
default_skill_trees에 포인트 투자 (전투/채굴 기본 강화)
    ↓
직업 모드에서 직업 선택
    ↓
skill_tree RPG Series 트리 해금
    ↓
두 트리를 병행 성장시키며 캐릭터 완성
```

> **참고: 포인트 배분이 틀려도 괜찮습니다**
>
> 스킬 창 우상단의 **리셋 버튼**으로 언제든지 전체 포인트를 환불받을 수 있습니다.
> 실험적으로 찍어보고 마음에 안 들면 리셋하는 것을 두려워하지 마세요.
