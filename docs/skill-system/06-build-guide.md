# 빌드 가이드

스킬 트리, Apotheosis 장비, 능력치를 어떻게 조합하면 좋을지
단계별 성장 로드맵과 빌드 방향을 제시합니다.

---

## 전체 성장 단계

```
[1단계] 게임 시작 \~ 첫 던전
  목표: 기본 생존 능력 확보

[2단계] 던전 탐험 \~ 직업 선택
  목표: 기본 트리 어느 정도 투자, 플레이 스타일 결정

[3단계] 직업 선택 \~ Apotheosis Frontier 티어
  목표: 직업 트리 해금, 기본 빌드 방향 확정

[4단계] Frontier \~ Summit 티어
  목표: Epic 장비 맞추기, 스킬 트리 핵심 노드 완성

[5단계] Summit \~ Pinnacle 티어
  목표: 최종 빌드 완성, Mythic 장비 파밍
```

---

## 1단계: 게임 시작 \~ 첫 던전

### 목표

- 생존 가능한 최소 스탯 확보
- 기본 채굴/전투 루틴 확립

### 스킬 트리 투자

```
전투 트리:
  ✅ 최대 체력 I, II (첫 포인트는 여기)
  ✅ 근접 공격력 I

채굴 트리:
  ✅ 채굴 속도 I (필요시)
```

### 장비 목표

- Apotheosis 오버월드 타워에서 Starter Affix 아이템 수집
- World Tier: Haven 활성화

---

## 2단계: 던전 탐험 \~ 직업 선택

### 목표

- 어떤 전투 스타일이 재미있는지 파악
- 직업 선택을 위한 준비

### 스킬 트리 투자

```
전투 트리:
  ✅ 최대 체력 II, III
  ✅ 근접 또는 원거리 공격력 II
  ✅ 방어력 I

채굴 트리:
  ✅ 채굴 속도 II, III
  ✅ 채굴 행운 I
```

### 이 단계에서 결정해야 할 것

**근접 vs 원거리 vs 마법** 중 어떤 스타일로 갈 것인지 결정합니다.
직업 선택의 기준이 됩니다.

| 플레이 스타일 | 추천 직업 |
|--------------|-----------|
| 근접 강타형 | Berserker, Witcher |
| 근접 탱커형 | Paladin |
| 원거리 | Archer, Rogue |
| 마법 공격 | Wizard, Elemental Wizard |
| 마법 지원 | Paladin (힐/버프) |
| 올라운더 | Rogue, Forcemaster |

---

## 3단계: 직업 선택 \~ Frontier 티어

### 목표

- 직업 트리 핵심 노드 빠르게 해금
- Uncommon 풀세트 완성 → Frontier 해금

### 스킬 트리 투자

```
직업 트리 우선:
  ✅ 직업 기본 노드 전부 해금
  ✅ 특화 갈림길 방향 결정
  ✅ 특화 초반 노드 2\~3개 해금

전투 트리 병행:
  ✅ 직업과 시너지 있는 노드 위주로 투자
```

### 특화 방향 선택 기준

```
혼자 플레이하는 경우:
  → 공격 특화 권장 (처치 속도가 중요)

파티 플레이하는 경우:
  → 탱커/힐러는 방어 특화 고려
```

---

## 4단계: Frontier \~ Summit 티어

### 목표

- 직업 트리 특화 노드 핵심 완성
- Rare → Epic 장비로 업그레이드
- 크리티컬 시스템 완성

### 스킬 트리 투자

```
직업 트리:
  ✅ 특화 핵심 패시브 노드 해금
  ✅ 주문 강화 노드 (마법 직업)

전투 트리:
  ✅ 크리티컬 확률 → 80% 이상 목표
  ✅ 크리티컬 데미지 증가
```

### Apotheosis 장비와의 시너지

이 단계에서 스킬 트리와 장비의 조합이 중요해집니다.

```
예시: 근접 딜러 빌드
  - Affix 장비: Cleaving + Life Steal 위주
  - 젬: Ruby (공격력) + Amethyst (크리티컬)
  - 스킬 트리: 크리티컬 특화
  결과: 범위 공격으로 여러 적을 한 번에 처치하며 체력 회복
```

---

## 5단계: Summit \~ Pinnacle 티어 (최종)

### 목표

- 스킬 트리 완성 (모든 원하는 노드 해금)
- Mythic 장비 파밍
- 능력치 최적화

### 스킬 트리 투자

```
모든 트리:
  ✅ 남은 포인트로 시너지 노드 채우기
  ✅ 크리티컬 확률 100% 달성 후
     → 크리티컬 데미지 극대화로 전환
```

## 빌드 완성도 자가 체크

아래 항목을 체크해 빌드 완성도를 확인하세요.

### 생존 체크

- [ ] 최대 체력 40 이상
- [ ] Apotheosis Epic 풀세트 완성
- [ ] 방어구 젬 세팅 완료

### 공격 체크

- [ ] 크리티컬 확률 80% 이상
- [ ] 직업 주력 스킬 강화 노드 해금
- [ ] 무기 Affix가 직업 스타일과 매칭

### 유틸 체크

- [ ] 이동 속도 불편하지 않은 수준
- [ ] 스킬 쿨다운 관리 가능
- [ ] 포션/회복 수단 확보
