# 능력치 연동 (puffish_attributes)

`puffish_attributes`는 스킬 트리에서 투자한 포인트를
**실제 게임 수치로 반영**하는 역할을 합니다.

또한 바닐라에 없는 다양한 세분화된 능력치를 새로 추가해
RPG다운 캐릭터 빌딩이 가능하게 해줍니다.

---

## 능력치 확인 방법

### 캐릭터 스탯 창

인벤토리를 열면 캐릭터 탭에서 전체 능력치를 확인할 수 있습니다.

- 각 능력치의 **현재 수치** 표시
- 항목 위에 마우스를 올리면 어디서 수치가 왔는지 소스 표시

### 아이템 툴팁

장비에 마우스를 올리면 착용 시 변화하는 능력치가 표시됩니다.

```
예시:
+5  공격력
+1.2  공격 속도
+3%  크리티컬 확률
-2  최대 체력  (단점)
```

`+`는 증가, `-`는 감소를 의미합니다.

## 추가 능력치 전체 목록

### 공격 능력치

| 능력치 | 기본값 | 설명 |
|--------|--------|------|
| **근접 공격력** | 직업/장비에 따라 다름 | 근접 공격 기본 데미지 |
| **원거리 공격력** | 직업/장비에 따라 다름 | 활, 석궁 등 원거리 데미지 |
| **공격 속도** | 4.0 | 초당 공격 횟수 |
| **크리티컬 확률** | 5% | 치명타 발동 확률 |
| **크리티컬 데미지** | 150% | 치명타 시 데미지 배율 |

### 방어 능력치

| 능력치 | 기본값 | 설명 |
|--------|--------|------|
| **최대 체력** | 20 | 최대 HP (하트 10개) |
| **방어력 (Armor)** | 0 | 받는 피해 % 감소 |
| **방어 강도 (Toughness)** | 0 | 강한 타격에 대한 추가 방어 |
| **넉백 저항** | 0% | 밀려남 감소 비율 |

### 이동 & 유틸 능력치

| 능력치 | 기본값 | 설명 |
|--------|--------|------|
| **이동 속도** | 0.1 | 걷기/달리기 속도 |
| **행운** | 0 | 드롭 및 인챈트 보너스 |
| **점프 높이** | 0.42 | 점프 최대 높이 |

## 능력치 계산 방식

능력치는 여러 소스가 **합산**됩니다.

```
최종 수치 = 기본값
          + 장비 Affix 보너스
          + 젬 보너스
          + 스킬 트리 보너스
          + 인챈트 보너스
          + 포션 효과 (임시)
```

예를 들어 크리티컬 확률이 27%라면:
- 기본 5%
- Affix 장비 +8%
- 젬 소켓 +6%
- 스킬 트리 +8%

이렇게 합산된 결과입니다.

## 능력치 상한선과 효율

### 크리티컬 확률 100% 달성 후

크리티컬 확률이 100%를 넘으면 모든 공격이 치명타입니다.
이 지점에서 **크리티컬 확률 투자를 중단**하고 크리티컬 데미지에 집중하세요.

```
크리티컬 확률 100% 도달 전: 확률 > 데미지
크리티컬 확률 100% 달성 후: 확률 투자 불필요, 데미지만 올리기
```

### 공격 속도 상한

공격 속도는 너무 높아지면 오히려 조작이 어려워집니다.
`0.4\~0.6` 수준이 전투에서 실용적인 범위입니다.

### 이동 속도 주의

이동 속도가 너무 높으면 조작이 어려워지고 낙사 위험이 증가합니다.
`0.2\~0.35` 수준이 적절합니다.

## 스킬 트리 투자의 실제 효과

스킬 창에서 노드를 찍으면 어떻게 반영되는지 예시입니다.

```
[스킬 창] 근접 공격력 증가 II 노드 해금
    ↓
[puffish_attributes] 근접 공격력 +8% 적용
    ↓
[인벤토리 스탯 창] 공격력 수치 실시간 업데이트
```

투자 후 인벤토리 스탯 창에서 수치 변화를 바로 확인할 수 있습니다.

## 능력치 투자 우선순위 가이드

### 근접 전투 빌드

```
우선순위: 최대 체력 → 공격력 → 크리티컬 확률 → 크리티컬 데미지 → 공격 속도
목표: 크리티컬 확률 80\~100%, 크리티컬 데미지 200% 이상
```

### 원거리/마법 빌드

```
우선순위: 최대 체력 → Spell Power 계열 → 크리티컬 확률 → 이동 속도
목표: 안전한 거리 유지를 위한 이동 속도 확보
```

### 탱커 빌드

```
우선순위: 최대 체력 → 방어력 → 방어 강도 → 넉백 저항
목표: 최대 체력 50+, 방어력 최대치 접근
```

## 자주 묻는 질문

**Q: 스탯 창에서 소수점 수치가 보이는데 정상인가요?**
A: 정상입니다. 여러 소스의 수치가 합산되기 때문입니다. 정확한 수치보다 방향성을 보세요.

**Q: 이동 속도가 너무 빨라서 조작하기 어렵습니다.**
A: 이동 속도 노드 투자를 줄이거나, 이동 속도 감소 효과가 있는 장비로 조절할 수 있습니다.

**Q: Apotheosis 능력치와 스킬 트리 능력치 중 어느 쪽이 더 중요한가요?**
A: 둘 다 합산되므로 어느 한쪽이 더 중요하다고 할 수 없습니다. Apotheosis Affix 장비로 기반을 만들고, 스킬 트리로 부족한 부분을 보완하는 방식이 가장 효율적입니다.
