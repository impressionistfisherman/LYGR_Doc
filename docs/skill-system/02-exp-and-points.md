# 경험치와 스킬 포인트

스킬 포인트는 스킬 트리의 노드를 해금하는 데 사용됩니다.  
포인트는 각 트리마다 **독립적으로 관리**됩니다.

---

## 중요: 트리별 포인트 독립

가장 처음에 알아야 할 핵심 사항입니다.

```
⚔️ 전투 트리 포인트   ←→   ⛏️ 채굴 트리 포인트   ←→   🏹 직업 트리 포인트
       (별개)                    (별개)                    (별개)
```

**전투만 한다고 채굴 트리 포인트가 쌓이지 않습니다.**  
각 트리는 해당 트리와 관련된 행동으로만 경험치가 쌓입니다.

---

## 경험치 획득 방법

### ⚔️ 전투 트리

| 행동 | 경험치 |
|------|--------|
| 몬스터 처치 | ★★★ 주요 획득처 |
| 보스 처치 | ★★★★ 대량 획득 |
| Apotheosis Invader 처치 | ★★★★ |
| 던전 클리어 | ★★★ |

### ⛏️ 채굴 트리

| 행동 | 경험치 |
|------|--------|
| 광석 채굴 | ★★★ 주요 획득처 |
| 돌류 블록 채굴 | ★★ |
| 나무 벌채 | ★★ |

### 🏹 직업 트리

직업마다 경험치 획득 조건이 다릅니다.

| 직업 | 주요 경험치 행동 |
|------|-----------------|
| Paladin | 적 처치, 아군 치유 |
| Rogue | 백스탭, 스텔스 처치 |
| Wizard | 주문 시전, 적 처치 |
| Berserker | 피해 주고받기 |
| Archer | 원거리 처치 |

## 레벨업 구조

```
경험치 축적
    ↓
스킬 레벨 1 상승
    ↓
스킬 포인트 1\~2개 획득
    ↓
스킬 창(K)에서 원하는 노드에 투자
```

초반에는 레벨업이 빠르지만 점점 필요 경험치가 증가합니다.  
초반에 기본 노드들을 빠르게 찍고, 중반부터 고급 노드에 투자하는 흐름이 일반적입니다.

---

## 포인트 환불 (리셋)

### 리셋 방법

스킬 창(K) 우상단에 **리셋 버튼**이 있습니다.

- 해당 트리의 포인트가 **전부 환불**됩니다
- 직업 트리 리셋 시 직업 특화 방향도 초기화됩니다
- 리셋 비용이 있을 수 있습니다 (모드팩 설정에 따라 다름)

:::tip 리셋을 활용하세요
처음부터 완벽한 빌드를 만들 필요가 없습니다.  
일단 여러 노드를 찍어보고, 플레이해보면서 마음에 안 들면 리셋하는 것이 가장 빠른 학습법입니다.
:::

## 초반 포인트 투자 우선순위

### 전투 트리 (초반)

생존을 위해 체력부터 올리는 것을 권장합니다.

```
1. 최대 체력 노드 (1\~3개) — 생존 기반 확보
2. 공격력 노드 (2\~3개)   — 처치 속도 향상
3. 방어력 노드 (1\~2개)   — 피해 경감
```

크리티컬, 공격 속도 등 고급 노드는 중반 이후에 투자해도 늦지 않습니다.

### 채굴 트리 (초반)

채굴을 자주 한다면 채굴 속도 노드를 먼저 찍는 것이 체감 효과가 큽니다.

```
1. 채굴 속도 노드 (2\~3개) — 일반 블록 채굴 속도 향상
2. 행운 노드 (1\~2개)      — 광석 추가 드롭 확률 증가
```

### 직업 트리 (직업 선택 후)

직업을 선택한 직후에는 직업 트리에 우선 투자하는 것을 권장합니다.

- 기본 트리는 언제든 쌓이지만 직업 트리는 직업 전용 능력을 빠르게 열어야 플레이가 재미있어짐
- 직업 트리 안에 있는 **주요 스킬 노드**를 먼저 목표로 삼고 포인트를 집중 투자

## 경험치 파밍 팁

스킬 레벨업이 막혔을 때 경험치를 빠르게 모으는 방법입니다.

### 전투 경험치

- **Apotheosis 스포너 팜** 구축 (07-spawners.md 참고)
- 던전 반복 탐험
- Apotheosis Invader 처치 (레벨당 경험치가 많음)

### 채굴 경험치

- **Silent Gear 해머(AOE 곡괭이)** 로 광맥 대량 채굴
- Create 모드 자동 채굴 장치 (중반 이후)
- 대규모 지하 채굴 세션

### 직업 경험치

직업별로 최적 경험치 방법이 다릅니다.

- **마법 직업 (Wizard 등):** 주문을 자주 시전하면서 전투
- **근접 직업 (Berserker 등):** 일반 전투 반복
- **힐러 (Paladin):** 아군이 있을 때 치유 주문 적극 활용
