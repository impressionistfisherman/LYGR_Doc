# 직업 전용 스킬 트리 (RPG Series)

`skill_tree (RPG Series)` 모드가 직업 모드와 연동해  
각 직업마다 **전용 고급 스킬 트리**를 제공합니다.

:::info 직업 선택 후 해금
직업 전용 트리는 해당 직업 모드에서 직업 아이템을 습득해야 해금됩니다.  
직업 선택 전에는 스킬 창(K)에 직업 탭이 표시되지 않습니다.
:::

---

## 기본 트리와의 차이점

| 항목 | 기본 트리 (default_skill_trees) | 직업 트리 (skill_tree RPG) |
|------|--------------------------------|---------------------------|
| 내용 | 체력, 공격력, 방어력 등 수치 증가 | 직업 고유 패시브, 주문 강화 |
| 노드 수 | 약 20~30개 | **100개 이상** |
| 특화 방향 | 없음 | 공격형 / 방어형 갈림길 |
| 해금 조건 | 처음부터 | 직업 선택 후 |

## 직업 트리 구조

### 공통 구성

모든 직업의 스킬 트리는 기본적으로 아래 구조를 따릅니다.

```
[기본 노드들]
    ↓
[중간 노드들]
    ↓
[특화 갈림길]
  ↙         ↘
[공격 특화]  [방어/지원 특화]
    ↓               ↓
[특화 전용 노드들] [특화 전용 노드들]
```

### 기본 노드 (모든 직업 공통 구역)

트리 초반부에 있는 기본 강화 노드들입니다.

- 직업 관련 기본 스탯 증가 (공격력, 체력 등)
- 직업 고유 스킬의 기본 효과 강화
- 특화 갈림길로 진입하기 위한 전제 조건 노드

### 특화 갈림길 (Specialisation)

트리 중반부에서 **공격형 또는 방어/지원형** 중 하나를 선택합니다.

```
⚔️ 공격 특화
  - 데미지 증가에 집중
  - 공격적인 스킬 강화 노드
  - 단, 생존력이 상대적으로 낮아짐

🛡️ 방어/지원 특화
  - 체력, 방어력, 생존력에 집중
  - 아군 지원 관련 노드 (일부 직업)
  - 단, 딜이 상대적으로 낮아짐
```

:::warning 특화 후 반대 방향 노드는 해금 불가
한쪽 특화를 선택하면 **반대 방향 노드는 포인트를 쓸 수 없습니다.**  
리셋을 통해 다른 방향으로 전환할 수 있으므로, 먼저 양쪽 트리를 스킬 창에서 미리 살펴보고 결정하는 것을 권장합니다.
:::

## 노드 종류 설명

### 스탯 노드

가장 기본적인 형태입니다.

```
예시: "공격력 +8%", "최대 체력 +5", "크리티컬 확률 +3%"
```

수치를 직접 올려주는 단순하지만 중요한 노드입니다.

### 패시브 스킬 노드 (Passive Skill)

게임 메커니즘 자체를 바꿔주는 특수 노드입니다.

```
예시:
- "적을 처치하면 잠시 이동 속도 증가"
- "체력이 30% 이하일 때 방어력 대폭 증가"
- "백스탭 공격 시 추가 피해"
```

같은 직업이라도 어떤 패시브 노드를 찍느냐에 따라 플레이 방식이 완전히 달라집니다.

### 주문 강화 노드 (Spell Modifier)

마법 직업(Wizard, Paladin 등)에만 있는 특수 노드입니다.

```
예시:
- "파이어볼 적중 시 화염 지속 시간 +2초"
- "힐 주문의 치유량 +20%"
- "아이스 볼트가 적중 시 슬로우 효과 추가"
```

같은 주문을 쓰더라도 이 노드들을 통해 스킬을 **자신만의 방식으로 커스터마이징**할 수 있습니다.

## 직업별 트리 특징 요약

| 직업 | 공격 특화 방향 | 방어 특화 방향 |
|------|---------------|---------------|
| **Paladin** | 성스러운 데미지 강화 | 방어막, 힐 강화 |
| **Rogue** | 크리티컬, 백스탭 극대화 | 회피, 독 강화 |
| **Wizard** | 광역 주문 데미지 | 방어막, 마나 효율 |
| **Berserker** | 분노 스택 극대화 | 피격 반사, 체력 |
| **Archer** | 원샷 특화, 관통 | 罠(트랩), 서포트 화살 |
| **Witcher** | 징표 + 검 콤보 | 회피, 물약 강화 |
| **Elemental Wizard** | 원소 폭발 데미지 | 원소 방벽 |

각 직업의 상세 스킬 트리 설명은 직업별 문서가 추가되면 별도 섹션에서 다룹니다.

## 트리 미리보기 방법

직업 선택 전에 어떤 스킬 트리가 있는지 미리 볼 수 있습니다.

1. 스킬 창(K) 열기
2. 직업 탭 선택 (직업 선택 전에는 잠겨 있지만 미리보기 가능)
3. 각 노드 위에 마우스를 올려 효과 확인

:::tip 직업 선택 전에 트리를 먼저 보세요
어떤 스킬 트리가 마음에 드는지 먼저 확인하고 직업을 선택하는 것도 좋은 방법입니다.  
직업의 외형이나 이름보다 실제 플레이를 결정하는 **스킬 트리**가 더 중요할 수 있습니다.
:::

## 자주 묻는 질문

**Q: 직업 트리 포인트는 기본 트리 포인트와 공유되나요?**  
A: 아닙니다. 직업 트리 포인트는 직업 관련 행동으로 별도로 획득합니다.

**Q: 직업을 바꾸면 직업 트리 포인트는 어떻게 되나요?**  
A: 이전 직업의 트리 포인트는 유지되지 않습니다. 새 직업의 트리를 처음부터 쌓아야 합니다.

**Q: 특화 방향을 잘못 골랐어요.**  
A: 스킬 창의 리셋 버튼으로 직업 트리를 초기화하면 다시 선택할 수 있습니다.
