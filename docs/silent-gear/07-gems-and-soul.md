# Silent's Gems & Gear Soul

Silent's Gems는 Silent Gear와 함께 사용하는 보석 소재 모드입니다.
16종의 보석을 채굴해 장비 재료로 활용하고, Gear Soul로 장비를 장기 성장시킬 수 있습니다.

---

### Gem Bag (젬 가방) — 필수 아이템

보석 채굴 시 자동 수집해주는 가방입니다.

**제작 방법:** JEI에서 `Gem Bag` 검색 → 초반에 바로 제작 권장

**기능:**
- 인벤토리에 있으면 채굴한 보석이 자동으로 가방에 수납
- 인벤토리 공간 크게 절약
- 가방 하나에 보석 여러 종류를 종류별로 자동 분류

### 보석 채굴 팁

```
오버월드 보석:
  → 일반 광석과 같은 깊이에서 발견 (Y=-50 이하 권장)
  → Prospector's Hammer로 위치 미리 파악
  → 보석마다 생성 깊이와 바이옴이 다름

Black Diamond (네더):
  → 네더 깊은 층에서 발견
  → 바스틸라/네더 요새 탐험 중 발견 빈도 높음

White Diamond (엔드):
  → 엔드 섬 탐험 중 발견
  → 엔더 드래곤 처치 후 탐험 권장
```

## Gear Soul 시스템

### Gear Soul이란?

Silent Gear 도구/무기에 장착하는 **장기 성장 아이템**입니다.

- 장비를 사용하면 소울이 경험치를 쌓고 레벨업
- 레벨업 시 추가 특성(Trait)이 자동 부여
- 같은 재료로 만든 장비라도 소울 레벨에 따라 성능 차이 발생

### Gear Soul 제작 방법

**Step 1 — Soul Gem 획득**

몬스터를 처치하면 일정 확률로 Soul Gem이 드롭됩니다.

```
주의: 자동 파밍(스포너 트랩 등)으로 처치한 몬스터에서는 드롭되지 않습니다.
      반드시 플레이어가 직접 처치해야 합니다.
```

Soul Gem은 **원소(Element) 속성**이 있으며, 세계마다 랜덤 배정됩니다.
어떤 몬스터에서 어떤 원소의 Soul Gem이 드롭되는지 초반에 파악해두면 파밍 목표가 생깁니다.

**Step 2 — Soul Gem → Gear Soul 합성**

Soul Gem 여러 개를 조합해 Gear Soul을 만듭니다.

```
JEI에서 "Gear Soul" 검색 → 조합 비율 확인
```

**Step 3 — Gear Soul 장착**

```
제작대:
  [ Silent Gear 장비 ] + [ Gear Soul ]
  → 소울이 장착된 장비 완성
```

## Gear Soul 레벨업

### 경험치 획득 방법

| 도구/무기 종류 | 경험치 획득 행동 |
|---------------|-----------------|
| **무기** | 몬스터 처치 |
| **채굴 도구** | 블록 채굴, 나무 벌채 |
| **방어구** | 피격 (데미지 받을 때) |

### 레벨업 시 부여되는 특성

레벨이 오를 때마다 랜덤으로 특성이 추가됩니다.

```
레벨 1 → 특성 1개 추가
레벨 2 → 특성 1개 추가
...
```

부여되는 특성은 Soul Gem의 **원소 속성**에 따라 달라집니다.

### 원소별 특성 방향

| 원소 | 주로 부여되는 특성 |
|------|-------------------|
| 불(Fire) | 화염 피해, 내화성 관련 |
| 물(Water) | 수중 관련, 회복 관련 |
| 땅(Earth) | 채굴 관련, 내구도 관련 |
| 바람(Air) | 이동속도, 공격속도 관련 |
| 번개(Lightning) | 크리티컬, 범위 관련 |

---

## 실전 Gear Soul 전략

### 어떤 원소를 골라야 할까?

```
채굴 도구: 땅(Earth) 원소 → 채굴 관련 특성 위주
근접 무기: 번개(Lightning) → 크리티컬 관련 특성
방어구: 물(Water) 또는 땅(Earth) → 회복, 내구도 관련
```

### 파밍 효율화

Soul Gem 파밍에 효율적인 몬스터를 파악한 뒤
Apotheosis 스포너 팜에서 **플레이어 직접 처치** 방식으로 대량 파밍하는 것이 가장 효율적입니다.

> **정보: 스포너 파밍과 Soul Gem**
>
> Soul Gem은 자동 파밍으로 드롭되지 않습니다.
> 낙하 트랩으로 체력 1을 만든 뒤 플레이어가 1타로 마무리하는 방식을 사용하세요.
