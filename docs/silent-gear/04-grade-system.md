# 등급 시스템 (Grade)

같은 재료라도 **등급(Grade)에 따라 최종 스탯이 달라집니다.**  
최고 장비를 만들려면 재료 등급을 올리는 과정이 필수입니다.

---

## 등급 목록

| 낮음 | | | | | | | | 높음 |
|------|-|-|-|-|-|-|-|------|
| E | D | C | B | A | S | SS | SSS | MAX |

| 등급 | 스탯 배율 | 획득 난이도 |
|------|-----------|-------------|
| **E** | \~70% | 최저 등급, 쉬움 |
| **D** | \~80% | — |
| **C** | \~90% | — |
| **B** | \~95% | 기본 등급 |
| **A** | 100% | 기본 목표 |
| **S** | \~110% | 중후반 목표 |
| **SS** | \~125% | 최종 장비 목표 |
| **SSS** | \~140% | 매우 희귀 |
| **MAX** | 최대 | 극희귀 |

:::info 초반 가이드
초반에는 등급을 신경 쓰지 않아도 됩니다.  
**B\~A 등급**으로도 던전 탐험에 충분하며,

최종 장비를 만들 준비가 됐을 때 S\~SS를 목표로 하세요.
:::

## Material Grader (재료 등급기)

재료의 등급을 올리는 전용 블록입니다.

### 제작 방법

JEI에서 `Material Grader`를 검색 → 조합법 확인 후 제작

### 기본 사용 방법

1. Material Grader 설치 후 우클릭으로 GUI 열기
2. 왼쪽 슬롯에 **등급을 올릴 재료** 투입
3. 오른쪽 슬롯에 **촉매(Catalyst)** 투입
4. 처리 완료 → 랜덤 등급의 재료 획득

---

## 촉매 (Catalyst)

촉매의 티어가 높을수록 높은 등급이 나올 확률이 올라갑니다.

### 촉매 종류

| 촉매 | 티어 | 높은 등급 확률 |
|------|------|---------------|
| **Stone Catalyst** | 1 (낮음) | A 이상 낮음 |
| **Iron Catalyst** | 2 (보통) | A 도달 가능 |
| **Diamond Catalyst** | 3 (높음) | S 도달 가능 |
| **Netherite Catalyst** | 4 (최고) | SS 이상 가능 |

### 촉매 제작 방법

JEI에서 `Catalyst`를 검색하면 각 티어별 조합법이 표시됩니다.

:::warning 촉매는 1회 소모
촉매는 사용할 때마다 1개씩 소모됩니다.  
비싼 Netherite Catalyst는 최종 장비 재료에만 사용하세요.
:::

## 등급 재도전 방법

이미 등급이 붙은 재료도 다시 Grader에 넣어 재등급을 시도할 수 있습니다.

```
낮은 등급 재료 → Material Grader → 새 등급 부여 (촉매 소모)
```

### 재도전 전략

- 목표 등급(S 이상)이 나올 때까지 반복
- 촉매 여유분이 충분할 때 진행
- Diamond Catalyst로 S를 목표로 → S 달성 후 Netherite로 SS 도전

---

## 등급에 따른 실제 성능 차이

예시: 철 Head Part 기준

| 등급 | 채굴 속도 | 내구도 | 공격력 |
|------|-----------|--------|--------|
| E | 4.2 | 294 | 2.1 |
| B | 5.7 | 399 | 2.85 |
| A | 6.0 | 420 | 3.0 |
| S | 6.6 | 462 | 3.3 |
| SS | 7.5 | 525 | 3.75 |

E등급과 SS등급의 차이가 **약 80%** 에 달합니다.  
최종 장비에서 등급이 얼마나 중요한지 알 수 있습니다.

## 효율적인 등급 파밍 전략

### 초반 전략
```
촉매 없이 기본 등급으로 장비 사용
→ 실용성 확보가 우선, 등급은 나중에
```

### 중반 전략
```
Iron Catalyst 다량 제작
→ 주요 재료(Head Part용) A등급 달성 목표
→ 나머지 파트(Rod, Tip)는 B\~A로 충분
```

### 후반 전략
```
Diamond Catalyst로 주요 재료 S등급 목표
→ 최종 무기/도구 Head Part 우선
→ 방어구는 Main Part 위주로 S 달성
```

### 최종 단계
```
Netherite Catalyst 제작
→ SS\~SSS 도전
→ 최고 성능 재료 확보 시 최종 장비 완성
```

## 재료 대량 확보 팁

등급 파밍은 재료를 많이 소모합니다.

- **Silent Gear 해머 (AOE 곡괭이)** 제작 후 대규모 채굴로 재료 확보
- Create 모드 자동 채굴 장치 활용 (중반 이후)
- Silent's Gems 보석의 경우 Gem Bag 제작 후 채굴 권장
