# 파트 교체 & 수리

Silent Gear의 가장 큰 장점 중 하나는
**장비를 버리지 않고 부분 업그레이드**할 수 있다는 점입니다.

---

## 파트 교체

### 교체란?

이미 완성된 장비의 파트를 더 좋은 재료로 만든 파트로 바꾸는 것입니다.

```
[기존] Iron Head + Wood Rod 곡괭이
            ↓  Diamond Head 준비
[이후] Diamond Head + Wood Rod 곡괭이
   (Wood Rod는 그대로 유지)
```

### 교체 방법

1. 교체할 파트를 새 재료로 미리 제작
2. **기존 장비 + 새 파트**를 제작대에 함께 올리기
3. 결과 슬롯에 파트가 교체된 장비 완성

```
제작대:
  [ 기존 곡괭이 ] [ 새 Diamond Head ]
  → Diamond Head로 교체된 곡괭이
```

### 교체 시 주의사항

> **주의: 기존 파트는 반환되지 않습니다**
>
> 교체 시 기존 파트는 사라집니다.
> 비싼 재료(Black Diamond 등)로 만든 파트를 교체할 때는 신중하게 결정하세요.

- 인챈트는 유지됩니다
- 내구도는 새 파트 기준으로 재계산됩니다
- Gear Soul은 유지됩니다

### 교체 우선순위 권장

```
초반 → 중반 업그레이드:
  Head Part 교체 우선 (성능 영향 가장 큼)
  → Rod Part는 나중에 교체해도 됨

중반 → 후반 업그레이드:
  재료 등급이 오르면 Head + Rod 모두 교체
  → 최고 성능 조합으로 완성
```

## 수리

### 방법 1 — 모루 수리 (권장)

제작대나 모루에서 해당 장비의 **주재료**로 수리할 수 있습니다.

```
모루:
  [ 수리할 장비 ] + [ 재료 ]
  → 내구도 회복
```

- 재료 1개당 내구도 일정량 회복
- 어떤 재료로 수리하는지는 Head Part 재료와 같음
- Apotheosis가 있으면 모루 수리 횟수 제한 없음

### 방법 2 — Mending 인챈트

Mending 인챈트가 적용된 장비는 **경험치 획득 시 자동으로 내구도 회복**됩니다.

- 경험치가 내구도 회복에 사용되므로 레벨업이 느려질 수 있음
- 파밍 중에는 유용하지만, 경험치가 필요한 상황(스킬 레벨업 등)에서는 단점

### 방법 3 — Renew / Lustrous 특성

- **Renew:** 시간이 지나면 내구도 자동 회복
- **Lustrous:** 경험치를 얻을 때마다 내구도 소량 회복

재료 선택 시 이 특성이 있는 재료를 Rod나 Tip에 사용하면
장기적으로 수리 걱정을 크게 줄일 수 있습니다.

## 장비 분해

더 이상 사용하지 않는 Silent Gear 장비는 분해해서 재료를 일부 회수할 수 있습니다.

### 방법

제작대에 장비를 혼자 올리면 분해 옵션이 나타납니다.
(일부 버전에서는 별도 분해 테이블 사용)

### 회수되는 것

- 파트 재료 일부 (전량 회수 불가)
- 내구도가 낮을수록 회수량 감소

---

## 업그레이드 전체 흐름

```
게임 초반:
  기본 재료(철, 나무)로 장비 제작
  → 일단 쓰면서 진행

중반:
  더 좋은 재료 확보 (다이아, 보석)
  → Head Part 우선 교체
  → Material Grader로 A등급 달성

중후반:
  S등급 재료 확보
  → 전체 파트 교체 + Gear Soul 장착

후반:
  SS 이상 등급 재료 + White/Black Diamond
  → 최종 파트 구성 완성
  → 인챈트 + Gear Soul 레벨업으로 마무리
```
