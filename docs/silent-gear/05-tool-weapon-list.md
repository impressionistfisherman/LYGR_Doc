# 도구 & 무기 목록

Silent Gear는 바닐라보다 훨씬 많은 종류의 도구와 무기를 추가합니다.

---

## ⛏️ 도구 목록

### Pickaxe (곡괭이)

- 구성: Head + Rod
- 역할: 기본 광석 채굴

### Shovel (삽)

- 구성: Head + Rod
- 역할: 흙/모래/자갈 채굴

### Axe (도끼)

- 구성: Head + Rod
- 역할: 나무 벌채 + 근접 공격 가능

### Hoe (괭이)

- 구성: Head + Rod
- 역할: 농지 개간

### Mattock (매톡) ⭐

- 구성: Head + Rod
- 역할: **삽 + 도끼 + 괭이 기능 통합**
- 추천: 하나로 여러 역할을 커버해서 인벤토리 절약에 최적

### Hammer (해머) ⭐⭐⭐

- 구성: Head + Rod + (선택) Tip
- 역할: **3×3 범위 채굴** (9블록 동시)
- 추천: 초반부터 만들면 채굴 효율이 바닐라 곡괭이의 9배

:::tip 해머를 가장 먼저 만드세요
Silent Gear 시스템을 배우면서 첫 번째로 만들 도구로 해머를 추천합니다.  
3×3 범위 채굴로 자원 수급 속도가 극적으로 올라갑니다.
:::

### Excavator (굴착기)

- 구성: Head + Rod
- 역할: **3×3 범위 삽** (땅/모래 9블록 동시)
- 추천: 건축 기초 작업, 넓은 면적 정지 작업

### Lumber Axe (벌목 도끼) ⭐⭐

- 구성: Head + Rod
- 역할: **나무 전체를 한 번에 벌채**
- 추천: TreeChop 모드가 있더라도 Lumber Axe가 훨씬 빠름

### Prospector's Hammer (탐광 해머) ⭐

- 구성: Head + Rod
- 역할: **근처 특정 광맥을 탐지**
- 사용법: 손에 들고 블록 우클릭 → 주변 특정 광물 위치 채팅창에 표시
- 추천: 초반 다이아몬드/에메랄드 탐색에 매우 유용

### Sickle (낫)

- 구성: Head + Rod
- 역할: 범위 농작물 수확

### Shears (가위)

- 구성: Blade + Rod
- 역할: 양모, 잎 블록 수집

---

## ⚔️ 무기 목록

### Sword (검)

- 구성: Blade + Guard + Rod
- 역할: 기본 근접 무기, 빠른 공격 속도

### Dagger (단검)

- 구성: Blade + Rod
- 역할: 매우 빠른 공격 속도, 낮은 데미지
- 추천: 크리티컬 확률 빌드, Rogue 직업

### Katana (카타나)

- 구성: Blade + Rod
- 역할: 긴 사거리, 빠른 공격
- 추천: 범위가 중요한 전투 스타일

### Greatsword (대검)

- 구성: Blade + Guard + Rod
- 역할: 높은 데미지, 느린 공격 속도
- 추천: 한 방 위주 빌드, Berserker 직업

### Machete (마체테)

- 구성: Blade + Rod
- 역할: 식물 빠른 제거 + 근접 공격

### Bow (활)

- 구성: Bowstring + Rod
- 역할: 원거리 공격
- 재료: Bowstring에는 실(String) 계열 재료 사용

### Crossbow (석궁)

- 구성: Bowstring + Rod
- 역할: 강력한 원거리 공격, 느린 장전

### Slingshot (슬링샷)

- 구성: Bowstring + Rod
- 역할: 다양한 투사체 발사 (돌, 눈덩이 등)

## 🛡️ 방어구 목록

모든 방어구는 **Main Part + Lining Part** 구성입니다.

| 방어구 | 구성 |
|--------|------|
| **Helmet** | Main + Lining |
| **Chestplate** | Main + Lining |
| **Leggings** | Main + Lining |
| **Boots** | Main + Lining |

### Lining Part

Lining은 방어구의 내부 재료로, 착용감 관련 특성(이동 속도, 착용 무게 등)에 영향을 줍니다.

- 가벼운 재료 → 이동 속도 페널티 감소
- 강한 재료 → 방어력 추가 향상

## 도구 선택 가이드

| 플레이 목적 | 추천 도구 |
|-------------|-----------|
| 빠른 자원 수급 | Hammer + Lumber Axe + Excavator |
| 광물 탐색 | Prospector's Hammer |
| 전투 위주 | Greatsword 또는 Katana |
| 빠른 연타 | Dagger + Sword |
| 원거리 전투 | Bow 또는 Crossbow |
| 농업/수집 | Mattock + Sickle |

:::tip 인벤토리 최소화 구성
Mattock(삽+도끼+괭이) + Hammer(3×3 채굴) + Lumber Axe(벌목)  
이 세 가지만 있으면 일상적인 활동 대부분을 커버할 수 있습니다.
:::
