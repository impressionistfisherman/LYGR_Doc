# 재료 & 특성 (Trait) 가이드

재료 선택은 Silent Gear 장비의 성능을 결정하는 가장 중요한 요소입니다.

---

## 재료 스탯 항목

모든 재료는 다음 항목의 수치를 가집니다.

| 항목 | 설명 |
|------|------|
| **내구도 (Durability)** | 장비가 얼마나 오래 사용되는지 |
| **채굴 속도 (Mining Speed)** | 블록 파괴 속도 (도구 전용) |
| **채굴 티어 (Harvest Tier)** | 캘 수 있는 블록 난이도 (철/다이아/네더라이트 등) |
| **공격 데미지 (Attack Damage)** | 무기 사용 시 기본 데미지 |
| **공격 속도 (Attack Speed)** | 초당 공격 횟수 |
| **방어력 (Armor)** | 방어구 사용 시 방어 수치 |
| **방어 강도 (Toughness)** | 강한 타격 방어 (방어구) |
| **마법 적성 (Enchantability)** | 인챈트 부여 확률/효율 |

## 주요 재료 스탯 비교표

### 도구/무기 Head 재료

| 재료 | 내구도 | 채굴속도 | 공격력 | 특성 |
|------|--------|----------|--------|------|
| **나무** | 낮음 | 2.0 | 낮음 | — |
| **돌** | 낮음 | 4.0 | 낮음 | — |
| **철** | 보통 | 6.0 | 보통 | — |
| **금** | 낮음 | 12.0 | 낮음 | — |
| **다이아몬드** | 높음 | 8.0 | 높음 | — |
| **네더라이트** | 최고 | 9.0 | 최고 | Fire Resistance |
| **뼈 (Bone)** | 낮음 | 5.0 | 낮음 | Jagged (크리티컬 시 추가 피해) |
| **슬라임** | 낮음 | 3.0 | 낮음 | Bouncy (넉백 증가) |
| **끈적이는 피스톤** | 보통 | 5.0 | 보통 | Magnetic |

### Silent's Gems 보석 재료 (요약)

| 보석 | 특징 | 주요 특성 |
|------|------|-----------|
| **Ruby** | 높은 공격력 | Ancient (경험치 추가 드롭) |
| **Sapphire** | 높은 채굴 속도 | Lustrous (내구도 회복) |
| **Emerald** | 균형 스탯 | Lucky (행운 증가) |
| **Topaz** | 높은 내구도 | Renew (자동 내구도 회복) |
| **Amethyst** | 높은 마법 적성 | Masterful (인챈트 레벨 증가) |
| **Black Diamond** | 최고 채굴속도+내구도 | Crushing + Renew | 
| **White Diamond** | 최고 공격력+채굴속도 | Ancient + Lustrous |

:::info Black/White Diamond는 희귀
Black Diamond는 네더, White Diamond는 엔드에서만 드롭됩니다.  
최종 장비에 사용하는 재료이므로 초반에 욕심내지 마세요.
:::

## 특성 (Trait) 전체 목록

### 좋은 특성

| 특성 이름 | 효과 |
|-----------|------|
| **Renew** | 시간이 지나면 내구도 자동 회복 |
| **Lustrous** | 경험치를 얻을 때 내구도 회복 |
| **Magnetic** | 드롭 아이템 자동 수집 (자석 효과) |
| **Crushing** | 광석 채굴 시 추가 드롭 가능 |
| **Ancient** | 몬스터 처치 시 추가 경험치 드롭 |
| **Lucky** | 행운 수치 증가 효과 |
| **Jagged** | 내구도가 낮아질수록 데미지 증가 |
| **Masterful** | 인챈트 적성 대폭 증가 |
| **Synergistic** | 같은 특성이 많을수록 효과 증가 |

### 주의가 필요한 특성

| 특성 이름 | 효과 | 주의 |
|-----------|------|------|
| **Brittle** | 내구도 낮을수록 속도 증가 | 내구도 관리 필요 |
| **Flammable** | 불에 닿으면 내구도 손상 | 네더 사용 시 위험 |
| **Aqueous** | 물속에서 채굴 속도 감소 없음 | 단독으론 약점 없음 |

### 부정적 특성

| 특성 이름 | 효과 | 대처법 |
|-----------|------|--------|
| **Flammable** | 용암/불 접촉 시 빠른 내구도 손상 | 네더 작업 시 해당 재료 파트 교체 |
| **Soft** | 채굴 속도 감소 | 해당 재료 사용 최소화 |

## 재료 조합 전략

여러 파트에 서로 다른 재료를 쓰면 **특성을 조합**할 수 있습니다.

### 도구 특화 조합 예시

**채굴 효율 극대화:**
```
Head: Black Diamond (Crushing + 최고 채굴속도)
Rod: Topaz (Renew — 내구도 자동 회복)
Tip: Emerald (Lucky — 추가 드롭)
```

**자원 파밍 최적화:**
```
Head: Sapphire (Lustrous + 높은 채굴속도)
Rod: 철 막대
Tip: Bone (Jagged — 몬스터 처치 시 추가 피해)
```

### 무기 특화 조합 예시

**근접 DPS 극대화:**
```
Blade: Ruby (Ancient + 높은 공격력)
Guard: Amethyst (Masterful — 인챈트 적성)
Rod: Topaz (Renew)
```

**장기 전투 생존형:**
```
Blade: Iron (균형 스탯)
Guard: Emerald (Lucky)
Rod: Topaz (Renew — 내구도 걱정 없음)
```

## 재료 정보 확인 체크리스트

재료를 선택하기 전에 확인할 사항입니다.

- [ ] Ctrl 키로 스탯 수치 확인
- [ ] 특성(Trait) 목록 확인 — 부정적 특성 없는지 체크
- [ ] 현재 제작할 파트 종류에 맞는 재료인지 확인
- [ ] 원하는 특성이 다른 파트와 조합 가능한지 확인
- [ ] 재료 등급 확인 (등급이 낮으면 Material Grader 사용)
